Руководство по созданию и размещению веб-форм

В руководстве описан процесс создания и размещения веб-формы на примере формы обратной связи. Такая веб-форма может быть использована администратором для получения сообщений и вопросов от посетителей сайта. В примере веб-форма содержит 5 полей: ваше имя, компания, e-mail, телефон, ваши вопросы.

В руководстве процесс создания веб-формы изложен в виде описания двух главных этапов. 

Создание и настройка веб-формы.

Пошаговые инструкции по созданию веб-формы, навигации по разделам, настройка веб-формы, создание вопросов веб-формы, добавление статусов веб-формы.

Размещение веб-формы на странице сайта

Рассказывается, как создать страницы для заполнения веб-формы, списка результатов, просмотра и редактирования результатов.
