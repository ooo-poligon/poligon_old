Рекомендации по конфигурированию веб-серверов

Это руководство предназначено для технических специалистов и системных администраторов.

Документ раскрывает специфические приёмы конфигурирования серверного ПО. Эти приемы позволяют в несколько раз увеличить производительность системы в целом, сократить время генерации страниц, увеличить устойчивость системы к пиковым нагрузкам.

В руководстве приведены рекомендации по настройке серверного ПО, которые выполняются сотрудниками компании "1C-Битрикс". Эти рекомендации эффективны при конфигурировании проектов с посещаемостью более 10 тысяч уникальных пользователей в день, либо при недостаточности системных ресурсов для обработки меньшей нагрузки. Все рекомендации относятся, в основном, к UNIX или Windows системам, использующим веб-сервер Apache.

Приведённые в документе рекомендации базируются на следующих подходах:

  • уменьшение потребляемой веб-сервером памяти;
  • создание двухуровневой конфигурации Front-end - Back-end;
  • конфигурирование PHP;
  • настройка базы данных MySQL;
  • настройка базы данных Oracle.
