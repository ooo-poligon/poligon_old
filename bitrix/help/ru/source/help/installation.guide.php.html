Руководство по установке

Руководство предназначено для пользователей системы "1C-Битрикс: Управление сайтом". В документе рассматриваются процедуры установки и регистрации продукта.

Установка

Описаны процессы установки системы разными способами:

  • после копирования файлов системы на сервер; 
  • с помощью скрипта BitrixSetup; 
  • с использованием Windows-инсталлятора (пробная версия CMS);
  • перенос сайта с локального компьютера на удалённый сервер (хостинг);
  • интеграция версии "статистика" в существующий сайт;
  • установка PHP под Windows с IIS 6.0: установка и проверка работы PHP;
  • дополнительная справочная информация (файл конфигурации веб-сервера .htaccess, работа с архивами tar.gz, необходимый уровень прав на хостинге и прочее)

Регистрация и загрузка исходного кода

  • регистрация пробной (demo) версии продукта;
  • регистрация закодированной версии продукта с коммерческим лицензионным ключом;
  • загрузка обновлений и исходных текстов.
