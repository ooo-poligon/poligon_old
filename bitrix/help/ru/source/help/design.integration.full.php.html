Руководство по интеграции с дизайном

В документе подробно рассматривается процедура интеграции продукта в новый или уже существующий сайт.

Шаблоны и сайты

Рассматриваются основные концепции, положенные в основу архитектуры системы "1C-Битрикс: Управление сайтом":

  • Многосайтовость, позволяющая создать несколько сайтов на одной копии продукта. Каждый сайт может характеризоваться собственным доменным именем, внешним видом, языком интерфейса и информационным наполнением.
  • Шаблоны дизайна, определяющие внешний вид сайта. Для каждого сайта может быть назначено неограниченное количество шаблонов. Их использование открывает широкие возможности для настройки внешнего вида сайта и изменения его дизайна в зависимости от различных условий.
