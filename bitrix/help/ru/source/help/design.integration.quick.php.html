Интеграция. Быстрый старт

Руководство предназначено для IT-специалистов компаний, специалистов по веб-технологиям и веб-мастеров, сотрудникам интернет-агентств и студий веб-дизайна. Цель данного документа - дать общее представление об интеграции и обеспечить специалиста необходимыми знаниями для внедрения своего дизайна в создаваемый проект.


В руководстве описываются общие правила интеграции дизайна с программным продуктом "1С-Битрикс: Управление сайтом" и построение индивидуальных решений.

Шаблоны и сайты

Рассматриваются основные концепции, положенные в основу архитектуры системы "1С-Битрикс: Управление сайтом":

  • Многосайтовость, позволяющая создать несколько сайтов на одной копии продукта. Каждый сайт может характеризоваться собственным доменным именем, внешним видом, языком интерфейса и информационным наполнением.
  • Шаблоны дизайна, определяющие внешний вид сайта. Для каждого сайта может быть назначено неограниченное количество шаблонов. Использование шаблонов открывает широкие возможности для настройки внешнего вида сайта и изменения его дизайна в зависимости от различных условий.

Структура шаблона. Управление шаблонами

Подробно описываются составные части шаблона сайта и основные операции над шаблонами: добавление, активация, редактирование.

Включаемые области и компоненты

Рассматривается концепция включаемых областей, на основе которой построена сборка динамических страниц сайта, управляемого системой "1С-Битрикс".

Программные компоненты

Обсуждается технология программных компонентов, которые используются для инкапсулирования и реализации часто используемых областей на сайте. Основная особенность компонентов – возможность управлять не только программным кодом, но также определёнными параметрами, определяющими поведение компонента.
