Руководство по созданию раздела новостей

В руководстве рассматривается пример использования модуля Информационный блок системы "1C-Битрикс: Управление сайтом" в качестве инструмента для создания на сайте раздела новостей. Данный модуль системы обладает широкими возможностями по созданию новостных блоков. Например, если вы хотите оповещать ваших посетителей о новостях интернет-магазина, то для этого может быть создан информационный блок Новости магазина. В нем будут представлены новые поступления товара, информация о конкурсах, о доставке товаров и т.д.. В руководстве приведён пример создания блока новостей Новости компании.

Последовательное создание блока новостей:

  • создание нового типа информационных блоков Новости (News);
  • создание информационного блока (Новости компании);
  • создание разделов информационного блока (в примере - Услуги);
  • добавление новостей.

Размещение новости в публичном разделе сайта

  • создание страницы со списком новостей;
  • создание страницы детального просмотра новостей.
