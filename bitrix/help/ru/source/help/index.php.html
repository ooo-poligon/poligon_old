Основные сведения

Справочная система "1C-Битрикс: Управление сайтом" предназначена как для пользователей продукта, так и для разработчиков сайтов. Некоторые разделы документации (например, Форум) могут также оказаться полезными посетителям сайтов.

Справочная документация представляет собой описание интерфейса системы управления сайтом, ее основных модулей и наиболее типичных операций. Также в ней содержится полное описание прикладного программного интерфейса "1C-Битрикс: Управление сайтом". Благодаря чему документация может использоваться программистами при разработке сайтов и заказных модулей.

Вместе с тем, представленная документация не содержит такой специфической информации, как запуск системы, её конфигурирование и т.п.. Сведения об этом доступны в виде отдельных руководств и могут быть быть загружены с сервера компании "1C-Битрикс".

В справочную систему включен обзор документов, доступных для загрузки на момент выпуска текущей версии. Каждому документу посвящена отдельная статья, в которой даётся краткое описание руководства и ссылка для его загрузки с сервера компании "1C-Битрикс".

Комплект доступных руководств постоянно расширяется. Наиболее полный набор доступен по следующему адресу: http://www.1c-bitrix.ru/sitemanager/doc.php.
