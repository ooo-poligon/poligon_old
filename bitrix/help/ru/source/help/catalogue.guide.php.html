Руководство по управлению каталогом товаров

В руководстве приводится пример использования модуля Информационные блоки для создания Каталога товаров. Представление каталога в публичном разделе сайта осуществляется с помощью визуальных компонентов. С помощью компонентов каталога осуществляется:

  • настройка внешнего вида элементов каталога;
  • сравнение по некоторым параметрам;
  • просмотр связанных элементов и т.д.  .

Функциональные возможности каталога:

  • детальный просмотр товаров;
  • получения списка товаров из других каталогов, связанных с данным товаром;
  • сравнения параметров различных товаров.

В документации описывается пример создания каталога товаров Книги и его представление в публичном разделе сайта. Процесс состоит из следующих этапов и процедур.

Создание Каталога товаров

  • создание типа информационного блока Каталог книг;
  • создание собственно каталогов Книги, Авторы и Рецензии;
  • добавление разделов каталога;
  • создание разделов и подразделов каталога;
  • добавление товаров в каталог.

Размещение каталога в публичном разделе сайта

  • создание страниц;
  • заполнение списка разделов и товаров каталога;
  • просмотр полного списка элементов раздела каталога;
  • детальный просмотр элементов каталога;
  • просмотр таблицы сравнения товаров;
  • просмотр связанных элементов (товаров).
