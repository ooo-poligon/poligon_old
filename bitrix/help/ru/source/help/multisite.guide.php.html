Руководство по конфигурированию многосайтовых систем

Документ описывает порядок конфигурирования программного продукта "1C-Битрикс: Управление сайтом" и серверного ПО для правильной работы многосайтовой версии программного продукта. Возможны две конфигурации под управлением веб-сервера Apache:

  • продукт и все сайты работают под управлением одной копии веб-сервера Apache;
  • каждый сайт работает под управлением отдельной копии веб-сервера Apache или отдельного виртуального веб-сервера.
