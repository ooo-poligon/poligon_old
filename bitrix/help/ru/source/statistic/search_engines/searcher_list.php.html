Поисковые системы

Форма, представленная на странице Веб-аналитика -> Поисковики -> Индексация -> Список, содержит информацию об индексации сайта различными поисковыми системами.

Фильтр

ПараметрОписание
Наименование* Отбор поисковой системы по ее названию.
ID* Отбор поисковой системы по ID.
Активен Отбор по значению флага активности.
Сохранять хиты Если флаг установлен, отображаются только поисковики, чьи хиты записываются в таблицу хитов поисковиков.
Включить в круговую диаграмму индексации по умолчанию Позволяет отобрать те поисковые системы, которые включаются в диаграмму индексации сайта, не включаются, или все.
Всего хитов Отбор поисковых систем по количеству произведенных с них хитов.
Дата последнего хита Отбор по дате последней индексации сайта поисковиком.
Период Временной интервал, за который следует отобразить количество проиндексированных страниц.
UserAgent* UserAgent искомых поисковых систем.
Показать все условияОтражает все поля фильтра.
Скрыть все условияСкрывает все поля фильтра, кроме полей по умолчанию.

* - для данных полей вы можете воспользоваться специальными логическими выражениями.

Чтобы установить фильтр по заданным критериям поиска, нажмите на кнопку Найти. Для отображения всех фраз нажмите на кнопку Отменить.

Контекстная панель

Кнопка Описание
Добавить Нажатие этой кнопки открывает форму добавления новой поисковой системы.
Диаграмма Открывает форму с круговой диаграммой индексации сайта поисковыми системами.
Графики Открывает форму с графиком индексации сайта поисковыми системами.
Настроить Позволяет перейти к диалогу настройки внешнего вида отчетной формы.
Excel Экспортирует данные из отображаемой таблицы в формат MS Excel.

Таблица

Поле Описание
Колонка флажков Поле предназначено для выбора поисковых систем, к которым предполагается применить какое-либо действие из панели действий, расположенной ниже таблицы.
Меню действий  Нажатие на кнопку действий в строке конкретной поисковой системы отображает меню, содержащее команды:
  • График - открывает форму с графиком индексации сайта этой поисковой системой;
  • Динамика - открывает форму с отчетом о динамике индексации сайта этой поисковой системой (количество хитов за каждый день);
  • Изменить - редактирование параметров поисковой системы;
  • Удалить - удаляет поисковую систему.
ID ID поисковой системы.
Наименование Имя поисковой системы. Ссылка на имени открывает форму с графиком индексации сайта этой поисковой системой.
UserAgent Строка UserAgent поисковой системы.
Сегодня Количество индексаций этой поисковой системой за текущий день. Ссылка на значении открывает форму с отчетом о динамике индексации сайта этой поисковой системой именно за этот день или период.
Вчера Количество индексаций за вчерашний день.
Позавчера Количество индексаций за позавчерашний день.
Всего Общее количество индексаций страниц этой поисковой системой. Ссылка на значении открывает форму с отчетом о динамике индексации сайта этой поисковой системой за все время ведения статистики.
Последний хит Дата последней индексации.
Выбрано Количество записей, отобранных из базы данных.
Отмечено Количество записей, отмеченных флажками.
