График индексации сайта

На данной странице (Веб-аналитика > Поисковики > Индексация > Графики) отображаются графики, иллюстрирующие индексацию сайта поисковыми системами.

Фильтр

ПараметрОписание
ПериодВременной интервал, за который необходимо построить график.
ПоисковикиВ этом списке можно выбрать поисковые системы, которые следует отразить на графике.

В выпадающем списке выберите метод построения графика: суммарный по всем указанным поисковым системам или по отдельности.
Показать все условияОтражает все поля фильтра.
Скрыть все условияСкрывает все поля фильтра, кроме полей по умолчанию.

Чтобы построить график по заданным критериям поиска, нажмите на кнопку Найти.

Контекстная панель

КнопкаОписание
СписокНажатие этой кнопки открывает форму со списком всех поисковых систем.
ДиаграммаОткрывает форму с диаграммой индексации сайта поисковыми системами.

Легенда графика

Внизу графика находится легенда, аналогичная легенде круговой диаграммы.
