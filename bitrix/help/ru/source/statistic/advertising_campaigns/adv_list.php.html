    Фильтр

    ПараметрОписание
    НайтиПоле, позволяющее найти рекламные кампании по их основным параметрам: идентификатору, referer1 или referer2. Это поле присутствует, даже если фильтр свернут.
    ID*ID рекламных кампаний, которые требуется отобразить.
    referer1 / referer2*Идентификаторы рекламных кампаний, соответственно referer1 и referer2.
    Длительность (дней)Интервал для фильтрации по длительности рекламной кампании. Длительность - время, прошедшее с момента первого захода по рекламной кампании до последнего прямого захода либо возврата.
    Сессий (всего)Интервал для фильтрации по суммарному значению сессий (на прямом заходе либо возврате).
    Посетителей (всего)Интервал для фильтрации по суммарному значению посетителей (на прямом заходе либо возврате).
    ПериодУстановка данного поля фильтра выводит дополнительный столбец в отчете по рекламным кампаниям, в котором содержатся данные за установленный период.
    ВнимательностьИнтервал для фильтрации по значению внимательности (среднее значение количества хитов за одну сессию).
    Посетителей за деньИнтервал для фильтрации по количеству посетителей за день (среднее значение количества посетителей за один день).
    ЗатратыИнтервал для фильтрации по полю рекламной кампании затраты. В данном поле может храниться денежная сумма, затраченная на рекламную кампанию.
    ДоходыИнтервал для фильтрации по полю доходы. Данное поле накапливает все денежные суммы по каждому событию, произошедшему в рамках рекламной кампании. Значение поля может быть изменено вручную.
    ПрибыльИнтервал для фильтрации по полю прибыль (разница между доходами и затратами на рекламную кампанию).
    Рентабельность (ROI)Интервал для рентабельности рекламной кампании, рассчитывается по формуле: (прибыль/затраты)*100%.
    ВалютаСписок валют, позволяющий отобразить денежные суммы в той или иной валюте по курсу на текущий день.
    Показать все условияОтражает все поля фильтра.
    Скрыть все условияСкрывает все поля фильтра, кроме полей по умолчанию.

    * - для данных полей вы можете воспользоваться специальными логическими выражениями.

    Меню действий Нажатие на кнопку действий в строке конкретной кампании отображает меню, содержащее команды:
    • Анализ рекламной кампании - открывает форму, отображающую детальную информацию по динамике рекламной кампании: сводную статистику, графики, финансовые показатели, события и проч.;
    • Изменить - открывает форму редактирования рекламной кампании;
    • Сбросить - сбрасывает счетчики рекламной кампании: кол-во посетителей, внимательность, финансовые индикаторы;
    • Удалить - удаляет рекламную кампанию;
    Следующие пункты меню присутствуют, если по данной рекламной кампании произошло по крайней мере одно событие (например, хит).
    • Разделы - открывает форму с отчетом и диаграммой посещаемости разделов и страниц посетителями, пришедшими по этой рекламной кампании;
    • Пути - открывает форму с отчетом и диаграммой путей, по которым прошли посетителями;
    • Динамика - отображает показатели динамики рекламной кампании по дням (сессии, посетители и т.п.);
    • Графики - открывает форму с графиками по рекламной кампании (посещения и события).

    referer1Идентификатор рекламной кампании – referer1.
    referer2Идентификатор рекламной кампании – referer2.
    НачалоВремя первого захода по рекламной кампании.
    ОкончаниеВремя последнего захода либо возврата по рекламной кампании.
    ПриоритетПриоритет рекламной кампании. Это значение вводится при создании (или редактировании) кампании.
    Зашли (сессий) сегодняКол-во сессий на прямом заходе (переходов) и возврате* за текущий день.

    Всего событийКоличество событий за период (если он установлен в фильтре).
    ВыбраноКоличество записей (рекламных кампаний), выбранных из базы данных.

    Дополнительная информация: рекламные кампании

    Под термином рекламная кампания подразумевается набор условий, выделяющих часть потока посетителей, приходящих на Ваш сайт.  Все условия равносильны и могут быть следующих типов:

    • Наличие в ссылке, ведущей на Ваш сайт, одного или двух параметров – referer1, referer2. Пример подобной ссылки: http://www.1c-bitrix.ru/?referer1=vesti&referer2=news.
    • Поисковые системы, с которых могут заходить посетители.
    • Маски любых ссылающихся страниц.
    • Маски страниц Вашего сайта, на которые приходят посетители.

    При определении рекламной кампании используется следующий алгоритм.

    1. Определяются те рекламные кампании, условия которых удовлетворяют параметрам посетителя.
    2. Среди выбранных рекламных кампаний берется кампания с наивысшим приоритетом (задается в настройках рекламной кампании). Если приоритеты одинаковы, то берется самая молодая рекламная кампания (с наибольшим ID).

    Если посетитель пришел на сайт с параметрами referer1 и/или referer2, и с такими идентификаторами не было найдено ни одной рекламной кампании, то автоматически создаётся новая рекламная кампания с указанными referer1, referer2 в качестве идентификаторов.
