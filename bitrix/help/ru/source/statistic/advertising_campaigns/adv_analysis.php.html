Сравнительный анализ рекламных кампаний

Данная форма позволяет показать на одной координатной сетке графики нескольких рекламных кампаний, что позволит в наглядной форме сравнить их эффективность.

Фильтр

ПараметрОписание
ПериодПериод времени, за который требуется вывести графики.
ID рекламной кампании*Маска идентификаторов искомых рекламных кампаний.
referer1*Маска для поиска рекламных кампаний по referer1.
referer2*Маска для поиска рекламных кампаний по referer2.
Рекламные кампанииВыберите в этом списке рекламные кампании, показатели которых требуется отобразить на графике. Для выбора нескольких кампаний удерживайте клавишу Ctrl.

Кнопка Добавить открывает окно со списком кампаний, в котором можно выбрать (установкой флажков и нажатием кнопки Выбрать) кампании для отображения. Кнопка Удалить исключает выбранные кампании из списка.
ID события*Маска идентификаторов для поиска событий рекламных кампаний.
event1*Маска для поиска событий по event1.
event2*Маска для поиска событий по event2.
Вы можете уточнить какие именно события вас интересуютВ этом списке можно выбрать события, которые требуется отобразить на графике. Для выбора нескольких событий, удерживайте клавишу Ctrl.

Кнопка Добавить открывает окно со списком событий для выбора. Кнопка Удалить исключает выбранные события из списка.
Показать все условияОтражает все поля фильтра.
Скрыть все условияСкрывает все поля фильтра, кроме полей по умолчанию.

* - для данных полей вы можете воспользоваться специальными логическими выражениями.

Чтобы отобразить данные с учётом фильтра, нажмите на кнопку Найти. Для отображения всех данных нажмите на кнопку Отменить.
