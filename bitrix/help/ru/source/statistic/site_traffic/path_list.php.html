Пути по сайту

В зависимости от того, какой пункт выбран в меню функций, эта форма может показывать диаграмму полных путей по сайту или их отрезков. Также можно отобразить диаграмму страниц, на которые уходили посетители с любого отрезка пути.

Фильтр

ПараметраОписание
ПериодПериод времени для построения диаграммы.
Количество страниц в путиИнтервал для количества страниц в каждом из путей для диаграммы полных путей.
Первая страница пути*Маска для любой страницы каждого из путей, попадающих в отчет. Данное поле фильтра нагляднее всего использовать в диаграмме полных путей.
Любая страница пути*Маска для любой страницы каждого из путей, попадающих в отчет. Данное поле фильтра нагляднее всего использовать в диаграмме полных путей.
Последняя страница пути*Маска для последней страницы каждого из путей, попадающих в отчет. Данное поле фильтра нагляднее всего использовать в диаграмме отрезков путей.
Рекламные кампанииПозволяет отфильтровать данные отчета по одной или нескольким рекламным кампаниям.
Тип данных для рекламных кампанийЕсли Вы выбрали одну или несколько рекламных кампаний, по которым Вы хотите отфильтровать отчет, то у Вас есть возможность выбрать тип данных. Вы можете построить диаграмму:
  • по суммарным значениям прямых заходов и возвратов;
  • на прямых заходах по рекламным кампаниям;
  • на возвратах.
Показать все условияОтражает все поля фильтра.
Скрыть все условияСкрывает все поля фильтра, кроме полей по умолчанию.

* - для данных полей вы можете воспользоваться специальными логическими выражениями.

Чтобы установить фильтр по заданным критериям поиска, нажмите на кнопку Найти. Для отображения всех данных нажмите на кнопку Отменить.

Контекстная панель

КнопкаОписание
Начало путейНажатие этой кнопки открывает отчет, в котором перечислены все страницы и разделы, с которых посетители начинали навигацию по сайту (начало путей). Кнопка отображается после выбора отрезка пути в списке полных путей.
Полные пути / 
Отрезки путей		Позволяет выбрать тип путей для отображения: только отрезки или полные пути.
НастроитьПозволяет перейти к диалогу настройки внешнего вида отчетной формы.
ExcelЭкспортирует данные из отображаемой таблицы в формат MS Excel.
