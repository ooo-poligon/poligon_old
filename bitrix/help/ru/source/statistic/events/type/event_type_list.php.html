Список

Данная форма отображает таблицу типов событий.

Тип события представляет собой заданное разработчиком сайта событие. Примерами типов событий могут служить:

  • уход на внешнюю платежную систему для оформления заказа;
  • событие, происходящее при нажатии ссылки купить и ведущей, например, на http://www.softkey.ru;
  • скачивание того или иного файла, происходящее при нажатии ссылки Скачать;
  • клик по баннеру, который показывается на Вашем сайте;
  • отсылка комментария на сайте;
  • переход в тот или иной раздел сайта.

Практически любое действие, происходящие на сайте, можно зафиксировать как событие в модуле статистики. Разработчики сайта могут получить подробное описание, как создавать события в разделе справочной системы Для разработчиков > События.

Помимо событий, происходящих на Вашем сайте, Вы можете регистрировать события, происходящие вне сайта. К числу таких событий можно отнести события оплата заказа либо отмена заказа (см. Список событий).

Показать все условияОтражает все поля фильтра.

* - для данных полей вы можете воспользоваться специальными логическими выражениями.

Чтобы установить фильтр по заданным критериям поиска, нажмите на кнопку Найти. Для отображения всех типов событий нажмите на кнопку Отменить.

Excel Экспортирует данные из таблицы в формат MS Excel.
