События

На данной форме представлена детальная информация по каждому событию, произошедшему как на сайте, так и вне сайта.

Фильтр

Наименование параметраОписание
Найти Поле, позволяющее найти события по их основным параметрам: идентификатору рекламной кампании, event1 или event2. Это поле присутствует, даже если фильтр свернут.
ID* ID события в базе.
ID типа события* ID типа события в базе.
Наименование типа события* Наименование типа события.
event1 / event2* Идентификаторы типа события: event1 и event2 соответственно.
event3* Дополнительный параметр события.
Дата Интервал, за который требуется отобразить события.
ID сессии* ID сессии, в рамках которой произошло событие.
ID посетителя* ID посетителя, совершившего события.
Страна* Страна посетителя сайта, инициировавшего событие.
ID рекламной кампании* ID рекламной кампании, в рамках которой произошло событие.
На возврате по рекламной кампании С помощью данного поля можно отфильтровать события, произошедшие на прямом заходе по рекламной кампании либо на возврате.
ID хита* В данном поле можно задать ID хита, инициировавшего занесение в базу данных события.
Откуда пришли* Сайт и страница, с которой пришли на ту страницу, где было инициировано требуемое событие.
Где произошло* Сайт и страница, на которой было инициировано требуемое событие.
Куда перенаправили* Поле позволяет отобрать события по странице, на которую перенаправляется посетитель после события.
Денежная сумма Интервал для отбора событий по значению их денежной суммы.
Валюта Поле позволяет отобрать события по валюте денежной суммы.
Логика между полями Переключатель, определяющий механизм поиска записей.
  • "И" - выводятся события, удовлетворяющие всем критериям фильтра.
  • "ИЛИ" - выводятся события, удовлетворяющие хотя бы одному критерию.
Показать все условияОтражает все поля фильтра.
Скрыть все условияСкрывает все поля фильтра, кроме полей по умолчанию.

* - для данных полей вы можете воспользоваться специальными логическими выражениями.

Чтобы установить фильтр по заданным критериям поиска, нажмите на кнопку Найти. Для отображения всех событий нажмите на кнопку Отменить.

Контекстная панель

Кнопка Описание

event1 Параметр типа события - event1.
event2 Параметр типа события - event2.
Дата Дата возникновения событие.
Сессия ID сессии, в течение которой произошло событие. Ссылка открывает список сессий с фильтром, установленным на эту сессию.
Посет. ID посетителя, инициировавшего событие. Ссылка открывает список посетителей с фильтром, установленным на этого посетителя.
Страна Страна посетителя, инициировавшего событие.
Рекл. камп. ID рекламной кампании, зайдя по которой, посетитель инициировал событие. Если ID выведен с символом *, посетитель инициировал событие на возврате по данной рекламной кампании, в противном случае - на прямом заходе.
Ссылка открывает список кампаний с фильтром, установленным на эту кампанию.
Хит ID хита, на котором произошло событие. Ссылка открывает список посетителей с фильтром, установленным на этого посетителя.
Сайт Сайт, на котором произошло событие.
Денежная сумма Денежная сумма события, указывается либо в базовой валюте модуля, либо в валюте, выбранной в фильтре.
