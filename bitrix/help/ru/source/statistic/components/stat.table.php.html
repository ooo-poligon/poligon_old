Компоненты 1.0::Таблица статистики

Данный компонент выводит основные статистические показатели сайта в виде небольшой таблицы, отчасти напоминающей статистические таблицы таких систем как SpyLog, HotLog и т.п.

Пример вызова

<?$APPLICATION->IncludeFile("statistic/stat_table/default.php", Array());?>

Компонент вызывается без параметров.
