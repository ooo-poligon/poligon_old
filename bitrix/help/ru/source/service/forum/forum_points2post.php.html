Баллы за сообщения

В этом разделе

  • Контекстная панель
  • Список баллов за сообщение
    •

    Форма Баллы за сообщения (Сервисы > Форумы > Баллы за сообщения) предназначена для управления количеством баллов, присваиваемых пользователю форума за каждое отправленное сообщение. При помощи этой формы можно легко настроить кумулятивную систему начисления баллов (т.е. количество баллов за одно сообщение будет увеличиваться в зависимости от общего числа отправленных сообщений).

    Контекстная панель

    Поле Описание
    Новая запись Добавление новой записи с количеством баллов за сообщение.
    Настроить Переход к диалогу настройки внешнего вида отчетной формы.
    Excel Экспорт данных из отображаемой таблицы в формат MS Excel.

    Список баллов за сообщение

    Поле Описание
    Колонка флажков Выбор записей, к которым предполагается применить какое-либо правило или действие.
    Действия Действия над записью о количестве баллов за сообщение:
    • Изменить параметры - редактирование записи;
    • Удалить - удаление записи.
    ID Идентификатор записи в системе.
    Необходимо сообщений Число сообщений, начиная с которого за каждое сообщение будет начисляться указанное в записи количество баллов.

    Например, если значение поля равно 10, то начиная с десятого сообщения посетителю будет начисляться новое (указанное в данной записи) количество баллов за каждое сообщение.
    Баллов за сообщение Количество баллов, начисляемое за сообщение, при достижении указанного в предыдущем поле числа сообщений.
