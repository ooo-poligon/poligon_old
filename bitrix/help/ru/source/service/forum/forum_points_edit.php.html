Создание и изменение звания

В этом разделе

  • Контекстная панель
  • Форма редактирования
  • Кнопки управления
    •

    Форма применяется как для создания новых, так и для редактирования существующих званий посетителей форума. 

    Контекстная панель

    Кнопка Описание
    Список званий Переход на страницу со списком званий.
    Добавить новое звание Переход к форме создания нового звания. Кнопка отображается при редактировании существующего звания.
    Удалить звание Удаление редактируемого звания.

    Форма редактирования

    Поле Описание
    ID Идентификатор звания.
    *Для получения звания необходимо баллов Количество баллов, требуемое для получения этого звания.
    Мнемонический код Символьный код звания. Поле предназначено для использования разработчиками.
    Имеет голосов Количество голосов, требуемое для получения этого звания.
    [Язык]
    *Название Название звания, как оно отображается на форуме. Название задается отдельно для каждого языка.

    * - Поля, обязательные для заполнения.

    Кнопки управления

    Кнопка Описание
    Сохранить Сохранение внесённых изменений. Переход на страницу со списком званий.
    Применить Сохранение внесённых изменений. Продолжение редактирования параметров звания.
    Отменить Отмена внесённых изменений. Возврат первоначальных значений параметров.
