Группы форумов

    Страница Группы форумов (Сервисы > Форумы > Группы форумов) содержит список существующих групп форумов, а также позволяет перейти к созданию и изменению групп форумов.

    Группы используются для группировки форумов в публичной части сайта. Например, можно объединить все форумы, имеющие отношение к различным вопросам по конкретному продукту, в одну группу: форум Лицензирование, форум Применение и пр.

    Каждый форум может быть привязан к одной из групп форумов, либо может остаться без привязки.

    Контекстная панель

    Кнопка Описание
    Новая группа форумов Переход к созданию новой группы форумов.
    Настроить Позволяет перейти к диалогу настройки внешнего вида отчетной формы.
    Excel Экспорт данных из отображаемой таблицы в формат MS Excel.

    Список групп форумов

    Поле Описание
    Колонка флажков Выбор групп, к которым предполагается применить какое-либо правило или действие.
    Действия Действие над группой форумов:
    • Изменить параметры группы - редактирование группы форумов;
    • Удалить группу - удаление группы форумов. Удаление группы не влечет удаление форумов, относящихся к данной группе.
    Код Уникальный код группы форумов.
    Название Название группы, показываемое в публичной части.
    Сорт. Относительный "вес", определяющий положение группы в списке. Чем меньше значение, тем выше в списке показывается группа.
