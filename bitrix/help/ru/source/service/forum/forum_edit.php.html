    Форма применяется как для создания новых, так и для редактирования существующих форумов. 

    Примечание: правами на создание новых форумов обладают только администраторы сайта.

    Контекстная панель

    Кнопка Описание
    Список форумов Переход на страницу со списком форумов.
    Создать новый форум Переход к форме создания нового форума. Кнопка отображается при редактировании существующего форума.
    Удалить форум Удаление редактируемого форума.

    Форма редактирования

    Закладка "Форум"

    Настройка общих параметров форума.

    Поле Описание
    Активен Признак активности форума. Неактивные форумы доступны только тем пользователям, которые имеют права на полный доступ к ним.
    Группа форумов Группа, к которой относится форум. Может быть не установлена (т.е. форум без группы).
    *Сортировка Относительный "вес", определяющий порядок показа форумов в рамках одной группы. Чем больше значение поля, тем ниже в списке будет расположен форум.
    *Имя форума Произвольное название форума.
    Описание Краткое описание форума. Описание будет показано в публичной части под названием форума.
    Сайты, на которых доступен форум / Шаблон пути к сообщению на сайте
    Сайт Сайты, на которых будет доступен форум.
    Шаблон пути к сообщению Шаблон пути к сообщениям форума для данного сайта. Это поле используется при отправке почтовых сообщений с форума и при индексировании сообщений форума в модуле поиска.

    Пример шаблона: 
    /forum/index.php?
    PAGE_NAME=read&FID=#FORUM_ID#&TID=#TOPIC_ID#&MID=#MESSAGE_ID##message#MESSAGE_ID#

    Переменные #FORUM_ID#, #TOPIC_ID# и #MESSAGE_ID# при использовании шаблона автоматически заменяются на реальные код форума, код темы и код сообщения.

    * - Поля, обязательные для заполнения.

    Закладка "Настройки"
    Поле Описание
    Позволить HTML Разрешение или запрещение использования HTML-кода в тексте сообщения. Не рекомендуется разрешать использование произвольного HTML-кода на форуме без особой необходимости, так как это существенно снижает безопасность форума.
    Замена новой строки на <br> (Для HTML)
    		Заменять перевод каретки (новую строку) на тег <br> в режиме HTML. По умолчанию в режиме HTML текст сообщения выводится "как есть", т.е. без преобразований.
    Позволить ссылки Разрешить использование гиперссылок в сообщении.
    Позволить теги B, U, I Разрешить выделение текста полужирным, подчеркиванием или курсивом.
    Позволить изображения Разрешить использование изображений в тексте сообщения. Изображения располагаются на сторонних сайтах и подключаются на форуме.
    Позволить списки Разрешить создание списков.
    Позволить цитирование Возможность цитировать сообщение другого пользователя.
    Позволить коды Использование кодов в сообщении.
    Позволить шрифты Возможность изменения цвета текста и шрифт.
    Позволить смайлики Разрешить использование смайликов в сообщении.
    Позволить загрузку Разрешить загрузку присоединенных к сообщению файлов указанного типа. Возможные варианты значений:
    • нет - загрузка запрещена;
    • изображения - загрузка только изображений (gif, jpg, png, jpe, jpeg);
    • файлов с указанными расширениями - загрузка только файлов с расширениями из поля Допустимые расширения загружаемых файлов;
    • любых файлов - загрузка любых файлов. Не разрешайте без крайней необходимости загрузку выполнимых файлов и скриптов, так как это может позволить злоумышленникам полностью разрушить ваш сайт.
    Допустимые расширения загружаемых файлов Расширения файлов, разрешенных к загрузке на сайт (см. описание предыдущего поля).
    Премодерируемый Включение режима премодерирования: сообщения пользователей будут публиковаться на сайте только после проверки модератором (администратором сайта или другим пользователем с соответствующими правами).
    Разрешить перенос тем модераторам других форумов Определяет, разрешено ли модераторам других форумов переносить темы.
    Запросить E-Mail у незарегистрированного пользователя Определяет, будет ли запрашиваться адрес почты у незарегистрированного пользователя при попытке добавления им сообщения.
    Использовать CAPTCHA Определяет, использовать ли защиту от автоматического заполнения.
    Сортировка по Сортировка "по умолчанию" тем в рамках одного форума в публичной части сайта. Выберите вариант сортировки сообщений, а также направление сортировки.

    Например, если выбрана сортировка по дате последнего сообщения и направление по убыванию, то последние по дате сообщения будут показаны выше, чем более ранние.
    Направление сортировки Направление сортировки для выбранного режима сортировки.
    Как регистрировать новые сообщения в модуле статистики**
    event1 Идентификатор типа события - event1
    event2 Идентификатор типа события - event2
    event3 Дополнительный параметр события - event3

    ** Секция служит для настройки идентификаторов, с использованием которых событие создания нового сообщения на форуме будет регистрироваться в модуле статистики.

    Закладка "Доступ"

    Предназначена для установки прав доступа к форуму для каждой из зарегистрированных на сайте групп пользователей. По умолчанию права имеют следующие значения.

    Поле Описание
    [Группа пользователей]

    Уровень доступа к форуму для данной группы пользователей:

    • Нет доступа - запрет на доступ к форуму;
    • Чтение - доступ на чтения сообщений форума без права создания новых тем;
    • Ответ - доступ к обсуждению существующих тем без права создания новых тем;
    • Новая тема - доступ к обсуждению существующих тем, а также возможность создания новых тем;
    • Модерирование - возможность модерировать форум, т.е. производить те действия, которые не повлекут реальной потери данных. Например, модераторы могут скрыть или показать сообщение, но не могут отредактировать или удалить его. Члены группы с данным уровнем прав имеют возможность подписаться на полученииы копии всех сообщений форума по почте.
    • Редактирование - доступ к управлению чужими сообщениями на форуме;
    • Полный доступ - полный доступ к форуму, включая доступ к настройкам форума и прав доступа.

    Примечание: Младшие права включаются в старшие (т.е. если группа имеет право на модерирование форума, то она тем более имеет право читать сообщения, отвечать на сообщения, создавать новые темы, но не имеет права на редактирование сообщений). При определении прав пользователя, который принадлежит к нескольким группам, берется максимальное из его прав.

    Кнопки управления

    Кнопка Описание
    Сохранить Сохранение внесённых изменений. Переход на страницу со списком форумов.
    Применить Сохранение внесённых изменений. Продолжение редактирования параметров форума.
    Отменить Отмена внесённых изменений. Возврат первоначальных значений параметров.
