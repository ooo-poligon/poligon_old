Подписки пользователя

В этом разделе

  • Форма поиска
  • Контекстная панель
  • Список подписок пользователя

    • На форме Подписки пользователя <логин_пользователя> (Рабочий стол > Сервисы > Форумы > Управление подпиской ) представлена информация обо всех подписках этого пользователя.

    Форма поиска

    Поиск подписок пользователя в соответствии с указанными условиями.

    Поле Описание
    Найти Поиск подписки пользователя по их основным параметрам: форуму или теме. Это поле присутствует, даже если фильтр свернут.
    Дата начала подписки Диапазон дат, в пределах которых искомые подписки могли быть начаты.
    ФорумВыбор форума, на котором осуществлять поиск
    Вид подписки Поле позволяет определить вид подписки.

    Для того чтобы отфильтровать подписки по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех подписок нажмите кнопку Отменить.

    Контекстная панель

    Кнопка Описание
    Список пользователей Переход на страницу со списком пользователей, имеющих подписки на форумах.
    Настроить Переход к диалогу настройки внешнего вида отчетной формы.
    Excel Экспорт данных из отображаемой таблицы в формат MS Excel.

    Список подписок пользователя

    Поле Описание
    Колонка флажков Выбор подписок, к которым предполагается применить какое-либо правило или действие.
    Действия Действия, которые можно применить к элементу списка:
    • Удалить - удаление подписки пользователя.
    ID Идентификатор подписки.
    Форум Название форума, к которому относится подписка.
    Тема Название подписки и непосредственно темы форума.
    Начало подписки Дата и время начала подписки.
    Последнее сообщение Идентификатор последнего отправленного пользователем сообщения.
