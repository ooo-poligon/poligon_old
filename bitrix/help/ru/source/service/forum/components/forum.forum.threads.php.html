Компоненты 1.0::Темы форума

<?$APPLICATION->IncludeFile("forum/forum_pieces/last_topics.php",Array(
     'FID' => '1',
     'NUM' => '10',
     'ORDER_BY' => 'TITLE',
     'ORDER_DIRECTION' => 'ASC',
     'PATH2MESSAGES' => '/forum/read.php',
     'CACHE_TIME' => '600',
     )
);?>

Компонент показывает список тем определённого форума, упорядоченных по заданному условию.

Параметры

FID
Идентификатор форума.
NUM
Количество тем, выводимых на странице.
ORDER_BY
Поле, по которому осуществляется сортировка, допустимые значения:
  • TITLE - заголовок темы,
  • POSTS - количество ответов,
  • USER_START_NAME - автор темы,
  • VIEWS - количество просмотров,
  • START_DATE - дата создания темы,
  • LAST_POST_DATE - дата последнего сообщения;
ORDER_DIRECTION
Направление сортировки, допустимые значения:
  • ASC - по возрастанию,
  • DESC - по убыванию;
PATH2MESSAGES
Путь к странице сообщений форума.
CACHE_TIME
Время кеширования данных в секундах.
