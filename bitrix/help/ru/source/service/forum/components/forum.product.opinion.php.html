Компоненты 1.0::Отзывы о продукте

<?$APPLICATION->IncludeFile("forum/forum_pieces/reviews.php", Array(
     'FORUM_ID' => '1',
     'PRODUCT_ID' => $arIBlockElement["ID"],
     'CACHE_TIME' => '600',
     )
);?>

Компонент служит для ввода и показа отзывов о продукте (элементе инфоблока). В инфоблоке, к элементам которого пристегиваются отзывы, должно существовать целочисленное свойство с мнемоническим кодом FORUM_TOPIC_ID.

Параметры

FORUM_ID
Идентификатор форума, в который сохраняются комментарии.
PRODUCT_ID
Идентификатор элемента инфоблока (код товара), к которому добавляются комментарии.
CACHE_TIME
Период (в секундах), на который кешируется результат. Установка этого параметра в ноль отключает кеширование.
