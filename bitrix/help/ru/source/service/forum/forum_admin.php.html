    Страница Список форумов (Сервисы > Форумы > Список форумов) содержит информацию о существующих форумах, позволяет создавать новые форумы и управлять существующими. Информация о форумах представлена в виде таблицы.

    Форма поиска

    Поиск форумов в соответствии с указанными условиями.

    Поле Описание
    Сайт Сайт, к которому относится форум.
    Активность Поиск сайта по признаку активности.
    Группа форумов Группа, к которой относится форум.

    Чтобы отобрать форумы по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех форумов нажмите кнопку Отменить.

    Контекстная панель

    Кнопка Описание
    Новый форум Переход к созданию нового форума.
    Настроить Переход к диалогу настройки внешнего вида отчетной формы.
    Excel Экспорт данных из отображаемой таблицы в формат MS Excel.

    Список форумов

    Поле Описание
    Колонка флажков Выбор форумов, к которым предполагается применить какое-либо правило или действие.
    Действия Действия, которые можно произвести над форумом:
    • Редактирование настроек форума;
    • Удаление форума.
    ID Идентификатор форума в системе.
    Имя Произвольное название форума.
    Активен Флаг активности форума. Если флаг не установлен, форум будет недоступен в публичной части.
    Сайт Сайт(ы), к которым относится данный форум.
    Сорт. Порядок сортировки форума. Влияет на расположение форума в списке.
