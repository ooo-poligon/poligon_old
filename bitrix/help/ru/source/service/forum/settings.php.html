    Закладка "Настройки"

    Наименование параметра Описание
    E-Mail для поля From писем с форума Адрес отправителя, подставляемый по-умолчанию в поле FROM писем, отправляемых с форума.
    Число форумов на страницу Количество форумов, выводимых на одной странице в публичном разделе сайта.
    Число тем на страницу Количество тем форума, выводимых на одной странице в публичном разделе сайта.
    Число сообщений на страницу Количество сообщений форума, выводимых на одной странице в публичном разделе сайта.
    Включить голосования Разрешение посетителям форума участвовать в голосовании для присвоения баллов авторам сообщений. Количество баллов, присвоенное посетителю форума, влияет на звание данного посетителя.
    Иконки на панель Разрешение на показ кнопок для управления форумом на административной панели.

    Примечание: Работают только при наличии константы FORUM_MODULE_PAGE.
    Показывать ICQ в контактах пользователя При отмеченной опции в контактах пользователя будет показан номер ICQ.
    Показывать имя хоста вместо IP-адреса При отмеченной опции вместо IP-адреса пользователя будет показано имя хоста.
    Хранить в БД код HTML Для того чтобы HTML-код хранился в базе данных, отметьте эту опцию.
    Пользовательские настройки
    Максимальный размер аватара (байт) Максимальный размер файла аватара в байтах.
    Максимальная ширина/высота аватара (пикселей) Максимально допустимые ширина и высота аватара в пикселях.
    Максимальный размер загружаемых файлов (байт) Максимально допустимый размер файлов, разрешенных для загрузки на форум.
    Настройки персональных сообщений
    Версия Указывается версия персональных сообщений.
    Размер ящика (сообщений) Количество сообщений, помещающихся в ящике.
    Поиск
    Количество документов, переиндексируемых за шаг Количество документов, переиндексируемых за шаг поиска.

    Закладка "Доступ"

    Форма позволяет настроить права групп пользователей на доступ к модулю форумов.

    Поле Описание
    По умолчанию Управление правами доступа всех групп пользователей, для которых установлен уровень доступа по умолчанию.
    [Группа пользователей] Уровень доступа для конкретной группы пользователей:
    • Закрыт– доступ к меню и административным файлам модуля закрыт;
    • Просмотр всех данных модуля– доступ к меню открыт, разрешен просмотр файлов модуля без права изменения;
    • Запись – полный доступ.

    Закладка "Фильтр нецензурных слов"

    Поле Описание
    Включить фильтр нецензурных слов Для использования на форуме фильтра нецензурных слов отметьте данную опцию. При ее включении становятся активными следующие два поля.
    Действия с найденными словами Выбирается одно из действий, совершаемое с найденными нецензурными словами:
    • ничего;
    • удалить;
    • заменить.
    Замена Если в качестве действия с найденными словами выбрано заменить, то необходимо указать словосочетание или слово, на которое будет заменено данное нецензурное слово.
    Языкозависимые параметры
    [Язык] Задаются словари слов и транслита для каждого из языков.
