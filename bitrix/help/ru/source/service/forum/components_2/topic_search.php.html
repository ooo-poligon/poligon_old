Компоненты 2.0::Поиск темы

Компонент выводит форму поиска тем форума. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:forum.topic.search","",Array(
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "0",
     "TOPIC_PER_PAGE" => "20",
     "URL_TEMPLATES_TOPIC_SEARCH" => "topic_search.php"
     )
);?>

Описание параметров

Шаблоны ссылок
URL_TEMPLATES_TOPIC_SEARCH
Адрес страницы поиска тем форума.
Настройки кеширования
CACHE_TYPE
Тип кеширования:
A - Авто: действует при включенном кешировании в течение заданного времени;
Y - Кешировать: необходимо определить время кеширования;
N - Не кешировать: кеширования нет в любом случае.
CACHE_TIME
Время кеширования, указанное в секундах.
Дополнительные настройки
TOPIC_PER_PAGE
Количество найденных тем на одной странице. Остальные будут выведены с помощью постраничной навигации.
