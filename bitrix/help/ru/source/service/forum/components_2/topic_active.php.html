Компоненты 2.0::Новые темы

Компонент список новых тем форума. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:forum.topic.active","",Array(
     "URL_TEMPLATES_READ" => "read.php?FID=#FID#&TID=#TID#&MID=#MID#",
     "URL_TEMPLATES_LIST" => "list.php?FID=#FID#",
     "URL_TEMPLATES_PROFILE_VIEW" => "profile_view.php?UID=#UID#",
     "FID" => $_REQUEST["FID"],
     "SET_TITLE" => "Y",
     "DATE_FORMAT" => "d.m.Y",
     "DATE_TIME_FORMAT" => "d.m.Y H:i:s",
     "SET_NAVIGATION" => "Y",
     "TOPICS_PER_PAGE" => "20",
     "MESSAGES_PER_PAGE" => "25",
     "PATH_TO_ICON" => "/bitrix/images/forum/icon/",
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "0",
     "WORD_LENGTH" => "50"
     )
);?>

Описание параметров

Основные параметры
FID
Идентификатор форума.
Шаблоны ссылок
URL_TEMPLATES_READ
Адрес страницы чтения темы форума.
URL_TEMPLATES_LIST
Адрес страницы со списком тем форума.
URL_TEMPLATES_PROFILE_VIEW
Адрес страницы просмотра профиля пользователя.
Настройки кеширования
CACHE_TYPE
Тип кеширования:
A - Авто: действует при включенном кешировании в течение заданного времени;
Y - Кешировать: необходимо определить время кеширования;
N - Не кешировать: кеширования нет в любом случае.
CACHE_TIME
Время кеширования, указанное в секундах.
Дополнительные настройки
SET_TITLE
При отмеченной опции в качестве заголовка страницы будет установлено Новые темы.
DATE_FORMAT
Формат показа даты.
DATE_TIME_FORMAT
Формат показа даты и времени.
SET_NAVIGATION
При отмеченной опции в навигационной цепочке будут отражен переход на страницу новых тем.
TOPICS_PER_PAGE
Количество тем, отображаемых на одной странице. Для отображения всех тем используется постраничная навигация.
MESSAGES_PER_PAGE
Количество сообщений, отображаемых на одной странице. Для отображения всех сообщений используется постраничная навигация.
PATH_TO_ICON
Путь относительно корня сайта к папке с иконками к темам.
WORD_LENGTH
Максимально допустимая длина словообразующих символов, следующих друг за другом.
Если длина слова превысит указанное число, то символы слова будут разбавлены разрывами (при просмотре сообщений разрывы не отображаются).
