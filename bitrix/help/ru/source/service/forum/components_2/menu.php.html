Компоненты 2.0::Меню

Компонент выводит меню форума. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:forum.menu","",Array(
     "TMPLT_SHOW_AUTH_FORM" => "LINK",
     "SHOW_ADD_MENU" => "N",
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "0",
     "URL_TEMPLATES_INDEX" => "index.php",
     "URL_TEMPLATES_LIST" => "list.php?FID=#FID#",
     "URL_TEMPLATES_PROFILE_VIEW" => "profile_view.php?UID=#UID#",
     "URL_TEMPLATES_SUBSCR_LIST" => "subscr_list.php",
     "URL_TEMPLATES_ACTIVE" => "active.php",
     "URL_TEMPLATES_SEARCH" => "search.php",
     "URL_TEMPLATES_HELP" => "help.php",
     "URL_TEMPLATES_USER_LIST" => "user_list.php",
     "URL_TEMPLATES_PM_LIST" => "pm_list.php",
     "URL_TEMPLATES_PM_EDIT" => "pm_edit.php?FID=#FID#&MID=#MID#&UID=#UID#&mode=#mode#",
     "URL_TEMPLATES_PM_FOLDER" => "pm_folder.php",
     "URL_TEMPLATES_RULES" => "rules.php",
     "FID" => $_REQUEST["FID"],
     "TID" => $_REQUEST["FID"],
     "PAGE_NAME" => "",
     "PATH_TO_AUTH_FORM" => "/auth.php"
     )
);?>

Описание параметров

Дополнительно
TMPLT_SHOW_AUTH_FORM
Авторизация может быть показана в виде:
LINK - ссылок;
INPUT - полей ввода.
SHOW_ADD_MENU
При отмеченной опции будут показаны дополнительные пункты меню. Например, пункт Отметить форумы прочитанными.
Основные параметры
FID
Идентификатор форума.
TID
Идентификатор темы.
PAGE_NAME
Имя страницы вызывающего компонента.
PATH_TO_AUTH_FORM
Путь к форме авторизации. Если не задан, то будет использовано значение по умолчанию.
Шаблоны ссылок
URL_TEMPLATES_INDEX
Адрес страницы со списком форумов.
URL_TEMPLATES_LIST
Адрес страницы со списком тем форума.
URL_TEMPLATES_PROFILE_VIEW
Адрес страницы просмотра профиля пользователя.
URL_TEMPLATES_SUBSCR_LIST
Адрес страницы подписки на форумы.
URL_TEMPLATES_ACTIVE
Адрес страницы списка новых тем форумов.
URL_TEMPLATES_SEARCH
Адрес страницы поиска по форумам.
URL_TEMPLATES_HELP
Адрес страницы помощи по форумам.
URL_TEMPLATES_USER_LIST
Адрес страницы со списком зарегистрированных пользователей форума.
URL_TEMPLATES_PM_LIST
Адрес страницы личных сообщений пользователя.
URL_TEMPLATES_PM_EDIT
Адрес страницы создания (редактирования) личных сообщений.
URL_TEMPLATES_PM_FOLDER
Адрес страницы управления директориями личных сообщений.
URL_TEMPLATES_RULES
Адрес страницы, содержащей правила форума.
Настройки кеширования
CACHE_TYPE
Тип кеширования:
A - Авто: действует при включенном кешировании в течение заданного времени;
Y - Кешировать: необходимо определить время кеширования;
N - Не кешировать: кеширования нет в любом случае.
CACHE_TIME
Время кеширования, указанное в секундах.
