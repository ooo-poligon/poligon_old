Компоненты 2.0::Перемещение сообщения

Данный компонент служит для перемещения сообщений форума. Компонент является стандартным и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:forum.message.move","",Array(
     "SET_TITLE" => "Y",
     "SET_NAVIGATION" => "Y",
     "URL_TEMPLATES_INDEX" => "index.php",
     "URL_TEMPLATES_LIST" => "list.php?FID=#FID#",
     "URL_TEMPLATES_READ" => "read.php?FID=#FID#&TID=#TID#&MID=#MID#",
     "URL_TEMPLATES_PROFILE_VIEW" => "profile_view.php?UID=#UID#",
     "URL_TEMPLATES_TOPIC_SEARCH" => "topic_search.php",
     "PATH_TO_SMILE" => "/bitrix/images/forum/smile/",
     "PATH_TO_ICON" => "/bitrix/images/forum/icon/",
     "WORD_LENGTH" => "50"
     )
);?>

Описание параметров

Шаблоны ссылок
URL_TEMPLATES_INDEX
Адрес страницы со списком форумов.
URL_TEMPLATES_LIST
Адрес страницы со списком тем форума.
URL_TEMPLATES_READ
Адрес страницы чтения темы.
URL_TEMPLATES_PROFILE_VIEW
Адрес страницы просмотра профиля пользователя.
URL_TEMPLATES_TOPIC_SEARCH
Адрес страницы поиска темы форума.
Дополнительные настройки
SET_TITLE
Данный параметр определяет будет установлен заголовок или нет.
SET_NAVIGATION
При отмеченной опции в навигационной цепочке будет отражен переход на страницу переноса сообщения.
PATH_TO_SMILE
Путь к папке со смайлами относительно корня сайта.
PATH_TO_ICON
Путь к папке с иконками к темам относительно корня сайта.
WORD_LENGTH
Максимально допустимая длина словообразующих символов, следующих друг за другом.
Если длина слова превысит указанное число, то символы слова будут разбавлены разрывами (при просмотре сообщений разрывы не отображаются).
