Компоненты 2.0::Изменение профиля пользователя

Компонент выводит форму редактирования профиля пользователя. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:forum.user.profile.edit","",Array(
     "USER_PROPERTY_NAME" => "",
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "0",
     "SET_TITLE" => "Y",
     "SET_NAVIGATION" => "Y",
     "USER_PROPERTY" => array(),
     "URL_TEMPLATES_PROFILE_VIEW" => "profile_view.php?UID=#UID#",
     "UID" => $_REQUEST["UID"
     )
);?>

Описание параметров

Дополнительно
USER_PROPERTY_NAME
Название закладки с дополнительными пользовательскими свойствами.
Основные параметры
UID
Идентификатор пользователя.
Шаблоны ссылок
URL_TEMPLATES_PROFILE_VIEW
Путь к странице просмотра профиля пользователя.
Настройки кеширования
CACHE_TYPE
Тип кеширования:
  • A - Авто: действует при включенном кешировании в течение заданного времени;
  • Y - Кешировать: необходимо определить время кеширования;
  • N - Не кешировать: кеширования нет в любом случае.

CACHE_TIME
Время кеширования, указанное в секундах.
Дополнительные настройки
SET_TITLE
При отмеченной опции в качестве заголовка страницы будет установлено <имя_пользователя> (изменение профиля) .
SET_NAVIGATION
При отмеченной опции будет добавлен пункт с именем пользователя в цепочку навигации.
USER_PROPERTY
Указываются пользовательские свойства, которые необходимо дополнительно отобразить.
