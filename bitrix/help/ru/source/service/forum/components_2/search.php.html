Компоненты 2.0::Поиск

Компонент выводит форму поиска по форуму. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:forum.search","",Array(
     "TOPICS_PER_PAGE" => "20",
     "SET_TITLE" => "Y",
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "0",
     "SET_NAVIGATION" => "Y",
     "URL_TEMPLATES_INDEX" => "index.php",
     "URL_TEMPLATES_READ" => "read.php?FID=#FID#&TID=#TID#&MID=#MID#"
     )
);?>

Описание параметров

Шаблоны ссылок
URL_TEMPLATES_INDEX
Адрес страницы со списком форумов.
URL_TEMPLATES_READ
Адрес страницы чтения темы форума.
Настройки кеширования
CACHE_TYPE
Тип кеширования:
A - Авто: действует при включенном кешировании в течение заданного времени;
Y - Кешировать: необходимо определить время кеширования;
N - Не кешировать: кеширования нет в любом случае.
CACHE_TIME
Время кеширования, указанное в секундах.
Дополнительные настройки
TOPICS_PER_PAGE
Количество тем, отображаемых на одной странице. Для отображения всех тем используется постраничная навигация.
SET_TITLE
При отмеченной опции в качестве заголовка страницы будет установлено Поиск.
SET_NAVIGATION
При отмеченной опции в навигационной цепочке будут отражен переход на страницу поиска.
