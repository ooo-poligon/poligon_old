Компоненты 2.0::Папка личных сообщений

Компонент служит для вывода списка личных папок пользователя. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:forum.pm.folder","",Array(
     "SET_TITLE" => "Y",
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "0",
     "SET_NAVIGATION" => "Y",
     "URL_TEMPLATES_PM_LIST" => "pm_list.php?FID=#FID#",
     "URL_TEMPLATES_PM_FOLDER" => "pm_folder.php"
     )
);?>

Описание параметров

Шаблоны ссылок
URL_TEMPLATES_PM_LIST
Путь к странице со списком персональных сообщений.
URL_TEMPLATES_PM_FOLDER
Путь к странице управления папками пользователя с личными сообщениями.
Настройки кеширования
CACHE_TYPE
Тип кеширования:
  • A - Авто: действует при включенном кешировании в течение заданного времени;
  • Y - Кешировать: необходимо определить время кеширования;
  • N - Не кешировать: кеширования нет в любом случае.

CACHE_TIME
Время кеширования, указанное в секундах.
Дополнительные настройки
SET_TITLE
При установленной опции в качестве заголовка страницы будет установлено Список папок.
SET_NAVIGATION
При отмеченной опции будет добавлен пункт с заголовком страницы в цепочку навигации.
