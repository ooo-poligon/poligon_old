Компоненты 2.0::Форма создания сообщения

Компонент выводит форму создания сообщения форума. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:forum.post_form","",Array(
     "MESSAGE_TYPE" => $_REQUEST["MESSAGE_TYPE"],
     "FID" => $_REQUEST["FID"],
     "TID" => $_REQUEST["TID"],
     "PAGE_NAME" => "read",
     "PATH_TO_SMILE" => "/bitrix/images/forum/smile/",
     "PATH_TO_ICON" => "/bitrix/images/forum/icon/",
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "0"
     )
);?>

Описание параметров

Основные параметры
MESSAGE_TYPE
Тип отображения формы редактирования (ответ, редактирование, новая тема).
FID
Идентификатор форума.
TID
Идентификатор темы.
PAGE_NAME
Имя страницы вызывающего компонента.
Настройки кеширования
CACHE_TYPE
Тип кеширования:
A - Авто: действует при включенном кешировании в течение заданного времени;
Y - Кешировать: необходимо определить время кеширования;
N - Не кешировать: кеширования нет в любом случае.
CACHE_TIME
Время кеширования, указанное в секундах.
Дополнительные настройки
PATH_TO_SMILE
Путь относительно корня сайта к папке со смайлами.
PATH_TO_ICON
Путь относительно корня сайта к папке с иконками к темам.
