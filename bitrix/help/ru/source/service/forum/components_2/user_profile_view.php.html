Компоненты 2.0::Пользователь (профиль)

Данный компонент служит для просмотра профиля пользователя форума. Компонент является стандартным и входит в дистрибутив модуля.

Пример вызова

<?$APPLICATION->IncludeComponent("bitrix:forum.user.profile.view","",Array(
     "USER_PROPERTY_NAME" => "",
     "SET_TITLE" => "Y",
     "SET_NAVIGATION" => "Y",
     "USER_PROPERTY" => array(),
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "0",
     "DATE_FORMAT" => "d.m.Y",
     "DATE_TIME_FORMAT" => "d.m.Y H:i:s",
     "URL_TEMPLATES_READ" => "read.php?FID=#FID#&TID=#TID#&MID=#MID#",
     "URL_TEMPLATES_PROFILE" => "profile.php?UID=#UID#",
     "URL_TEMPLATES_USER_LIST" => "user_list.php",
     "URL_TEMPLATES_PM_LIST" => "pm_list.php",
     "URL_TEMPLATES_MESSAGE_SEND" => "message_send.php?TYPE=#TYPE#&UID=#UID#",
     "URL_TEMPLATES_USER_POST" => "user_post.php?UID=#UID#&mode=#mode#",
     "UID" => $_REQUEST["UID"]
     )
);?>

Описание параметров

Дополнительно
USER_PROPERTY_NAME
Название закладки с дополнительными свойствами пользователя.
Основные параметры
UID
Идентификатор пользователя.
Шаблоны ссылок
URL_TEMPLATES_READ
Адрес страницы чтения темы форума.
URL_TEMPLATES_PROFILE
Адрес страницы редактирования профиля пользователя.
URL_TEMPLATES_USER_LIST
Адрес страницы со списком зарегистрированных пользователей форума.
URL_TEMPLATES_PM_LIST
Адрес страницы личных сообщений пользователя.
URL_TEMPLATES_MESSAGE_SEND
Адрес страницы отправки сообщений.
URL_TEMPLATES_USER_POST
Адрес страницы со списком всех сообщений пользователя.
Настройки кеширования
CACHE_TYPE
Тип кеширования:
A - Авто: действует при включенном кешировании в течение заданного времени;
Y - Кешировать: для кеширования необходимо определить только время кеширования;
N - Не кешировать: кеширования нет в любом случае.
CACHE_TIME
Время кеширования, указанное в секундах.
Дополнительные настройки
SET_TITLE
[Y/N] При отмеченной опции в качестве заголовка страницы будет установлено имя пользователя.
SET_NAVIGATION
[Y/N] При отмеченной опции в навигационной цепочке будут отражен переход на страницу профиля пользователя.
USER_PROPERTY
Указываются свойства пользователя, которые будут показаны на закладке с названием, заданным в параметре USER_PROPERTY_NAME.
DATE_FORMAT
Формат показа даты.
DATE_TIME_FORMAT
Формат показа даты и времени.
