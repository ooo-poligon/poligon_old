Компоненты 2.0::Список тем

Компонент выводит список всех тем форума. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:forum.topic.list","",Array(
     "TMPLT_SHOW_ADDITIONAL_MARKER" => "",
     "TMPLT_SHOW_BOTTOM" => "SET_BE_READ",
     "WORD_WRAP_CUT" => "23",
     "SET_TITLE" => "Y",
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "0",
     "DATE_FORMAT" => "d.m.Y",
     "DATE_TIME_FORMAT" => "d.m.Y H:i:s",
     "SET_NAVIGATION" => "Y",
     "PATH_TO_ICON" => "/bitrix/images/forum/icon/",
     "TOPICS_PER_PAGE" => "10",
     "MESSAGES_PER_PAGE" => "3",
     "URL_TEMPLATES_INDEX" => "index.php",
     "URL_TEMPLATES_LIST" => "list.php?FID=#FID#",
     "URL_TEMPLATES_READ" => "read.php?FID=#FID#&TID=#TID#&MID=#MID#",
     "URL_TEMPLATES_PROFILE_VIEW" => "profile_view.php?UID=#UID#",
     "URL_TEMPLATES_MESSAGE_APPR" => "message_appr.php?FID=#FID#",
     "URL_TEMPLATES_TOPIC_NEW" => "topic_new.php?FID=#FID#",
     "URL_TEMPLATES_SUBSCR_LIST" => "subscr_list.php?FID=#FID#",
     "URL_TEMPLATES_TOPIC_MOVE" => "topic_move.php?FID=#FID#&TID=#TID#",
     "FID" => $_REQUEST["FID"],
     "USE_DESC_PAGE" => "Y",
     "WORD_LENGTH" => "50"
     )
);?>

Описание параметров

Дополнительно
TMPLT_SHOW_ADDITIONAL_MARKER
Надпись, которая пишется рядом с темой, в которой есть непрочитанные вами сообщения.
TMPLT_SHOW_BOTTOM
Надпись снизу списка тем.
WORD_WRAP_CUT
Действие со словом, длина которого превышает заявленную. Например:
  • 0 - разорвать: в слово будут вставлены разделительные символы, но при просмотре они не отображаются;
  • 23 - обрезать: слово будет обрезано до 23 символов.
Основные параметры
FID
Идентификатор форума.
USE_DESC_PAGE
Отмеченная опция позволяет использовать обратную постраничную навигацию для списка тем.
Шаблоны ссылок
URL_TEMPLATES_INDEX
Адрес страницы со списком форумов.
URL_TEMPLATES_LIST
Адрес страницы со списком тем форума.
URL_TEMPLATES_READ
Адрес страницы чтения темы форума.
URL_TEMPLATES_PROFILE_VIEW
Адрес страницы просмотра профиля пользователя.
URL_TEMPLATES_MESSAGE_APPR
Адрес страницы с неодобренными сообщениями форума. 
URL_TEMPLATES_TOPIC_NEW
Адрес страницы создания новой темы.
URL_TEMPLATES_SUBSCR_LIST
Адрес страницы подписки на форум. 
URL_TEMPLATES_TOPIC_MOVE
Адрес страницы переноса темы форума.
Настройки кеширования
CACHE_TYPE
Тип кеширования:
A - Авто: действует при включенном кешировании в течение заданного времени;
Y - Кешировать: необходимо определить время кеширования;
N - Не кешировать: кеширования нет в любом случае.
CACHE_TIME
Время кеширования, указанное в секундах.
Дополнительные настройки
SET_TITLE
При отмеченной опции в качестве заголовка страницы будет установлено название форума, список тем которого просматривается.
DATE_FORMAT
Формат показа даты.
DATE_TIME_FORMAT
Формат показа даты и времени.
SET_NAVIGATION
[Y/N] При отмеченной опции в навигационной цепочке будут отражен переход на страницу списка тем форума.
PATH_TO_ICON
Путь к папке с иконками к темам относительно корня сайта.
TOPICS_PER_PAGE
Количество тем форума на одной странице. Остальные будут выведены с помощью постраничной навигации.
MESSAGES_PER_PAGE
Количество сообщений на одной странице. Остальные будут выведены с помощью постраничной навигации.
WORD_LENGTH
Максимально допустимая длина словообразующих символов, следующих друг за другом.
Если длина слова превысит указанное число, то символы слова будут разбавлены разрывами (при просмотре сообщений разрывы не отображаются).
