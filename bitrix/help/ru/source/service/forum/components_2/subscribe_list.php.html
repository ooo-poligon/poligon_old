Компоненты 2.0::Список подписки

Компонент выводит список подписок пользователя на форум. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:forum.subscribe.list","",Array(
     "SET_TITLE" => "Y",
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "0",
     "DATE_FORMAT" => "d.m.Y",
     "DATE_TIME_FORMAT" => "d.m.Y H:i:s",
     "SET_NAVIGATION" => "Y",
     "TOPICS_PER_PAGE" => "20",
     "URL_TEMPLATES_LIST" => "list.php?FID=#FID#",
     "URL_TEMPLATES_READ" => "read.php?FID=#FID#&TID=#TID#&MID=#MID#",
     "URL_TEMPLATES_PROFILE_VIEW" => "profile_view.php?UID=#UID#",
     "URL_TEMPLATES_SUBSCR_LIST" => "subscr_list.php",
     "FID" => $_REQUEST["FID"]
     )
);?>

Описание параметров

Основные параметры
FID
Идентификатор форума.
Шаблоны ссылок
URL_TEMPLATES_LIST
Адрес страницы со списком тем форума.
URL_TEMPLATES_READ
Адрес страницы чтения темы форума.
URL_TEMPLATES_PROFILE_VIEW
Адрес страницы просмотра профиля пользователя.
URL_TEMPLATES_SUBSCR_LIST
Адрес страницы подписки на форумы.
Настройки кеширования
CACHE_TYPE
Тип кеширования:
A - Авто: действует при включенном кешировании в течение заданного времени;
Y - Кешировать: необходимо определить время кеширования;
N - Не кешировать: кеширования нет в любом случае.
CACHE_TIME
Время кеширования, указанное в секундах.
Дополнительные настройки
SET_TITLE
При отмеченной опции в качестве заголовка страницы будет установлено Список подписки на сообщения форума.
SET_NAVIGATION
При отмеченной опции в навигационной цепочке будут отражен переход на страницу списка подписки.
DATE_FORMAT
Формат показа даты.
DATE_TIME_FORMAT
Формат показа даты и времени.
TOPICS_PER_PAGE
Количество тем, отображаемых на одной странице. Для отображения всех тем используется постраничная навигация.
