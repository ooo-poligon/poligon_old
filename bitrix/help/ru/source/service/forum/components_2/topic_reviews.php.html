Компоненты 2.0::Тема (отзывы)

Данный компонент выводит форму для создания отзыва к инфоблоку, а также и весь список отзывов. Компонент является стандартным и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:forum.topic.reviews","",Array(
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "0",
     "CACHE_RANGE" => "OPTIMAL",
     "DATE_TIME_FORMAT" => "d.m.Y H:i:s",
     "MESSAGES_PER_PAGE" => "25",
     "USE_CAPTCHA" => "Y",
     "PREORDER" => "Y",
     "PATH_TO_SMILE" => "/bitrix/images/forum/smile/",
     "FORUM_ID" => "2",
     "ELEMENT_ID" => $_REQUEST["ELEMENT_ID"],
     "URL_TEMPLATES_READ" => "read.php?FID=#FID#&TID=#TID#&MID=#MID#"
     )
);?>

Описание параметров

Основные параметры
FORUM_ID
Идентификатор форума для отзывов.
ELEMENT_ID
Идентификатор элемента инфоблока.
Шаблоны ссылок
URL_TEMPLATES_READ
Адрес страницы чтения темы.
Настройки кеширования
CACHE_TYPE
Тип кеширования:
A - Авто: действует при включенном кешировании в течение заданного времени;
Y - Кешировать: для кеширования необходимо определить только время кеширования;
N - Не кешировать: кеширования нет в любом случае.
CACHE_TIME
Время кеширования, указанное в секундах.
CACHE_RANGE
Степень кеширования:
  • MAX - максимально;
  • OPTIMAL - оптимально.
Дополнительные настройки
DATE_TIME_FORMAT
Формат показа даты и времени.
MESSAGES_PER_PAGE
Количество сообщений на одной странице. Остальные будут выведены с помощью постраничной навигации.
USE_CAPTCHA
[Y|N] При отмеченной опции будет использоваться защита от автоматического заполнения.
PREORDER
[Y|N] При отмеченной опции сообщения будут выведены в прямом порядке.
PATH_TO_SMILE
Путь к папке со смайлами относительно корня сайта.
