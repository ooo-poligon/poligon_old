Компоненты 2.0::RSS форума

Компонент служит для экспорта данных форума в формате RSS. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:forum.rss","",Array(
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "86400",
     "TYPE_RANGE" => Array("RSS2"),
     "TYPE" => $_REQUEST["TYPE"],
     "CACHE_RSS" => "1800",
     "FID_RANGE" => Array("1"),
     "COUNT" => "30",
     "TN_TITLE" => "",
     "TN_DESCRIPTION" => "",
     "PATH_TO_SMILE" => "/bitrix/images/forum/smile/",
     "URL_TEMPLATES_LIST" => "list.php?FID=#FID#",
     "URL_TEMPLATES_READ" => "read.php?FID=#FID#&TID=#TID#&MID=#MID#",
     "URL_TEMPLATES_PROFILE_VIEW" => "profile_view.php?UID=#UID#",
     "URL_TEMPLATES_RSS" => "rss.php?TYPE=#TYPE#&MODE=#MODE#&IID=#IID#"
     )
);?>

Описание параметров

Основные параметры
TYPE_RANGE
Указываются необходимы форматы экспорта данных форума: RSS 0.92, RSS 2.0 и Atom 0.3.
TYPE
Тип RSS.
CACHE_RSS
Время кеширования ленты, указанное в секундах.
FID_RANGE
Указываются идентификаторы тех форумов, на которых разрешить RSS.
COUNT
Количество элементов форума для экспорта.
TN_TITLE
Название ленты (при пустом поле ввода будет использовано значение по умолчанию).
TN_DESCRIPTION
Описание ленты (при пустом поле ввода будет использовано значение по умолчанию).
Шаблоны ссылок
URL_TEMPLATES_LIST
Адрес страницы со списком тем форума.
URL_TEMPLATES_READ
Адрес страницы чтения темы форума.
URL_TEMPLATES_PROFILE_VIEW
Адрес страницы просмотра профиля пользователя.
URL_TEMPLATES_RSS
Адрес страницы RSS форума.
Настройки кеширования
CACHE_TYPE
Тип кеширования:
A - Авто: действует при включенном кешировании в течение заданного времени;
Y - Кешировать: необходимо определить время кеширования;
N - Не кешировать: кеширования нет в любом случае.
CACHE_TIME
Время кеширования, указанное в секундах.
Дополнительные настройки
PATH_TO_SMILE
Путь относительно корня сайта к папке со смайлами.
