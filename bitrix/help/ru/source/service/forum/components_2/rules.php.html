Компоненты 2.0::Правила

Компонент служит для создания страницы с правилами форума. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:forum.rules","",Array(
     "SET_TITLE" => "Y",
     "SET_NAVIGATION" => "Y",
     "CONTENT" => "",
     "URL_TEMPLATES_INDEX" => "index.php"
     )
);?>

Описание параметров

Шаблоны ссылок
URL_TEMPLATES_INDEX
Адрес страницы со списком форумов.
Дополнительные настройки
SET_TITLE
При установленной опции в качестве заголовка страницы будет установлено Правила форума.
SET_NAVIGATION
При отмеченной опции будет добавлен пункт с заголовком страницы в цепочку навигации.
CONTENT
Путь к файлу, содержащему текст раздела. Если значение параметра не указано, то будет использовано значение по умолчанию.
