Компоненты 2.0::Чтение темы

Компонент выводит список сообщений темы форума. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:forum.topic.read","",Array(
     "FID" => $_REQUEST["FID"],
     "TID" => $_REQUEST["TID"],
     "MID" => $_REQUEST["MID"],
     "SET_TITLE" => "Y",
     "SET_NAVIGATION" => "Y",
     "DATE_FORMAT" => "d.m.Y",
     "DATE_TIME_FORMAT" => "d.m.Y H:i:s",
     "MESSAGES_PER_PAGE" => "25",
     "PATH_TO_SMILE" => "/bitrix/images/forum/smile/",
     "URL_TEMPLATES_READ" => "read.php?FID=#FID#&TID=#TID#&MID=#MID#",
     "URL_TEMPLATES_LIST" => "list.php?FID=#FID#",
     "URL_TEMPLATES_MESSAGE_MOVE" => "message_move.php?FID=#FID#&TID=#TID#&MID_ARRAY=#MID_ARRAY#",
     "URL_TEMPLATES_PROFILE_VIEW" => "profile_view.php?UID=#UID#",
     "URL_TEMPLATES_TOPIC_NEW" => "topic_new.php?FID=#FID#",
     "URL_TEMPLATES_SUBSCR_LIST" => "subscr_list.php?FID=#FID#",
     "URL_TEMPLATES_TOPIC_MOVE" => "topic_move.php?FID=#FID#&TID=#TID#",
     "URL_TEMPLATES_INDEX" => "index.php",
     "URL_TEMPLATES_PM_EDIT" => "pm_edit.php",
     "URL_TEMPLATES_MESSAGE_SEND" => "message_send.php?UID=#UID#",
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "0",
     "USE_DESC_PAGE" => "Y",
     "WORD_LENGTH" => "50"
     )
);?>

Описание параметров

Основные параметры
FID
Идентификатор форума.

TID
Идентификатор темы.

MID
Идентификатор сообщения.

USE_DESC_PAGE
При отмеченной опции для вывода списка сообщений будет использована обратная постраничная навигация.
Шаблоны ссылок
URL_TEMPLATES_READ
Адрес страницы чтения темы форума.
URL_TEMPLATES_LIST
Адрес страницы со списком тем форума.
URL_TEMPLATES_MESSAGE_MOVE
Адрес страницы переноса сообщений форума.
URL_TEMPLATES_PROFILE_VIEW
Адрес страницы просмотра профиля пользователя.
URL_TEMPLATES_TOPIC_NEW
Адрес страницы создания новой темы форума.
URL_TEMPLATES_SUBSCR_LIST
Адрес страницы подписки на форум.
URL_TEMPLATES_TOPIC_MOVE
Адрес страницы переноса темы форума.
URL_TEMPLATES_INDEX
Адрес страницы списка форумов.
URL_TEMPLATES_PM_EDIT
Адрес страницы редактирования личных сообщений пользователя.
URL_TEMPLATES_MESSAGE_SEND
Адрес страницы отправки сообщения.
Настройки кеширования
CACHE_TYPE
Тип кеширования:
A - Авто: действует при включенном кешировании в течение заданного времени;
Y - Кешировать: необходимо определить время кеширования;
N - Не кешировать: кеширования нет в любом случае.
CACHE_TIME
Время кеширования, указанное в секундах.
Дополнительные настройки
SET_TITLE
При отмеченной опции в качестве заголовка страницы будет установлено название просматриваемой темы.
SET_NAVIGATION
При отмеченной опции будет добавлен пункт с заголовком страницы в цепочку навигации.
DATE_FORMAT
Формат показа даты.
DATE_TIME_FORMAT
Формат показа даты и времени.
MESSAGES_PER_PAGE
Количество сообщений на одной странице. Остальные будут выведены с помощью постраничной навигации.
PATH_TO_SMILE
Путь к папке со смайлами относительно корня сайта.
WORD_LENGTH
Максимально допустимая длина словообразующих символов, следующих друг за другом.
Если длина слова превысит указанное число, то символы слова будут разбавлены разрывами (при просмотре сообщений разрывы не отображаются).
