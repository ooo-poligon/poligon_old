Компоненты 2.0::Письмо

Компонент производит отправку письма (сообщения) пользователю. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:forum.message.send","",Array(
     "SET_TITLE" => "Y",
     "SET_NAVIGATION" => "Y",
     "URL_TEMPLATES_PROFILE_VIEW" => "profile_view.php?UID=#UID#",
     "UID" => $_REQUEST["UID"],
     "SEND_MAIL" => "E",
     "SEND_ICQ" => "A"
     )
);?>

Описание параметров

Основные параметры
UID
Параметр содержит код, в котором передается идентификатор пользователя. Значение по умолчанию: $_REQUEST["UID"].
SEND_MAIL
Указывается группа, пользователи которой могут отправлять письмо (e-mail) из профиля:
  • A - никто;
  • E - авторизованные пользователи;
  • U - все пользователи, но с учётом того, что неавторизованные должны заполнить поле CAPTCHA;
  • Y - все пользователи.
SEND_ICQ
Указывается группа, пользователи которой могут отправлять сообщение (iсq) из профиля.
Шаблоны ссылок
URL_TEMPLATES_PROFILE_VIEW
Адрес страницы просмотра профиля пользователя.
Дополнительные настройки
SET_TITLE
При установленной опции в качестве заголовка страницы будет установлено Новое при создании нового сообщения или <тема_сообщения> (редактирование) при изменении сообщения.
SET_NAVIGATION
При отмеченной опции будет добавлен пункт с заголовком страницы в цепочку навигации.
