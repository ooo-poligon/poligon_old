Модерирование форума

Модерирование форума выполняется из публичного раздела сайта. При просмотре сообщений форума для пользователей, обладающих правом на модерирование сообщений данного форума, рядом с каждым сообщением будет отображаться кнопка Показать/Скрыть. С помощью этой кнопки модератор может скрыть сообщение от других посетителей форума, либо сделать сообщение доступным для просмотра.

Если форум является премодерируемым, то все вновь создаваемые сообщения данного форума являются скрытыми. Для того чтобы сообщения стали доступны посетителям форума, модератор форума (либо администратор сайта) должен разрешить показ данных сообщений.
