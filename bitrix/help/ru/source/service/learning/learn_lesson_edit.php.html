    • Для создания и редактирования уроков применяется форма одного и того же вида. В зависимости от того, создаете вы новый урок или редактируете существующий, некоторые поля могут присутствовать или отсутствовать на форме.

    Контекстная панель

    Кнопка Описание
    Список Ссылка для перехода на страницу со списком уроков.
    Создать Ссылка для перехода на страницу создания нового урока. Отображается только на странице редактирования уже существующего урока.
    Удалить Удаление урока, просматриваемого на данной странице. Команда доступна только на странице редактирования уже существующего урока.

    Поля формы

    Вкладка "Урок"

    Поле Описание
    ID Идентификатор урока.

    Данное поле отображается только в режиме редактирования существующего урока.
    Изменена Дата и время создания или изменения урока.

    Данное поле отображается только в режиме редактирования существующего урока.
    Запись активна Признак активности урока. Если это поле отмечено, то урок и его содержимое доступно для показа в публичной части.
    Родительский раздел Указывается глава, в которой будет находиться урок.
    *Название Название урока.
    Индекс сортировки Вес сортировки урока; применяется при выводе урока в списке.
    * - Поля, обязательные для заполнения.

    Вкладка "Анонс"

    Поле Описание
    Text / HTML Выбор формата создания краткого описания урока.
    Использовать визуальный редактор При условии использования формата HTML, для создания анонса урока будет использоваться визуальный редактор.
    Изображение Поле позволяет в краткое описание урока добавить изображение с помощью кнопки Обзор...

    Вкладка "Содержание"

    Поле Описание
    Text / HTML Выбор формата создания содержимого урока.
    Использовать визуальный редактор При условии использования формата HTML, для создания содержимого урока будет использоваться визуальный редактор.
    Изображение Поле позволяет в детальное описание урока добавить изображение с помощью кнопки Обзор...

    Кнопки управления

    Кнопка Описание
    Сохранить Сохранение внесённых изменений. Возврат к списку уроков.
    Применить Сохранение внесённых изменений. Продолжение редактирования урока.
    Отменить Отмена внесённых изменений. Возврат к списку уроков.
