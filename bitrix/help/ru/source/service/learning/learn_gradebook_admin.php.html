    Фильтр

    Задание параметров отбора результатов, отображаемых в списке.

    Поле Описание
    Учащийся Отбор результатов прохождения тестов указанного учащегося.
    ID Идентификатор результата.
    Тест Выбор теста, по которому имеются результаты о его прохождении. Если указано значение любой, то отображаются результаты по всем тестам.
    Пройден Отображение всех результатов или только результатов по успешному прохождению тестов, или только результатов по неудачно пройденным тестам.
    Баллов Указывается интервал количества баллов, которые заработали учащиеся в ходе прохождения тестирования. Если какое-то из полей оставлено пустым (или оба), то граница считается неограниченной.

    Чтобы отфильтровать результаты по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех результатов нажмите кнопку Отменить.

    Контекстная панель

    Кнопка Описание
    Настроить Переход к диалогу настройки внешнего вида отчетной формы.
    Excel Экспорт данных из отображаемой таблицы в формат MS Excel.

    Список результатов обучения

    Колонка Описание
    Колонка флажков Выбор результатов, к которым предполагается применить какое-либо действие.
    Меню
    действий
    		Доступные действия:
    • Удалить - удалить результат.
    ID Идентификатор результата.
    Тест Название теста, по которому получен данный результат.
    Учащийся Указывается идентификатор, логин, имя и фамилия пользователя.
    Баллов Количество баллов, которые заработал учащийся во время прохождения тестирования.
    Макс. баллов Максимальное количество баллов, которое можно заработать во время прохождения теста.
    Попыток Количество попыток прохождения теста.
    Пройден Указывается удачно пройден тест или нет.
