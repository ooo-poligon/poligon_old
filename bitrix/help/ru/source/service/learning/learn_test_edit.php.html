    • Для создания и редактирования тестов курса применяется форма одного и того же вида. В зависимости от того, создаете вы новый тест или редактируете существующий, некоторые поля могут присутствовать или отсутствовать на форме.

    Контекстная панель

    Кнопка Описание
    Список Ссылка для перехода на страницу со списком тестов.
    Создать Ссылка для перехода на страницу создания нового теста. Отображается только на странице редактирования уже существующего теста.
    Удалить Удаление просматриваемого теста. Команда доступна только на странице редактирования уже существующего теста.

    Поля формы

    Вкладка "Тест"

    Поле Описание
    ID Идентификатор теста.

    Данное поле отображается только в режиме редактирования существующего теста.
    Изменена Дата и время создания или изменения теста.

    Данное поле отображается только в режиме редактирования существующего теста.
    Запись активна Признак активности теста. Если это поле отмечено, то тест и его содержимое доступно для показа в публичной части.
    *Название Название теста.
    Индекс сортировки Вес сортировки теста; применяется при выводе теста в списке.
    В тесте участвуют Количество вопросов, участвующих в тесте:
    • все вопросы курса;
    • либо указанное количество вопросов со всего курса;
    • либо указанное количество вопросов из каждой главы;
    • или же указанное количество вопросов из каждого урока.
    Включать вопросы для самопроверки При отмеченной опции в тест будут включены вопросы для самопроверки.
    Случайный порядок вопросов Возможность вывода вопросов в случайном порядке.
    Случайный порядок ответов Возможность вывода вариантов ответов на вопрос в случайном порядке.
    Количество попыток Задание количества попыток прохождения теста. Если указано 0 или строка пуста, то количество попыток неограничено.
    Ограничение времени Задается количество минут для прохождения теста. Если указано 0 или строка пуста, то прохождение теста во времени не ограничено.
    Автоматическая проверка результатов Если данная опция отмечена, то результаты тестирования будут проверены автоматически.
    Для прохождения теста необходимо правильно ответить на Задается процент вопросов, на которые необходимо правильно ответить для прохождения теста.
    Тип прохождения теста Выбирается один из трех возможных типов прохождения теста:
    • Запретить переход к следующему вопросу без ответа на текущий вопрос, пользователь не может изменять свои ответы.
    • Разрешить переход к следующему вопросу без ответа на текущий вопрос, пользователь не может изменять свои ответы.
    • Разрешить переход к следующему вопросу без ответа на текущий вопрос, пользователь может изменять свои ответы.
    * - Поля, обязательные для заполнения.

    Вкладка "Описание"

    Поле Описание
    Text / HTML Выбор формата создания описания теста.
    Использовать визуальный редактор При условии использования формата HTML, для создания описания теста будет использоваться визуальный редактор.

    Кнопки управления

    Кнопка Описание
    Сохранить Сохранение внесённых изменений. Возврат к списку тестов.
    Применить Сохранение внесённых изменений. Продолжение редактирования теста.
    Отменить Отмена внесённых изменений. Возврат к списку тестов.
