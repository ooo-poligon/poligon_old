    • Для создания и редактирования учебного курса применяется форма одного и того же вида. В зависимости от того, создаете вы новый курс или редактируете существующий, некоторые поля могут присутствовать или отсутствовать на форме.

    Контекстная панель

    Кнопка Описание
    Список Ссылка для перехода на страницу со списком учебных курсов.
    Создать Ссылка для перехода на страницу создания нового курса. Отображается только на странице редактирования уже существующего курса.
    Удалить Удаление курса, просматриваемого на данной странице. Команда доступна только на странице редактирования уже существующего курса.

    Поля формы

    Вкладка "Курс"

    Настройка основных параметров.

    ПолеОписание
    ID Идентификатор курса.

    Поле отображается только в режиме редактирования существующего курса.
    Изменена Дата и время создания или изменения курса.

    Поле отображается только в режиме редактирования существующего курса.
    Запись активна Признак активности курса. Если это поле отмечено, то курс и его содержимое доступно для показа в публичной части.
    Период активности (DD.MM.YYYY) Указывается период активности курса. Если какое-то из полей оставлено пустым, то граница периода считается неограниченной.
    Символическое имя Произвольное символическое имя курса.
    *Сайты Указываются сайты, на которых данный курс будет активным.
    *Название Название курса.
    Индекс сортировки Вес сортировки курса; применяется при выводе курса в списке.
    * - Поля, обязательные для заполнения.

    Вкладка "Анонс"

    Создание анонса курса

    ПолеОписание
    Text / HTML Выбор формата создания краткого описания курса.
    Использовать визуальный редактор При условии использования формата HTML, для создания анонса курса будет использоваться визуальный редактор.
    Изображение Поле позволяет в краткое описание курса добавить изображение с помощью кнопки Обзор...

    Вкладка "Подробно"

    Создание детального описания курса.

    ПолеОписание
    Text / HTML Выбор формата создания детального описания курса.
    Использовать визуальный редактор При условии использования формата HTML, для создания детального описания курса будет использоваться визуальный редактор.
    Изображение Поле позволяет в детальное описание курса добавить изображение с помощью кнопки Обзор...

    Вкладка "Доступ"



    ПолеОписание
    Доступы групп пользователей Назначение различных прав доступа к курсам для разных групп пользователей:
    • нет доступа - доступ к курсам закрыт;
    • чтение - курсы доступны на чтение;
    • изменение - право на редактирование курсов;
    • полный доступ - разрешено редактировать курсы и управлять уровнем доступа к курсам пользователей других групп.

    Кнопки управления

    КнопкаОписание
    Сохранить Сохранение внесённых изменений. Возврат к списку курсов.
    Применить Сохранение внесённых изменений. Продолжение редактирования курса.
    Отменить Отмена внесённых изменений. Возврат к списку курсов.
