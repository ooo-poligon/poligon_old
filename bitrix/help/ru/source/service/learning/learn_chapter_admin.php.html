    Фильтр

    Задание параметров отбора глав курса, отображаемых в списке.

    Поле Описание
    Название Отбор глав по названию.
    Активен Отображение всех глав или только активных, или только неактивных.
    Глава Выбирается уровень, к которому относится искомая глава. При значении любой, т.е. будут отображены все главы.

    Чтобы отфильтровать главы курса по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех глав нажмите кнопку Отменить.

    Контекстная панель

    Кнопка Описание
    Добавить главу Переход к форме создания новой главы.
    Добавить урок Переход к форме создания нового урока в курсе.
    Настроить Переход к диалогу настройки внешнего вида отчетной формы.
    Excel Экспорт данных из отображаемой таблицы в формат MS Excel.

    Список глав

    Колонка Описание
    Колонка флажков Выбор глав, к которым предполагается применить какое-либо действие.
    Меню
    действий
    		Доступные действия:
    • Добавить главу - создать новую главу;
    • Добавить урок - создать новый урок;
    • Изменить - отредактировать главу;
    • Удалить - удалить главу.
    ID Идентификатор главы.
    Дата изм. Дата и время последней модификации главы.
    Название Название главы.
    Сорт. Вес сортировки главы; применяется при выводе главы в списке.
    Акт. Признак активности главы.
    Главы Количество подглав в данной главе. При нажатии на число будет осуществлен переход к странице со списком подглав этой главы.
    Символ "+" служит для перехода к созданию новой главы.
    Уроки Количество уроков в данной главе. При нажатии на число будет осуществлен переход к странице со списком уроков этой главы.
    Символ "+" служит для перехода к созданию нового урока.
    Символическое имя Мнемоническое имя главы, данное латиницей.
