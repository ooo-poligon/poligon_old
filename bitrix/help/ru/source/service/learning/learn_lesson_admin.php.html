    Фильтр

    Задание параметров отбора уроков, отображаемых в списке. Следующая таблица описывает поля фильтра.

    Поле Описание
    Название Отбор уроков по названию.
    ID Идентификатор урока.
    Кем создан Указывается автор урока, либо значение любой, т.е. будут отображены уроки всех авторов.
    Глава Выбор главы, к которому относится урок. При значении любая, т.е. будут отображены уроки всех глав.
    Активность Отображение всех уроков или только активных, или только неактивных.

    Чтобы отфильтровать уроки по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех уроков нажмите кнопку Отменить.

    Контекстная панель

    Кнопка Описание
    Добавить урок Служит для перехода к форме создания нового урока.
    Настроить Переход к диалогу настройки внешнего вида отчетной формы.
    Excel Экспорт данных из отображаемой таблицы в формат MS Excel.

    Список уроков

    Колонка Описание
    Колонка флажков Выбор уроков, к которым предполагается применить какое-либо действие.
    Меню
    действий
    		Допустимые действия:
    • Добавить вопрос - создать новый вопрос;
    • Изменить - отредактировать урок;
    • Удалить - удалить урок.
    ID Идентификатор урока.
    Дата изм. Дата и время последней модификации урока.
    Название Название урока.
    Сорт. Вес сортировки урока; применяется при выводе урока в списке.
    Акт. Признак активности урока.
    Глава Название главы, к которой относится урок.
    Вопросы Количество вопросов в данном уроке. При нажатии на число будет осуществлен переход к странице со списком вопросов этого урока.
    Символ "+" служит для перехода к созданию нового вопроса.
