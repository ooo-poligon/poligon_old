Сертификация

  • Фильтр

  • Список сертификатов

    • Информация о пользователях, получивших сертификаты (Рабочий стол > Сервисы > Обучение > Сертификация). Сертификат выдается пользователю после успешного тестирования.

    Фильтр

    Задание параметров отбора сертификатов, отображаемых в списке.

    Поле Описание
    Учащийся Отбор сертификатов, выданных указанному учащемуся.
    ID Идентификатор сертификата.
    Курс Выбирается курс, по которому выданы сертификаты. Если указано значение любой, то отображаются сертификаты по всем курсам.
    Активен Отображение всех сертификатов или только активных, или только неактивных.
    Баллов Указывается интервал количества баллов, которые заработали учащиеся. Если какое-то из полей оставлено пустым (или оба), то граница считается неограниченной.
    Сертифицировался Online Отображение всех сертификатов или только тех, которые получили пользователи Onine, либо только тех, которые получили пользователи Offline.

    Чтобы отфильтровать сертификаты по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех сертификатов нажмите кнопку Отменить.

    Контекстная панель

    Кнопка Описание
    Настроить Переход к диалогу настройки внешнего вида отчетной формы.
    Excel Экспорт данных из отображаемой таблицы в формат MS Excel.

    Список сертификатов

    Колонка Описание
    Колонка флажков Выбор сертификатов, к которым предполагается применить какое-либо действие.
    Меню
    действий
    		Доступные действия:
    • Удалить - удалить сертификат.
    ID Идентификатор сертификата.
    Дата изм. Дата и время получения сертификата.
    Курс Название курса, по которому получен данный сертификат.
    Учащийся Указывается идентификатор, логин, имя и фамилия пользователя.
    Баллов Количество заработанных баллов.
    Макс. баллов Максимальное количество баллов.
    Акт. Признак активности сертификата.
    Сорт. Вес сортировки сертификата. Равен весу сортировка курса, за который получен данный сертификат.
    Вес определяет положение сертификата в списке, чем меньше вес, тем выше в списке располагается сертификат.
    Online Отметка о том, что учащийся сертифицировался Online или нет.
