    Для создания и редактирования глав учебного курса применяется форма одного и того же вида. В зависимости от того, создаете вы новую главу или редактируете существующую, некоторые поля могут присутствовать или отсутствовать на форме.

    Контекстная панель

    Кнопка Описание
    Список Ссылка для перехода на страницу со списком глав учебного курса.
    Создать Ссылка для перехода на страницу создания новой главы. Отображается только на странице редактирования уже существующей главы.
    Удалить Удаление главы, просматриваемой на данной странице. Команда доступна только на странице редактирования уже существующей главы.

    Поля формы

    Вкладка "Глава"

    Поле Описание
    ID Идентификатор главы.

    Данное поле отображается только в режиме редактирования существующей главы.
    Изменена Дата и время создания или изменения главы.

    Данное поле отображается только в режиме редактирования существующей главы.
    Запись активнаУстановка статуса активности
    Родительский раздел Указывается уровень, в котором находится глава.
    *Название Название главы.
    Индекс сортировки Вес сортировки главы; применяется при выводе главы в списке.
    Символическое имя Мнемоническое имя главы.
    * - Поля, обязательные для заполнения.

    Вкладка "Анонс"

    Поле Описание
    Text / HTML Выбор формата создания краткого описания главы.
    Использовать визуальный редактор При условии использования формата HTML, для создания анонса главы будет использоваться визуальный редактор.
    Изображение Поле позволяет в краткое описание главы добавить изображение с помощью кнопки Обзор...

    Вкладка "Содержание"

    Поле Описание
    Text / HTML Выбор формата создания детального описания главы.
    Использовать визуальный редактор При условии использования формата HTML, для создания детального описания главы будет использоваться визуальный редактор.
    Изображение Поле позволяет в детальное описание главы добавить изображение с помощью кнопки Обзор...

    Кнопки управления

    Кнопка Описание
    Сохранить Сохранение внесённых изменений. Возврат к списку глав курса.
    Применить Сохранение внесённых изменений. Продолжение редактирования параметров главы.
    Отменить Отмена внесённых изменений. Возврат к списку глав курса.
