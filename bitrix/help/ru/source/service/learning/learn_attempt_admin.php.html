    Фильтр

    Задание параметров отбора попыток, отображаемых в списке.

    Поле Описание
    Учащийся Отбор попыток прохождения тестов указанного учащегося.
    ID Идентификатор попытки.
    Тест Выбор теста, по которому необходимо отобразить попытки о прохождении. Если указано значение любой, то отображаются попытки по всем тестам.
    Дата начала Указывается период начала тестирования. Если какое-то из полей оставлено пустым (или оба), то временная граница считается неограниченной.
    Дата конца Указывается период окончания тестирования. Если какое-то из полей оставлено пустым (или оба), то временная граница считается неограниченной.
    Тест пройден Отображение всех попыток или только успешных попыток, или только неудачных попыток.
    Статус Выбирается один или несколько статусов попытки прохождения тестирования.
    Баллов Указывается интервал количества баллов, которые заработали учащиеся в ходе прохождения тестирования. Если какое-то из полей оставлено пустым (или оба), то граница считается неограниченной.

    Чтобы отфильтровать попытки по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех попыток нажмите кнопку Отменить.

    Контекстная панель

    Кнопка Описание
    Настроить Переход к диалогу настройки внешнего вида отчетной формы.
    Excel Экспорт данных из отображаемой таблицы в формат MS Excel.

    Список попыток

    Колонка Описание
    Колонка флажков Выбор попыток, к которым предполагается применить какое-либо действие.
    Меню
    действий
    		Доступные действия:
    • Результаты - переход к форме с подробным описанием результатов прохождения теста этой попытки;
    • Удалить - удалить попытку.
    ID Идентификатор попытки.
    Учащийся Указывается идентификатор, логин, имя и фамилия пользователя.
    Тест Название теста, который учащийся пытается пройти или уже прошел.
    Дата начала Дата и время начала тестирования.
    Дата конца Дата и время окончания тестирования. Если тестирование ещё не закончено, то данное поле пусто.
    Статус Статус, в котором находится процесс тестирования.
    Вопросов Количество вопросов в тесте.
    Тест пройден Указывается удачно пройден тест или нет.
    Баллов Количество баллов, которые заработал учащийся во время прохождения тестирования.
    Макс. баллов Максимальное количество баллов, которое можно заработать во время прохождения теста.
