Импорт курса

Форма (Рабочий стол > Сервисы > Обучение > Импорт) служит для загрузки учебного курса на сайт.

Вкладка "Файл данных " - 1 шаг

Поле Описание
*Файл данных С помощью кнопки Открыть в поле указывается имя и путь к загружаемому файлу данных.
*Сайт Выбор сайтов, на которые необходимо загрузить учебный курс.
* - Поля, обязательные для заполнения.

Для загрузки курса нажмите кнопку Загрузить данные , после чего будет осуществлен переход на вкладку Результат.

Вкладка "Результат" - 2 шаг

Вывод сообщение о завершении импорта.

Для повтора импорта воспользуйтесь кнопкой Назад, откроется вкладка Файл данных.
