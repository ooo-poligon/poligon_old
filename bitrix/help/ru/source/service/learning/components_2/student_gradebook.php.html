Компоненты 2.0::Журнал студента

Компонент отображает результаты прохождения тестов. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:learning.student.gradebook","",	Array(
     "TEST_DETAIL_TEMPLATE" => "course/index.php?COURSE_ID=#COURSE_ID#&TEST_ID=#TEST_ID#",
     "COURSE_DETAIL_TEMPLATE" => "course/index.php?COURSE_ID=#COURSE_ID#&INDEX=Y",
     "TEST_ID_VARIABLE" => "TEST_ID",
     "SET_TITLE" => "Y"
     )
);?>
Шаблоны ссылок
TEST_DETAIL_TEMPLATE
Путь к странице прохождения теста.
COURSE_DETAIL_TEMPLATE
Путь к странице с детальным просмотром курса.
Дополнительные настройки
TEST_ID_VARIABLE
Имя переменной, которой передается ID теста.
SET_TITLE
При отмеченной опции в качестве заголовка страницы будет установлено Результаты тестирования.
