Компоненты 2.0::Дерево курса

Компонент выводит иерархию глав и уроков. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:learning.course.tree","",Array(
     "COURSE_ID" => $_REQUEST["COURSE_ID"],
     "COURSE_DETAIL_TEMPLATE" => "course/index.php?COURSE_ID=#COURSE_ID#",
     "CHAPTER_DETAIL_TEMPLATE" => "chapter.php?CHAPTER_ID=#CHAPTER_ID#",
     "LESSON_DETAIL_TEMPLATE" => "lesson.php?LESSON_ID=#LESSON_ID#",
     "SELF_TEST_TEMPLATE" => "self.php?LESSON_ID=#LESSON_ID#",
     "TESTS_LIST_TEMPLATE" => "course/test_list.php?COURSE_ID=#COURSE_ID#",
     "TEST_DETAIL_TEMPLATE" => "course/test.php?COURSE_ID=#COURSE_ID#&TEST_ID=#TEST_ID",
     "CHECK_PERMISSIONS" => "Y",
     "SET_TITLE" => "Y"
     )
);?>
Основные параметры
COURSE_ID
Идентификатор курса.
Шаблоны ссылок
COURSE_DETAIL_TEMPLATE
Путь к странице с детальным просмотром курса.
CHAPTER_DETAIL_TEMPLATE
Путь к странице главы курса.
LESSON_DETAIL_TEMPLATE
Путь к странице урока курса.
SELF_TEST_TEMPLATE
Путь к странице с тестом для самопроверки.
TESTS_LIST_TEMPLATE
Путь к странице списка тестов курса.
TEST_DETAIL_TEMPLATE
Путь к странице прохождения теста.
Дополнительные настройки
CHECK_PERMISSIONS
При выборе значения Y будет проверяться право на доступ к курсу.
SET_TITLE
При отмеченной опции в качестве заголовка страницы будет установлено название главы курса.
