Компоненты 2.0::Детальный вывод урока

Компонент производит детальный вывод урока курса. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:learning.lesson.detail","",Array(
     "COURSE_ID" => $_REQUEST["COURSE_ID"],
     "LESSON_ID" => $_REQUEST["LESSON_ID"],
     "SELF_TEST_TEMPLATE" => "self.php?COURSE_ID=#COURSE_ID#&LESSON_ID=#LESSON_ID#",
     "CHECK_PERMISSIONS" => "Y",
     "SET_TITLE" => "Y",
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "3600"
     )
);?>

Описание параметров

Основные параметры
COURSE_ID
Идентификатор курса.
LESSON_ID
Идентификатор урока.
Шаблоны ссылок
SELF_TEST_TEMPLATE
Путь к странице с тестом для самопроверки.
Настройки кеширования
CACHE_TYPE
Тип кеширования:
  • A - Авто: действует при включенном кешировании в течение заданного времени;
  • Y - Кешировать: необходимо определить время кеширования;
  • N - Не кешировать: кеширования нет в любом случае.
CACHE_TIME
Время кеширования, указанное в секундах.
Дополнительные настройки
CHECK_PERMISSIONS
При выборе значения Y будет проверяться право на доступ.
SET_TITLE
При отмеченной опции в качестве заголовка страницы будет установлено название урока.
