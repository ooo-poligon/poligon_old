Компоненты 2.0::Профиль студента

Компонент отображает результаты прохождения тестов. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:learning.student.profile","",Array(
     "TRANSCRIPT_DETAIL_TEMPLATE" => "certification/?TRANSCRIPT_ID=#TRANSCRIPT_ID#",
     "SET_TITLE" => "Y"
     )
);?>
Шаблоны ссылок
TRANSCRIPT_DETAIL_TEMPLATE
Путь к странице с результатами сертификации.
Дополнительные настройки
SET_TITLE
При отмеченной опции в качестве заголовка страницы будет установлено Анкета специалиста.
