Компоненты 2.0::Контрольный тест

Компонент отображает контрольный тест курса. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:learning.test","",Array(
     "COURSE_ID" => $_REQUEST["COURSE_ID"],
     "TEST_ID" => $_REQUEST["TEST_ID"],
     "PAGE_NUMBER_VARIABLE" => "PAGE",
     "GRADEBOOK_TEMPLATE" => "../gradebook.php?TEST_ID=#TEST_ID#",
     "PAGE_WINDOW" => "10",
     "SHOW_TIME_LIMIT" => "Y",
     "SET_TITLE" => "Y"
     )
);?>

Описание параметров

Основные параметры
COURSE_ID
Идентификатор курса.
TEST_ID
Идентификатор теста.
Шаблоны ссылок
GRADEBOOK_TEMPLATE
Путь к странице с результатами тестирования (путь к журналу).
Дополнительные настройки
PAGE_NUMBER_VARIABLE
Имя переменной для идентификатора вопроса теста.
PAGE_WINDOW
Количество вопросов, отображаемых в навигационной цепочке.
SHOW_TIME_LIMIT
При отмеченной опции будет отображаться счетчик времени.

Примечание: если в настройках теста ограничение по времени не установлено, то счетчик не будет отображаться в любом случае.
SET_TITLE
При отмеченной опции в качестве заголовка страницы будет установлено имя пользователя.
