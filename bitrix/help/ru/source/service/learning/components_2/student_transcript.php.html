Компоненты 2.0::Резюме студента

Компонент отображает информацию о студенте и список полученных им сертификатов. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:learning.student.transcript","",Array(
     "TRANSCRIPT_ID" => $_REQUEST["TRANSCRIPT_ID"],
     "SET_TITLE" => "Y"
     )
);?>
Основные параметры
TRANSCRIPT_ID
Идентификатор резюме.
Дополнительные настройки
SET_TITLE
При отмеченной опции в качестве заголовка страницы будет установлено имя пользователя.
