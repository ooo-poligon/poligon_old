Компоненты 2.0::Список курсов

Компонент выводит список курсов. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:learning.course.list","",Array(
     "SORBY" => "SORT",
     "SORORDER" => "ASC",
     "COURSE_DETAIL_TEMPLATE" => "course/index.php?COURSE_ID=#COURSE_ID#&INDEX=Y",
     "CHECK_PERMISSIONS" => "Y",
     "COURSES_PER_PAGE" => "20",
     "SET_TITLE" => "Y"
     )
);?>
Шаблоны ссылок
COURSE_DETAIL_TEMPLATE
Путь к странице с детальным просмотром курса.
Дополнительные настройки
SORBY
Поле, по которому будет происходить сортировка курсов на странице:
  • ID - идентификатор курса;
  • NAME - название;
  • ACTIVE_FROM - дата начала активности;
  • SORT - индекс сортировки;
  • TIMESTAMP_X - дата последнего изменения.
SORORDER
Направление сортировки курсов:
  • asc - сортировка по возрастанию;
  • desc - сортировка по убыванию.
CHECK_PERMISSIONS
При выборе значения Y будет проверяться право на доступ к курсу.
COURSES_PER_PAGE
Количество курсов, выводимых на странице. Остальные будут выведены с помощью постраничной навигации.
SET_TITLE
При отмеченной опции в качестве заголовка страницы будет установлено Список курсов.
