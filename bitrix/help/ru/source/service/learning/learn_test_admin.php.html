    Фильтр

    Фильтр используется для задания параметров отбора тестов, отображаемых в списке.

    Поле Описание
    Название Отбор тестов по названию.
    IDОтбор тестов по ID
    СайтОтбор тестов для конкретного сайта
    Активность Отображение всех тестов или только активных, или только неактивных.
    Показать все условияОдновременное отображение всех условий фильтра
    Скрыть все условияСокрытие всех условий

    Чтобы отфильтровать тесты по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех тестов нажмите кнопку Отменить.

    Контекстная панель

    Кнопка Описание
    Добавить тест Служит для перехода к форме создания нового теста.
    Настроить Позволяет перейти к диалогу настройки внешнего вида отчетной формы.
    Excel Экспортирует данные из отображаемой таблицы в формат MS Excel.

    Список тестов

    Колонка Описание
    Колонка флажков Выбор тестов, к которым предполагается применить какое-либо действие.
    Меню
    действий
    		Допустимые действия:
    • Добавить главу - добавление главы учебного курса;
    • Добавить урок - добавление урока;
    • Изменить - отредактировать тест;
    • Удалить - удалить тест.
    ID Идентификатор теста.
    Дата изм. Дата и время последней модификации теста.
    Название Название теста.
    Сорт. Вес сортировки теста; применяется при выводе теста в списке.
    Акт. Признак активности теста.
    СайтСайт, к кторому относится тест
    ГлавыГлава теста
    УрокиУрок теста
