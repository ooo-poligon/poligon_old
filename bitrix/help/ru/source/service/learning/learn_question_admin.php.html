    Фильтр

    Задание параметров отбора вопросов, отображаемых в списке.

    Поле Описание
    Название Отбор вопросов по названию.  Данное поле отображается всегда, его невозможно скрыть.
    Урок Указывается урок, к которому относится вопрос, либо выбирается значение любой для отображения вопросов всех уроков.
    Для самопроверки Отображение всех вопросов или только вопросов для самопроверки, или только вопросов не для самопроверки.
    Активность Отображение всех вопросов или только активных, или только неактивных.

    Чтобы отфильтровать вопросы по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех вопросов нажмите кнопку Отменить.

    Контекстная панель

    Кнопка Описание
    Добавить вопрос Служит для перехода к форме создания нового вопроса.
    Настроить Переходк диалогу настройки внешнего вида отчетной формы.
    Excel Экспортирует данные из отображаемой таблицы в формат MS Excel.

    Список вопросов

    Колонка Описание
    Колонка флажков Выбор вопросов, к которым предполагается применить какое-либо действие.
    Меню
    действий
    		доступные действия:
    • Изменить - отредактировать вопрос;
    • Удалить - удалить вопрос.
    ID Идентификатор вопроса.
    Акт. Признак активности вопроса.
    Дата изм. Дата и время последней модификации вопроса.
    Название Текст самого вопроса.
    Сорт. Вес сортировки вопроса; применяется при выводе вопроса.
    Самопров. Принадлежность вопроса: для самопроверки или не для самопроверки.
    Тип Указывается тип вопроса: одиночный или множественный выбор выриантов ответа.
