  • Фильтр
  • Контекстная панель
  • Список курсов

    • Управление имеющимися учебными курсами (Рабочий стол > Сервисы > Обучение > Курсы).

    Фильтр

    Задание параметров отбора обучающих курсов, отображаемых в списке.

    Поле Описание
    Название Отбор учебных курсов по названию.
    Поле отображается всегда, его невозможно скрыть.
    ID Идентификатор искомого курса.
    Сайт Сайт, к которому принадлежит курс.
    Активность Отображение всех курсов или только активных, или только неактивных.

    Чтобы отфильтровать обучающие курсы по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех курсов нажмите кнопку Отменить.

    Контекстная панель

    Кнопка Описание
    Добавить курс Служит для создания нового учебного курса.
    Настроить Переход к диалогу настройки внешнего вида отчетной формы.
    Excel Экспортирует данные из отображаемой таблицы в формат MS Excel.

    Список курсов

    Колонка Описание
    Колонка флажков Выбор курсов, к которым предполагается применить какое-либо действие.
    Меню
    действий
    		Допустимые действия:
    • Добавить главу - переход к форме создания новой главы в курсе;
    • Добавить урок - переход к форме создания нового урока в курсе;
    • Изменить - переход к редактированию курса;
    • Удалить - удалить учебный курс.
    ID Идентификатор курса.
    Дата изм. Дата и время последней модификации курса.
    Название Название курса.
    Сорт. Вес сортировки курса; применяется при выводе курса в списке всех курсов.
    Акт. Признак активности курса.
    Сайт Сайт, к которому принадлежит курс.
    Главы Количество глав в курсе. При нажатии на число будет осуществлен переход к странице со списком всех глав курса.
    Символ "+" служит для перехода к созданию новой главы.
    Уроки Количество уроков во всем курсе. При нажатии на число будет осуществлен переход к странице со списком всех уроков курса.
    Символ "+" служит для перехода к созданию нового урока.
    Тесты Количество тестов в курсе. При нажатии на число будет осуществлен переход к странице со списком всех тестов курса.
    Символ "+" служит для перехода к созданию нового теста.
    Символическое имя Символическое имя курса.
