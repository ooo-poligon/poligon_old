Контроллер::Создание и редактирование клиента

Форма (Сервисы > Контроллер > Сайты > Редактировать или Добавить) предназначена для регистрации новых клиентов и редактирования параметров существующих.

Контекстная панель

Кнопка Описание
Вернуться к списку клиентов Переход на страницу со списком клиентов.
Добавить нового клиента Переход на страницу регистрации нового клиента. Кнопка отображается только при редактировании существующего клиента.
Отсоединить клиента Отключение клиента от контроллера. Кнопка отображается только при редактировании существующего неотсоединенного клиента.
Удалить текущий Удаление клиента.
Примечание: если при удалении клиент не был отсоединен, то запись о этом клиенте из контроллера будет удалена, а сам сайт будет работать с теми настройками, которые были установлены контроллером. Если же перед удалением клиент был отсоединен, то на сайте будут возвращены настройки, установленные до подключения к контроллеру.
Кнопка отображается только при редактировании существующего клиента.

Форма редактирования

Поле Описание
ID Идентификатор сайта-клиента. Поле отображается только при редактировании существующего клиента и изменению не подлежит.
Отсоединен от контроллера Отметка о том, что клиент отсоединен от контроллера. Поле отображается только при редактировании существующей записи о клиенте если клиент отключен и изменению не подлежит.
Дата создания Дата и время, а также логин, имя и фамилия пользователя, который зарегистрировал клиента. Поле отображается только при редактировании существующего клиента.
Последнее изменение Дата и время, а также логин, имя и фамилия последнего пользователя, изменившего параметры клиента. Поле отображается только при редактировании существующего клиента.
*Название Название клиента.
*Уникальный код Уникальный код клиента, генерируемый автоматически.
*URL клиента URL-адрес клиента.
*Группа Указывается группа, к которой принадлежит клиент.
*Логин администратора подключаемого сервера Логин администратора подключаемого сайта-клиента. Поле отображается только при регистрации нового клиента.
*Пароль администратора подключаемого сервера Пароль администратора подключаемого сайта-клиента. Поле отображается только при регистрации нового клиента.
Доступность для посещения
Текущий статусСтатус клиента. Поле отображается только при редактировании существующего клиента и изменению не подлежит.
Пробный периодОтображение срока демовресии
Признак активности клиента. Чтобы сайт-клиент был открыт для доступа посетителям, необходимо отметить данную опцию.
Период активности Указывается период, в течение которого активен клиент.
Общее ядро При отмеченной опции будет использована жесткая привязка клиента к контроллеру: локальные сайты не содержат ядра продукта, а используют подключенное символическими ссылками (symlink) общее ядро (мастер-копию) с головного сайта.
В противном случае будет использована мягкая привязка: сайт подключается со своим ядром.
Описание Произвольное описание клиента.

Кнопки управления

Кнопка Описание

Применить Сохранение внесенных изменений. Продолжение редактирования параметров клиента.
