Контроллер::Управление группами сайтов

На странице Управление группами сайтов (Сервисы > Контроллер > Группы) представлена таблица, содержащая список групп сайтов-клиентов. Страница служит для добавления новых групп, а также для перехода к редактированию существующих.

Фильтр

Фильтр используется для задания параметров отбора групп сайтов, отображаемых в списке. Следующая таблица описывает поля фильтра.

Поле Описание
Название Поиск группы по названию.
ID Поиск группы по идентификатору.
Период изменения Поиск группы по изменениям за указанный период.
Период создания Отбор групп, которые были созданы в указанный период.

Для того чтобы отфильтровать группы по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех групп нажмите кнопку Отменить.

Контекстная панель

Кнопка Описание
Добавить Служит для добавления новой группы сайтов.
Настроить Позволяет перейти к диалогу настройки внешнего вида отчетной формы.
Excel Экспорт данных из отображаемой таблицы в формат MS Excel.

Колонка флажков Поле предназначено для выбора групп сайтов, к которым предполагается применить какое-либо действие.
Меню
действий
При нажатии на кнопку действий конкретной группы отображается меню, позволяющее:
  • Редактировать - отредактировать группу сайтов;
  • Удалить - удалить группу.
    Примечание: удаление группы, к которой относится хотя бы один сайт, а также группы, используемой по умолчанию, невозможно.
