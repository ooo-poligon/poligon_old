Контроллер::Удаленное выполнение команд

Форма (Сервисы > Контроллер > Удаленное выполнение команд) служит для задания php-скрипта, который будет выполняться на удаленном сервере.

Форма фильтра

Поле Описание
Сайт Выбирается сайт, на котором будет выполнен скрипт. При выборе значения все скрипт будет выполнен на всех сайтах указанной группы.
Группа Выбирается группа, на сайте (сайтах) которой должен быть выполнен скрипт.

Вкладка "PHP-скрипт"

Поле Описание
[поле типа textarea] Поле служит для ввода произвольного php-скрипта.
Переслать файлыВыбор в полях исходного и конечного путей для пересылки файла
Выполнить через планировщик задач При отмеченной опции команда будет выполнена не сразу, а поставлена в очередь задач.

Кнопки управления

Поле Описание
Выполнить Выполнение указанного в поле скрипта или постановка задачи в очередь.
Очистить Очистка поля для создания нового скрипта.
