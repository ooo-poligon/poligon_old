Контроллер::Задачи для выполнения на клиентах

Список задач, выполняемых на клиентах контроллера. Могут быть выполнены следующие задачи:
  • установка групповых настроек;
  • обновление сайта;
  • обновление счетчика;
  • выполнение команды;
  • закрытие/открытие клиентов.

Фильтр

Фильтр используется для задания параметров отбора задач, отображаемых в списке.

Поле Описание
Статус выполнения Поиск задач по статусам их выполнения. Возможен множественный выбор статусов.
ID Поиск задачи по идентификатору.
Клиент Название клиента, на котором должны выполниться (выполнялись) задачи.
Операция Поиск задачи по названию. Если выбрано значение любая, то будут отображены все задачи.
Период выполнения Отбор задач, с указанным периодом их выполнения.
Период изменения Отбор задач, с указанным периодом их изменения.
Период создания Отбор задач, с указанным периодом их создания.

Чтобы отфильтровать задачи по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех задач нажмите кнопку Отменить.

Контекстная панель

Кнопка Описание
Обновить список Обновляет список задач.
Настроить Позволяет перейти к диалогу настройки внешнего вида отчетной формы.
Excel Экспортирует данные из отображаемой таблицы в формат MS Excel.

Список задач

Колонка Описание
Колонка флажков Выбор задач, к которым предполагается применить какое-либо действие.
Меню
действий
Доступные действия:
  • Повторить - повторить выполнение задачи.
  • Отменить - удалить задачу из списка.
Клиент Название клиента, на котором выполнена задача.
Задача Название задачи.
Состояние Один из статусов выполнения задачи: новая, выполнена частично, успешно выполнена или ошибка выполнения.
Выполнено Дата и время выполнения задачи.
Аргумент Аргумент задачи. Например, для задачи Выполнить команду в качестве аргумента выступает php-скрипт.
Результат Указывается результат выполнения задачи или описание ошибки выполнения задачи.
URL клиента URL-адрес клиента.
Дата изменения Дата и время изменения задачи.
Дата создания Дата и время создания задачи.
ID Идентификатор задачи.
