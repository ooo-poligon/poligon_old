Контроллер::Настройки модуля

Форма (Настройки > Настройки продукта > Настройки модулей > Контроллер) предназначена для задания общесистемных параметров модуля Контроллер сайтов.

Закладка "Настройки"

Поле Описание
Группа по умолчанию для присоединяющихся сайтов Название группы, к которой будут присоединяться сайты при их присоединении к контроллеру.
Интервал для автоматической установки групповых настроек на клиентские сайты, по умолчанию, минут Задается интервал по умолчанию (в минутах) для автоматической установки групповых настроек на клиентские сайты. Если указано 0, то настройки клиентов не будут обновляться автоматически.
Путь к папке с общим ядром (относительно корня сайта-контроллера) Для возможности обновления клиентов с общим файловым ядром продукта, необходимо указать путь к папке с общим ядром.

Закладка "Доступ"

Форма служит для определения прав групп пользователей на доступ к модулю Контроллер сайтов.

Поле Описание
По умолчанию Управление уровнем доступа групп пользователей, для которых установлено право по умолчанию.
[Группа пользователей] Управление правом доступа отдельных групп пользователей. Возможны следующие уровни доступа:
  • Доступ запрещен - доступ к модулю закрыт;
  • Только чтение - разрешается только просмотр файлов и параметров модуля;
  • Добавление новых сайтов - добавление новых сайтов, позволяет безопасно вводить имя-пароль на клиентских машинах при присоединении;
  • Управление сайтами - управление сайтами, т.е. их добавление, обновление, отсоединение, административный вход на них, просмотр журнала работы. Нельзя удалять сайты и изменять групповые настройки;
  • Полный доступ - полный доступ без каких-либо ограничений.
