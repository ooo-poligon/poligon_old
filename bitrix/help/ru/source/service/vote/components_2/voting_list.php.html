Компоненты 2.0::Список вопросов

Вывод списка вопросов. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:voting.list","",Array(
     "CHANNEL_SID" => "BOOKS_VOTE",
     "VOTE_FORM_TEMPLATE" => "vote_new.php?VOTE_ID=#VOTE_ID#",
     "VOTE_RESULT_TEMPLATE" => "vote_result.php?VOTE_ID=#VOTE_ID#"
     )
);?>

Описание параметров

Основные параметры
CHANNEL_SID
Символьный идентификатор группы опросов.
Шаблоны ссылок
VOTE_FORM_TEMPLATE
Указывается путь к странице с формой вопроса. В качестве параметра страницы необходимо передавать идентификатор вопроса.

VOTE_RESULT_TEMPLATE
Указывается путь к странице с диаграммой результатов опроса. В качестве параметра страницы необходимо передавать идентификатор опроса.
