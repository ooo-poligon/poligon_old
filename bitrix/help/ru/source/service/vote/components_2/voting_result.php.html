Компоненты 2.0::Результаты опроса

Вывод результата опроса в виде диаграммы. Диаграмма может быть представлена в трех видах: по умолчанию, гистограмма или круговая диаграмма.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:voting.result","",Array(
     "VOTE_ID" => "2",
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "1200",
     "QUESTION_DIAGRAM_5" => "histogram",
     "QUESTION_DIAGRAM_7" => "histogram"
     )
);?>

Описание параметров

Основные параметры
VOTE_ID
Идентификатор опроса. Задается явно или с помощью кода.
Настройки кэширования
CACHE_TYPE
Тип кеширования:
  • A - Авто: действует при включенном кешировании в течение заданного времени;
  • Y - Кешировать: необходимо определить время кеширования;
  • N - Не кешировать: кеширования нет в любом случае.

CACHE_TIME
Время кеширования, указанное в секундах.
Дополнительно

Эта группа параметров отображается только в том случае, если параметр VOTE_ID задан явно и для этого опроса созданы вопросы.

QUESTION_DIAGRAM_5
Тип диаграммы для вопроса с идентификатором равным 5.

QUESTION_DIAGRAM_7
Тип диаграммы для вопроса с идентификатором равным 7.
