Компоненты 1.0::Список опросов

Вывода списка опросов. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова

<?
$APPLICATION->IncludeFile("vote/vote_list/default.php", array(
    "CHANNEL_SID" => "ANKETA",
    "VOTE_URL" => "vote_new.php",
    "RESULTS_URL" => "vote_result.php"
));
?>

Описание параметров

CHANNEL_SID
Символьный код группы опросов (если не задан, то будут выводиться опросы всех групп привязанных к текущему сайту).
VOTE_URL
Адрес страницы, на которой выводится форма для голосования (участия в опросе).
RESULTS_URL
Адрес страницы, на которой выводятся диаграммы результатов опроса.
