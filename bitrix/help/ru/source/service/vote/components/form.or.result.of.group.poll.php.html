Компоненты 1.0::Форма/результаты текущего опроса в группе

Вывод формы текущего опроса заданной группы. Если пользователь уже голосовал по данному опросу, то данный компонент выводит диаграмму его результатов. Как правило, используется для главной страницы сайта. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова

<?
$APPLICATION->IncludeFile("vote/vote_new/current_channel.php", 
array(
    "CHANNEL_SID" => “PHONE”
));
?>

Описание параметров

CHANNEL_SID
Символьный код группы опросов.
