Компоненты 1.0::Вывод формы опроса

Вывод пустой формы опроса, предназначенной для заполнения. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова

<?
$APPLICATION->IncludeFile("vote/vote_new/default.php", 
array(
    "VOTE_ID" => $VOTE_ID,
    "BACK_REDIRECT_URL" => "vote_result.php"
));
?>

Описание параметров

VOTE_ID
ID опроса.
BACK_REDIRECT_URL
Адрес страницы на которую будет перенаправлен пользователь после голосования (если параметр оставить незаполненным, пользователь не будет перенаправляться после голосования).
