Посетители

Информация по посетителям сайта, участвовавшим в опросах.

Фильтр

Задание параметров отбора посетителей, которые отображаются в списке.

Наименование параметраОписание
ID, имя, фамилия пользователя* Маска для имени и фамилии авторизованного пользователя, либо ID пользователя.
ID* ID посетителя для модуля опросов.
Дата первого опроса Интервал для даты когда посетитель впервые принял участие в опросах.
Дата последнего опроса Интервал для даты когда посетитель впервые принял участие в опросах.
ID посетителя в модуле статистики* ID посетителя выданный модулем статистики.
IP адрес* Маска для последнего IP адреса с которым голосовал посетитель.
Число голосований Интервал для количество голосований данного посетителя.
Опрос* Опрос, в котором участвовали посетители.

* - для данных полей вы можете воспользоваться специальными логическими выражениями.

Для инициализации фильтра необходимо нажать кнопку Найти. Для отображения всех опросов нажмите кнопку Отменить.
