Настройки

Форма (Настройки > Настройки продукта > Настройки модулей > Опросы, голосования) предназначена для установки глобальных настроек модуля.

Вкладка "Настройки"

Наименование параметра Описание
Использовать HTML редактор (только для IE 5.0 или FireFox 1.0 и выше): Флаг использования HTML редактора при редактировании описания для опроса.
Поддерживать старые шаблоны (default.php, bluebar.php) в форме редактирования вопросов: Флаг использования шаблонов старой версии при редактировании вопросов.

Вкладка "Доступ"

На вкладке редактирования прав доступа к модулю можно задать следующие права:

Закрыт - доступ к меню и административным файлам модуля закрыт.
Просмотр всех данных модуля - доступ к меню открыт, в административных файлах разрешен просмотр всего без права сохранения модификаций в базе.
Запись -полный доступ.

Все вышеперечисленные права являются наследуемыми.
