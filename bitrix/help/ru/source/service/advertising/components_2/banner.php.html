Компоненты 2.0::Баннер

Компонент выводит баннер заданного типа. Компонент стандартный и входит в дистрибутив модуля.

Внимание! Компонент не учитывает таргетинг по ключевым словам, для этого необходимо использовать функцию $APPLICATION->ShowBanner().

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:advertising.banner","",Array(
     "TYPE" => "LEFT",
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "0"
     )
);?>

Описание параметров

Основные параметры
TYPE
Указывается один из типов баннера, созданных в системе.
Настройки кеширования
CACHE_TYPE
Тип кеширования:
  • A - Авто: действует при включенном кешировании в течение заданного времени;
  • Y - Кешировать: необходимо определить время кеширования;
  • N - Не кешировать: кеширования нет в любом случае.

CACHE_TIME
Время кеширования, указанное в секундах.
