    Кнопка Описание

    График CTR баннера Просмотр графика отклика для данного баннера.

    Форма редактирования

    Форма предназначена для создания новых и изменения параметров существующих баннеров. Для удобства работы форма разделена на несколько закладок. Каждая закладка содержит логически сгруппированный набор полей для управления параметрами баннера.

    Закладка "Баннер"

    Служит для задания основных параметров баннера.

    Поле Описание
    Статус баннера Индикатор, определяющий статус показа баннера:
  • - баннер не показывается;
  • - баннер показывается.
    • Поле отображается в режиме редактирования существующего баннера.
    Создан Дата и время создания баннера, а также имя пользователя, создавшего баннер.

    Поле отображается только в режиме редактирования существующего баннера.
    Изменен Дата и время изменения баннера, а также имя пользователь, выполнившего изменения.

    Поле отображается только в режиме редактирования существующего баннера.
    Контракт Контракт, к которому приписывается баннер. Перечень доступных контрактов представлен в виде поля со списком.
    Активен Признак активности баннера. Для показа в публичном разделе доступны только активные баннеры.
    Интервал показа Период времени, в течение которого баннер может показываться на сайте. По истечении этого периода, баннер становится не активным.
    Имя Произвольное имя баннера.
    Группа Произвольная группа баннера (в выпадающем списке вы можете выбрать одну из ранее созданных групп).
    Тип Тип баннера.
    Вес (приоритет) Относительный "вес", влияющий на вероятность выбора баннера для показа в публичном разделе сайта.
    Что показывать
    Изображение Поле для загрузки изображения с локального компьютера. Это изображение будет показываться в виде баннера.
    Текст всплывающей подсказки Текст подсказки, которая будет появляться при наведении указателя мыши на изображении.
    Flash-баннер не содержит ссылку Установка флага в данное поле позволяет при нажатии на flash-баннер выполнять переход по ссылке, указанной в поле URL для перехода при клике на изображении, а также выполнять регистрацию события клика по баннеру в модуле статистики.

    Обратите внимание!
    Выбор данной опции накладывает ограничения на использование интерактивности flash-баннеров. Т.е. если flash-баннер содержит некоторую интерактивность (например, появление сообщения или эффекта движения при наведении указателем мыши на баннер), то при выборе этой опции данная интерактивность будет недоступна.

    URL для перехода при клике на изображении Адрес страницы, которая будет открыта по клику на баннере.
    Где развернуть URL <a target= Параметр, определяющий, в каком окне будет показана страница, открытая по клику на баннер:
    • _self – открыть в текущем окне (по умолчанию);
    • _blank – открыть в новом окне;
    • _parent – открыть в своем фрейме;
    • _top  - во всем текущем окне браузера.
    Код баннера
    Код баннера - произвольный текст или HTML-код, который будет отображаться в качестве баннера.

    В коде баннера можно использовать макросы #RANDOM1#, #RANDOM2#, #RANDOM3#, #RANDOM4#, #RANDOM5#, которые будут заменены случайными числами в момент показа в публичной части сайта данного кода. Также можно использовать следующие шаблоны:
    • #BANNER_NAME# - имя баннера;
    • #BANNER_ID# - ID баннера;
    • #CONTRACT_ID# - ID контракта;
    • #TYPE_SID# - тип баннера.
    Статус баннера
    Статус Статус баннера, определяющий возможность показа баннера на сайте:
    • подтвержден – в данном статусе баннер доступен для показа на сайте;
    • на рассмотрении – данный статус не позволяет показывать баннер на сайте и означает, что баннер необходимо либо перевести в статус подтвержден, либо в статус отклонен;
    • отклонен – в данном статусе баннер не может быть показан на сайте.
    Комментарий к статусу Пояснение к изменению статуса баннера.
    Оповестить об изменении статуса владельцев контракта Если опция выбрана, то при изменении статуса баннера владельцу контракта, к которому относится баннер, будет отправлено почтовое сообщение по шаблону Изменился статус баннера.
    Закладка "Ограничения"

    Служит для задания ограничений на показ баннера.

    Поле Описание
    Фиксировать число показов баннера Ограничение числа показов баннера.
    Показывать баннеры с равномерной скоростью При отмеченной опции баннер будет показываться с равномерной скоростью.
    Необходимо настроить следующие параметры:
    • указать интервал показа;
    • отметить опцию фиксировать число показов баннера;
    • задать максимальное число показов баннера.
    Максимальное число посетителей, для которых будет показан баннер Максимальное число посетителей, для которых будет показан баннер.

    При редактировании баннера рядом с данным полем отображается число посетителей, которым был показан баннер. Чтобы обнулить показатели счетчика, установите флаг в поле сбросить и нажмите кнопку Применить.
    Максимальное число показов данного баннера одному посетителю Установка ограничения на число показов баннера одному посетителю. Считается, что оптимальным количество показов баннера одному посетителю равно 4.
    Максимально число показов баннера Максимально число показов баннера, при достижении которого баннер становится неактивным.

    При редактировании баннера рядом с данным полем отображается число выполненных показов баннера. Чтобы обнулить показатели счетчика, установите флаг в поле сбросить и нажмите кнопку Применить.
    Фиксировать клики на данный баннер Регистрация кликов по баннеру в модуле статистики.
    Максимальное число кликов на баннер Максимальное число кликов на баннер, при достижении которого баннер становится неактивным.

    При редактировании баннера рядом с данным полем отображается число выполненных кликов по баннеру. Для того чтобы обнулить показатели счетчика, установите флаг в поле сбросить и нажмите кнопку Применить.
    Отклик баннера (CTR) (%) Значение показателя отклика данного баннера.
    Закладка "Таргетинг"

    Настройка таргетинга рекламы. Диапазон доступных значений полей зависит от настроек контракта, к которому относиться данный баннер.

    Поле Описание
    Сайты Сайты, на которых будет показываться баннер.
    Разделы и страницы, на которых баннер обязательно будет показан Адреса разделов или страниц сайта, на которых должен показываться баннер. Например: /ru/catalog/ или/ru/catalog/index.php.

    Адреса страниц могут быть заданы с использованием параметров. Каждый адрес раздела или страницы должен начинаться с новой строки.
    Разделы и страницы, на которых баннер не будет показан Адреса разделов или страниц сайта, на которых не должен показываться данный баннер.

    Адреса страниц могут быть заданы с использованием параметров. Каждый адрес раздела или страницы должен начинаться с новой строки.
    Ключевые слова Ключевые слова для баннера, влияющие на возможность показа баннера на страницах сайта:
    • Если для страницы сайта заданы желательные ключевые слова, то баннер будет показан на странице в том случае, если хотя бы одно ключевое слово баннера присутствует в наборе желательных ключевых слов страницы;
    • Если для страницы сайта заданы обязательные ключевые слова, то баннер будет показан на странице в том случае, если набор ключевых слов баннера совпадает с набором обязательных ключевых слов страницы.
    Более подробная информация приводится в руководстве Управление показом баннеров с помощью ключевых слов.
    Группы пользователей С помощью данного поля вы можете выбрать группы пользователей, для которых будет (или не будет) показываться данный баннер. Для этого выберите необходимые группы пользователей и установите переключатель в одно из значений: или Не показывать баннер для выбранных групп.
    Показывать только для посетителей следующих стран Страны, посетителям из которых будет показываться данный баннер.
    Показать только для посетителей, приходящих по следующим рекламным кампаниям Поле доступно только администраторам модуля.

    Показ баннера для посетителей, пришедших по определённой рекламной компании. Регистрация рекламных компаний выполняется с помощью модуля статистики.
    Показывать посетителям Выбор, каким посетителям показывать баннер:
    • всем - показывать баннер всем посетителям сайта;
    • только новым - показывать баннер пользователям, впервые пришедшим на сайт;
    • только вернувшимся - показывать баннер пользователям, вернувшимся на сайт.
    Время и дни недели для показа Часы и дни недели, по которым баннер доступен для показа. На диапазон доступных значений влияют ограничения, заданные для контракта, к которому относится баннер.
    Закладка "Статистика"

    Параметры регистрации кликов на баннер в модуле статистики. Управление параметрами доступно только администраторам сайта.

    Поле Описание
    Фиксировать в статистике Разрешение регистрирации события клика по баннеру в модуле Статистики.
    Event1 Идентификатор типа события - event1.
    Event2 Идентификатор типа события - event2.
    Event3 Дополнительный параметр события.

    Для создания идентификаторов типа события event1, event2, event3 можно использовать следующие макросы:
    • #BANNER_NAME# - имя баннера;
    • #BANNER_ID# - ID баннера;
    • #CONTRACT_ID# - ID контракта;
    • #TYPE_SID# - тип баннера.
    Закладка "Комментарий"

    Создание произвольного комментария к баннеру.

    Кнопки управления

    Кнопка Описание
    Сохранить Сохранение внесённых изменений. Переход на страницу со списком баннеров.
    Применить Сохранение внесённых изменений. Продолжение редактирования параметров баннера.
    Отменить Отмена внесённых изменений. Возврат первоначальных значений параметров.
