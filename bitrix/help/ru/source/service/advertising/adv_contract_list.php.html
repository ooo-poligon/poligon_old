    На странице Контракты (Сервисы > Реклама > Контракты) представлен список рекламных контрактов.

    Рекламный контракт – набор ограничений, связанных с показом баннеров. К рекламному контракту может быть привязано произвольное число пользователей, имеющих различные права на этот контракт.  

    Владельцы контракта– пользователи, привязанные к контракту и имеющие различные права на него. Пользователь может быть привязан к контракту только в том случае, если он входит в группу, для которой в настройках модуля рекламы установлен уровень доступа Работа с контрактами (рекламодатель).

    Отклик баннеров контракта (CTR) – отношение суммарного числа кликов на баннеры контракта к суммарному числу показов этих баннеров. Выражается в процентах и рассчитывается по следующей формуле: (K * 100) / S, где K – суммарное количество кликов на все баннеры контракта, S – суммарное количество показов всех баннеров контракта. Контракт с CTR выше 10% считается удачным.

    Вес контракта - число, влияющее на вероятность выбора контракта при показе баннеров. При показе баннера в публичной части сайта сначала выбирается контракт, баннер которого будет показан. При этом, чем выше вес контракта, тем больше вероятность того, что данный контракт будет выбран. После того, как определён контракт, система выбирает баннер, принадлежащий данному контракту.

    Форма поиска

    Поиск рекламного контракта в соответствии с указанными условиями.

    Поле Описание
    *ID Идентификатор (ID) требуемого контракта.
    Изменено Время возможного изменения требуемых баннеров.
    Индикатор Это поле позволяет отобрать контракты по значению индикатора:
    • красный – контракт неактивен либо истек в соответствии со своими ограничениями;
    • зеленый – контракт активен и не истек.
    Сайты Позволяет отобразить только те контракты, баннеры которых показываются только на указанных сайтах.
    Количество баннеров Интервал количества баннеров, приписанных к требуемым контрактам.
    Посетителей Интервал числа уникальных посетителей, которым были показаны баннеры требуемых контрактов.
    Показано Интервал суммарного количества показов баннеров требуемых контрактов.
    Кликнули Интервал суммарного количество кликов на баннеры требуемых контрактов.
    Отклик баннеров контракта (CTR) (%) Интервал откликов баннеров требуемых контрактов (CTR).
    *ID, логин, имя владельца ID и/или логин и/или ФИО пользователя, приписанного к требуемым контрактам.
    *Заголовок Поиск по заголовку требуемых контрактов.
    *Описание Поиск по описанию требуемых контрактов.
    *Административный комментарий Поиск по комментарию администратора.

    * - для данных полей вы можете воспользоваться специальными логическими выражениями. При поиске по данному полю вы можете пользоваться флагом в чекбоксе, расположенным справа от поля для поиска точного совпадения с искомой информацией.

    Чтобы отобрать рекламные контракты по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех контрактов нажмите кнопку Отменить.

    Контекстная панель

    Кнопка Описание
    Добавить контракт Переход к форме создания нового рекламного контракта.
    Настроить Переход к диалогу настройки внешнего вида отчетной формы.
    Excel Экспорт данных из отображаемой таблицы в формат MS Excel.

    Список рекламных контрактов

    Поле Описание
    Колонка флажков Поле предназначено для выбора контрактов, к которым предполагается применить какое-либо правило или действие.
    Действия Действия над контрактом:
    • Изменить - редактирование рекламного контракта;
    • Просмотреть - просмотр параметров контракта;
    • График - просмотр графика CTR для данного контракта;
    • Удалить - удаление контракта. Удаление контракта, используемого по умолчанию (с ID=1), невозможно.
    ID Идентификатор контракта.
    Инд. Индикатор контракта:
  • – контракт не активен, либо истек в соответствии со своими ограничениями;
  • – контракт активен.
    • Изменено Дата и время последнего изменения контракта.
    Сайты Сайты, на которых действует данный контракт.
    Сорт. Порядок показа контрактов в списке.
    Акт. Признак активности контракта. Для показа в публичном разделе сайта могут быть выбраны баннеры только активного контракта.
    Вес Относительный "вес" контракта. Влияет на вероятность выбора для показа в публичном разделе баннера, принадлежащего к данному контракту.
    Заголовок Заголовок контракта.
    Описание Описание контракта.
    Баннеров Количество баннеров, принадлежащих контракту. Ссылка с количеством баннеров позволяет перейти к списку этих баннеров.
    Посетителей Количество уникальных посетителей, которым показаны баннеры контракта.
    Посетителей макс. Максимальное количество уникальных посетителей, которым будут показаны баннеры контракта.
    Показано Количество показов баннеров контракта.
    Максим. показов Максимальное количество показов баннеров контракта, при достижении которого контаракт становится неактивным.
    Кликнули Количество кликов по баннерам данного конракта.
    Максим. кликов Максимальное количество кликов на баннеры контракта, при достижении которого контракт становится неактивным.
    CTR (%) Значение показателя суммарного отклика баннеров контракта.
