    Форма настроек модуля Рекламы (Настройки > Настройки продукта > Модули) предназначена для задания общесистемных параметров модуля.

    Закладка "Настройки"

    Настройка общих параметров модуля.

    Название параметра Описание параметра
    Ограничения контрактов не будут использованы при показе баннеров.
    Использовать HTML редактор (только для IE 5.0 или FireFox 1.0 и выше) Разрешить или запретить использование визуального HTML редактора при редактировании кода баннеров.
    Сколько дней хранить динамику по баннерам Ограничить количество дней, отводимых для хранения данных по показам и кликам на баннеры.
    Ширина графика Ширина графиков (в пикселях).
    Высота графика Высота графиков (в пикселях).
    Диаметр круговой диаграммы Диаметр круговых диаграмм (в пикселях).
    На сколько дней запоминать, что посетитель посмотрел тот или иной баннер Количество дней, в течение которых посетитель считается просмотревшим тот или иной баннер.
    Полное имя файла для учета кликов на баннер Путь к файлу, содержащему скрипт для сбора статистики по кликам на баннеры. Скрипт по умолчанию (/bitrix/rk.php) регистрирует клики в модуле Статистики, если последний установлен; иначе, предпринимается попытка зарегистрировать клик в модуле Рекламы.
    Подкаталог для хранения загруженных баннеров Имя подкаталога (относительно каталога /upload/), в котором хранятся все изображения, загружаемые при создании баннеров.

    Закладка "Доступ"

    Настройка прав групп пользователей на доступ к модулю рекламы.

    Поле Описание
    По умолчанию Управление правами доступа всех групп пользователей, для которых установлен уровень доступа по умолчанию.
    [Группа пользователей] Уровень доступа для конкретной группы пользователей:
    • Доступ закрыт - доступ запрещён;
    • Рекламодатель - доступ к меню и административным файлам открыт, пользователь видит только те контракты и принадлежащие им баннеры, на которые у него есть соответствующие права. Данный уровень прав используется для групп рекламодателей, размещающих свои баннеры на вашем сайте.
    • Менеджер баннеров- доступ к управлению баннерами контракта (без права изменения параметров контракта), в настройках которого для данных пользователей установлено право Управление баннерами.
    • Демо-доступ - пользователь видит все контракты, типы баннеров, баннеры, может строить любые графики, но без права модификации чего-либо (право обычно используется в демонстрационном режиме, когда пользователь может все видеть, но не имеет права что либо изменять).
    • Администратор рекламы - полный доступ, без каких либо ограничений.
