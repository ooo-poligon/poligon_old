    На странице Баннеры (Сервисы > Реклама > Баннеры) представлен список рекламных баннеров, имеющихся на сайте.

    Рекламный баннер – это графическое изображение или текстовый блок рекламного характера, клик на котором обычно приводит к открытию страницы с описанием рекламируемого товара или услуги. Баннер в системе "1С-Битрикс: Управление сайтом" характеризуется следующими признаками:

    • типом (тип имеет уникальный идентификатор, используемый в шаблоне сайта для указания места размещения баннера);
    • контрактом (рекламный баннер обязательно должен быть приписан к какому-либо контракту);
    • группой (рекламный баннер может входить в ту или иную группу).

    Также баннер может находиться в одном из трех статусов:

    • подтвержден – в данном статусе баннер доступен для показа на сайте;
    • на рассмотрении – данный статус не позволяет показывать баннер на сайте и означает, что баннер необходимо либо перевести в статус подтвержден, либо в статус отклонен;
    • отклонен – в данном статусе баннер не может быть показан на сайте.

    При переводе баннера в тот или иной статус можно указать причину смены статуса в поле Комментарий к статусу.

    Отклик баннера (CTR) – отношение числа кликов на баннер к числу показов этого баннера. Выражается в процентах и рассчитывается по следующей формуле: (K * 100) / S, где K – число кликов на баннер, S – число показов баннера. Баннер с CTR выше 10% считается очень удачным.

    Вес баннера - число, влияющее на вероятность выбора баннера при показе. При показе баннера в публичной части сайта сначала выбирается контракт, баннер которого будет показан. При этом, чем выше вес контракта, тем больше вероятность того, что данный контракт будет выбран. После того, как определён контракт, система выбирает баннер, принадлежащий данному контракту. Чем выше вес баннера, тем больше вероятность выбора данного баннера для показа.

    Группа баннера– баннеры, обладающие каким-либо общим признаком (например, рекламируемый объект), могут быть объединены в группу. Название группы задается произвольно. Группировка баннеров применяется для удобства управления баннерами, а также при построении отчетов по рекламным компаниям.

    Форма поиска

    С помощью данной формы вы можете выполнить поиск баннера в соответствии с указанными условиями.

    Поле Описание
    *ID Идентификатор баннера.
    Индикатор Поле позволяет отобрать баннеры по значению индикатора:
    • – баннер неактивен (например, истек заданный период активности);
    • – баннер активен.
    На значение данного индикатора влияет индикатор контракта, к которому принадлежит баннер. Если контракт имеет красный индикатор (контракт не активен), то все баннеры данного контракта будут также иметь красный индикатор.
    Статус Статус баннера.
    Сайты Сайты, для показа на которых предназначен баннер.
    Посетителей Интервал количества посетителей, просмотревших тот или иной баннер.
    Показано Интервал количества показов баннера.
    Кликнули Интервал количества кликов на баннер.
    Отклик баннера (CTR) (%) Интервал значений показателя отклика баннера.
    *Группа Группа, к которой относиться баннер.
    *Контракт Контракт баннера.
    *Тип Тип баннера.
    *Имя Имя баннера.
    *Код Код баннера.
    *Комментарий Комментарий к баннеру.

    * - для данных полей вы можете воспользоваться специальными логическими выражениями. При поиске по данному полю вы можете пользоваться флагом в чекбоксе, расположенным справа от поля для поиска точного совпадения с искомой информацией.

    Чтобы отобрать баннеры по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех баннеров нажмите кнопку Отменить.

    Контекстная панель

    Список баннеров

    Поле Описание
    Колонка флажков Выбор баннеров для примения какого-либо правила или действия.
    Действия Действия над баннером:
    • Изменить - редактирование баннера;
    • Просмотреть - просмотр параметров баннера;
    • График CTR - просмотр графика отклика для данного баннера;
    • Копировать - копирование баннера в новый;
    • Удалить баннер - удаление баннера.
    ID Идентификатор баннера.
    Инд. Индикатор активности баннера:
  • – баннер неактивен (например, истек заданный период активности);
  • – баннер активен.
    • На значение данного индикатора влияет индикатор контракта, к которому принадлежит баннер. Если контракт имеет красный индикатор (контракт не активен), то все баннеры данного контракта будут также иметь красный индикатор.
    Сайты Сайты, на которых может быть показан данный баннер.
    Акт. Признак активности баннера:
    • Да - баннер активен;
    • Нет - баннер неактивен.
    Вес Относительный "вес" баннера, влияющий на вероятность выбора баннера для показа на сайте.
    Имя Имя баннера.
    Группа Группа, к которой принадлежит данный баннер.
    Тип Название типа баннера.
    Контракт Название контракта, к которому приписан баннер.
    Статус Текущий статус баннера.
    Посетителей Количество посетителей, просмотревших баннер.
    Посетителей макс. Максимальное число посетителей, для которых будет показан баннер.
    Показано Количество выполненных показов баннера.
    Максим. показов Максимально число показов баннера, при достижении которого баннер становится неактивным.
    Последний показ Дата и время последнего показа баннера.
    Кликнули Количество кликов по данному баннеру.
    Максим. кликов Максимальное число кликов на баннер, при достижении которого баннер становится неактивным.
    Последний переход Дата и время последнего перехода по баннеру.
    CTR (%) Значение показателя отклика данного баннера.
    Дата первого показа Дата и время первого показа баннера.
    Равномерность ротации Отметка о равномерности показа баннеров.
