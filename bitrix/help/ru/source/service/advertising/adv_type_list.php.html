    На странице Типы баннеров (Сервисы > Реклама > Типы баннеров) представлен список используемых в системе типов баннеров.

    Тип баннера - именованная область в HTML-шаблоне сайта, в которой выводятся баннеры. Имя области в шаблоне сайта совпадает с именем типа баннера, которые требуется показывать в данной области.

    Форма поиска

    Поиск типа баннеров в соответствии с указанными условиями.

    Поле Описание
    *ID Идентификатор типа баннера.
    Изменено Дата изменения типа баннера.
    Активен Признак активности типа баннеров.
    *Имя Имя типа.
    *Описание Описание типа.

    * - для данных полей вы можете воспользоваться специальными логическими выражениями. При поиске по полю вы можете пользоваться чекбоксом, расположенным справа от поля для поиска точного совпадения с искомой информацией.

    Чтобы отобрать типы баннеров по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех типов баннеров нажмите кнопку Отменить.

    Контекстная панель

    Кнопка Описание
    Добавить тип Переход к форме создания нового типа баннеров.
    Настроить Переход к диалогу настройки внешнего вида отчетной формы.
    Excel Экспорт данных из отображаемой таблицы в формат MS Excel.

    Список типов баннеров

    Поле Описание
    Колонка флажков Выбор типов баннеров, к которым предполагается применить какое-либо правило или действие.
    Действия Действия над типом баннеров:
    • Изменить - редактирование типа баннеров;
    • Просмотреть - просмотр параметров типа баннеров;
    • График CTR - просмотр графика отклика для баннеров данного типа;
    • Удалить - удаление типа баннеров.
    ID Символьный идентификатор типа баннеров.
    Изменено Дата и время последнего изменения типа баннеров.
    Акт. Признак активности типа баннеров.
    Сорт. Относительный "вес", влияющий на порядок вывода типов баннеров в списке.
    Имя Произвольное имя типа баннеров.
    Описание Произвольное описание типа баннеров.
    Баннеров Количество баннеров данного типа. Ссылка с количеством баннеров позволяет перейти к списку баннеров этого типа.
