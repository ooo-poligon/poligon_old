    Контекстная панель

    Кнопка Описание
    Список типов Переход на страницу со списком типов баннеров.
    График CTR типа Просмотр графика CTR для баннеров данного типа.
    Просмотреть тип Переход к форме просмотра параметров типа баннера.
    Добавить тип Переход к форме создания нового типа баннера. Кнопка отображается при редактировании существующего типа баннера.
    Удалить тип Удаление текущего типа баннера.

    Форма редактирования

    Форма применяется для создания и редактирования типов баннеров.

    Примечание для разработчиков
    После создания нового типа следует изменить HTML-шаблон сайта, выделив в нем место для отображения баннеров нового типа путём вызова $APPLICATION->ShowBanner(...).

    Поле Описание
    Создан Дата и время создания типа баннера, а также имя пользователя, создавшего данный тип.
    Изменен Дата и время последнего изменения типа баннера, а также имя пользователя, выполнившего изменение.
    Активен Признак активности типа.
    Порядок сортировки Относительный "вес", влияющий на порядок вывода типов баннера в списке.
    * Символьный идентификатор (ID) Символьный идентификатор типа. Для создания идентификатора допустимо использование латинских букв, цифр, символа подчеркивания (_). Используется для определения места показа баннеров данного типа в шаблоне сайта (см. ShowBanner).
    Имя Произвольное имя типа баннера.
    Описание Произвольное описание типа баннера.
    * - Поля, обязательные для заполнения.

    Кнопки управления

    Кнопка Описание
    Сохранить Сохранение внесённых изменений. Переход на страницу со списком типов баннера.
    Применить Сохранение внесённых изменений. Продолжение редактирования параметров типа баннера.
    Отменить Отмена внесённых изменений. Возврат первоначальных значений параметров.
