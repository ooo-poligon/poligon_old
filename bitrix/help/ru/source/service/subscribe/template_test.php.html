Проверка шаблона рассылки

В этом разделе

  • Контекстная панель
  • Поля формы
  • Кнопки управления

    • Переход на Страницу проверки шаблона осуществляется по кнопке Проверить, расположенной в Контекстном меню на странице редактирования автоматической рассылки.

    На Странице проверки шаблона можно осуществить просмотр сообщения, сформированного на основании выбранного шаблона, в том виде, в котором оно будет отправлено подписчикам рассылки.

    Контекстная панель

    Ссылка Описание
    Список Ссылка на страницу со списком рассылок.
    Добавить Ссылка на страницу создания новой рассылки.
    Редактировать Переход к редактированию проверяемой рассылки.
    Удалить Удаление проверяемой рассылки.

    Поля формы - вкладка "Проверка"

    Поле Описание
    Название рассылки Название проверяемой рассылки.
    Название шаблона Название проверяемого шаблона для автоматической генерации выпусков.
    Описание шаблона Выводится описание шаблона.
    Интервал времени
    *С (DD.MM.YYYY HH:MI:SS) Указывается нижняя граница периода, за который будет осуществляться проверка шаблона. Т.е. дата и время последней генерации сообщений рассылки на основе выбранного шаблона.
    Для новой автоматической рассылки по умолчанию подставляется дата и время ее создания.
    *По (DD.MM.YYYY HH:MI:SS) Указывается верхняя граница периода, за который будет осуществляться проверка шаблона. Т.е. дата и время предстоящей генерации сообщений рассылки на основе выбранного шаблона.

    Пример
    Если в настройках рассылки указана дата генерации сообщений - 1 числа каждого месяца, то для проверки вида сообщений рассылки, например, в августе данные поля будут заполнены следующим образом:
    - в поле начала действия шаблона будет указана дата предыдущего выпуска: 1 июля (01.07.2005);
    - в поле окончания действия шаблона будет указана дата предстоящего выпуска: 1 августа (01.08.2005).

    * - Поля, обязательные для заполнения.

    Для проверки шаблона нужно нажать кнопку Проверить. Будет выведена таблица, содержащая информацию:

    Массив входных параметров
    [параметры шаблона в базе данных системы] Значения параметров шаблона, используемые для его идентификации в системе.
    Тело письма
      Сообщение в том виде, в котором оно будет получено подписчиками.
    Параметры выпуска
    [ параметры выпуска ] Значения параметров выпуска данной рассылкие, заданные на странице создания/редактирования выпуска.

    Кнопки управления

    Кнопка Описание
    Добавить выпуск Сообщение, сформированное на Странице проверки шаблона, может быть добавлено в список выпусков сайта. После формирования выпуска, информация из сообщения будет представлена в виде статического текста.
    Вы сможете отредактировать текст сообщения, а также отправить сформированное сообщение вручную или создать автоматический выпуск.
