Список рассылок

В этом разделе

  • Кнопки управления
  • Поля формы

    • Форма является основным средством создания списков рассылки. Важно понимать, что сами по себе списки рассылки не содержат новостных сообщений или подписчиков. Они лишь удобное средство группировки подписчиков, получающих определённые новостные сообщения. Например, вы можете создать список рассылки "Новости компании" и предлагать посетителям подписаться на него; в дальнейшем, при создании новостного сообщения, вы можете указать, что это сообщение должно отправляться по списку рассылки "Новости компании" - сообщение будет отослано тем подписчикам, которые подписаны на эту рассылку.

    Фильтр

    Используется для задания параметров отбора списков рассылок, которые отображаются в списке.

    ПолеОписание
    ID Идентификатор рассылки, под которым она зарегистрирован в базе данных.
    Сайт Сайт, на котором действуют требуемые рассылки.
    Активность Отбор рассылки, чей статус активен или неактивен, либо отображение как активных, так и неактивных рассылок.
    Публикация Отбор рассылок, отображаемых на сайте (т.е. предложенных посетителям для подписки).
    Автоматическая Отбор автоматических либо неавтоматических рассылок.

    Чтобы отфильтровать рассылки по заданным критериям нажмите кнопку Найти. Для отображения всех рассылок нажмите кнопку Отменить.

    Контекстная панель

    Кнопка Описание
    Добавить Служит для добавления новой рассылки.
    Настроить Позволяет перейти к диалогу настройки внешнего вида отчетной формы.
    Excel Экспортирует данные из отображаемой таблицы в формат MS Excel.

    Поля формы

    Таблица списков рассылок отображает списки рассылки.

    Колонка Описание
    Колонка флажков Поле выбора списков рассылки, к которым применяется какое-либо правило или действие.
    Меню
    действий
    		Отображение меню, позволяющее Изменить или Удалить правило.
    ID Идентификатор рассылки, под которым оно зарегистрировано в базе данных. 
    Название Название рассылки, отображаемое посетителям.
    Сайт Сайт рассылки.
    Сорт. Вес сортировки списка рассылки. Чем больше это значение, тем выше в видимом списке располагается эта рассылка.
    Акт. Статус активности рассылки. Если рассылка активна, пользователи могут подписаться на нее.
    Публ. Является ли данная рассылка публичной. Т.е. разрешен или запрещён показ ссылки для подписки на данную рассылку в Публичном разделе сайта.
    Авт. Признак автоматической генерации выпусков, относящихся к данной рассылке. Если значение поля - Да, то выпуски данной рассылки формируются автоматически по расписанию и на основании выбранного шаблона.
    Генерация Если рассылка является автоматической (поле Авт. имеет значение Да), то в этом поле выводится дата и время последней генерации выпусков данной рассылки.
    Если генерация выпусков не происходила, то в поле отображается дата и время, в которое рассылка стала автоматической и активной.
