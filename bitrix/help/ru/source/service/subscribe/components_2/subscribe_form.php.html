Компоненты 2.0::Форма подписки

Компонент служит для построения включаемой в дизайн сайта формы для подписки на рассылки. Право на подписку имеют как зарегистрированные, так и анонимные пользователи (в случае наличия установленной опции в настройках модуля Подписка, рассылка).

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:subscribe.form","",Array(
     "USE_PERSONALIZATION" => "Y",
     "PAGE" => "#SITE_DIR#personal/subscribe/subscr_edit.php",
     "SHOW_HIDDEN" => "N",
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "3600"
     )
);?>

Описание параметров

Основные параметры
USE_PERSONALIZATION
При отмеченной опции будут определяться имеющиеся подписки текущего пользователя.
Источник данных
SHOW_HIDDEN
При установленной опции в списке рассылок выводятся все активные рассылки вне зависимости от опции Выводить в списке публичных рассылок в настройках рассылки.
Шаблоны ссылок
PAGE
Указывается путь от корня сайта (#SITE_DIR) к странице редактирования подписки.
Настройки кеширования
CACHE_TYPE
Тип кеширования:
  • A - Авто: действует при включенном кешировании в течение заданного времени;
  • Y - Кешировать: необходимо определить время кеширования;
  • N - Не кешировать: кеширования нет в любом случае.

CACHE_TIME
Время кеширования, указанное в секундах.
