Настройки

Форма (Настройки > Настройки модулей > Подписка, рассылки) предназначена для задания общесистемных параметров модуля подписки и рассылки.

Настройка параметров модуля - вкладка "Настройки"

Параметр Описание
Разрешить анонимную подписку Если флаг установлен, то посетителям сайта позволено подписываться на рассылки без предварительной регистрации на сайте.
Показывать ссылки на авторизацию при анонимной подписке Если флаг установлен и текущий пользователь имеет статус анонимного пользователя (то есть, вход пользователем не был произведен), отображается ссылка, предлагающая посетителю войти на сайт с использованием его регистрационных данных.
Публичный раздел, где находится страница редактирования подписки Путь к папке относительно корня сайта в Публичном разделе, в которой находится страница редактирования подписки. При задании пути допустимо использование макроса #LANG_DIR# 
Использовать визуальный HTML-редактор Если опция включена, при создании выпусков новостей по умолчанию используется HTML-редактор.
Интервал в секундах для пошаговой рассылки в режиме "Персонально подписчику" (0 - рассылать за один шаг) Интервал отправки писем подписчикам. Режим Персонально подписчику предполагает отправку индивидуальных сообщений каждому подписчику с адресом подписчика в поле Кому каждого такого письма.
Максимальное количество адресов в поле BCC не в режиме "Персонально подписчику" Количество адресов, которое может быть выведено в поле BCC в случае неиспользования режима "Персонально подписчику". Если в поле указано "0", то отправляется одно письмо всем подписчикам.
Адрес отправителя по умолчанию Адрес отправителя, по умолчанию используемый во всех создаваемых сообщениях.
Адрес получателя по умолчанию Адрес получателя, по умолчанию используемый в поле Кому каждого создаваемого сообщения.
Кодировки сообщения в рассылку Перечень допустимых таблиц кодов символов для создания сообщений.
Разрешить 8-битные символы в заголовке письма Если опция включена, в заголовке письма используются символы расширенного ASCII (0-255). В противном случае, такие символы (обычно национальные, например, символы русского алфавита) выводятся в кодировке Base-64.
Дополнительный параметр для передачи функции mail Для более детальной настройки указывается дополнительный параметр функции mail.
Отправлять картинки в виде вложений в письмо Если опция включена, то изображения отправляются как вложения. В противном случае в тело письма вставляется ссылка на картинку.
Через сколько дней удалять неподтвержденные подписки (0 - не удалять)  Указание количества дней по истечении которых подписчик, не подтвердивший подписку, будет удален из базы данных.
Метод автоматической рассылки
  • Агент - автоматическая рассылка осуществляется внутренними средствами системы;
  • Крон - используется при установке системы на сервер под управлением ОС Unix; предусматривает использование демона cron ОС Unix.
Количество писем для автоматической рассылки агентом за один запуск в режиме “Персонально подписчику” Количество писем выпуска (для которого задано свойство Отправлять письмо персонально каждому получателю), отправляемое агентом за один запуск.
Применяется для снижения нагрузки на сервер.
Метод автоматической генерации выпусков
  • Агент - автоматическая генерация осуществляется внутренними средствами системы;
  • Крон - используется при установке системы на сервер под управлением ОС Unix; предусматривает использование демона cron ОС Unix.
Интервал проверки необходимости генерации выпусков в секундах Интервал времени по истечении которого будет проверяться необходимость генерации выпусков.
Интервал времени задается в секундах.

Настройка уровней доступа к модулю - вкладка "Доступ"

Форма позволяет настроить права доступа для всех имеющихся в системе типов пользователей, кроме администратора, который всегда имеет полный доступ. Настроив уровень доступа "по умолчанию", можно легко изменить политику доступа сразу для всех пользователей, которым присвоен уровень доступа "по умолчанию".

В качестве уровней доступа возможен выбор между:

  • закрыт - доступ невозможен;
  • просмотр всех данных модуля - доступ открыт только на просмотр файлов, но не на их изменение;
  • запись - доступ открыт и на чтение, и на изменение файлов..
Поле Описание
По умолчанию Изменение настроек "по умолчанию".
[Группа пользователей] Изменение настроек доступа для конкретной группы.
