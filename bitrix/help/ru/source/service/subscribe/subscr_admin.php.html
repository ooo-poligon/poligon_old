    • Форма Подписчики (Сервисы > Рассылки > Подписчики) предназначена для управления пользователями, подписанными на получение новостей. С помощью этой формы можно:

    • в форме наглядной таблицы просматривать данные подписчиков;
    • создавать новых подписчиков;
    • редактировать параметры подписчиков;
    • удалять подписчиков.

    Форма Подписчики содержит две основные области: фильтр и список подписчиков.

    Фильтр

    Задание параметров отбора подписчиков, которые отображаются в списке.

    • идентификатор подписчика;
    • диапазон дат, в пределах которых подписчик могло быть добавлен или изменен в базе данных;
    • адрес электронной почты;
    • статус анонимности;
    • имя;
    • другие критерии.

    Нижеследующая таблица описывает поля фильтра.

    Поле Описание
    ID Идентификатор подписчика, под которым он зарегистрирован в базе данных.
    Поле позволяет использовать язык запросов для задания сложных условий.
    Дата добавления Диапазон дат, в пределах которых искомые подписчики могли быть добавлены в базу данных.
    Дата изменения Диапазон дат, в пределах которых данные искомых подписчиков могли быть изменены.
    E-Mail Адрес электронной почты подписчика, используемый для отсылки сообщений.
    Анонимный Поле позволяет включать в отчет подписчиков:
    • анонимных (не зарегистрированных на сайте) подписчиков;
    • зарегистрированных на сайте подписчиков;
    • всех подписчиков, как анонимных, так и зарегистрированных.
    ID пользователя Идентификатор пользователя, под которым он зарегистрирован на сайте. (Сравните с ID.)
    Поле значимо лишь для зарегистрированных (не анонимных пользователей).
    Подписчик Персональные данные пользователя, по которым следует произвести отбор.
    Поле значимо лишь для зарегистрированных (не анонимных пользователей).
    Подтверждено Позволяет отобразить подписчиков: 
    • подтвердивших свою подписку;
    • не подтвердивших подписки;
    • всех подписчиков.
    Активно Поле позволяет включать в отчет подписчиков, чей статус подписки:
    • активен (подписчик получает сообщения);
    • не активен (подписчик не получает сообщения);
    • всех подписчиков.
    Формат Фильтрация подписчиков по желаемому формату получаемых сообщений.
    Рассылка Отображение подписчиков, которые подписаны на выбранные рассылки. Для выбора нескольких рассылок воспользуйтесь клавишей Ctrl при нажатии кнопки мыши.

    Чтобы отфильтровать сообщения по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех сообщений, нажмите кнопку Отменить.

    Контекстная панель

    Кнопка Описание
    Добавить Добавление нового подписчика.
    Настроить Переход к диалогу настройки внешнего вида отчетной формы.
    Excel Экспорт данных из отображаемой таблицы в формат MS Excel.

    Список подписчиков

    Фильтрация списка подписчиков согласно установленным критериям. Данные подписчика могут быть отредактированы и удалены.

    Колонка Описание
    Колонка флажков Выбор подписчиков, к которым предполагается применить какое-либо правило или действие.
    Меню
    действий
    		Меню, позволяющее изменить или удалить подписчика.
    ID Идентификатор подписчика, под которым он зарегистрирован в базе данных. 
    Добавлено Дата добавления данного подписчика в базу данных.
    Адрес Адрес электронной почты подписчика, используемый для отсылки сообщений.
    Подписчик Данные о подписчике: анонимный или его регистрационные данные.
    Подтв. Статус подтверждения подписки. Да - подписчик подтвердил желание получать сообщения подписанных рассылок, и нет - в противном случае.
    Акт. Статус активности подписки. 
    Если подписка активна, то сообщения данному подписчику отсылаются.
    Формат Желаемый подписчиком формат сообщений.
    Изменено Дата изменения параметров подписки.
    Время подтверждения Дата создания подписки.
    Код подтверждения Код подтверждения подписки.

    Как создать подписчика

    Для создания подписчика служит кнопка Добавить, открывающая форму добавления/редактирования адреса.

    Как изменить данные подписчика

    Для изменения регистрационных данных подписчика дважды кликните на строке соответствующего подписчика.
