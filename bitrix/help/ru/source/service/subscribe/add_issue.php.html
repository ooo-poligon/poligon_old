    Для создания и редактирования сообщений рассылки применяется форма одного типа. В зависимости от того, создаете вы новое сообщение или редактируете существующее, некоторые поля могут присутствовать или отсутствовать на форме.

    С помощью формы редактирования сообщений можно:

    • создать новое или изменить существующее сообщение рассылки;
    • выбрать подписчиков для получения выпуска рассылки;
    • выбрать пользователей сайта для получения выпуска рассылки;
    • добавить дополнительные адреса получателей выпуска рассылки;
    • настроить параметры сообщения;
    • отправлять сообщения рассылки подписчикам.

    Контекстная панель

    Ссылка Описание
    Список Переход на страницу со списком выпусков.
    Добавить Переход на страницу создания нового выпуска. Отображается только на странице редактирования уже существующего выпуска.
    Копировать Команда копирования выпуска доступна только на странице редактирования уже существующего выпуска.
    Удалить данный выпуск Удаление выпуска, просматриваемого на данной странице. Команда доступна только на странице редактирования уже существующего выпуска.

    Поля формы

    Вкладка "Выпуск"

    ПолеОписание
    Информация о состоянии выпуска
    Дата модификации Дата и время последнего редактирования сообщения.

    Данное поле отображается только в режиме редактирования существующего сообщения.
    Дата отправки Дата и время отправки сообщения (если сообщение было отправлено).

    Данное поле отображается только в режиме редактирования существующего сообщения.
    Отчет об отправке Список адресов, на которые сообщение было отправлено или не отправлено. Выводится, если такой список существует (при статусах Отправлено, Отправлено с ошибками, Остановлен).

    Данное поле отображается только в режиме редактирования существующего сообщения.
    Текущий статус
    • черновик - выпуск может быть сохранен в данном статусе или переведен в него после остановки или завершения отправки (при этом будет очищена очередь отправки выпуска);
    • отправлено - отправка выпуска прошла успешно;
    • отправлено с ошибками - во время отправки выпуска произошли ошибки;
    • остановлен - отправка выпуска была прервана.
    Перевести в статус Смена статуса сообщения. Например, чтобы отослать сообщение повторно, можно изменить его статус с Отправлено на Черновик и затем отправить.

    Данное поле отображается только в режиме редактирования существующего сообщения.
    Поля письма
    От кого Информация об отправителе сообщения в виде, в котором его увидит пользователь. Обязательно для заполнения.
    Кому Поле Кому заголовка сообщения, как его увидит пользователь, получивший сообщение. Обязательно для заполнения.
    Тема Тема сообщения. Обязательно для заполнения.
    Текст письма Собственно текст сообщения.
    Text / HTML Формат сообщения. В текстовом режиме отсутствует возможность форматирования текста и вставки рисунков. Режим HTML позволяет использовать любое форматирование сообщения.
    Использовать визуальный редактор Опция действительна только для формата сообщения HTML, включается использование  визуального редактора.

    Вкладка "Получатели"

    ПолеОписание
    Адреса получателей
    Рассылки Список существующих рассылок. Выбор рассылок для посылки сообщений.
    Формат подписки Поле позволяет выборочно отправлять сообщения по формату получения.
    (любой) - Отправить сообщение всем подписчикам.
    Текст - Отправить сообщение тем подписчикам, которые выбрали простой текст в качестве формата подписки.
    HTML - Отправить сообщение тем подписчикам, которые выбрали текст HTML в качестве формата подписки.
    Группы пользователей Группы пользователей, для которых предназначено сообщение. Выбор группы из списка имеющихся.
    Фильтр на выбранные адреса
    Фильтр по E-Mail Ограничение получателей сообщения рассылки путём задания фильтра.

    Для установки фильтра могут быть использованы специальные логические выражения

    Пример фильтра: %@bitrixsoft.ru
    Посмотреть выбранные адреса получателей Просмотр выбранных адресов получателей сообщения.
    Дополнительные адреса получателей
    Дополнительные адреса электронной почты, на которые будет отправлено данное сообщение.

    Вкладка "Вложения"

    ПолеОписание
    Имена файлов вложений и кнопка Обзор... Выбор файлов для отправки получателям выпуска в виде вложений.

    Вкладка "Параметры"

    ПолеОписание
    Кодировка письма Кодировка  сообщения.
    Отправлять письмо персонально каждому получателю (рекомендуется) Если эта опция включена, каждое сообщение будет формироваться и отправляться каждому пользователю отдельно.
    Если опция выключена, одно сообщение будет отправлено сразу всем пользователям с указанием их адресов в поле Скрытая копия (BCC).

    Рекомендуется отправлять сообщения персонально, чтобы гарантировать их доставку подписчикам.
    Отправлять автоматически в указанное время Разрешается или запрещается автоматическая отправка сообщения в соответствии с указанными в поле Дата и время отправки параметрами.
    Дата и время отправки Дата и время отправки сообщения.

    Кнопки управления

    КнопкаОписание
    Режим создания нового сообщения
    Добавить Сохраняет сообщение и переходит к отображению списка сообщений.
    Применить Сохраняет сообщение. Редактирование сообщения продолжается.
    Сбросить Сбрасывает все введенные после загрузки формы данные.
    Отправить Нажмите эту кнопку для отправки сообщения.
    Режим редактирования имеющегося сообщения
    Добавить Сохраняет сообщение и переходит к отображению списка сообщений.
    Применить Сохраняет сообщение. Редактирование сообщения продолжается.
    Сбросить Сбрасывает все введенные после загрузки формы данные.
    Отправить Отправка сообщения. Отображается в форме редактирования сообщений, находящихся в статусе Черновик.
    Продолжить отправку Команда доступна только для сообщения, находящихся в статусе Остановлен.
    Повторить для ошибочных Повторная отправка сообщения по адресам, предыдущая отправка на которые прошла с ошибками. Доступна только для сообщений, находящихся в статусе Отправлено с ошибками.
