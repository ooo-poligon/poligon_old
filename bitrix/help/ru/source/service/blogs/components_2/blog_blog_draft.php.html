Компоненты 2.0::Черновики сообщений блога

Компонент выводит неопубликованные сообщения текущего пользователя блога. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:blog.blog.draft","",Array(
     "PATH_TO_BLOG_CATEGORY" => "",
     "PATH_TO_POST_EDIT" => "blog_p_edit.php?page=post_edit&blog=#blog#&post_id=#post_id#",
     "PATH_TO_SMILE" => "/bitrix/images/blog/smile/",
     "BLOG_VAR" => "blog",
     "POST_VAR" => "post_id",
     "PAGE_VAR" => "page",
     "BLOG_URL" => $blog,
     "SET_NAV_CHAIN" => "Y",
     "SET_TITLE" => "Y"
     )
);?>

Описание параметров

Источник данных
BLOG_URL
Код, в котором передается идентификатор блога.
Шаблоны ссылок
PATH_TO_BLOG_CATEGORY
Путь к странице блога с фильтром по категории.
PATH_TO_POST_EDIT
Путь к странице редактирования сообщения блога.
Дополнительные настройки
PATH_TO_SMILE
Путь к папке со смайликами относительно корня сайта.
SET_NAV_CHAIN
Добавление в цепочку навигации пункта с названием блога.
SET_TITLE
Установка в качестве заголовка страницы Недописанные сообщения блога <название_блога>.
BLOG_VAR
Имя переменной, которой передается ID блога.
POST_VAR
Имя переменной, которой передается ID сообщения блога.
PAGE_VAR
Имя переменной, которой передается страница блога.
