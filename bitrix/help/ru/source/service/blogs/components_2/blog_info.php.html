Компоненты 2.0::Информация о блоге

Компонент выводит информацию о блоге со списком категорий. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:blog.info","",Array(
     "PATH_TO_BLOG" => "blog_blog.php?page=blog&blog=#blog#",
     "PATH_TO_BLOG_CATEGORY" => "blog_blog.php?page=blog&blog=#blog#&category=#category#",
     "PATH_TO_USER" => "blog_user.php?page=user&user_id=#user_id#",
     "BLOG_VAR" => "blog",
     "USER_VAR" => "user_id",
     "PAGE_VAR" => "page",
     "BLOG_URL" => $blog,
     "CATEGORY_ID" => $category,
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "86400",
     "BLOG_PROPERTY_LIST" => array()
     )
);?>

Описание параметров

Основные параметры
CATEGORY_ID
Код, в котором передается идентификатор категории для фильтрации.
Источник данных
BLOG_URL
Код, в котором передается идентификатор блога.
Шаблоны ссылок
PATH_TO_BLOG
Путь к главной странице конкретного блога.
PATH_TO_BLOG_CATEGORY
Путь к странице блога с фильтром по категории.
PATH_TO_USER
Путь к странице профиля пользователя.
Настройки кэширования
CACHE_TYPE
Тип кэширования:
  • A - Авто: действует при включенном кэшировании в течение заданного времени;
  • Y - Кешировать: еобходимо определить время кэширования;
  • N - Не кэшировать: кэширования нет в любом случае.

CACHE_TIME
Время кэширования, указанное в секундах.
Дополнительные настройки
BLOG_PROPERTY_LIST
Пользовательские свойства блога, которые необходимо дополнительно отобразить.
BLOG_VAR
Имя переменной, которой передается ID блога.
USER_VAR
Имя переменной, которой передается ID пользователя блога.
PAGE_VAR
Имя переменной, которой передается страница блога.
