Компоненты 2.0::Меню блогов

Компонент выводит меню блогов. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова

<?$APPLICATION->IncludeComponent("bitrix:blog.menu","",Array(
	"PATH_TO_BLOG" => "blog_blog.php?page=blog&blog=#blog#", 
	"PATH_TO_BLOG_INDEX" => "blog_s.php", 
	"PATH_TO_DRAFT" => "blog_b_draft.php?page=draft&blog=#blog#", 
	"PATH_TO_POST_EDIT" => "blog_p_edit.php?page=post_edit&blog=#blog#&post_id=#post_id#", 
	"PATH_TO_USER_FRIENDS" => "blog_friends.php?page=user_friend&user_id=#user_id#", 
	"PATH_TO_USER_SETTINGS" => "blog_user_set.php?page=user_settings&blog=#blog#", 
	"PATH_TO_GROUP_EDIT" => "blog_user_gr.php?page=group_edit&blog=#blog#", 
	"PATH_TO_BLOG_EDIT" => "blog_b_edit.php?page=blog_edit&blog=#blog#", 
	"PATH_TO_CATEGORY_EDIT" => "blog_category.php?page=category_edit&blog=#blog#", 
	"PATH_TO_USER" => "blog_user.php?page=user&user_id=#user_id#", 
	"BLOG_VAR" => "blog", 
	"POST_VAR" => "post_id", 
	"PAGE_VAR" => "page", 
	"USER_VAR" => "user_id", 
	"BLOG_URL" => $blog, 
	"SET_NAV_CHAIN" => "Y" 
	)
);?>

Описание параметров

Источник данных
BLOG_URL
Путь блога. Параметр содержит код, в котором передается идентификатор блога.
Шаблоны ссылок
PATH_TO_BLOG
Путь к главной странице конкретного блога.
PATH_TO_BLOG_INDEX
Путь к главной странице.
PATH_TO_DRAFT
Путь к странице с недописанными сообщениями блога.
PATH_TO_POST_EDIT
Путь к странице редактирования сообщения блога.
PATH_TO_USER_FRIENDS
Путь к странице сообщений друзей пользователя.
PATH_TO_USER_SETTINGS
Путь к странице редактирования списка друзей пользователя.
PATH_TO_GROUP_EDIT
Путь к странице настройки групп пользователей блога.
PATH_TO_BLOG_EDIT
Путь к странице редактирования настроек блога.
PATH_TO_CATEGORY_EDIT
Путь к странице настроек категорий сообщений блога.
PATH_TO_USER
Путь к странице профиля пользователя.
Дополнительные настройки
SET_NAV_CHAIN
Добавление пункта в цепочку навигации.
BLOG_VAR
Имя переменной, которой передается ID блога.
POST_VAR
Имя переменной, которой передается ID сообщения.
PAGE_VAR
Имя переменной, которой передается страница блога.
USER_VAR
Имя переменной, которой передается ID пользователя.
