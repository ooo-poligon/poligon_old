Компоненты 2.0::Получение Trackback'ов к сообщению

Компонент получает Trackback к сообщению. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:blog.post.trackback.get","",Array(
     "BLOG_URL" => $blog,
     "PATH_TO_POST" => "blog_post.php?page=post&blog=#blog#&post_id=#post_id#",
     "BLOG_VAR" => "blog",
     "POST_VAR" => "post_id",
     "PAGE_VAR" => "page",
     "ID" => $post_id
     )
);?>

Описание параметров

Источник данных
BLOG_URL
Код, в котором передается идентификатор блога.
ID
Код, в котором передается идентификатор сообщения блога.
Шаблоны ссылок
PATH_TO_POST
Путь к странице просмотра сообщения блога.
Дополнительные настройки
BLOG_VAR
Имя переменной, которой передается ID блога.
POST_VAR
Имя переменной, которой передается ID сообщения блога.
PAGE_VAR
Имя переменной, которой передается страница блога.
