Компоненты 2.0::Профиль пользователя

Компонент позволяет просмотреть и изменить профиль пользователя. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:blog.user","",Array(
     "PATH_TO_BLOG" => "blog_blog.php?page=blog&blog=#blog#",
     "PATH_TO_USER" => "blog_user.php?page=user&user_id=#user_id#",
     "PATH_TO_USER_EDIT" => "",
     "PATH_TO_SEARCH" => "blog_search.php",
     "BLOG_VAR" => "blog",
     "PAGE_VAR" => "page",
     "USER_VAR" => "user_id",
     "ID" => $user_id,
     "SET_TITLE" => "Y",
     "USER_PROPERTY" => array()
     )
);?>

Описание параметров

Источник данных
ID
Код, в котором передается идентификатор пользователя.
Шаблоны ссылок
PATH_TO_BLOG
Путь к главной странице блога.
PATH_TO_USER
Путь к странице профиля пользователя.
PATH_TO_USER_EDIT
Путь к странице редактирования профиля пользователя.
PATH_TO_SEARCH
Путь к странице поиска.
Дополнительные настройки
SET_TITLE
Установка в качестве заголовка страницы Профиль пользователя <имя_пользователя>.
USER_PROPERTY
Дополнительные свойства пользователя, которые необходимо отобразить.
BLOG_VAR
Имя переменной, которой передается ID блога.
PAGE_VAR
Имя переменной, которой передается страница блога.
USER_VAR
Имя переменной, которой передается ID пользователя блога.
