Компоненты 2.0::Ссылки на RSS блога

Компонент выводит ссылки на RSS. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:blog.rss.link","",Array(
     "PATH_TO_RSS" => "",
     "PATH_TO_RSS_ALL" => "blog_rss_all.php?page=rss_all&blog=#blog#&type=#type#",
     "BLOG_VAR" => "blog",
     "PAGE_VAR" => "page",
     "GROUP_VAR" => "group_id",
     "BLOG_URL" => $blog,
     "GROUP_ID" => $group_id,
     "RSS1" => "Y",
     "RSS2" => "Y",
     "ATOM" => "Y",
     "MODE" => "S"
     )
);?>

Описание параметров

Основные параметры
RSS1
Отображение ссылки на RSS .92.
RSS2
Отображение ссылки на RSS 2.0.
ATOM
отображение ссылки на ATOM .3.
MODE
Ссылки на RSS показывать для:
  • B - блога;
  • S - блогов сайта;
  • G - блогов группы.
Источник данных
BLOG_URL
Код, в котором передается идентификатор блога.
GROUP_ID
Код, в котором передается идентификатор группы блогов.
Шаблоны ссылок
PATH_TO_RSS
Путь к странице RSS блога.
PATH_TO_RSS_ALL
Путь к странице общего RSS блогов.
Дополнительные настройки
BLOG_VAR
Имя переменной, которой передается ID блога.
PAGE_VAR
Имя переменной, которой передается страница блога.
GROUP_VAR
Имя переменной, которой передается ID группы блогов.
