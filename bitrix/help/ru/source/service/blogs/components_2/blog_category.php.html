Компоненты 2.0::Теги блога

Компонент позволяет редактировать теги сообщений блога. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:blog.category","",Array(
     "BLOG_URL" => $blog,
     "SET_TITLE" => "Y"
     )
);?>

Описание параметров

Источник данных
BLOG_URL
Путь блога. Параметр содержит код, в котором передается идентификатор блога.
Дополнительные настройки
SET_TITLE
В качестве заголовка страницы будет установлено Категории блога <название_блога>.
