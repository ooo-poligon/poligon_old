Компоненты 2.0::Редактирование блога

Компонент позволяет изменить параметры и настройки блога. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:blog.blog.edit","",Array(
     "PATH_TO_BLOG" => "blog_blog.php?page=blog&blog=#blog#",
     "PATH_TO_BLOG_EDIT" => "blog_b_edit.php?page=blog_edit&blog=#blog#",
     "BLOG_VAR" => "blog",
     "PAGE_VAR" => "page",
     "BLOG_URL" => $blog,
     "SET_TITLE" => "Y",
     "BLOG_PROPERTY" => array()
     )
);?>

Описание параметров

Источник данных
BLOG_URL
Код, в котором передается идентификатор блога.
Шаблоны ссылок
PATH_TO_BLOG
Путь к главной странице блога.
PATH_TO_BLOG_EDIT
Путь к странице редактирования блога.
Дополнительные настройки
SET_TITLE
Установить в качестве заголовка страницы  Редактирование блога <название_блога>.
BLOG_PROPERTY
Отображение дополнительных пользовательских свойств блога.
BLOG_VAR
Имя переменной, которой передается ID блога.
PAGE_VAR
Имя переменной, которой передается страница блога.
