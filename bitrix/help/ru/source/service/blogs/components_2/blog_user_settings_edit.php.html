Компоненты 2.0::Настройка прав пользователя блога

Компонент позволяет настраивать привязку пользователя к группам блога. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:blog.user.settings.edit","",Array(
     "PATH_TO_USER" => "blog_user.php?page=user&user_id=#user_id#",
     "PATH_TO_USER_SETTINGS" => "blog_user_set.php?page=user_settings&blog=#blog#&user_id=#user_id#",
     "BLOG_VAR" => "blog",
     "USER_VAR" => "user_id",
     "PAGE_VAR" => "page",
     "BLOG_URL" => $blog,
     "ID" => $user_id,
     "SET_TITLE" => "Y"
     )
);?>

Описание параметров

Источник данных
BLOG_URL
Код, в котором передается идентификатор блога.
ID
Код, в котором передается идентификатор пользователя.
Шаблоны ссылок
PATH_TO_USER
Путь к странице профиля пользователя.
PATH_TO_USER_SETTINGS
Путь к странице настроек пользователей.
Дополнительные настройки
SET_TITLE
Установка в качестве заголовка страницы Изменение прав пользователя блога <название_блога>.
BLOG_VAR
Имя переменной, которой передается ID блога.
USER_VAR
Имя переменной, которой передается ID пользователя блога.
PAGE_VAR
Имя переменной, которой передается страница блога.
