Компоненты 2.0::Блоги группы

Компонент выводит все блоги указанной группы. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:blog.group.blog","",Array(
     "BLOG_COUNT" => "10",
     "PATH_TO_BLOG" => "blog_blog.php?page=blog&blog=#blog#",
     "PATH_TO_POST" => "blog_post.php?page=post&blog=#blog#&post_id=#post_id#",
     "PATH_TO_USER" => "blog_user.php?page=user&user_id=#user_id#",
     "BLOG_VAR" => "blog",
     "POST_VAR" => "post_id",
     "USER_VAR" => "user_id",
     "PAGE_VAR" => "page",
     "ID" => $group_id,
     "SHOW_BLOG_WITHOUT_POSTS" => "Y",
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "86400",
     "SET_TITLE" => "Y"
     )
);?>

Описание параметров

Источник данных
ID
Код, в котором передается идентификатор группы блогов.
Внешний вид
BLOG_COUNT
Количество блогов на странице. Остальные будут выведены с помощью постраничной навигации.
SHOW_BLOG_WITHOUT_POSTS
Отображение блогов, которые не имеют сообщений.
Шаблоны ссылок
PATH_TO_BLOG
Путь к главной странице блога.
PATH_TO_POST
Путь к странице детального просмотра сообщения блога.
PATH_TO_USER
Путь к странице профиля пользователя.
Настройки кеширования
CACHE_TYPE
Тип кеширования:
  • A - Авто: действует при включенном кешировании в течение заданного времени;
  • Y - Кешировать: необходимо определить время кеширования;
  • N - Не кешировать: кеширования нет в любом случае.

CACHE_TIME
Время кеширования, указанное в секундах.
Дополнительные настройки
SET_TITLE
Установка в качестве заголовка страницы названия группы блогов.
BLOG_VAR
Имя переменной, которой передается ID блога.
POST_VAR
Имя переменной, которой передается ID сообщения блога.
USER_VAR
Имя переменной, которой передается ID пользователя блога.
PAGE_VAR
Имя переменной, которой передается страница блога.
