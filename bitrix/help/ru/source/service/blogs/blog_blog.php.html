    На странице Блоги (Сервисы > Блоги > Блоги) представлена таблица со списком блогов. Она служит для добавления новых блогов, а также для перехода к управлению их параметрами.

    Фильтр

    С помощью формы выполнется поиск блогов в соответствии с указанными условиями.

    Параметр Описание
    Название Название блога.
    Активность Поиск можно задать по активным, неактивным или всем блогам.
    Путь Путь к блогу для данного сайта.
    Группа Указывается группа, по которой будет проходить поиск, либо поиск будет выполнен по всем группам.
    Владелец Имя владельца блога.

    Для того чтобы отобрать блоги по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех блогов нажмите кнопку Отменить.

    Контекстная панель

    Кнопка Описание
    Новый блог Переход к форме создания нового блога.
    Настроить Переход к диалогу настройки внешнего вида отчетной формы.
    Excel Экспортирует данные из таблицы в формат MS Excel.

    Список блогов

    Поле Описание
    Колонка флажков Поле для выбора блогов, к которым предполагается применить какое-либо правило или действие.
    Действия Действия над блогом:
    • Изменить параметры блога - переход к форме редактирования блога;
    • Удалить блог - удаление блога.
    ID Идентификатор блога.
    Название Указывается имя пользователя блога.
    Дата создания Дата создания блога.
    Дата последнего обновления Дата последнего обновления блога.
    Активность Признак активности блога. Если блог неактивен, то он не может быть использован.
    Владелец Пользователь, который является владельцем блога.
    Путь Название блога латницей, которое используется при указании пути к блогу.
    Группа Название группы, к которой привязан блог.
