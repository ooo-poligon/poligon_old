Группы блогов

В этом разделе

  • Фильтр
  • Контекстная панель
  • Список групп блогов

    • На странице Группы блогов (Сервисы > Блоги > Группы блогов) представлена таблица, содержащая список групп блогов. Она служит для добавления новых групп, а также для перехода к редактированию существующих.

    Фильтр

    Параметр Описание
    Сайт Поиск блогов по заданному сайту или по всем сайтам.

    Для того чтобы отобрать группу по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех групп блогов нажмите кнопку Отменить.

    Контекстная панель

    Кнопка Описание
    Новая группа Переход к форме создания новой группы блогов.
    Excel Экспортирует данные из таблицы в формат MS Excel.
    Настроить Позволяет перейти к диалогу настройки внешнего вида отчетной формы.

    Список групп блогов

    Поле Описание
    Колонка флажков Выбор группы блогов, к которой предполагается применить какое-либо правило или действие.
    Действия Действия над группой:
    • Изменить параметры группы - переход к форме редактирования группы блогов;
    • Удалить группу - удаление группы блогов.
    ID Идентификатор группы.
    Название Название группы .
    Сайт Привязка группы блогов к указанному сайту.
