Компоненты 2.0::Просмотр результата

Компонент служит для просмотра результата заполнения веб-формы. Обычно используется в сочетании с компонентом Список результатов. При самостоятельном использовании данного компонента необходимо явно указывать ID результата, чтобы соответствующий результат веб-формы был выведен в публичном разделе. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:form.result.view","",Array(
     "SEF_MODE" => "Y",
     "RESULT_ID" => $_REQUEST["RESULT_ID"],
     "SHOW_ADDITIONAL" => "N",
     "SHOW_ANSWER_VALUE" => "N",
     "SHOW_STATUS" => "Y",
     "EDIT_URL" => "result_edit.php",
     "CHAIN_ITEM_TEXT" => "",
     "CHAIN_ITEM_LINK" => "",
     "SEF_FOLDER" => "/communication/web-forms/",
     "SEF_URL_TEMPLATES" => Array(
         "view" => "#RESULT_ID#/"
         ),
     "VARIABLE_ALIASES" => Array(
         "view" => Array(),
         )
     )
);?>

Описание параметров

Источник данных
RESULT_ID
Идентификатор результата. По умолчанию указано $_REQUEST["RESULT_ID"].
Управление адресами страниц
SEF_MODE
Включение поддержки ЧПУ и полей настройки адресов ЧПУ.
SEF_FOLDER
Каталог ЧПУ: путь до папки, с которой работает компонент. Этот путь может как совпадать с физическим путём, так и не совпадать.
SEF_URL_TEMPLATES
Указывается адрес следующей страницы:
  • view - просмотр результата.
VARIABLE_ALIASES
Имена переменных, в которых передаются названия действий.
Параметры компонента
SHOW_ADDITIONAL
Вывод дополнительных поля формы (при их наличии).
SHOW_ANSWER_VALUE
Вывод для показа значения параметра ANSWER_VALUE вопроса веб-формы (выводится рядом со значением ответа).
SHOW_STATUS
Отображение на странице со списком результатов будет статуса текущего результата.
EDIT_URL
Указывается адрес страницы редактирования результата.
CHAIN_ITEM_TEXT
Название дополнительного пункта в навигационной цепочке (если оставить незаполненным, то в навигационную цепочку пункт не добавляется).
CHAIN_ITEM_LINK
Можно задать ссылку, которая будет показана на дополнительном пункте меню в навигационной цепочке.
