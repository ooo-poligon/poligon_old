Компоненты 1.0::Редактирование результата

Компонент предназначен для редактирования результата заполнения произвольной веб-формы, созданной в административном интерфейсе модуля Веб-формы. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова

<?
$APPLICATION->IncludeFile("form/result_edit/default.php", array(
    "RESULT_ID" => $_REQUEST["RESULT_ID"],
    "EDIT_ADDITIONAL" => "N",
    "EDIT_STATUS" => "Y",
    "LIST_URL" => "result_list.php",
    "VIEW_URL" => "result_view.php",
    "CHAIN_ITEM_TEXT" => "Список анкет",
    "CHAIN_ITEM_LINK" => "result_list.php?WEB_FORM_ID=".$_REQUEST["WEB_FORM_ID"],
    ));
?>

Описание параметров

RESULT_ID
ID результата.
EDIT_ADDITIONAL
Вывод на редактирование поля результата веб-формы (не путать с вопросами).
EDIT_STATUS
Вывод выпадающего списка для смены статуса результата веб-формы.
LIST_URL
Адрес страницы для перенаправления пользователя после нажатия на кнопку Сохранить (если данный параметр оставить незаполненным, то пользователь останется на той же странице).
VIEW_URL
Адрес страницы для просмотра результата.
CHAIN_ITEM_TEXT
Название дополнительного пункта в навигационной цепочке (если оставить незаполненным, то в навигационную цепочку пункт не добавляется).
CHAIN_ITEM_LINK
Ссылка на дополнительном пункте меню в навигационной цепочке.
