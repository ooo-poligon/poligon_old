Компоненты 1.0::Вывод списка результатов

Компонент предназначен для вывода таблицы-списка результатов заполнения произвольной веб-формы, созданной в административном интерфейсе модуля Веб-формы. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова

<?
$APPLICATION->IncludeFile("form/result_list/default.php", array(
    "WEB_FORM_ID" => $_REQUEST["WEB_FORM_ID"], 
    "VIEW_URL" => "result_view.php",
    "EDIT_URL" => "result_edit.php",
    "NEW_URL" => "result_new.php",
    "SHOW_ADDITIONAL" => "N",
    "SHOW_ANSWER_VALUE" => "N",
    "SHOW_STATUS" => "Y",
    "NOT_SHOW_FILTER" => "",
    "NOT_SHOW_TABLE" => ""
    ));
?>

Описание параметров

WEB_FORM_ID
ID веб-формы, результаты которой необходимо вывести.
VIEW_URL
Адрес страницы для просмотра результата (если параметр не задан, ссылка на просмотр результата не выводится).
EDIT_URL
Адрес страницы для редактирования результата (если параметр не задан, ссылка на редактирование результата не выводится).
NEW_URL
Адрес страницы для заполнения формы и создания нового результата.
SHOW_ADDITIONAL
Вывод для показа поля результата веб-формы (не путать с вопросами).
SHOW_ANSWER_VALUE
Вывод для показа значение параметра ANSWER_VALUE вопроса веб-формы (выводится в круглых скобках рядом со значением ответа).
SHOW_STATUS
Вывод текущего статуса результата веб-формы.
NOT_SHOW_FILTER
Символьные коды полей и вопросов веб-формы, которые не нужно выводить в фильтре (через запятую).
NOT_SHOW_TABLE
Символьные коды полей и вопросов веб-формы, которые не нужно выводить в таблице (через запятую).
