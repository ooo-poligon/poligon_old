Вывод формы для заполнения

Универсальный компонент предназначен для вывода произвольной веб-формы, созданной в административном интерфейсе модуля Веб-формы. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова

<?
$APPLICATION->IncludeFile("form/result_new/default.php", array(
    "WEB_FORM_ID" => $_REQUEST["WEB_FORM_ID"],
    "LIST_URL" => "result_list.php",
    "EDIT_URL" => "result_edit.php",
    "CHAIN_ITEM_TEXT" => "Список анкет",
    "CHAIN_ITEM_LINK" => "result_list.php?WEB_FORM_ID=".$_REQUEST["WEB_FORM_ID"]
));
?>

Описание параметров

WEB_FORM_ID
ID веб-формы.
LIST_URL
Адрес страницы для перенаправления пользователя после нажатия на кнопку Сохранить (если данный параметр оставить незаполненным, то пользователь останется на той же странице).
EDIT_URL
Адрес страницы для перенаправления пользователя после нажатия на кнопку Применить (если данный параметр оставить незаполненным, то пользователь останется на той же странице).
CHAIN_ITEM_TEXT
Название дополнительного пункта в навигационной цепочке (если оставить незаполненным, то в навигационную цепочку пункт не добавляется).
CHAIN_ITEM_LINK
Ссылка на дополнительном пункте меню в навигационной цепочке.
