Компоненты 1.0::Просмотр результата

Компонент предназначен для просмотра результата заполнения произвольной веб-формы, созданной в административном интерфейсе модуля Веб-формы. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова

<?
$APPLICATION->IncludeFile("form/result_view/default.php", array(
    "RESULT_ID" => $_REQUEST["RESULT_ID"],
    "SHOW_ADDITIONAL" => "N",
    "SHOW_ANSWER_VALUE" => "N",
    "SHOW_STATUS" => "Y",
    "EDIT_URL" => "result_edit.php",
    "CHAIN_ITEM_TEXT" => "Список анкет",
    "CHAIN_ITEM_LINK" => "result_list.php?WEB_FORM_ID=".$_REQUEST["WEB_FORM_ID"],
    ));
?>

Описание параметров

RESULT_ID
ID результата.
SHOW_ADDITIONAL
Вывод для показа поля результата веб-формы (не путать с вопросами).
SHOW_ANSWER_VALUE
Вывод для показа значение параметра ANSWER_VALUE вопроса веб-формы (выводится в круглых скобках рядом со значением ответа).
SHOW_STATUS
Вывод текущего статуса результата веб-формы.
EDIT_URL
Адрес страницы для редактирования результата.
CHAIN_ITEM_TEXT
Название дополнительного пункта в навигационной цепочке (если оставить незаполненным, то в навигационную цепочку пункт не добавляется).
CHAIN_ITEM_LINK
Ссылка на дополнительном пункте меню в навигационной цепочке.
