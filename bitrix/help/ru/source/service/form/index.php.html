Описание модуля

Модуль веб-форм позволяет создавать произвольное количество веб-форм, а также осуществлять гибкое управление параметрами созданных форм. Основное назначение веб-форм - это организация обратной связи с посетителями сайта. Например, веб-формы могут использоваться для:

  • создания и хранения комментариев посетителей сайта (например, для создания отзыва о статье, товаре каталога и т.д.);
  • для получения запросов посетителей сайта (например, для заполнения заявок на участие в семинаре);
  • заполнения и хранения анкет посетителей сайта;
  • обработки другой информации, подразумевающей использование форм для ввода данных.

Интеграция с Модулем статистики системы позволяет анализировать динамику заполнения форм, а также получать другую информацию о посетителях, заполнивших формы.

Интеграция с Модулем продаж позволяет организовать продажи различных услуг и товаров посредством заполнения анкет веб-форм и дальнейшей пересылки данных анкет в Модуль продаж.
