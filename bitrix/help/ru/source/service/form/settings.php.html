Настройки


Форма (Настройки > Настройки продукта > Настройки модулей) предназначена для задания общесистемных параметров Модуля веб-форм.

Закладка "Настройки"

Поле Описание
Использовать HTML редактор (только для IE 5.0 или FireFox 1.0 и выше) Для редактирования свойств и вопросов формы будет загружаться встроенный HTML редактор.
Использовать упрощённый режим редактирования форм Создание веб-форм в упрощённом режиме (без использования статусов результатов веб-формы, полей и т.д.).
Максимальное кол-во записей выбираемых в SQL запросах Ограничение на количество записей, выводимых в общем списке результатов веб-форм.
Количество результатов на одной странице Количество результатов веб-форм для показа на одной странице.
Право по умолчанию для вновь создаваемых веб-форм Права по умолчанию для групп пользователей на доступ ко всем вновь создаваемым веб-формам:
  • доступ закрыт - форма и ее результаты полностью недоступны;
  • заполнение формы - посетитель может только заполнить и сохранить форму;
  • работа со своим результатом в соответствии с его статусом - посетитель может работать только со своими результатами в соответствии с правами, заданными для каждого из статусов, в котором находится тот или иной результат;
  • работа со всеми результатами в соответствии с их статусами - посетитель может работать со всеми результатами в соответствии с правами, заданными для каждого из статусов, в котором находится тот или иной результат;
  • просмотр параметров формы - включает все вышеописанные права, а также возможность просмотра параметров веб-формы;
  • полный доступ - включает все вышеописанные права, а также права на изменение настроек формы.

Закладка "Доступ"

Форма служит для определения прав групп пользователей на доступ к Модулю веб-форм.

Поле Описание
По умолчанию Управление уровнем доступа групп пользователей, для которых установлено право по умолчанию.
[Группа пользователей]

Управление правом доступа отдельных групп пользователей. Возможны следующие уровни доступа:

  • По умолчанию - используется уровень доступа установленный в поле По умолчанию.
  • Доступ закрыт - доступ к меню и всем административным файлам модуля закрыт;
  • Доступ открыт - доступ к меню и административным файлам открыт; дальнейшие права доступа к формам регламентируются правами, заданными в каждой конкретной форме;
  • Полный доступ - открытие доступа к меню и административным файлам модуля, а также право на создание новых форм. В то же время, доступ к существующим формам регламентируется правами, заданными в каждой конкретной форме.
