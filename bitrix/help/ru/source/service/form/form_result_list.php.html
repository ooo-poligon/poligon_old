    Cтраница отображает результаты заполнения форм посетителями вашего сайта. Доступ к списку статусов осуществляется через кнопку Результаты Основного контекстного меню конкретной веб-формы (Сервисы > Веб-формы > Настройка форм > Конкретная форма).

    Фильтр

    Поиск результатов заполнения веб-формы, соответствующих указанным параметрам.

    Поле Описание
    ID* ID результата.
    Дата создания Дата создания результата.
    Дата изменения Дата изменения результата.
    Посетитель был зарегистрирован? Регистрировался ли посетитель, заполнявший форму на сайте.
    Пользователь был авторизован? Был ли посетитель авторизован по своим регистрационным данным в момент заполнения формы.
    ID пользователя* ID, логин, имя, фамилия посетителя заполнявшего форму.
    ID посетителя* ID посетителя в модуле статистики.
    ID сессии* ID сессии в модуле статистики.
    Статус Статус результата. Данное условие поиска доступно только в полном режиме редактирования веб-форм.
    ID статуса ID статуса результата. Данное условие поиска доступно только в полном режиме редактирования веб-форм.

    * - для данных полей вы можете воспользоваться специальными логическими выражениями.

    Чтобы отобрать результаты веб-формы по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех результатов нажмите кнопку Отменить.

    Контекстная панель

    На странице отображены две контекстные панели: основная, аналогичная Основной контекстной панели веб-формы и дополнительная, для Списка результатов. Дополнительная контекстная панель:

    Кнопка Описание
    Добавить Переход к форме создания нового результата.
    Настроить Позволяет перейти к диалогу настройки внешнего вида отчетной формы.
    Excel Экспортирует данные из отображаемой таблицы в формат MS Excel.

    Список результатов

    Поле Описание
    Колонка флажков Поле предназначено для выбора результатов, к которым предполагается применить какое-либо правило или действие.
    Действия Действия над результатом:
    • Просмотр - просмотр результата;
    • Изменить - редактирование результата;
    • Удалить - удаление результата.
    ID Идентификатор результата.
    Статус Статус, в котором находится результат. Данное поле отображается только в полном режиме редактирования форм.
    Изменено Дата последнего изменения результата.
    Кем изменено Логин, имя, фамилия пользователя, создавшего либо изменившего результат (если посетитель не зарегистрирован, то выводится надпись не зарегистрирован).
    Посетитель ID посетителя в модуле статистики.
    Сессия ID сессии в модуле статистики.
    [Вопросы формы] Ответы на вопросы формы.

    Редактирование результата

    Отредактировать результат можно с помощью одного из шаблонов, в том числе и с помощью шаблона default.php, поставляемого вместе с дистрибутивом модуля. После редактирования будет обновлена дата изменения результата. Также вы можете сохранить выбранный шаблон в качестве шаблона по умолчанию для редактирования результатов данной формы.

    Настройки, определяющие права групп пользователей на редактирование результатов устанавливаются в настройках каждой из веб-форм, а в случае работы в полном режиме также в настройках статусов.

    Просмотр результата

    Просмотреть данные результата можно с помощью одного из шаблонов, в том числе и с помощью шаблона default.php, поставляемого вместе с дистрибутивом модуля.

    Настройки, определяющие права групп пользователей на просмотр результатов устанавливаются в настройках каждой из веб-форм, а в случае работы в полном режиме также в настройках статусов.
