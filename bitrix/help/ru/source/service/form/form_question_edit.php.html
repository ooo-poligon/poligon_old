Создание и редактирование вопроса

  • Контекстная панель

    Для создания и изменения вопроса формы применяется страница одного типа. Большинство параметров вопросов и ответов можно настроить, воспользовавшись визуальным редактором шаблона оформления формы.

    Контекстная панель

    На странице отображены две контекстные панели: основная, аналогичная Основной контекстной панели веб-формы и дополнительная, для формы редактирования вопросов. Дополнительная контекстная панель:

    Кнопка Описание
    Создать Переход на страницу создания нового вопроса формы. Кнопка отображается при редактировании существующего вопроса.
    Копировать Копирование текущего вопроса в новый. Кнопка отображается при редактировании существующего вопроса.
    Удалить Удаление текущего вопроса. Кнопка отображается при редактировании существующего вопроса.

    Упрощённая форма редактирования

    Закладка "Свойства"
    Поле Описание
    Дата изменения Дата последнего изменения вопроса. Поле отображается при редактировании существующего вопроса.
    Активен Флаг активности вопроса.
    Порядок сортировки Относительный "вес", определяющий положение вопроса в общем списке.
    Изображение Изображение, которое будет выведено рядом с вопросом.
    Вопрос Текст вопроса.
    Поле ответа обязательно для заполнения Флаг обязательности ответа на данный вопрос формы.
    Тип поля для ответа Тип поля для ответа на данный вопрос:
    • text - однострочный текст;
    • textarea- многострочный текст;
    • multiselect - список вариантов с возможностью множественного выбора;
    • dropdown - список вариантов с возможностью одиночного выбора;
    • radio - набор radio кнопок (одиночный выбор);
    • checkbox - набор checkbox кнопок (множественный выбор);
    • image - поле для ввода изображения;
    • date - поля для ввода даты со встроенным календарем;
    • file - поле для добавления файла;
    • password - поле для ввода пароля;
    • email - поле для ввода электронного адреса;
    • url - поле для ввода адреса сайта;
    • hidden - скрытое поле.
    Ответ Поле меняется в зависимости от выбранного типа ответа.
    Для поля типа textarea указывается количество столбцов и строк.

    Для полей типа radio, checkbox, dropdown, multiselect поле Ответ выглядит следующим образом:
    Поле Описание
    Варианты ответа Вариант ответа на вопрос.
    Порядок Относительный "вес", определяющий порядок следования вариантов ответа в списке.
    По умолчанию Вариант ответа, отмеченный в этом поле, по умолчанию будет использоваться в качестве ответа на вопрос.

    Для поля типа hidden из выпадающего списка выбирается тип поля ответа, а для остальных полей задается размер поля ответа.
    Закладка "Валидаторы"

    Добавление валидаторов для используемых полей ответов.

    Поле Описание
    Доступные валидаторы для поля типа <тип_поля> Из выпадающего списка для поля указанного типа выбирается один из доступных валидаторов. Для его назначения служит кнопка Добавить валидатор, расположенная рядом с полем.
    Закладка "Комментарий"

     Созданный служебный комментарий будет отображаться только в Административном разделе.

    Полная форма редактирования

    Закладка "Свойства"
    Поле Описание
    Дата изменения Дата последнего изменения вопроса. Поле отображается при редактировании существующего вопроса.
    Активен Флаг активности вопроса.
    Порядок сортировки Относительный "вес", определяющий положение вопроса в общем списке.
    * Символьный идентификатор Символьный код вопроса. Задается символами латинского алфавита.
    Обязателен Признак обязательности ответа на данный вопрос.

    * - Поля, обязательные для заполнения.

    Закладка "Вопрос"
    Поле Описание
    Изображение Изображение, которое будет показываться рядом с вопросом.
    Текст вопроса Текст создаваемого вопроса.
    Закладка "Ответ"

    Форма позволяет создать поля, используемые для ввода ответа на вопрос, либо создать возможные варианты ответов на вопрос.

    Поле Описание
    ID Идентификатор поля для ответа.
    Текст
    ANSWER_TEXT    		 Текст ответа на вопрос.

    Используется для полей типа dropdown, multiselect, radio, checkbox. Для остальных типов полей необходимо поставить пробел.
    Значение
    ANSWER_VALUE    		 Дополнительный параметр, который может быть использован для идентификации конкретного ответа (например, в списках).
    Тип поля Тип поля для ответа на данный вопрос:
    • text - однострочный текст;
    • textarea- многострочный текст;
    • multiselect - список вариантов с возможностью множественного выбора;
    • dropdown - список вариантов с возможностью одиночного выбора;
    • radio - набор radio кнопок (одиночный выбор);
    • checkbox - набор checkbox кнопок (множественный выбор);
    • image - поле для ввода изображения;
    • date - поля для ввода даты со встроенным календарем;
    • file - поле для добавления файла;
    • password - поле для ввода пароля;
    • email - поле для ввода электронного адреса;
    • url - поле для ввода адреса сайта
    • hidden - скрытое поле.
    Ширина Ширина полей типа text, textarea, image, date. Задается количеством столбцов.
    Высота Высота полей типа textarea, multiselect. Задается количеством строк.
    Параметры Данное поле может использоваться для вывода стиля элемента ввода, а также для задания специальных идентификаторов:
    • SELECTED - при задании этого идентификатора поле ввода будет выделено; работает для полей следующих типов: multiselect, dropdown, radio, checkbox.
    • NOT_ANSWER - идентификатор используется для пометки тех пунктов элементов управления, которые не являются вариантами ответа на вопрос. В том случае, если этот идентификатора задан и если ответ на вопрос обязателен, выбор посетителем данного ответа не будет считаться ответом на обязательный вопрос формы. Работает для типов полей: dropdown, radio, multiselect, checkbox.
    • NOW_DATE -выводит в поле типа date текущую дату.
    • NOW_TIME -выводит в поле типа date текущее время.
    Сорт. Относительный "вес" определяющий положение варианта ответа в общем списке.
    Акт. Признак активности поля. Для отображения в публичном разделе выбираются только активные варианты ответа.
    Удалить Для того чтобы удалить вариант ответа, установите флаг в это поле и нажмите кнопку Применить.
    Закладка "Валидаторы"

    Добавление валидаторов для используемых полей ответов.

    Поле Описание
    Доступные валидаторы для поля типа <тип_поля> Выбирается один из доступных валидаторов. Для его назначения служит кнопка Добавить валидатор, расположенная рядом с полем.
    Закладка "Результаты"

    Закладка служит для настройки параметров показа результатов ответа вопрос.

    Поле Описание
    Показать в HTML-таблице результатов Если отметить данный флаг, то ответы на данный вопрос будут отображены в HTML-таблице результатов формы.
    Показать в Excel-таблице результатов Если отметить данный флаг, то ответы на данный вопрос будут отображены в Excel-таблице результатов формы.
    Заголовок столбца в таблице Текст, который будет выводиться в таблице результатов в качестве заголовка столбца, содержащего ответы на данный вопрос.
    Закладка "Фильтр"
    Поля в фильтре
    для вводимого значения Тип полей фильтра таблицы результатов для поиска ответов, вводимых пользователем в следующих типах полей: text, textarea.
    Возможны следующие типы полей в фильтре:
    • однострочное текстовое поле;
    • числовой интервал;
    • интервал дат;
    • флаг существования.
    для параметра ANSWER_TEXT Тип полей фильтра для поиска значений параметра ANSWER_TEXT в ответах посетителя.
    Возможны следующие типы полей в фильтре:
    • однострочное текстовое поле;
    • числовой интервал;
    • выпадающий список из всех значений ANSWER_TEXT;
    • флаг существования.
    для параметра ANSWER_VALUE Тип полей фильтра для поиска значений параметра ANSWER_VALUE в ответах посетителя.
    Возможны следующие типы полей в фильтре:
    • однострочное текстовое поле;
    • числовой интервал;
    • выпадающий список из всех значений ANSWER_TEXT;
    • флаг существования.
    Подпись к полю фильтра Текст, который будет выводиться рядом с выбранными полями фильтра по таблице результатов.
    Закладка "Комментарий"

    Созданный служебный комментарий будет отображаться только в Административном разделе.

    Кнопки управления

    Кнопка Описание
    Сохранить Сохранение параметров вопроса. Переход к списку вопросов.
    Применить Сохранение параметров вопроса. Продолжение редактирования вопроса.
    Отменить Отмена внесённых изменений. Возврат первоначальных значений параметров.
