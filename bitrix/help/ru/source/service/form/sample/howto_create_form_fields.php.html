Создание полей веб-формы

Поле формы подразумевает хранение каких-либо данных, определяемых после сохранения заполненной формы (после сохранения результата).

Для создания поля выполните следующие действия:
  1. Перейдите в полный режим редактирования веб-форм. Для этого откройте страницу настроек модуля веб-форм (Настройки > Настройки продукта > Настройки модулей) и снимите флаг в поле Использовать упрощённый режим редактирования форм. Затем сохраните выполненные настройки с помощью кнопки Сохранить.
  2. Откройте административный раздел: Сервисы > Веб-формы > Настройка форм.
  3. Для формы, поля к которой вы хотите создать, в столбце Поля нажмите ссылку [+].
  4. На открывшейся форме установите флажок в поле Активен для того, чтобы значение данного поля отображалось в результатах веб-форм;
  5. Укажите порядок сортировки.
  6. В поле Символьный идентификатор укажите символьный код поля.
  7. Выберите тип данного поля:
    • text (текст)
    • integer (число)
    • date (дата)
  8. Введите заголовок поля.
  9. Откройте закладку Результаты.
  10. Отметьте поля Показать в HTML-таблице результатов и Показать в Excel-таблице результатов, если данные, хранимые в этом поле, нужно отображать в соответствующих таблицах результатов и если нужно выводить в фильтре поля для фильтрации результатов по значениям, хранящимся в данном поле. Поля фильтра будут выбраны в соответствии с типом поля:
    • типу text соответствует однострочное текстовое поле фильтра;
    • типу integer соответствует интервал чисел;
    • типу date интервал дат.

    В однострочном текстовом поле фильтра у вас есть возможность использовать логические выражения в синтаксис которых входят:

    • & (логическое И)
    • | (логическое ИЛИ)
    • ~ (логическое отрицание)
    • () круглые скобки

    Приоритеты логических операций в полях фильтра идентичны приоритетам логических операций в математике.

  11. В поле Заголовок столбца в таблице вы можете ввести текст, который будет выводиться в таблице результатов в качестве заголовка столбца.
  12. Откройте закладку Фильтр.
  13. Укажите типы полей для проведения выборки в таблице результатов по значениям данного поля.
  14. В поле Подпись к полю фильтра вы можете ввести текст, который будет выводиться рядом с выбранными полями фильтра по таблице результатов.
  15. На закладке Комментарий вы можете ввести служебный комментарий к полю.
  16. Сохраните поле с помощью кнопки Сохранить или Применить.
