Создание вопросов веб-формы

Под вопросом формы понимается совокупность самого текста вопроса и полей ввода, предназначенных для ответа на него. Для того чтобы создать вопросы формы, выполните следующее:

  1. Откройте административный раздел: Сервисы > Веб-формы > Настройка форм.
  2. Для формы, вопросы к которой вы хотите создать, в столбце Вопросы нажмите ссылку [+].


  3. На открывшейся форме отметьте, активен ли вопрос.
  4. Укажите порядок сортировки.
  5. Установите флажок в поле Обязателен, если ответ на данный вопрос обязателен.
  6. Если необходимо, загрузите изображение для вопроса с локального компьютера.
  7. Введите текст вопроса в виде обычного текста или HTML-кода. При редактировании можно использовать HTML-редактор.


  8. Выберите тип поля для ответа на данный вопрос. В зависимости от выбранного типа поля укажите необходимые параметры в поле Ответ (в данном случае указана длина поля для ответа на вопрос).
  9. С помощью закладки Валидаторы можете назначить имеющиеся валидаторы для выбранного типа поля.
  10. Откройте закладку Комментарий и введите служебный комментарий к вопросу.
  11. Для сохранения вопроса нажмите кнопку Сохранить.
