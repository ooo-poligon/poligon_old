    Как добавить правило
  • Как изменить параметры правила

    • Форма Правила обработки почты (Сервисы > Почта > Правила) отображает созданные вами правила, которые могут быть применены к сообщениям. Форма содержит две основные области: фильтр и список правил.

    Фильтр

    Используется для задания параметров отбора правил, отображаемых в списке.

    Поле Описание
    Название Отбор правил с определённым названием.
    ID Идентификатор искомого правила.
    Почтовый ящик Название почтового ящика, правила которого следует отобразить.
    Активность Отображает все правила, или только активные или не активные.

    Чтобы отфильтровать правила по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех сообщений нажмите кнопку Отменить.

    Контекстная панель

    Кнопка Описание
    Добавить Добавление нового правила обработки почты. В открывшемся меню выберите, какой тип правила следует добавить. 
    Настроить Переход к диалогу настройки внешнего вида отчетной формы.
    Excel Экспорт данных из отображаемой таблицы в формат MS Excel.

    Список правил

    Колонка Описание
    Колонка флажков Выбор правил, к которым предполагается применить какое-либо действие.
    Меню
    действий
    • Изменить - отредактировать правило;
    • Журнал - просмотреть журнал работы почтовой системы по этому правилу;
    • Удалить - удалить правило.
    ID Идентификатор правила.
    Дата изм. Дата и время последней модификации правила.
    Название Название правила и список действий, осуществляемых правилом.
    Акт. Признак активности правила.
    Сорт. Вес сортировки правила; применяется при выводе правила в списках.
    Ящик. Название почтового ящика, к которому применяется это правило.

    Как добавить правило

    Для создания нового правила нажмите кнопку Добавить на контекстной панели и выберите предустановленный шаблон для правила: 

    • ручная настройка - все параметры правила изменяемы и доступны для изменения и настройки;
    • добавление сообщения в техподдержку - специализированный шаблон облегчающий создание правил для фильтрации сообщений, адресованных в службу технической поддержки.

    В открывшейся форме Новое правило установите параметры правила и сохраните его.

    Как изменить параметры правила

    Для изменения параметров правила дважды кликните строку требуемого правила в списке правил.
