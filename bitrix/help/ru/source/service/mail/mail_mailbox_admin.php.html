Почтовые ящики

  • Фильтр

  • Как изменить параметры почтового ящика

    • Форма Почтовые ящики (Сервисы > Почта > Почтовые ящики) отображает созданные вами учетные записи электронной почты, используемые для доставки корреспонденции. Форма содержит две основные области: фильтр и список ящиков.

    Фильтр

    Используется для задания параметров отбора учетных записей электронной почты, отображаемых в списке.

    Поле Описание
    Название Отбор учетных записей по названию.
    ID Идентификатор искомого почтового ящика.
    Активность Отображает все ящики, или только активные или не активные.
    Сайт Сайт, в котором искомый ящик активен.

    Чтобы отфильтровать почтовые ящики по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех почтовых ящиков нажмите кнопку Отменить.

    Контекстная панель

    Кнопка Описание
    Добавить Создание и установка параметров нового почтового ящика. 
    Настроить Переход к диалогу настройки внешнего вида отчетной формы.
    Excel Экспорт данных из отображаемой таблицы в формат MS Excel.

    Список ящиков

    Колонка Описание
    Колонка флажков Выбор правил, к которым предполагается применить какое-либо действие.
    Меню
    действий
    • Правила - отображает правила, которые назначены этому ящику;
    • Добавить новое правило - добавляет новое правило для этого ящика;
    • Журнал - позволяет посмотреть журнал событий по данному ящику;
    • Письма - отображает все сообщения, пришедшие на этот ящик;
    • Изменить - редактирование параметров учетной записи;
    • Удалить - удаление учетной записи.
    ID Идентификатор ящика в базе данных.
    Дата изм. Дата последнего изменения ящика.
    Название Название ящика и ссылка на изменение его параметров.
    Акт. Признак активности почтового ящика.
    Сайт Сайт, в котором данный ящик активен.

    Как добавить ящик

    Для создания нового почтового ящика нажмите кнопку Добавить. В открывшейся форме редактирования свойств почтового ящика установите параметры новой учетной записи.

    Как изменить параметры почтового ящика

    Для изменения параметров правила дважды кликните строку требуемой учетной записи в списке почтовых ящиков.
