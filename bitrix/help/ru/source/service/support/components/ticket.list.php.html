Компоненты 1.0::Список обращений

Компонент предназначен для вывода списка обращений. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeFile("support/ticket_list/default.php",
Array(
    'TICKET_EDIT_URL' => 'ticket_edit.php',
    'TICKETS_PER_PAGE' => '50',
    )
);?>

Описание параметров

TICKET_EDIT_URL
Адрес страницы редактирования существующих обращений и создания новых.
TICKETS_PER_PAGE
Количество обращений на одной странице.
