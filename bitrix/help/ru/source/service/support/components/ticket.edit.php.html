Компоненты 1.0::Редактирование обращения

Компонент предназначен для вывода формы редактирования существующего обращения либо создания нового. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeFile("support/ticket_edit/default.php", 
 Array(
   'ID' => $_REQUEST["ID"],
   'TICKET_LIST_URL' => 'ticket_list.php',
   'TICKET_MESSAGE_EDIT_URL' => 'ticket_message_edit.php',
   'MESSAGES_PER_PAGE' => '50',
   )
);?>

Описание параметров

ID
ID существующего обращения (если не задан, будет выведена форма для создания нового обращения).
TICKET_LIST_URL
Адрес страницы со списком обращений.
TICKET_MESSAGE_EDIT_URL
Адрес страницы для редактирования сообщения.
MESSAGES_PER_PAGE
Количество сообщений на одной странице.
