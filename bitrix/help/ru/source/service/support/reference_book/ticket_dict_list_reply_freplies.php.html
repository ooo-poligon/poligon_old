    • Форма служит для настройки справочников техподдержки. Справочники используются для вывода категорий, критичностей, статусов, оценок, источников, сложности обращений, а также часто используемых ответов.

    Форма настройки справочников техподдержки по умолчанию отображает справочник, выбранный в левой панели административного раздела (категории, критичности и т.д.) путём автоматической установки поля Тип фильтра по конкретному справочнику. Вы можете отобразить одновременно все справочники, сбросив фильтр нажатием кнопки Отменить.

    Фильтр

    Наименование параметра Описание
    Найти Поиск по наиболее типичным критериям, выбираемым в списке: заголовку и описанию. 
    *ID Поиск по ID.
    Сайты Отображать записи справочника только для указанных сайтов.
    *Заголовок Заголовок записи справочника для отображения.
    *Описание Маска для текста описания.
    *Символьный код Символьный код типа справочника.
    *Ответственный Сотрудник техподдержки, ответственный за новые обращения, созданные в соответствующей категорией или с той или иной критичностью.
    По умолчанию Определяет, отображать ли только справочники по умолчанию, не по умолчанию или все.

    * - эти поля допускают использование специальных логических выражений.

    Контекстная панель

    Кнопка Описание
    Добавить Добавление новой записи справочника. В открывшемся меню выберите, какой тип записи следует добавить: категорию, критичность, статус, оценку ответов, частый ответ, источник или сложность.
    Настроить Переход к диалогу настройки внешнего вида отчетной формы.
    Excel Экспорт данные из отображаемой таблицы в формат MS Excel.

    Действия над записями

    Панель Действия позволяет осуществить различные операции одновременно над несколькими записями. Чтобы применить то или иное действие к записям, следует пометить их галочками в левом столбце таблицы.

    Предусмотрены следующие действия.

    • Редактировать отмеченные записи - позволяет быстро отредактировать записи, не открывая форму редактирования. Доступные для изменения значения записей будут представлены в виде полей для ввода.
    • Удалить записи - удаляет отмеченные записи справочника.

    Как создать запись в справочнике

    Чтобы создать новую запись в справочнике, нажмите кнопку Добавить, расположенную на контекстной панели. В открывшейся форме Новая запись справочника задайте значения новой записи.

    Как изменить запись в справочнике

    Чтобы отредактировать существующую запись в справочнике, дважды щелкните мышью на строке требуемой записи. В открывшейся форме b задайте значения записи.
