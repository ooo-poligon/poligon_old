Уровни техподдержки (SLA)

В этом разделе

  • Фильтр
  • Записи справочника
  • Действия над записями

    • Форма (Сервисы > Техподдержка > Уровни поддержки) служит для создания и настройки уровней техподдержки. Уровни используются для ранжирования и определения приоритетов тех или иных пользователей на получение техподдержки.

    Фильтр

    Параметр Описание
    *Название Название уровня для отображения.
    *ID Идентификатор уровня для отображения.
    Сайт Отображать только уровни для определённого сайта.
    *Описание Маска для текста описания.

    * - эти поля допускают использование специальных логических выражений.

    Контекстная панель

    Кнопка Описание
    Добавить Добавление нового уровня техподдержки.
    Настроить Переход к диалогу настройки внешнего вида отчетной формы.
    Excel Экспорт данных из отображаемой таблицы в формат MS Excel.

    Записи справочника

    Таблица отображает записи в выбранном для отображения справочнике. Для сортировки таблицы по тому или иному полю вы можете кликнуть по заголовку таблицы.

    Колонка Описание
    Колонка флажков Выбор всех или только определённых записей для применения к ним действий при помощи панели Действия внизу таблицы.
    Меню
    действий
    		При нажатии на кнопку конкретного уровня, отображает меню, позволяющее:
    • Изменить - отредактировать параметры уровня;
    • Удалить - удалить запись.
    ID ID записи в базе.
    Приоритет Приоритет уровня техподдержки.
    Сайт Сайт, на котором действует данный уровень.
    Название Название уровня техподдержки.
    Описание Описание уровня техподдержки.
    Время реакции Максимально возможное время реакции (ответа) на сообщения авторов обращений.
    Группы пользователей Группы пользователей, для которых действуют правила уровня техподдержки.
    Ответственный Ответственный по умолчанию за обращения этого уровня.
    Выбрано Количество записей, выбранных для отображения.
    Отмечено Количество записей, отмеченных флажком в первом поле.

    Действия над уровнями

    Панель Действия позволяет осуществить различные операции одновременно над несколькими уровнями техподдержки. Для того чтобы применить то или иное действие к уровням, следует пометить их галочками в левом столбце таблицы.

    Предусмотрены следующие действия.

    • Редактировать отмеченные записи - позволяет быстро отредактировать уровни техподдержки, не открывая форму редактирования. Доступные для изменения значения записей будут представлены в виде полей для ввода.
    • Удалить записи - удаляет отмеченные уровни техподдержки.
