Компоненты 2.0::Список обращений

Компонент предназначен для вывода фильтра и списка обращений в техподдержку. Компонент является стандартным и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:support.ticket.list","",Array(
     "TICKET_EDIT_TEMPLATE" => "ticket_edit.php?ID=#ID#",
     "TICKETS_PER_PAGE" => "50",
     "SET_PAGE_TITLE" => "Y"
     )
);?>

Описание параметров

TICKET_EDIT_TEMPLATE
Адрес страницы для редактирования выбранного обращения.
TICKETS_PER_PAGE
Количество обращений на одной странице.
SET_PAGE_TITLE
Определяет устанавливать заголовок страницы (значение параметра равно Y) или нет (значение N). Если значение параметра равно Y, то в качестве заголовка страницы будет установлено Список обращений.
