Компоненты 2.0::Техподдержка (комплексный компонент)

Комплексный компонент создает полноценную систему техподдержки, с помощью которой можно просмотреть список обращений, добавить своё сообщение или написать новое обращение в техподдержку. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:support.ticket","",Array(
     "SEF_MODE" => "Y",
     "TICKETS_PER_PAGE" => "50",
     "MESSAGES_PER_PAGE" => "20",
     "SET_PAGE_TITLE" => "Y",
     "SEF_FOLDER" => "/examples/",
     "SEF_URL_TEMPLATES" => Array(
         "ticket_list" => "index.php",
         "ticket_edit" => "#ID#.php"
     ),
     "VARIABLE_ALIASES" => Array(
         "ticket_list" => Array(),
         "ticket_edit" => Array(),
  	)
     )
);?>

Описание параметров

SEF_MODE
При значении параметра Y включается поддержка ЧПУ и становятся доступными параметры настройки адресов ЧПУ:
  • SEF_FOLDER - каталог ЧПУ: путь до папки с которой работает компонент. Этот путь может как совпадать с физическим путём, так и не совпадать.
  • SEF_URL_TEMPLATE -> ticket_list - адрес страницы со списком обращений.
  • SEF_URL_TEMPLATE -> ticket_edit - адрес страницы редактирования обращений.
VARIABLE_ALIASES
Имена переменных. В режиме без поддержки ЧПУ указывается переменная, которой будет передаваться идентификатор обращения (например, ID).
TICKETS_PER_PAGE
Количество обращений на одной странице.
MESSAGES_PER_PAGE
Количество сообщений на одной странице.
SET_PAGE_TITLE
Определяет устанавливать заголовок страницы (значение параметра равно Y) или нет (значение N).
