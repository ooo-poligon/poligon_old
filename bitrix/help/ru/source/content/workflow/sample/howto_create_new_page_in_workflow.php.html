Создание новой страницы через документооборот

Для создания новой страницы через Документооборот в Меню команд кнопки   раздела Редактирование выберите команду Создать страницу. Откроется форма создания страницы.



  1. Выберите шаблон страницы (например, Стандартная страница или Включаемая область для Раздела или Страницы).
  2. Укажите статус, который будет присвоен документу после сохранения. Например, Draft (Черновик).
  3. В поле Полное имя файла укажите полное имя файла. Например, /ru/about/contacts.php. Либо воспользуйтесь кнопкой для выбора файла.
  4. Введите Заголовок страницы. Например, Контактная информация:  
  5. Разместите на странице необходимую информацию. Для выбора режима редактирования нужно отметить флагом соответствующее поле.
  6. Перейдите на вкладку Загруженные файлы и загрузите файлы (изображения и т.п.), используемые на новой странице.
    Для использования в документе ссылок на какие-либо файлы (jpg, gif, pdf и т.д.) пользователю необходимо загрузить их в документ. Т.е. если вы используете в тексте какие-либо изображения, то необходимо указать их название и ссылку на них в поле Загруженные файлы. После публикации документа все загруженные в него файлы переносятся в указанные каталоги.


  7. На вкладке Комментарий разместите дополнительную информацию для других пользователей.
  8. Нажмите кнопку Сохранить.
