    На странице История изменений (Контент > Документооборот > История) представлена история изменений документов. Каждое сохранение документа оставляет запись в истории. Максимальное количество последних записей в истории документа можно указать в настройках модуля. Также в настройках модуля указывается максимальное время хранения истории.

    Форма поиска

    Формы позволяет выполнить поиск записи из истории изменений в соответствии с указанными условиями.

    Поле Описание
    ID Идентификатор записи в истории изменений (возможно перечисление через запятую).
    ID документа Идентификатор документа (возможно перечисление через запятую).
    Дата изменения Дата изменения документа (дата создания записи истории редактирования).
    Кем изменен Пользователь, выполнявший изменение документа (создавший запись истории редактирования).
    Имя файла Имя файла документа.
    Заголовок Заголовок документа.
    Содержимое Предполагаемое содержание (текст) документа.
    Статус документа Статус документа.
    Логика между полями Установление логики между полями формы поиска.
    Показать все условияОтображение сразу всех условий
    Скрыть все условияСокрытие сразу всех условий

    Чтобы отобрать записи по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех записей нажмите кнопку Отменить.

    Список записей

    Поле Описание
    Колонка флажков Выбор записей, к которым предполагается применить какое-либо правило или действие.
    Действия Действия над записью в истории изменений:
    • Просмотр - просмотр записи;
    • Удалить - удаление записи.
    Дата изменения Дата изменения документа (т.е. дата создания записи в истории изменений).
    Кем изменен Пользователь, выполнивший изменение документа.
    Имя файла Имя файла документа.
    Заголовок Заголовок документа.
    Статус Статус документа в момент создания записи в истории изменений.

    История изменений позволяет сравнить изменения, внесенные в документы. Все сравнения производятся относительно более раннего документа. Зеленым отмечается то, что добавлено в более позднем документе, красным зачеркнутым - то что удалено из более ранего документа. Для сравнения:
    1. Установите флажки в чекбоксах тех документов, сравнение которых необходимо.
    2. Нажмите на кнопку Сравнить.
