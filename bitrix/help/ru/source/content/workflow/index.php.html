Описание модуля

Модуль Документооборот предназначен для организации последовательной обработки статических страниц, а также динамической составляющей сайта. Использование этого модуля позволяет наладить командную работу над содержимым сайта.

Модуль реализует следующую идеологию.

  1. Документы каждого раздела сайта могут быть модифицированы в режиме документооборота. Для этого пользователь должен иметь, как минимум, право Документооборот на данный раздел.
  2. Понятие документ подразумевает копию статической страницы сайта в модуле документооборота. У одной статической страницы не может быть более одного документа. В свою очередь, редактирование статической страницы представляет собой процесс модификации документа (копии данной страницы) в модуле документооборота. В процессе модификации возможна смена статуса документа. Конечным статусом является статус #1, при достижении которого содержимое документа переносится в статическую страницу (в файл). При этом порядок смены статусов может быть произвольным. В случае создания новой страницы в первую очередь создаётся документ, затем после его публикации создаётся непосредственно сама страница.
  3. Для использования в документе ссылок на какие-либо файлы (jpeg, gif, pdf и т.д.) пользователю необходимо загрузить их в документ. После публикации документа все загруженные в него файлы переносятся в указанные каталоги.
  4. В процессе модификации документа каждое его изменение записывается в историю документа. При необходимости отменить изменения, сделанные в документе, пользователь должен скопировать необходимую ему часть документа из его истории.
