    Форма Список документов (Контент > Документооборот > Документы) отображает документы, включенные в документооборот. Под документом понимается копия страницы сайта, находящейся в стадии редактирования в режиме документооборота. После достижения документом статуса #1 (момент публикации) все изменения переносятся из документа в файл (в редактируемую страницу).

    Форма поиска

    Форма позволяет выполнить поиск документа в соответствии с указанными условиями.

    Поле Описание
    ID документа Идентификатор документа.
    Индикатор доступа к документу Индикатор, показывающий, в каком состоянии редактирования находится документ.
    • зеленый– документ свободен для редактирования; 
    • желтый – документ временно заблокирован вами. Помните, кроме вас его никто не сможет редактировать. Максимальное время блокировки устанавливается в настройках модуля; 
    • красный– документ редактируется в данный момент другим пользователем.
    Дата изменения Дата последнего изменения документа.
    Кем изменен Пользователь, выполнивший последнее изменение документа.
    Сайт Сайт, в котором отображается редактируемый через модуль документ.
    Имя файла Поиск по имени файла документа.
    Заголовок Поиск по заголовку документа.
    Содержимое Произвольный текст, который может содержаться в документе.
    Статус Статус документа.
    Логика между полями Установление логики между полями формы поиска.
    Показать все условияПоказ полей сразу всех условий
    Скрыть все условияСокрытие полей сразу всех условий

    Чтобы отобрать документы по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех документов нажмите кнопку Отменить.

    Контекстная панель

    Кнопка Описание
    Добавить Переход к созданию документа в режиме документооборота.
    Настроить Позволяет перейти к диалогу настройки внешнего вида отчетной формы.
    Excel Экспортирует данные из отображаемой таблицы в формат MS Excel.

    Список документов

    Поле Описание
    Колонка флажков Выбор документов, к которым предполагается применить какое-либо правило или действие.
    Действия Действия над документом:
    • Разблокировать - Снятие блокировки с документа
    • Изменить - редактирование документа в режиме документооборота;
    • Просмотр - просмотр документа;
    • История - просмотр истории изменения документа по его ID;
    • История файла - просмотр истории изменения документа по его имени;
    • Удалить - удаление записи о документе из модуля документооборот.
    Инд. Индикатор, показывающий, в каком состоянии редактирования находится документ. 
  • – документ свободен для редактирования;
  • – документ временно заблокирован вами. Помните, кроме вас его никто не сможет редактировать. Максимальное время блокировки устанавливается в настройках модуля;
  • – документ редактируется в данный момент другим пользователем.
    • Дата изменения Дата последнего изменения документа.
    Кем изменен Пользователь, выполнивший последнее изменение документа.
    Заголовок Заголовок документа.
    Имя файла Имя файла документа.
    Статус Текущий статус документа (например, Черновик).
    Сайт Сайт, на котором отображается документ.
