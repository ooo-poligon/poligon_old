    Форма Настройки предназначена для управления общесистемными параметрами модуля Документооборот. Для удобства работы форма разделена на несколько закладок, каждая из которых содержит определённый набор параметров модуля.

    Закладка "Настройки"

    Служит для управления общими параметрами модуля Документооборот.

    Поле Описание
    Использовать HTML редактор (только для IE 5.0 или FireFox 1.0 и выше) Позволяет использовать встроенный HTML-редактор при редактировании документов в режиме документооборота.
    Сохранять историю изменений файлов при обычном редактировании Позволяет сохранять историю изменения документов, редактирование которых выполняется в обычном режиме (без использования документооборота).
    Максимальное время блокировки (мин.) Максимальное время блокировки документа пользователем. заблокированный документ недоступен для редактирования другими пользователями.
    Сколько дней хранить документ после его публикации Период времени (в днях), в течении которого копия документа будет храниться в истории модуля документооборота после того, как документ был переведен в статус Опубликован.
    Если значение поля - отрицательное число, то время хранения копии документа не ограничено. Чекбокс Очистить позволяет удалить записи свыше установленного периода.
    Максимальное количество последних записей в истории Максимальное количество записей об изменении документа или элемента информационного блока, хранимое в истории модуля документооборота. Чекбокс Очистить позволяет удалить записи свыше установленного периода
    Сколько дней хранить историю Максимальное количество дней, отводимых для хранения истории изменений документа и элемента информационного блока. Отрицательное значение поля позволяет хранить данные об изменении документов в течение неограниченного периода времени. Чекбокс Очистить позволяет удалить записи свыше установленного периода
    Группа администраторов документооборота Группа пользователей-администраторов документооборота, обладающих правом на редактирование, перемещение, удаление документов в любом из статусов, а также на разблокирование любых документов.

    Закладка "Доступ"

    Служит для управления уровнем прав доступа групп пользователей к ресурсам модуля Документооборот.

    Примечание: права на доступ к модулю Документооборот проверяются только на страницах административной части этого модуля.

    Поле Описание
    По умолчанию Уровень доступа групп пользователей, для которых установлено право по умолчанию.
    Группа пользователей Уровень прав группы пользователей на доступ к ресурсам модуля:
    • Закрыт - доступ запрещён;
    • Просмотр без права модификации - просмотр файлов модуля без права изменения;
    • Участие в документообороте - право на редактирование файлов в режиме документооборота;
    • Полный доступ - право на изменение документов в любом из статусов, а также на управление параметрами модуля (в т.ч. правами доступа других пользователей).

    Кнопки управления

    Кнопка Описание
    Сохранить Сохранение параметров модуля.
    Сбросить Сбросить внесённые изменения, которые ещё не сохранены.
    По умолчанию Установить значения по умолчанию.
