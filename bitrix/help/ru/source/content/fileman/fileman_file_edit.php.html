Редактирование файла

Форма редактирования


Редактирование файла можно осуществить двумя способами: через Форму редактирования в Менеджере файлов и через визуальный редактор с помощью кнопки   на Административной панели, которая вызовет визуальный редактор. Различие между этими способами в том, что редактирование через Форму редактирования  позволяет изменить не только содержание файла, но и его свойства. Если вам нужно изменить только содержание, то удобнее это сделать через кнопку , но если нужно менять еще и свойства, то лучше редактировать через Форму редактирования.

Сама форма состоит из трех закладок:
  • Редактирование
  • Свойства
  • Меню

Закладка "Редактирование"

В поле Заголовок страницы можно изменить заголовок. Ниже поля располагается HTML редактор, в котором производится редактирование содержимого страницы.

Закладка "Свойства"

Закладка предназначена для задания и редактирования значений свойств страницы, а также для управления значениями тегов META: ключевыми словами (keywords) и описанием (description) страницы.

Поле Описание
Описание Название свойства.
Ключевые слова
Поле для задания значения свойства.
Дополнительный заголовокТекст, который будет виден в заголовке окна браузера
Мета-теги robotsзначения тега robots

Примечание: Кнопка Ещё добавляет пустую строку для создания нового свойства текущей страницы (данное свойство будет использоваться только для текущей страницы). При создании свойства в поле Код необходимо указать код (мнемоническое имя) свойства. Код свойства задается символами латинского алфавита.

Закладка "Меню"

Форма позволяет добавить пункт, ссылающийся на созданную страницу, в меню текущего раздела сайта (т.е. раздела, в котором создаётся страница).

Поле Описание
Добавить страницу в меню Добавление в меню текущего раздела сайта пункта, ссылающегося на созданную страницу.
