    Загрузка файлов

    Вы можете загрузить на сайт любой файл с локального компьютера. Файл будет загружен в каталог, который открыт в текущий момент в менеджере файлов.

    Поля формы

    Поле Описание
    Имя файла Имя, которое будет присвоено файлу после загрузки на сайт. По умолчанию указывается оригинальное имя загружаемого файла.
    Файл для загрузки С помощью кнопки Обзор в данном поле указывается имя и путь к загружаемому файлу.

    Кнопки управления

    Кнопка Описание
    Сохранить Загрузка выбранных файлов. Переход в папку, в которую была выполнена загрузка.
    Применить Загрузка выбранных файлов. Очистка полей формы для выбора следующих файлов для загрузки.
    Отменить Отмена загрузки файлов. Переход в папку, открытую в данный момент в менеджере файлов.

    Пример загрузки изображений на сайт

    1. Откройте Менеджер файлов и перейдите в папку /images/ (все изображения рекомендуется загружать в эту папку).
    2. Выберите пункт меню Загрузить файл.
    3. Нажмите кнопку Обзор и выберите файлы на локальном диске. После этого в поле Имя файла появится название выбранного файла, а в поле Файл для загрузки будет указан путь к данному файлу на вашем локальном диске. При необходимости измените имя, которое будет присвоено файлу после загрузки.
    4. Нажмите кнопку Загрузить файлы.
