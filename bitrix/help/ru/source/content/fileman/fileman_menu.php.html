Меню

Использование меню обеспечивает удобство навигации по разделам и страницам сайта. Система "1C-Битрикс: Управление сайтом" позволяет полностью управлять меню веб-сайта без каких либо ограничений:
  • добавлять новые пункты меню, ссылающиеся на страницы и разделы сайта;
  • изменять названия и ссылки пунктов меню;
  • управлять условиями показа и параметрами пунктов меню;
  • удалять пункты меню.

Количество и типы используемых на страницах сайта меню определяются шаблоном дизайна сайта. Вид меню в публичном разделе сайта определяется шаблоном меню.

Типы меню

Тип меню - это условное обозначение меню. Тип меню используется для организации показа меню на странице сайта (т.е. в шаблоне сайта определяется область для показа меню, а затем с помощью компонента Меню или другого программного кода в данную область подключается меню необходимого типа), в качестве префикса для обозначения файлов, относящихся к данному меню, а также при построении карты сайта.

Обычно на сайте используется два типа меню:

  • Главное меню (обычно верхнее), используемое для навигации по разделам сайта;
  • Дополнительное меню (обычно левое или правое), используемое для навигации по подразделам и страницам раздела сайта.

Внимание!
По умолчанию меню родительского раздела используется во всех подразделах сайта, если для них не создано собственное меню.
Например, главное меню сайта предназначено для отображения ссылок на основные разделы сайта. Поэтому обычно главное меню создаётся только для корневой папки сайта и отображается во всех разделах сайта.
Второстепенное меню используется для отображения ссылок на документы и подразделы внутри раздела. Поэтому обычно второстепенное меню создаётся отдельно для каждого раздела сайта.

На сайте могут использоваться также и другие типы меню, например, нижнее. Используемые на сайте типы меню указываются в настройках модуля Управление структурой.

Редактирования меню

Приступить к управлению меню можно двумя способами: непосредственно из разделов Административной панели или из Панели управления.
