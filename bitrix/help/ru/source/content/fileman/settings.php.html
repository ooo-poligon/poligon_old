    Форма Настройки модуля Управления структурой сайта (Настройки > Настройки продукта > Настройки модулей > Управление структурой) предназначена для управления глобальными настройками модуля. Для удобства выполнения настроек форма разделена на три закладки: Настройки, Визуальный редактор и Доступ.

    Закладка "Настройки"

    Настройка общих параметров модуля.

    Поле Описание
    Редактор страниц по умолчанию Режим редактирования страниц по умолчанию. Доступны следующие режимы:
    • Текстовый редактор - отображает сам текст файла, без включения PHP кода. Однако, в этом режиме отображаются все HTML-теги.
    • Редактирование PHP скрипта - отображает исходный текст файла. Режим предназначен исключительно для использования разработчиками.
    • Визуальный HTML редактор - открывает встроенный визуальный редактор HTML, позволяющий форматировать текст аналогично Microsoft Word.
    Расширения исполняемых файлов Указываются расширения исполняемых файлов через запятую для контроля доступа.
    Разрешенные компонентыКомпоненты, которые будут показаны в визуальном редакторе. По умолчанию поле пустое и отображаются все компоненты. Поле - разрешающее, то есть  введенные имена компонентов отображаются, остальные - нет.
    Настройки для сайтов
    Использовать индивидуальные настройки для каждого сайта При включенной опции для каждого из созданных в системе сайтов могут быть заданы индивидуальные Типы меню и Типы свойств.
    Настройки для сайта В выпадающем списке доступны все созданные в системе сайты. Выбор доступен при отмеченной опции Использовать индивидуальные настройки для каждого сайта.
    Типы меню В данной мини-форме указываются используемые на сайте типы меню и их описания.
    Поле Описание
    Тип Тип (мнемоническое имя) используемого на сайте меню. Обычно мнемонические имена типов меню назначаются, исходя из расположения меню на странице сайта (например, top, left и т.д.). Имя типа меню указывается латинскими буквами.
    Название Произвольное название (описание) типа меню.
    Количество дополнительных  параметров меню Количество полей для задания значений параметров, обрабатываемых в шаблоне показа меню и влияющих на представление меню.
    Типы свойств Форма служит для создания свойств страниц сайта и задания их описания.
    Поле Описание
    Тип Тип (мнемоническое имя) свойства. Указывается латинскими буквами.
    Название Произвольное название (описание) свойства. Указанное название будет отображаться в форме задания значений свойств для папки или страницы.

    Мнемонические имена типов свойств, используемых для управления метаданными страницы, должны соответствовать названиям соответствующих мета-тегов в языке HTML. Например, ключевые слова - keywords, описание - description.

    Закладка "Визуальный редактор"

    Настройка визуального HTML редактора.

    Поле Описание
    Разрешить в визуальном HTML редакторе выводить стили без названий Позволяет при редактировании страницы в режиме HTML редактора выводить в списке доступных CSS стилей в том числе и те стили, для которых не заданы названия.

    Примечание: Названия стилей задаются в файле .styles.php, размещённом в папке соответствующего шаблона сайта.
    Отображать внешний вид стилей в выпадающем спискеНазвания стилей будут отображаться в соответствии с их оформлением
    По умолчанию открывать HTML редактор в полноэкранном режиме Использование полноэкранного режима по умолчанию.
    Настройки преобразования содержимого
    Скрывать исходный код компонентов 2.0 при просмотре страницы в режиме кода Установленный чекбокс позволяет скрыть код от  просмотра
    Группы символов, автоматически заменяемые на HTML-сущности при сохранении страницы Чекбоксы, определяющие подмену символов:
    • Дополнительные символы европейских алфавитов (умляуты и пр.)
    • Символы греческого алфавита
    • Другие символы, имеющие замену в виде html-сущностей
    Настройки системы проверки орфографии
    Использовать для проверки орфографии MS Word (если установлен) При включенной опции проверка орфографии будет проводиться с помощью редактора Microsoft Word, если он установлен на компьютере пользователя.
    Использовать для проверки pSpell/aSpell Опция доступна, если на сервере установлено расширение php - pSpell и имеются библиотеки aSpell (система словарей). Они доступны по адресу http://aspell.sourceforge.net/.
    Установить словарь и загрузить дополнительные словари можно по ссылкам.

    Закладка "Доступ"

    Настройка уровней доступа групп пользователей к ресурсам модуля.

    Поле Описание
    По умолчанию Уровень доступа к модулю Управление структурой для всех групп пользователей, для которых установлено право доступа по умолчанию.
    [Группа пользователей] Изменение настроек доступа для конкретной группы. Возможно задание следующих уровней доступа:
    • по умолчанию - действует уровень, установленный в поле по умолчанию.
    • доступ закрыт - доступ пользователей к файлам и папка невозможен;
    • только к доступным папкам - доступ открыт к тем папкам, в свойствах которых разрешен доступ для данной группы пользователей;
    • просмотр настроек модуля - просмотр только настроек модуля без возможности изменения.
    • полный доступ - можно все

    Кнопки управления

    Кнопка Описание
    Сохранить Сохранение внесённых изменений.
    Сбросить Отмена внесённых изменений. Возврат первоначальных значений параметров.
    По умолчанию Установка значений свойств модуля равными значениям по умолчанию.
