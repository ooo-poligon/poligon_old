Элементы информационного блока

В этом разделе

  • Форма поиска
  • Контекстная панель
  • Список элементов

    • На странице представлен список элементов разделов информационного блока или самого блока. Элемент раздела - сущность, не поддающаяся дальнейшей классификации в пределах созданной вами иерархии. Например, элемент-товар Коврик для мыши раздела Аксессуары. Поля формы отличаются для разных типов инфоблоков от описанного по порядку элементов и их числу. Здесь приведено самое общее описание элементов.

    Форма поиска

    Форма позволяет выполнить поиск элементов раздела или инфоблока, соответствующих указанным условиям.

    ПолеОписание
    НазваниеНазвание элемента. Допустимо использование операторов сложного поиска.
    РазделРодительский раздел элемента.
    ID (начальный и конечный)Отрезок ID для выборки элементов.
    Дата измененияДата и время последнего изменения.
    Символьный код Мнемонический код элемента.
    Внешний код Символьный код, используемый для связи элемента с внешним источником данных.
    Кто изменилПоследний изменивший элемент.
    Дата создания Дата создания элемента.
    Кто создал Пользователь, создавший элемент.
    СтатусСтатус элемента (при использовании документооборота).
    АктивенАктивность элемента раздела. Активные элементы доступны для просмотра в публичной части сайта.
    ОписаниеОписание элемента (фильтр по описанию для анонса и детальному описанию). Допустимо использование операторов сложного поиска.
    [Свойство]Поля для фильтрации элементов по значениям свойств, для которых было задано свойство "Выводить на странице списка элементов поле для фильтрации по этому свойству".
    Показать все условияПоказ всех условий одновременно
    Скрыть все условияСокрытие всех условий одновременно.

    Чтобы отобрать элементы информационного блока по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех элементов нажмите кнопку Отменить.

    Контекстная панель

    КнопкаОписание
    На один уровень вверхПозволяет перейти в вышележащую папку.
    Добавить элемент Переход к форме создания нового элемента инфоблока.
    Добавить подразделПереход к форме создания нового раздела инфоблока.
    Свойства инфоблока Переход к форме настройки свойств информационного блока, к которому относятся элементы, представленные на странице.
    НастроитьПереход к диалогу настройки внешнего вида отчетной формы.
    ExcelЭкспорт данных из отображаемой таблицы в формат MS Excel.

    Список элементов

    ПолеОписание
    Колонка флажковПоле предназначено для выбора элементов, к которым предполагается применить какое-либо правило или действие.
    ДействияДоступные действия:
    • Разблокировать - освободить заблокированный элемент;
    • Изменить - редактировать элемент инфоблока;
    • История изменений - просмотреть историю изменения элемента (возможно только при наличии установленного модуля Документооборот);
    • Просмотреть - перейти к просмотру элемента в публичном разделе сайта;
    • Удалить - удалить элемент инфоблока.
    НазваниеНазвание элемента.
    Акт.Активность элемента. Если поле отмечено, то элемент доступен для показа в публичной части сайта.
    Сорт.Относительный "вес", определяющий положение элемента в списке.
    Символьный код Мнемонический код элемента.
    Внешний код Символьный код, используемый для связи элемента с внешним источником данных.
    Дата изм.Дата и время последнего изменения элемента.
    ЭлементовДанное поле доступно для отображения только при отмеченной опции Совместный просмотр секций и элементов на странице настроек модуля.
    ПодразделовДанное поле доступно для отображения только при отмеченной опции Совместный просмотр секций и элементов на странице настроек модуля.
    Начало активности Дата начала активности.
    Окончание активности Дата окончания активности.
    Кем измененаЛогин и имя пользователя, который совершил последние изменения.
    Дата созданияВремя и дата создания данной записи.
    Кем создана Логин и имя пользователя, который создал запись о элементе.
    Количество показов Количество показов данного элемента на сайте.
    Дата первого показа Время и дата первого показа элемента на сайте.
    Картинка для анонса Изображение, указанное в анонсе элемента.
    Описание для анонса Краткое описание элемента для анонса.
    Детальная картинка Изображение, которое выводится в детальном описании элемента.
    Детальное описание Детальное описание элемента.
    IDИдентификатор элемента.
    СтатусСтатус элемента (при использовании документооборота). Например, Опубликован.
    Блокировка Значение поля информирует о доступности элемента для редактирования:
  • - элемент доступен для редактирования;
  • - элемент заблокирован вами. Для того чтобы освободить элемент, выберите пункт Разблокировать в меню столбца Действие;
  • - элемент заблокирован другим пользователем (т.е. редактирование элемента выполняется другим пользователем).
    • Кем заблокирована Логин и имя пользователя, который заблокировал данную запись.
    Когда заблокирована Время и дата произведения блокировки записи.
    Комментарий администратора Произвольный комментарий, который отображается только в административном разделе.
    [Свойство]Выводится значение этого свойства, если оно указано.
    [Тип цены]Величина стоимости в соответствии с данным типом цены.
