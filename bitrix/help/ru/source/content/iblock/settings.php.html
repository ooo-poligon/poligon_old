    • Управление общими параметрами модуля Информационные блоки осуществляется с помощью формы Настройки модуля: Настройки > Настройки продукта > Настройки модулей > Информационные блоки.

    Поля формы

    ПолеОписание
    Использовать визуальный редактор для изменения HTML содержимогоДля редактирования наполнения информационных блоков используется HTML-редактор, позволяющий изменять форматирование текста (размер, шрифт и т.д.), вставлять изображения и т.д.
    Показывать код загрузки из внешних источниковВ Административном разделе отображается код, при помощи которого система связывает данные вашего сайта с данными из других источников (например, других каталогов в Интернете).
    Путь к экспортируемым RSS файламПуть к директории относительно корня сайта, в которой будут сохраняться файлы RSS.
    Совместный просмотр секций и элементовРежим совместного просмотра секций и элементов напоминает функционирование проводника Windows: элементы и подразделы определённого раздела могут быть просмотрены лишь внутри этого раздела.
    Максимальное количество разделов в менюОпределяется максимальное количество отображаемых разделов информационных блоков в левом меню административного раздела. Позволяет избегать перегруженности меню.
