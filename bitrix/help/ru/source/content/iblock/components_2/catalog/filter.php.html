Каталог::Фильтр по элементам

Компонент выводит форму фильтра для фильтрации элементов информационых блоков. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова

<?$APPLICATION->IncludeComponent("bitrix:catalog.filter","",Array(
     "IBLOCK_TYPE" => "books",
     "IBLOCK_ID" => "30",
     "FILTER_NAME" => "arrFilter",
     "FIELD_CODE" => Array("NAME"),
     "PROPERTY_CODE" => Array("YEAR","PUBLISHER"),
     "PRICE_CODE" => Array("WHOLESALE","RETAIL"),
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "3600",
     "LIST_HEIGHT" => "5",
     "TEXT_WIDTH" => "20",
     "NUMBER_WIDTH" => "5"
     )
);?>

Описание параметров

Источник данных
IBLOCK_TYPE
Тип информационного блока.
IBLOCK_ID
Идентификатор информационного блока выбранного типа.
FILTER_NAME
Имя массива со значениями фильтра для фильтрации элементов. Если имя не указано, то будет использоваться значение по умолчанию.
FIELD_CODE
Необходимые для показа поля фильтра.
PROPERTY_CODE
Свойства инфоблока, которые будут выступать в качестве полей фильтра.
Внешний вид
LIST_HEIGHT
Высота списков множественного выбора, отображаемых в фильтре.
TEXT_WIDTH
Ширина текстового поля ввода в форме фильтра.
NUMBER_WIDTH
Ширина полей ввода для числовых интервалов.
Настройки кеширования
CACHE_TYPE
Тип кеширования:
  • A - Авто: действует при включенном кешировании в течение заданного времени;
  • Y - Кешировать: необходимо определить время кеширования;
  • N - Не кешировать: кеширования нет в любом случае.
CACHE_TIME
Время кеширования, указанное в секундах.
Цены
PRICE_CODE
Указываются типы цен для выводимых элементов. Если не задан ни один из типов, то цена товара и кнопки Купить и В корзину показаны не будут.
