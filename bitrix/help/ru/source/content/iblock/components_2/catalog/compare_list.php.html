Каталог::Список сравниваемых элементов каталога

Компонент выводит список сравниваемых элементов каталога в виде небольшой таблицы. Он предоставляет возможность удаления элемента из списка и возможность сравнения элементов. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова

<?$APPLICATION->IncludeComponent("bitrix:catalog.compare.list","",Array(
     "IBLOCK_TYPE" => "books",
     "IBLOCK_ID" => "30",
     "DETAIL_URL" => "element.php?IBLOCK_ID=#IBLOCK_ID#&SECTION_ID=#SECTION_ID#&ELEMENT_ID=#ELEMENT_ID#",
     "COMPARE_URL" => "compare.php?IBLOCK_ID=#IBLOCK_ID#",
     "NAME" => "CATALOG_COMPARE_LIST"
     )
);?>

Описание параметров

Основные параметры
IBLOCK_TYPE
Указывается тип информационного блока.
IBLOCK_ID
Идентификатор информационного блока выбранного типа.
Шаблоны ссылок
DETAIL_URL
Путь к странице с детальным описанием элемента инфоблока.
Дополнительные настройки
COMPARE_URL
Путь к странице с таблицей сравнения.
NAME
Имя переменной, в которой передается список сравниваемых элементов. По умолчанию CATALOG_COMPARE_LIST.
