Каталог::Структура разделов

Компонент выводит список разделов инфоблока с указанным количеством элементов в каждом разделе. Компонент стандартный и входит в дистрибутив модуля.

Пример вызова

<?$APPLICATION->IncludeComponent("bitrix:catalog.section.list","",Array(
     "IBLOCK_TYPE" => "books",
     "IBLOCK_ID" => "30",
     "SECTION_ID" => $_REQUEST["SECTION_ID"],
     "SECTION_URL" => "section.php?IBLOCK_ID=#IBLOCK_ID#&SECTION_ID=#SECTION_ID#",
     "COUNT_ELEMENTS" => "Y",
     "DISPLAY_PANEL" => "N",
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "3600"
     )
);?>

Описание параметров

Основные параметры
IBLOCK_TYPE
Идентификатор типа информационного блока.
IBLOCK_ID
Идентификатор информационного блока.
SECTION_ID
Задается код, в котором передается ID раздела. Значением по умолчанию параметра является ={$_REQUEST[" SECTION _ID"]}.
COUNT_ELEMENTS
[Y|N] При отмеченной опции отображается количество элементов, содержащихся в разделе.
Шаблоны ссылок
SECTION_URL
Путь к странице с содержимым раздела.
Настройки кэширования
CACHE_TYPE
Тип кеширования:
  • A - Авто: действует при включенном кешировании в течение заданного времени;
  • Y - Кешировать: необходимо определить время кеширования;
  • N - Не кешировать: кеширования нет в любом случае.
CACHE_TIME
Время кеширования, указанное в секундах.
Дополнительные настройки
DISPLAY_PANEL
[Y/N] При выборе значения "Y" в Административную панель будут добавлены кнопки управления инфоблоками указанного типа.
