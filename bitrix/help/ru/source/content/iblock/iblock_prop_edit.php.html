Детальное редактирование свойств инфоблока

В этом разделе
  • Поля формы
  • Кнопки управления

    • Форма позволяет выполнить детальную настройку свойств инфоблока. Поля некоторых формы могут отличаться для разных свойств. Для открытия формы нужно:

    1. Выполнить команду Контент > Информ. блоки > Типы информ. блоков > Название типа инфоблоков > Название инфоблока;
    2. В колонке команд выбрать команду Изменить;
    3. Перейдите в закладку Свойства
    4. В строке с нужным свойством элемента намжите на кнопку . Откроется форма Свойство детально.

    Поля формы

    ПолеОписание
    Поля, общие для всех типов свойств
    IDИдентификатор свойства.
    Символьный код Мнемонический код свойства, состоящий из латинских символов и цифр.
    Внешний код Символьный код, используемый для связи свойства с внешним источником данных. Поле отображается при включенной опции Показывать код загрузки из внешних источников в настройках модуля Информационные блоки.
    НазваниеНазвание свойства элементов инфоблока.
    Свойство активно Признак активности свойства (если свойство активно, оно может быть выбрано для показа в публичной части сайта).
    ТипТип свойства:
    • строка - значение свойства задается в виде текстовой строки;
    • число - значение свойства задается в виде числа;
    • список - значение свойства выбирается из списка;
    • файл - в качестве значения свойства используется файл;
    • привязка к разделам - задание связи между элементом данного инфоблока и разделами другого информационного блока;
    • привязка к элементам - задание связи между элементами информационных блоков.
    • привязка к файлу на сервере - задание связи между элементом данного инфоблока и файлом на сервере.
    • привязка к элементам в виде списка - задание связи между элементами в списке
    • дата/время - значение свойства задается в виде даты/времени.
    • привязка к теме форума - с помощью данного свойства можно задать связь между элементом данного инфоблока и темами форума.
    • HTML/текст - значение свойства задается в виде текста с HTML-тегами.
    • привязка к пользователю - с помощью данного свойства можно задать связь между элементом данного инфоблока и пользователями системы.
    • привязка к элементам по XML_ID - привязка хранится как строка и значением является xml_id привязанного элемента.
    МножественноеПризнак множественности свойства, т.е. наличия одновременно нескольких значений данного свойства.
    Например, свойство элементов инфоблока Мобильные телефоны Цвет панели может одновременно иметь следующие значения: синий, красный, черный и т.д.
    Количество полей для ввода новых множественных значенийЕсли данное свойство является множественным, то в этом определяется количество полей для ввода значений свойства (т.е. количество возможных значений свойства).
    Порядок сортировки Задается "вес" свойства, определяющий его положение в общем списке свойств.
    Тип свойства "Строка", "Число"
    Значение по умолчанию Значение данного свойства, присваиваемое всем вновь создаваемым элементам информационного блока по умолчанию.
    Размер поля для ввода значения Определяется длина и ширина поля для ввода значения данного свойства в форме создания/редактирования элементов инфоблока.
    Длина и ширина поля задаются числом столбцов и строк соответственно.
    Значения свойств участвуют в поиске Разрешается или запрещается индексация значений данного свойства для внутреннего поиска.
    Выводить на странице списка элементов поле для фильтрации по этому свойству Определяет возможность вывода в фильтре на странице со списком элементов отдельного поля, с помощью которого будет осуществляться фильтрация элементов по значениям данного свойства.

    Поле в фильтре будет иметь то же название и тип, что и данное свойство. Например, для свойства Год выпуска типа список данное поле в фильтре будет называться Год выпуска, а доступные значения поля будут представлены в виде списка.
    Выводить поле для описания значения Определяется, будет ли выводиться поле для описания данного свойства в форме создания/редактирования элементов информационного блока.
    Тип свойства "Список"
    Внешний вид Форма представления списка доступных значений свойства:
    • список - значение свойства выбирается из списка;
    • флажки - значение свойства указывается с помощью флажка, устанавливаемого в поле рядом с нужным значением.
    Количество строк (для внешнего вида "список")Высота поля со списком в строках (используется, если в предыдущем поле выбрано значение список).
    Значения свойств участвуют в поиске Разрешается или запрещается индексация значений данного свойства для внутреннего поиска.
    Выводить на странице списка элементов поле для фильтрации по этому свойству Определяет возможность вывода в фильтре на странице со списком элементов отдельного поля, с помощью которого будет осуществляться фильтрация элементов по значениям данного свойства.

    Поле в фильтре будет иметь то же название и тип, что и данное свойство. Например, для свойства Год выпуска типа список данное поле в фильтре будет называться Год выпуска, а доступные значения поля будут представлены в виде списка.
    Значения списка

    Данное поле служит для создания списка доступных значений свойства:

    ID ЗначениеСорт.Умолч.
    [1][2][3][4]
    ПолеОписание
    [1] ID Идентификатор значения свойства в списке. Задается символами латинского алфавита и цифрами (не должен начинаться с цифры).
    [2] Значение Поле для создания возможного значения свойства.
    [3] Сорт. Относительный "вес" возможного значения, определяющий положение значения в списке.
    [4] Умолч. Признак того, что для новых элементов инфоблока по умолчанию настраиваемое свойство будет принимать данное значение.
    Тип свойства "Файл"
    Типы загружаемых файлов Список расширений (через запятую) доступных для загрузки файлов.
    Размер поля для ввода (столбцов) Длина поля для ввода значения свойства. Задается количеством столбцов.
    Выводить поле для описания значения Определяется, будет ли выводиться поле для описания данного свойства в форме создания/редактирования элементов информационного блока.
    Тип свойства "Привязка к разделам "
    Информационный блок Информационный блок, к разделам которого может осуществляться привязка элементов данного инфоблока.
    Выводить на странице списка элементов поле для фильтрации по этому свойству Определяет возможность вывода в фильтре на странице со списком элементов отдельного поля, с помощью которого будет осуществляться фильтрация элементов по значениям данного свойства.

    Поле в фильтре будет иметь то же название и тип, что и данное свойство. Например, для свойства Год выпуска типа список данное поле в фильтре будет называться Год выпуска, а доступные значения поля будут представлены в виде списка.
    Тип свойства "Привязка к элементам "
    Информационный блок Информационный блок, к элементам которого может осуществляться привязка элементов данного инфоблока.
    Выводить на странице списка элементов поле для фильтрации по этому свойству Определяет возможность вывода в фильтре на странице со списком элементов отдельного поля, с помощью которого будет осуществляться фильтрация элементов по значениям данного свойства.

    Поле в фильтре будет иметь то же название и тип, что и данное свойство. Например, для свойства Год выпуска типа список данное поле в фильтре будет называться Год выпуска, а доступные значения поля будут представлены в виде списка.
