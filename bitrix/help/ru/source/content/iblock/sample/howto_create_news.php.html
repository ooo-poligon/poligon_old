Добавление новости

Чтобы добавить новость, выполните следующие действия.

  1. Откройте страницу типа информационных блоков Новости: Контент > Новости > раздел Новостей, в котором необходимо создать новость. Например, инфоблок Новости компании
  2. На странице со списком новостей выберите пункт Контекстного меню Добавить элемент. Вы перейдете к форме создания нового элемента (новости).
  3. Установите флажок в поле Запись активна, чтобы новость была доступна для просмотра в публичном разделе сайта.
  4. Укажите период активности записи. Нажмите на кнопку , расположенную рядом с полями Период активности, чтобы использовать календарь для выбора даты. Вы можете заранее добавить новость в административный раздел. Показ новости начнется с указанной вами даты.


  5. Введите название новости.
  6. Перейдите на закладку Анонс и создайте описание новости для анонса.
  7. Укажите полный текст новости. Для этого перейдите на закладку Подробно.
  8. На закладке Группы выберите раздел инфоблока Новости компании, к которому относится новость.
  9. Откройте закладку Дополнительно. Здесь вы можете указать индекс сортировки и символьный код новости.
  10. Нажмите кнопку Сохранить, чтобы добавить новость и вернуться к списку новостей.
