Экспорт в формате XML

Форма Экспорт XML (Контент > Информ. блоки > Экспорт > XML) предназначена для экспорта данных из инфоблока в XML файл.

Поле Описание
Файл для выгрузки Путь относительно корня сайта к файлу, в который будет выгружен инфоблок. Достаточно задать только имя и инфоблок будет выгружен по указанному пути в файл <имя_файла>.xml.
Информационный блок Указывается информационный блок, который необходимо экспортировать.
Длительность шага в секундах Количество секунд выполнения шага экспорта. Если указано "0", то экспорт инфоблока будет осуществлен за один шаг.

При нажатии на кнопку Экспортировать начинается экспорт инфоблока и над формой настроек отображается ход процесса выгрузки. Кнопка Остановить экспорт служит для остановки процесса выгрузки информационного блока в случае необходимости. После завершения операции выводится соответствующее сообщение.
