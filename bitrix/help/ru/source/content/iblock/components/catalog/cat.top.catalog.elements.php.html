Каталог::TOP элементов каталога

Компонент предназначен для вывода TOP'а элементов каталога в табличном порядке. Как правило, используется на главной странице сайта.

Пример вызова

<?
$APPLICATION->IncludeFile("iblock/catalog/main_page.php", Array(
	"IBLOCK_TYPE"		=> "catalog",
	"IBLOCK_ID"		=> "21",
	"ELEMENT_COUNT"            => "6",
	"LINE_ELEMENT_COUNT"	=> "2",
	"ELEMENT_SORT_FIELD"	=> "shows",
	"ELEMENT_SORT_ORDER"	=> "desc",
	"arrPROPERTY_CODE" => Array(
		"YEAR",
		"STANDBY_TIME",
		"TALKTIME",
		"WEIGHT",
		"STANDART",
		"SIZE",
		"BATTERY"
		),
	"PRICE_CODE"		=> "RETAIL",
	"BASKET_URL"		=> "/personal/basket.php",
	"CACHE_TIME"		=> "3600",
	"ACTION_VARIABLE"		=> "action",
	"PRODUCT_ID_VARIABLE"	=> "id",
	"SHOW_DESCRIPTION"		=> "N",
	"USE_PRICE_COUNT"		=> "N",
	"SHOW_PRICE_COUNT"		=> "1",
));
?>

Описание параметров

IBLOCK_TYPE
Тип инфоблока.

IBLOCK_ID
ID инфоблока.

ELEMENT_COUNT
Максимальное количество элементов, выводимых в каждом разделе.

LINE_ELEMENT_COUNT
Количество элементов, выводимых в одной строке таблицы.

ELEMENT_SORT_FIELD
Сортировка может быть выполнена по:
  • shows – среднее количество просмотров элемента (популярность элемента);
  • sort – по индексу сортировки;
  • timestamp_x – по дате изменения;
  • name – по заголовку;
  • id – по ID элемента;
  • active_from – по дате активности “с”;
  • active_to – по дате активности “по”.

ELEMENT_SORT_ORDER
Порядок сортировки элементов, допустимы следующие значения:
  • asc – по возрастанию;
  • desc – по убыванию.

arrPROPERTY_CODE
Массив символьных кодов свойств.

PRICE_CODE
Символьный код типа цены элемента.

BASKET_URL
URL, ведущий на страницу с корзиной покупателя.

CACHE_TIME
(сек.) время кеширования значений выбираемых из базы.

ACTION_VARIABLE
Название переменной, в которой передается действие, например, BUY и др.

PRODUCT_ID_VARIABLE
Название переменной, в которой передается код товара для покупки.

SHOW_DESCRIPTION
Показывать описание для списка.

USE_PRICE_COUNT
Использовать вывод цен с диапазонами. При отмеченной опции выводятся цены всех типов на товары.

SHOW_PRICE_COUNT
Указывается, для какого количества товаров выводить цены.
