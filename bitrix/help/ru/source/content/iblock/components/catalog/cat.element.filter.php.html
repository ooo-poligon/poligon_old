Каталог::Фильтр элементов

Компонент предназначен для вывода формы фильтра элементов.

Пример вызова

<?
$APPLICATION->IncludeFile("iblock/catalog/element_filter.php", Array(
	"IBLOCK_TYPE"		=> "catalog",
	"IBLOCK_ID"		=> "21",
	"arrFIELD_CODE"		=> Array("NAME"),
	"arrPROPERTY_CODE"	=> Array(
		"YEAR",
		"WEIGHT",
		"STANDART",
		"BATTERY",
		"WAP",
		"VIBRO",
		"VOICE",
		"PC"
		),
	"arrPRICE_CODE"		=> Array("RETAIL"),
	"CURRENCY_CODE"	=> "\$",
	"SAVE_IN_SESSION"	=> "N",
	"FILTER_NAME"		=> "arrFilter",
	"LIST_HEIGHT"		=> "5",
	"TEXT_WIDTH"		=> "24",
	"NUMBER_WIDTH"	=> "5",
	"CACHE_TIME"		=> "3600",
	));
?>

Описание параметров

IBLOCK_TYPE
Тип инфоблока.

IBLOCK_ID
ID инфоблока.

arrFIELD_CODE
Массив идентификаторов полей; доступны следующие значения:
  • ID – ID элемента (по данному полю будет доступна сложная логика);
  • NAME – заголовок элемента (по данному полю будет доступна сложная логика);
  • PREVIEW_TEXT – анонс элемента (по данному полю будет доступна сложная логика);
  • DETAIL_TEXT – детальное описание элемента (по данному полю будет доступна сложная логика);
  • SEARCHABLE_CONTENT – заголовок и описания элемента (по данному полю будет доступна сложная логика);
  • SECTION_ID – выпадающий список разделов инфоблока;
  • DATE_ACTIVE_FROM – интервал для даты активности "с";
  • DATE_ACTIVE_TO – интервал для даты активности "по".

arrPROPERTY_CODE
Массив символьных кодов свойств.

arrPRICE_CODE
Массив символьных кодов типов цен.

CURRENCY_CODE
Обозначение валюты для полей фильтра связанных с ценами.

SAVE_IN_SESSION
[Y|N] Y - сохранять значения полей фильтра в сессии.

FILTER_NAME
Имя выходного массива со значениями фильтра.

LIST_HEIGHT
Высота списков множественного выбора.

TEXT_WIDTH
Ширина однострочных текстовых полей ввода.

NUMBER_WIDTH
Ширина полей ввода для числовых интервалов.

CACHE_TIME
(сек.) время кеширования значений, выбираемых из базы и необходимых для построения формы фильтра.
