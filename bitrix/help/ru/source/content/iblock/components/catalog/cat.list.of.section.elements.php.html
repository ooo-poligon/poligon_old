Каталог::Элементы раздела в виде списка

Компонент предназначен для вывода элементов раздела в виде табличного списка. Над списком располагается дерево подразделов, позволяющее переходить в другие разделы. Как правило компонент используется на странице, отображающей содержимое того или иного раздела каталога.

Пример вызова

<?
$APPLICATION->IncludeFile("iblock/catalog/section_2.php", Array(
"IBLOCK_TYPE"		=> "catalog",
"IBLOCK_ID"		=> "22",
"SECTION_ID"		=> $_REQUEST["SECTION_ID"],
"PRICE_CODE"		=> "RETAIL",
"BASKET_URL"		=> "/personal/basket.php",
"PAGE_ELEMENT_COUNT"	=> "50",
"ELEMENT_SORT_FIELD"	=> "sort",
"ELEMENT_SORT_ORDER"	=> "asc",
"arrPROPERTY_CODE"         => array(),
"FILTER_NAME"		=> "arrFilter",
"CACHE_FILTER"		=> "N", 
"CACHE_TIME"		=> "3600",
"DISPLAY_PANEL"		=> "Y",
));
?>

Описание параметров

IBLOCK_TYPE
Тип инфоблока.

IBLOCK_ID
ID инфоблока.

SECTION_ID
ID раздела.

PRICE_CODE
Символьный код типа цены.

BASKET_URL
URL, ведущий на страницу с корзиной покупателя.

PAGE_ELEMENT_COUNT
Количество элементов на странице.

ELEMENT_SORT_FIELD
Сортировка может быть выполнена по:
  • shows – среднее количество просмотров элемента (популярность элемента);
  • sort – по индексу сортировки;
  • timestamp_x – по дате изменения;
  • name – по заголовку;
  • id – по ID элемента;
  • active_from – по дате активности "с";
  • active_to – по дате активности "по".

ELEMENT_SORT_ORDER
Сортировка может быть выполнена по:
  • asc – по возрастанию;
  • desc – по убыванию.

arrPROPERTY_CODE
Массив символьных кодов свойств раздела.

FILTER_NAME
Имя массива со значениями фильтрации элементов.

CACHE_FILTER
[Y|N] кешировать значения, выбираемые из базы (если установлен фильтр по ним).

CACHE_TIME
(сек.) время кеширования значений выбираемых из базы.

DISPLAY_PANEL
Добавлять в Административную панель кнопки для данного компонента.
