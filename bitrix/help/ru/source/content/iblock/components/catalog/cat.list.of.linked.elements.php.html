Каталог::Список связанных элементов

Компонент предназначен для вывода списка связанных элементов.

Пример вызова

<?
$APPLICATION->IncludeFile("iblock/catalog/link_element_list.php", Array(
"ELEMENT_ID"		=> $_REQUEST["ID"],
"LINK_IBLOCK_TYPE"		=> "catalog",
"LINK_IBLOCK_ID"           => "22",
"LINK_PROPERTY_SID"        => "THEMES",
"PAGE_LINK_ELEMENT_COUNT"	=> "50",
"LINK_ELEMENT_SORT_FIELD"	=> "sort",
"LINK_ELEMENT_SORT_ORDER"	=> "desc",
"arrPROPERTY_LINK_CODE"	=> array(),
"LINK_PRICE_CODE"		=> "RETAIL",
"CATALOG_URL"              => "/catalog/",
"SECTION_URL"              => "/catalog/accessory/section.php?",
"BASKET_URL"               => "/personal/basket.php"
"CACHE_TIME"               => "3600",
"DISPLAY_PANEL"            => "Y",
));
?>

Описание параметров

ELEMENT_ID
ID родительского элемента для связанных элементов.

LINK_IBLOCK_TYPE
Тип инфоблока связанных элементов.

LINK_IBLOCK_ID
ID инфоблока связанных элементов.

LINK_PROPERTY_SID
Символьный код свойства родительского элемента, в котором хранится связь.

PAGE_LINK_ELEMENT_COUNT
Количество связанных элементов на одной странице (для постраничного вывода).

LINK_ELEMENT_SORT_FIELD
Сортировка может быть выполнена по:
  • shows – среднее количество просмотров элемента (популярность элемента);
  • sort – по индексу сортировки;
  • timestamp_x – по дате изменения;
  • name – по заголовку;
  • id – по ID элемента;
  • active_from – по дате активности “с”;
  • active_to – по дате активности “по”.

LINK_ELEMENT_SORT_ORDER
Сортировка может быть выполнена по:
  • asc – по возрастанию;
  • desc – по убыванию.

arrPROPERTY_LINK_CODE
Массив выбираемых символьных кодов свойств для связанных элементов.

LINK_PRICE_CODE
Символьный код типа цены для связанных элементов.

CATALOG_URL
Начальная страница каталога.
SECTION_URL
Страница с разделами каталога связанных элементов.

BASKET_URL
URL, ведущий на страницу с корзиной покупателя.

CACHE_TIME
(сек.) время кеширования значений выбираемых из базы.

DISPLAY_PANEL
Добавлять в Административную панель кнопки для данного компонента.
