Каталог::Двухуровневая структура разделов (с подсчетом количества элементов в каждом разделе)

Компонент предназначен для вывода списка разделов и их подразделов, с подсчетом количества элементов в каждом. Как правило, используется на главной странице каталога.

Пример вызова

<?
$APPLICATION->IncludeFile("iblock/catalog/sections_top_2.php", Array(
"IBLOCK_TYPE"		=> "catalog",
"IBLOCK_ID"		=> "22",
"SECTION_SORT_FIELD"	=> "sort",
"SECTION_SORT_ORDER"	=> "asc",
"SECTION_URL"		=> "/catalog/accessory/section.php?",
"CACHE_TIME"		=> "3600",
"DISPLAY_PANEL"       	=> "Y",
));
?>

Описание параметров

IBLOCK_TYPE
Тип инфоблока.

IBLOCK_ID
ID инфоблока.

SECTION_SORT_FIELD
Сортировка может быть выполнена по:
  • sort – по значнию индекса сортировки;
  • timestamp_x – по дате модификации параметров раздела;
  • name – по загол,овку раздела;
  • id – по ID разделу;
  • depth_level – уровень вложенности раздела.

SECTION_SORT_ORDER
Сортировка может быть выполнена по:
  • asc – по возрастанию;
  • desc – по убыванию.

SECTION_URL
URL, ведущий на страницу с содержимым раздела.

CACHE_TIME
(сек.) время кеширования значений, выбираемых из базы.

DISPLAY_PANEL
(Y/N) Добавлять ли в административную панель кнопки для данного компонента.
