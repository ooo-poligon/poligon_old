Лента новостей

Компонент предназначен для показа списка новостей без разделения по информационным блокам. Возможна выборка из всех или нескольких информационных блоков.

Пример вызова

<?
$APPLICATION->IncludeFile("iblock/news/news_line.php", Array(
    "IBLOCK_TYPE"   => "news",
    "IBLOCK"        => Array("1"),
    "NEWS_COUNT"    => "20",
    "SORT_BY1"      => "ACTIVE_FROM",
    "SORT_ORDER1"   => "DESC",
    "SORT_BY2"      => "SORT",
    "SORT_ORDER2"   => "ASC",
    "CACHE_TIME"    => "0"
));
?>

Описание параметров

IBLOCK_TYPE
Тип информационного блока (используется только для проверки).

IBLOCK
Код информационного блока.

NEWS_COUNT
Количество новостей в странице.

SORT_BY1
Поле для первой сортировки новостей:
  • sort – по индексу сортировки;
  • timestamp_x – по дате изменения;
  • name – по заголовку;
  • active_from – по дате активности "с".

SORT_ORDER1
Направление для первой сортировки новостей:
  • asc – по возрастанию;
  • desc – по убыванию.

SORT_BY2
Поле для второй сортировки новостей:
  • sort – по индексу сортировки;
  • timestamp_x – по дате изменения;
  • name – по заголовку;
  • active_from – по дате активности "с".

SORT_ORDER2
Направление для второй сортировки новостей:
  • asc – по возрастанию;
  • desc – по убыванию.

CACHE_TIME
(сек.) время кеширования (0 – не кешировать).
