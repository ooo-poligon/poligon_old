Список новостей

Компонент предназначен для вывода списка новостей одного или нескольких информационных блоков.

Пример вызова

<?
$APPLICATION->IncludeFile("iblock/news/news.php", Array(
"IBLOCK_TYPE"	=> "news",
"ID"	=>	Array("1"),
"SECTION_ID"	=>	"220",
"NEWS_COUNT"	=> "10",
"SORT_BY1"	=> "ACTIVE_FROM",
"SORT_ORDER1"	=> "DESC",
"SORT_BY2"	=> "SORT",
"SORT_ORDER2"	=> "ASC",
"arrPROPERTY_CODE"	=> Array("SOURCE"),
"DETAIL_PAGE_URL"	=> "/content/news/index.php?news=#ID#",
"INCLUDE_IBLOCK_INTO_CHAIN"	=> "Y",
"FILTER"	=> "FILTER"
"CACHE_TIME"	=> "0"
"DISPLAY_PANEL"	=> "Y",
));
?>

Описание параметров

IBLOCK_TYPE
Тип информационного блока (используется только для проверки).

ID
Код информационного блока.

SECTION_ID
Код группы. Указывается, если выводятся элементы определнной группы. Если выводятся элементы всех групп, то параметр не определяется.

NEWS_COUNT
Количество новостей в странице.

SORT_BY1
Поле для первой сортировки новостей:
  • sort – по индексу сортировки;
  • timestamp_x – по дате изменения;
  • name – по заголовку;
  • active_from – по дате начала активности.

SORT_ORDER1
Направление для первой сортировки новостей:
  • asc – по возрастанию;
  • desc – по убыванию.

SORT_BY2
Поле для второй сортировки новостей:
  • sort – по индексу сортировки;
  • timestamp_x – по дате изменения;
  • name – по заголовку;
  • active_from – по дате начала активности.

SORT_ORDER2
Направление для второй сортировки новостей:
  • asc – по возрастанию;
  • desc – по убыванию.

arrPROPERTY_CODE
Указываются свойства, которые будут выводится для элементов.

DETAIL_PAGE_URL
URL страницы детального просмотра (по умолчанию - из настроек инфоблока).

INCLUDE_IBLOCK_INTO_CHAIN
[Y|N] Включать инфоблок/группу в цепочку навигации.

FILTER
Задается имя переменной, с помощью которой передается массив параметров из фильтра. Поле может быть оставлено пустым, тогда используется значение по умолчанию.

CACHE_TIME
(сек.) время кеширования значений, выбираемых из базы (0 – не кешировать).

DISPLAY_PANEL
[Y|N] Добавлять ли в Административную панель кнопки для данного компонента.
