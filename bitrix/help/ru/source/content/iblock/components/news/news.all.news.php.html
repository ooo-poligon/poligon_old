Все новости

Компонент предназначен для вывода списка всех новостей из всех информационных блоков одного типа с группировкой по информационным блокам.

Пример вызова

<?
$APPLICATION->IncludeFile("iblock/news/index.php", Array(
"IBLOCK_TYPE"       => "news",
"IBLOCK_SORT_BY"    => "SORT",
"IBLOCK_SORT_ORDER" => "ASC",
"NEWS_COUNT"        => "5",
"SORT_BY1"          => "ACTIVE_FROM",
"SORT_ORDER1"       => "DESC",
"SORT_BY2"          => "SORT",
"SORT_ORDER2"       => "ASC",
"CACHE_TIME"        => "0"
));
?>

Описание параметров

IBLOCK_TYPE
Тип информационного блока.

IBLOCK_SORT_BY
Поле для сортировки информационных блоков:
  • sort – по индексу сортировки;
  • id – по идентификатору;
  • name – по названию.
IBLOCK_SORT_ORDER
Направление для сортировки информационных блоков:
  • asc – по возрастанию;
  • desc – по убыванию.

NEWS_COUNT
Количество новостей в каждом блоке.

SORT_BY1
Поле для первой сортировки новостей:
  • sort – по индексу сортировки;
  • timestamp_x – по дате изменения;
  • name – по заголовку;
  • active_from – по дате активности "с".

SORT_ORDER1
Направление для первой сортировки новостей:
  • asc – по возрастанию;
  • desc – по убыванию.

SORT_BY2
Поле для второй сортировки новостей:
  • sort – по индексу сортировки;
  • timestamp_x – по дате изменения;
  • name – по заголовку;
  • active_from – по дате активности "с".

SORT_ORDER2
Направление для второй сортировки новостей:
  • asc – по возрастанию;
  • desc – по убыванию.

CACHE_TIME
(сек.) время кеширования (0 – не кешировать).
