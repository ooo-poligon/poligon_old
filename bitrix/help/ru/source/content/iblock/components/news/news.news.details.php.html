Новость детально

Компонент предназначен для показа новости детально.

Пример вызова

<?
$APPLICATION->IncludeFile("iblock/news/detail.php", Array(
    "ID"			=> $_REQUEST["ID"],
    "IBLOCK_TYPE"		=> "news",
    "IBLOCK_ID"			=> "33",
    "arrPROPERTY_CODE"		=> Array("SOURCE"),
    "META_KEYWORDS"		=> "KEYWORDS",
    "META_DESCRIPTION"		=> "DESCRIPTION",
    "LIST_PAGE_URL"		=> "/content/news/index.php",
    "INCLUDE_IBLOCK_INTO_CHAIN"	=> "Y",
    "CACHE_TIME"		=> "3600"
    "DISPLAY_PANEL"		=> "Y"
    ));
?>

Описание параметров

ID
ID элемента.

IBLOCK_TYPE
Тип информационного блока (используется только для проверки).

IBLOCK_ID
Код информационного блока.

arrPROPERTY_CODE
Свойства, значения которых будут выведены для элемента.

META_KEYWORDS
Ключевые слова страницы будут установлены из указанного здесь свойства.

META_DESCRIPTION
Описание страницы будет установлено из указанного здесь свойства.

LIST_PAGE_URL
URL страницы просмотра списка элементов (по умолчанию - из настроек инфоблока).

INCLUDE_IBLOCK_INTO_CHAIN
[Y|N] Включение инфоблока в цепочку навигации.

CACHE_TIME
(сек.) время кеширования (0 – не кешировать).

DISPLAY_PANEL
[Y|N] Добавлять ли в Административную панель кнопки для данного компонента.
