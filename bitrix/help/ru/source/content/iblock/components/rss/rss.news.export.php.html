RSS новости (экспорт)

Данный компонент предназначен для создания страницы, которая отдает выбранные новости вашего сайта в формате RSS.

Пример вызова

<?
$APPLICATION->IncludeFile("iblock/rss/rss_out.php", Array(
"ID"         => "1",
"NUM_NEWS"   => "20",
"NUM_DAYS"   => "30",
"YANDEX"     => "N",
"CACHE_TIME" => "600"
));
?>

Описание параметров

ID
ID информационного блока, новости которого экспортируются.

NUM_NEWS
Количество новостей для экспорта.

NUM_DAYS
Количество дней для экспорта.

YANDEX
Экспортировать в диалект Яндекса.

CACHE_TIME
(сек.) время кеширования (0 – не кэшировать).
