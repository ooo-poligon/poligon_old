    • Эта форма - основное средство для редактирования и добавления элементов информационных блоков вручную. Например, с ее помощью можно добавлять новые товары в торговый каталог, публиковать новости и т.п. Для добавления элемента инфоблока перейдите в раздел, в котором нужно создать элемент. Нажмите не кнопку Добавить элемент на Контекстной панели. откроется форма создания элемента.

    Контекстная панель

    КнопкаОписание
    Список элементов Переход на страницу со списком элементов данного инфоблока.
    Добавить новый элемент Переход на страницу с формой создания нового элемента инфоблока. Отображается при редактировании уже существующего элемента.
    УдалитьУдаление редактируемого элемента инфоблока.

    Форма редактирования

    Число закладок формы, их вид, название и поля на них могут меняться в зависимости от того инфоблок какого типа используется. Ниже приведено самое общее описание формы.

    Форма состоит из шести закладок:

    • Элемент
    • Анонс
    • Подробно
    • Группы
    • Дополнительно
    • Документооборот

    Закладка "Элемент"
    ПолеОписание
    Запись активнаПризнак активности элемента. Активный элемент доступен для отображения в публичной части сайта.
    Период активностиПериод активности элемента, по истечении которого элемент станет неактивным (т.е. недоступным для показа в публичной части сайта):
    • Если конечная дата отсутствует, элемент признается бессрочно активным.
    • Если отсутствует начальная дата, то элемент считается активным с момента его сохранения.
    *НазваниеНазвание элемента, как оно отображается в публичной части.
    Значения свойств
    Источник Источник информации
    Ключевые слова
    		Ключевые слова для создаваемого элемента
    ОписаниеПроизвольное описание элемента
    Торговый каталог
    Секция отображается, если в настройках модуля Торговый каталог для данного информационного блока была установлена опция "Является торговым каталогом"
    Вкладка Цены
    Вкладка служит для задания стоимости товара для разных типов цен.
    ПолеОписание
    *Базовая цена (Базовая)Базовая цена элемента (товара).
    *Валюта базовой ценыВалюта, в которой указана базовая цена.
    Тип цены Перечень доступных типов цен товара.
    Тип наценки Поле для выбора типа наценки, используемой при формировании стоимости товара для данного типа цены.
    Сумма Стоимость товара с учётом выбранного типа наценки. Стоимость товара рассчитывается по формуле [Цена] = [Базовая цена] + [Базовая цена] * [Процент наценки] / 100.
    Если тип наценки не выбран, то стоимость товара для данного типа цены можно ввести вручную.
    Валюта Наименование валюты, в которой указана стоимость товара для данного типа цены.
    Вкладка Цены (расш.)
    Вкладка служит для задания стоимости товара для разных типов цен с указанием дополнительных параметров, влияющих на стоимость.
    ПолеОписание
    Тип цены Перечень доступных типов цен товара.
    ОтПоля служат для задания связи между количеством приобретаемых товаров и их стоимостью.
    Например, можно указать, что при приобретении от 1 до 5 единиц товара, товар будет продаваться по цене 500 руб./шт, а при приобретении от 6 до 10 единиц товара, стоимость товара будет равна 400 руб./шт.
    Количество товара может быть задано как одним числом, так и в виде диапазона.
    До
    Тип наценки Поле для выбора типа наценки, используемой при формировании стоимости товара для данного типа цены.
    Выбор наценки для базового типа цены не производится.
    Цена за единицу Стоимость единицы товара для данного типа цены, при условии приобретения указанного товаров.
    Валюта Наименование валюты, в которой указана стоимость товара для данного типа цены.

    Кнопка "Ещё" позволяет добавить новую стpоку для указания стоимости товаров и условий ее применения.
    Вкладка Параметры
    На вкладке указываются дополнительные сведения о продаваемом товаре, а также (если каталог настроен на продажу контента) параметры подписки на контент сайта.
    ПолеОписание
    Количество на складе Количество товаров на складе в данный момент. Если установлена опция Уменьшать количество при заказе, это значение вручную изменить нельзя.
    Уменьшать при заказе Если опция установлена, при каждом заказе количество товаров, хранимое в базе, будет соответственно уменьшаться.
    Вес Вес единицы товара в граммах.
    Параметры подписки
    Тип оплаты Задается тип оплаты приобретения контента сайта. Доступно три типа оплаты:
    • одноразовая - означает, что товар приобретается покупателем один раз. В дальнейшем никаких движений по приобретенному товару не предусматривается;
    • регулярная (или "подписка") - означает, что по истечении заданного периода времени покупателю будет выставлен новый счет на продление использования товара;
    • пробный период ("триал") - предусматривает приобретение пробной версии товара (например, пробного доступа). По истечении указанного периода покупателю будет выставлен счет на покупку полной версии товара.
    Длина периода оплаты Указывается длительность периода времени, по истечении которого системой будет выставлен новый счет на приобретение товара. Значение поля задается для не одноразовых типов оплаты (регулярная и пробный период).
    Единица времени периода оплаты Определяются единицы времени, используемые для задания продолжительности периода оплаты (месяц, сутки, час и т.д.).
    Пробная для В поле для товара с типом оплаты пробный период указывается полная версия товара. По истечении пробного периода пользователю будет предложено приобрести указанный товар (полную версию).
    Продление без оформления заказа Опция доступна для типа оплаты регулярная. Если она выбрана, то при продлении товара (пользования товаром) будет рассчитана сумма заказа. Но сам заказ не будет сформирован в базе системы. Опция может быть полезна при проведении небольших регулярных списаний с внутреннего счета покупателя.
    Вкладка Группы
    Вкладка Группы отображается для элементов торгового каталога, настроенного на режим продажи контента. Служит для управления продажей прав доступа к контенту сайта.
    ПолеОписание
    ВключитьПоле для выбора группы пользователей, к которой будет осуществляться привязка покупателей данного товара.
    Группа пользователей Название группы пользователей, к которой будет осуществляться привязка покупателей данного товара.
    Длительность приписывания Длительность привязки покупателей к выбранной группе. По истечении указанного периода покупатель будет удален из списка пользователей данной группы.
    Если значение поля равно 0, то покупатели приписываются к группе на неограниченный период.
    Вкладка Скидки
    На вкладке Скидки вы можете выбрать виды скидок, которые будут применены к данному товару.
    ПолеОписание
    ID Идентификатор скидки.
    Сайт Идентификатор сайта, на товары которого распространяется действие данной скидки.
    Активность Признак активности скидки. Если скидка не является активной, она не может быть применена.
    НазваниеНазвание скидки.
    Величина Размер скидки.
    Действие Доступные действия над скидками. С помощью ссылки Изменить вы можете перейти к редактированию параметров скидки.

    * Поля, обязательные для заполнения.

    Закладка "Анонс"

    Данная форма служит для задания краткого описание элемента. Созданное описание может использоваться, например, для характеристики каждого элемента при показе списка элементов в публичной части сайта.

    Описание для анонса
    ИзображениеИзображение малого размера, ассоциируемое с данным элементом для предпросмотра.
    Text / HTMLВыбор формата создания и вывода краткого описания элемента инфоблока.
    Использовать визуальный редакторЕсли эта опция включена, при использовании формата HTML отображаются панели управления, содержащая опции форматирования текста.
    Закладка "Подробно"

    Форма служит для создания подробного описания элемента инфоблока. Например, полного текста новости, детального описания товара и т.д.

    ПолеОписание
    ИзображениеКрупноформатное изображение, ассоциируемое с данным элементом.
    Text / HTMLВыбор формата создания и вывода полного описания элемента инфоблока.
    Использовать визуальный редакторЕсли эта опция включена, при использовании формата HTML отображается панель управления, содержащая опции форматирования текста.
    Закладка "Разделы"

    Данная форма служит для выбора одного или нескольких разделов инфоблока, к которым относится элемент.

    ПолеОписание
    РазделыРазделы, к которым привязан элемент. Поле не отображается, если для данного типа информационного блока в настройках модуля снят флажок "Содержит разделы" либо в инфоблоке не создан ни один раздел.
    Закладка "Дополнительно"
    ПолеОписание
    Индекс сортировкиВес сортировки элемента. Чем меньше вес сортировки, тем выше отображается элемент.
    Символьный кодМнемонический код элемента.
    ТегиСлова или словосочетания, с помощью которых формируется поиск по тегам.
    Закладка "Документооборот"
    ПолеОписание
    СозданаДата и время создания записи элемента, а также имя пользователя, создавшего элемент.
    ИзмененаДата и время последнего изменения  записи элемента, а также имя пользователя, изменившего элемент.
    ЗаблокированаДата и время блокирования элемента и имя пользователя, заблокировавшего элемент.
    СтатусТекущий статус элемента. Создание и настройка параметров статусов выполняется в модуле документооборота.
    Комментарии
    Произвольный комментарий к статусу документа (например, описание причины изменения статуса).

    Кнопки управления

    КнопкаОписание
    СохранитьСохранение внесённых изменений. Возврат к списку элементов текущего информационного блока.
    ПрименитьПрименение внесённых изменений. Продолжение редактирования элемента.
    ОтменитьОтмена внесённых изменений. Возврат к списку элементов текущего информационного блока.
