Разделы информационного блока

В этом разделе

  • Форма поиска
  • Контекстная панель
  • Список разделов

    • Если при создании типа информационного блока вы указали, что инфоблоки данного типа имеют иерархическую структуру, то помимо конкретных элементов (например, товаров), каждый инфоблок также может содержать разделы и подразделы.

    Например, информационный блок Торговый каталог может содержать раздел Компьютерная техника, который, в свою очередь, может включать подразделы Принтеры и Сканеры.

    С помощью данной формы вы можете управлять элементами иерархии (разделами и подразделами) выбранного информационного блока.

    Форма поиска

    С помощью данной формы вы можете выполнить поиск разделов инфоблока, соответствующих указанным параметрам.

    ПолеОписание
    НазваниеНазвание раздела. Допустимо использование операторов сложного поиска.
    Раздел-родительВышележащий (родительский) раздел.
    ID раздела
    		Идентификатор раздела.
    Дата изменения Дата последнего изменения параметров раздела.
    Символьный код Мнемонический код раздела.
    Внешний код Внешний код раздела, используемый для связи раздела с внешними источниками данных.
    Кто изменилАвтор изменений
    Дата созданияДата создания раздела
    Кто создалАвтор раздела
    Статус в документооборотеСтатус (если имеется)
    АктивностьПозволяет отобрать активные или неактивные разделы информационного блока.
    ОписаниеПоиск по описанию
    Автор(ы)Авторы блоков в разделе
    Показать все условияОдновременный показ всех условий
    Скрыть все условияОдновременное сокрытие всех условий.

    Чтобы отобрать разделы информационного блока по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех разделов нажмите кнопку Отменить.

    Контекстная панель

    КнопкаОписание
    Список элементов Переход к списку элементов информационного блока. Кнопка доступна при снятом флаге Совместный просмотр секций и элементов в настройках модуля Информационные блоки.
    Добавить подраздел Переход к форме создания нового раздела или подраздела для данного информационного блока.
    Добавить элемент Переход к форме создания нового элемента для данного информационного блока.
    Свойства инфоблока Переход к форме управления свойствами информационного блока, список разделов которого представлен на странице.
    НастроитьПозволяет перейти к диалогу настройки внешнего вида отчетной формы.
    ExcelЭкспортирует данные из отображаемой таблицы в формат MS Excel.

    Список разделов

    ПолеОписание
    Колонка флажковПоле предназначено для выбора разделов, к которым предполагается применить какое-либо правило или действие.
    Действия Доступные действия:
    • Ссылка с названием раздела - переход к списку подразделов данного раздела;
    • Ссылка с названием элементов - переход к списку элементов данного раздела;
    • Изменить - редактирование раздела;
    • Удалить - удаление раздела.
    НазваниеНазвание раздела.
    Акт.Признак активности раздела. Если поле отмечено, информация раздела доступна для отображения в публичной части сайта.
    Сорт.Вес сортировки раздела. Определяет последовательность, в которой будут показаны разделы в публичной части сайта.
    ЭлементовКоличество элементов в данном разделе. Ссылка "+" позволяет добавить новый элемент в данный раздел.
    ПодразделовКоличество подразделов в данном разделе. Ссылка "+" позволяет добавить новый подраздел в данный раздел.
    Дата изм.Дата последнего изменения раздела.
    IDID раздела.
    СтатусСтатус раздела, если есть
    БлокировкаСтатус блокировки
