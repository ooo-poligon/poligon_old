Описание модуля

Торговый каталог - модуль системы "1С-Битрикс: Управление сайтом", позволяющий:

  • создавать торговые каталоги товаров, услуг и всевозможного цифрового контента;
  • осуществлять продажу товаров, услуг и цифрового контента через интернет-магазин;
  • осуществлять продажу прав доступа к ресурсам сайта;
  • управлять ценами на товары и услуги торговых каталогов;
  • загружать и выгружать данные о товарах, заказах, продажах в форматах CommerceML, Froogle, Yandex и CSV (формат последнего можно настраивать при загрузке);
  • загружать и выгружать данные в любой из форматов в соответствии с созданными пользователем профилями.

Торговый каталог создаётся путём соответствующей настройки Каталога товаров.

Важно!
Каталоги товаров формируются на основе информационных блоков, поэтому для использования модуля Торгового каталога необходимо наличие установленного в системе модуля Информационных блоков.

Настройка параметров и свойств Торгового каталога осуществляется на страницах раздела Торговый каталог (Административный раздел сайта -> Магазин -> Торговый каталог), а также на странице настроек модулей (Настройки -> Настройки продукта -> Торговый каталог).

Для того чтобы часть информационных блоков начала работать в режиме торговых каталогов, необходимо выбрать эти информационные блоки в поле Является торговым каталогом на странице настроек модуля Торговый каталог (пункт меню Настройки -> закладка Торговый каталог), закладка Каталоги. Управление ценами товаров Торгового каталога осуществляется с помощью специальных форм, расположенных на страницах настройки информационных блоков (модуль Информационные блоки).

Важно! Для корректной работы модуля Торговый каталог необходимо наличие установленного модуля Валюты.

Если вы хотите, чтобы в вашем интернет-магазине осуществлялась продажа цифрового контента, представленного в инфоблоках системы, отметьте соответствующие информационные блоки в поле Продажа контента.

На странице настроек модуля Торговый каталог вы также можете:

  • указать информационные блоки, данные из которых будут выгружаться в XML-файлы для системы Яндекс.Товары (см. статью Яндекс.Товары);
  • выбрать группы пользователей, права доступа к ресурсам сайта которых будут доступны для продажи.

Основные возможности модуля торгового каталога:

Многовалютность
Система позволяет гибко управлять валютами цены. Список доступных валют определяется администратором сайта. Для каждой валюты устанавливаются курсы (с возможностью загрузки с сайта Центрального Банка РФ), а также формат вывода для каждого сайта.
Типы цен
К одному товару можно привязать несколько цен разных типов. К каждому типу цены можно ограничить доступ на просмотр и на покупку для разных групп посетителей. Например, можно ввести розничный тип цен, который будет доступен для просмотра и покупки всем посетителям сайта, и оптовый тип цен, который будет доступен для просмотра всем посетителям, но покупать товары по которому смогут только члены группы "Оптовые покупатели".
Система наценок
Один из типов цен торгового каталога называется базовым. Все остальные цены товара могут рассчитываться в процентном отношении к базовой цене, либо задаваться вручную. Для цен, рассчитываемых как наценка к базовой цене, возможен автоматический пересчет.
Загрузка данных в формате CommerceML
Модуль позволяет загружать каталог товаров и цены в формате CommerceML. CommerceML - это единый стандарт обмена коммерческой информацией в формате XML, поддерживаемый программой 1C. Стандарт учитывает отечественную специфику и включает несколько универсальных решений, необходимых для российских Интернет-компаний и торговых организаций.
Загрузка и выгрузка данных в CSV файл
Модуль позволяет загружать и выгружать каталог товаров и цены в CSV файл.
Связь с модулем Интернет-магазин
Пользователь, совершающий покупку, помещает товар в корзину по самой выгодной из доступных для него цен на данный товар.
