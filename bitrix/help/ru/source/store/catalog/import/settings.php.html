    • Система 1С-Битрикс: Управление сайтом предусматривает наполнение и изменение торговых каталогов сайта с помощью загрузки данных из внешних файлов. Для настройки параметров импорта служит форма Настройка импорта: Магазин -> Торговый каталог -> Настройка импорта.

    Вверху страницы отображен путь к каталогу, где находятся скрипты импорта. По умолчанию /bitrix/php_interface/include/catalog_import/.

    Поле Описание
    [1] Действия над профилем импорта Операции, которые могут быть применены к профилю:
    • Импортировать - запуск профиля импорта;
    • Добавить в меню - добавления ссылки на профиль импорта в левое меню административной части сайта;
    • Создать агента - создание PHP функции-агента для автоматического импорта с использованием данного профиля;
    • Повесить на cron - привязка профиля к системной утилите cron для автоматического импорта;
    • Удалить профиль - удаление профиля.
    [2] Название Название типа профиля импорта, например, CommerceML.

    Примечание
    Для импорта данных из файлов формата CSV доступно два типа профиля:

    Import CSV(new) - данный формат импорта доступен при установленном в системе модуле Торговый каталог версии 4.0.5 и старше. Этот формат позволяет загружать данные с учётом группировки цен товаров, в зависимости от количества приобретаемых товаров, а также снимает ограничение с количества объединяемых таблиц, содержащих информацию о товарах. Кроме того, данный формат позволяет загружать цены товара в валюте, указанной для каждой цены.

    Import CSV- данный формат загрузки используется в системах с версией модуля Торговый каталог ниже 4.0.5. Этот формат накладывает ограничение на количество объединяемых таблиц с информацией о товарах (не более 30), а также не позволяет осуществлять загрузку групп цен товаров. Кроме того, этот формат позволяет загружать цены товара только в одной, единой для всех цен товара, валюте.
    По умолчанию данный тип импорта не доступент. Если вы хотите импортировать данные в формате Import CSV, то вам необходимо добавить соответствующий пункт в административное меню системы вручную.
    [3] Файл Имя файла - PHP-скрипта, осуществляющего импорт.
    [4] Профиль Название профиля импорта.

    Профиль "по умолчанию" каждый раз перед началом операции предлагает указать параметры импорта. При использовании пользовательского профиля, импорт осуществляется автоматически в соответствии с параметрами, указанными при создании профиля.
    [5] В меню Указывает, помещен ли данный профиль для быстрого доступа в левое меню административной части. Поместить профиль в левое меню можно при помощи действия Добавить в меню в колонке Действия.
    [6] В агентах Указывает, существует ли агент, осуществляющий в случае необходимости (по истечении определённого интервала времени) автоматическую загрузку элементов каталога. Создать агента для профиля можно при помощи действия Создать агента в колонке Действия.
    [7] В cron'е Указывает, привязан ли профиль к утилите cron для автоматической периодической загрузки данных. Привязка профиля к утилите cron осуществляется при помощи действия Привязать к cron в колонке Действия.
    [8] Использован Дата и время последнего импорта с использованием данного профиля.
