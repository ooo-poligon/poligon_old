Выгрузка каталога из 1С на сайт

Для выгрузки каталога из «1С» на сайт необходимо выполнить следующие действия.

  1. Откройте диалоговое окно Выгрузка пакета коммерческих предложений (меню Сервис -> Обмен данными в формате CommerceML -> Выгрузка пакета коммерческих предложений).
  2. Введите срок действия ваших коммерческих предложений.
  3. Выберите в поле По каталогу каталог, который хотите выгрузить на сайт.
  4. Установите флаг Выгружать каталог товаров.
  5. Укажите, какие типы цен должны быть выгружены на сервер.
  6. Если вы хотите выгрузить только некоторые товары каталога, выберите их с помощью кнопки "Подбор номенклатуры".
  7. Если вы хотите выгружать на сервер непосредственно из 1С, установите флаг Отправить на сайт и выберите сайт, на который хотите загрузить каталог.
  8. Если вы хотите выгрузить каталог через файл, то укажите имя файла, в котором будет сохранен ваш каталог.
  9. Нажмите кнопку "Выгрузить".
