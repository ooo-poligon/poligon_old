Компоненты::Таблица цен на товар

Компонент показывает цены указанного товара с учётом прав на доступ текущего покупателя.

Синтаксис вызова


$APPLICATION->IncludeFile("catalog/price_table.php", 
    Array(
    "PRODUCT_ID" => $GLOBALS["ID"],
    // Код товара
    "PRICE_TYPE_OLD" => "1",
    // Тип "старой" цены
    "PRICE_TYPE_NEW" => "7",
    // Тип "новой" цены
    "BASKET_PAGE" => "basket.php",
    // Страница корзины
    "CACHE_TIME" => "600",
    // Время кэширования данных (секунд)
    )
);

Параметры

PRODUCT_ID
Код товара, цены которого нужно показать.
PRICE_TYPE_OLD
Тип "старой" цены. Требуется для организации представления типа "старая цена: 100 р., новая цена: 90 р., вы экономите: 10 р.".
PRICE_TYPE_NEW
Тип "новой" цены.
BASKET_PAGE
Скрипт корзины.
CACHE_TIME
Период (в секундах), на который кешируется результат. Установите это поле в ноль для отключения кеширования.
