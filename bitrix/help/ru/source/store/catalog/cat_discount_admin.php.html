    На странице Скидки и купоны (Магазин -> Торговый каталог -> Скидки) представлена таблица, содержащая список всех доступных в системе скидок, используемых для приобретения товаров по льготной цене. Данная страница служит для добавления новых скидок, а также для перехода к управлению их параметрами.

    Форма поиска

    С помощью данной формы вы можете выполнить поиск скидок в соответствии с указанными условиями.

    Поле Описание
    Сайт Указывается сайт, к товарам которого могут быть применены выбираемые скидки.
    Активность Признак активности скидки или купона. Если скидка (купон) активна, то она может быть использована при покупке товара.
    Активна в период В данных полях можно указать период активности скидки или купона. Период активности может быть задан одним из следующих способов:
    • с помощью указания количества дней, прошедших со дня присвоения скидке или купону статуса Активна;
    • с помощью задания дат начала и конца периода активности.
    Название Название скидки.
    Купон Код купона.
    На продление Указывается, какие скидки будут участвовать в поиске:
    • да - в поиске будут участвовать купоны и скидки, используемые при продлении подписки на контент сайта;
    • нет - в поиске не будут участвовать купоны и скидки, используемые при продлении подписки на контент сайта;
    • все- в поиске будут участвовать все купоны и скидки, в т.ч. используемые для продления подписки.

    Для того чтобы отобрать скидки и купоны по заданным критериям поиска, нажмите кнопку "Найти". Для отображения всех скидок и купонов нажмите кнопку "Отменить".

    Контекстная панель

    Кнопка Описание
    Новая скидка Переход к форме добавления новой скидки.
    Excel Экспортирует данные из отображаемой таблицы в формат MS Excel.
    Настроить Позволяет перейти к диалогу настройки внешнего вида отчетной формы.

    Список скидок

    Поле Описание
    Колонка флажков Поле предназначено для выбора скидок, к которым предполагается применить какое-либо правило или действие.
    Действия Действия над скидкой:
    • Изменить параметры скидки - редактирование купона или скидки;
    • Удалить скидку - удаление купона или скидки.
    ID Идентификатор скидки.
    Сайт Название сайта(ов), к товарам которого может быть применена данная скидка или купон.
    Активна с Дата начала активности скидки или купона.
    по Дата окончания активности скидки или купона.
    Акт. Признак активности скидки или купона. Если скидка неактивна, то она не может быть применена.
    Название Название скидки.
    Сорт. Относительный "вес" скидки. Используется для вывода скидки в общем списке.
    Величина Размер предоставляемой скидки.
    Продл. Признак использования скидки для продления подписки на контент сайта.
