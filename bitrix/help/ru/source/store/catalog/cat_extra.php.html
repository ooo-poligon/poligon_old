    • Форма Типы наценок (Магазин -> Торговый каталог -> Наценки) служит для управления наценками и скидками в системе (далее просто наценка). 

    Наценка - заранее установленная в относительном выражении (например, в процентах) величина, на которую уменьшается или увеличивается цена товара для конкретного покупателя или группы покупателей.

    Типом наценки называется поименованная величина наценки (напр., Оптовая скидка). Имя вводится для упрощёния управления этими величинами.

    Если вы часто устанавливаете одни и те же проценты наценок для ваших товаров, введите эти проценты как типы наценок. В этом случае при выборе типа наценки для товаров система автоматически рассчитает цену на данные товары, прибавив сумму наценки к базовой цене товара. При изменении величины наценки все цены, сформированные с использованием данной наценки, могут быть автоматически пересчитаны системой.

    Список типов наценок

    Поле Описание
    Колонка флажков Поле предназначено для выбора наценок, к которым предполагается применить какое-либо правило или действие.
    Действия Действия над наценкой:
    • Изменить параметры наценки- редактирование наценки;
    • Удалить наценку - удаление наценки.
    ID Идентификатор типа наценки в базе данных.
    Название наценки Произвольное название типа наценки. Для удобства работы рекомендуется задавать названия, отражающие смысл создаваемой наценки.
    Процент Процент наценки от базовой цены товара. Может задаваться как положительным (наценка), так и отрицательным (скидка) числом.
    Пересчитать цены Указывается, будут ли цены, сформированные с использованием данной наценки, пересчитаны автоматически при изменении величины наценки.
