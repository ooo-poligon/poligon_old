Ручной экспорт каталогов в Yandex Simple (профиль по умолчанию)

Для выгрузки данных в форматах Yandex Simple с использованием профиля по умолчанию выберите ссылку Экспортировать напротив соответствующего профиля по умолчанию.

Для выгрузки каталога выполните следующий действия:

  1. После открытия пошагового мастера выберите торговые каталоги для экспорта;
  2. Укажите имя файла для размещения PHP-скрипта, при обращении к которому система сгенерирует XML файл и передаст его запрашивающему узлу;
  3. Нажмите на кнопку Экспортировать для начала экспорта.
