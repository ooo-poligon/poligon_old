Ручной экспорт каталогов в CSV (профиль по умолчанию)

Выгрузка данных в CSV формате позволяет организовать обмен данными между сайтом и другими программами (например, Microsoft Excel).

Для выгрузки каталога в файл CSV формата выполните следующие действия:

  1. Выберите действие Экспортировать в меню действий соответствующего профиля по умолчанию;
  2. Выберите информационный блок для выгрузки и нажмите на кнопку Далее для перехода на второй шаг;
  3. Задайте свойства формата файла: какой разделитель полей использовать при формировании файла;
  4. Задайте соответствие полей в файле полям в базе. Для этого каждому полю в базе, которое вы хотите видеть в CSV-файле, поставьте в соответствие его порядковый номер в CSV-файле. Если поле не должно присутствовать в CSV-файле, то вам следует снять соответствующий флаг Выгружать;
  5. Введите имя CSV-файла (с полным путём относительно корня сайта), в который будет выгружен каталог. Если такого файла не существует, то он будет создан. Если такой файл существует, то он будет перезаписан;
  6. Нажмите на кнопку Начать выгрузку для начала экспорта.
