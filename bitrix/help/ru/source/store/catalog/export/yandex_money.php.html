Автоматический экспорт каталогов в Яндекс.Товары

Модуль торгового каталога имеет возможность выгрузки каталогов товаров в XML файл, подходящий для размещения в системе Яндекс.Товары.

Укажите в настройках модуля Торговый каталог каталоги, которые необходимо выгружать в XML файл. XML файл будет пересоздаваться с периодичностью, указанной в переменной Период экспорта в Яндекс.Товары. Готовый файл будет располагаться по адресу http://ваш_сайт/upload/yandex.php (вместо подстроки ваш_сайт должен быть указан реальный адрес сайта, подстрока upload берется из переменной Папка по умолчанию для загрузки файлов настроек Главного модуля).

Готовый файл http://ваш_сайт/upload/yandex.php может принимать дополнительные параметры referer1 и referer2, которые служат для формирования ссылок на товары с учётом рекламных компаний. Например, если запросить адрес

http://ваш_сайт/upload/yandex.php?referer1=wizard&referer2=soft

то в выдаваемом XML файле ссылки на страницу с детальной информацией о товаре будут иметь вид

http://ваш_сайт/catalog/detail.php?ID=8192&r1=wizard&r2=soft

Если параметр referer1 не задан, то он принимает значение "yandext". Например,

http://ваш_сайт/catalog/detail.php?ID=8192&r1=yandext&r2=
