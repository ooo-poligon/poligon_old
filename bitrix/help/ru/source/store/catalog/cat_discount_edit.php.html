    Данная форма служит для создания новой или редактирования существующей скидки.

    Примечание.
    Купон представляет собой разновидность скидки. Его отличие заключается в том, что купон - это уникальная последовательность символов, случайно сгенерированная. Правом на получение скидки по купону обладает покупатель, которому каким-либо образом была сообщена данная уникальная последовательность символов.

    Контекстная панель

    Кнопка Описание
    Список скидок Переход на страницу, содержащую полный список скидок и купонов.
    Купоны этой скидки Переход к форме купонов на скидки с установленным фильтром по названию скидки.
    Добавить новую скидку Переход к форме создания новой скидки или купона. Кнопка отображается при редактировании существующей скидки.
    Удалить скидку Удаление редактируемой скидки или купона.

    Форма редактирования

    Закладка "Скидка"

    Поле Описание
    ID Уникальный идентификатор скидки.
    Дата последнего изменения Дата и время последнего изменения параметров скидки или купона.
    Сайт Название сайта, к товарам которого применяется скидка.
    Активность Признак активности скидки.
    Период активности С помощью on-line календаря вы можете указать дату начала и окончания действия скидки.
    *Название Название скидки.
    Индекс сортировки Относительный "вес" скидки. Используется для вывода элемента в общем списке.
    *Тип скидки Форма задания скидки:
    • в процентах- скидка задается в процентах от стоимости товара;
    • в виде фиксированной суммы - будет указана сумма скидки.
    *Величина скидки Размер скидки. В зависимости от выбранного типа скидки, величина скидки указывается в процентах от стоимости товара или в виде фиксированной суммы.
    Валюта скидки Укажите валюту скидки, если она задана в виде фиксированной суммы.
    Максимальная сумма скидки Максимально возможный размер скидки. Чаще всего значение данного поля задается для типаскидки "в процентах". Максимальная сумма скидки указывается в основной валюте скидки, выбранной в предыдущем поле.
    Применяется к продлению подписки Флаг в данном поле означает, что скидка или купон будет применяться при продлении подписки на доступ к контенту сайта.
    Краткое описание Данное поле служит для задания краткого описания скидки. Размер описания может составлять не более 255 символов.
    * - Поля, обязательные для заполнения.

    Закладка "Ограничения"

    Поле Описание
    Купон Данное поле предназначено для выбора применяемого купона:
    • один многоразовый купон - создаётся один многоразовый купон. Автоматическая генерация кода осуществляется с помощью кнопки "Генерировать".
    • много купонов - создаются одноразовые купоны. В поле для автоматической генерации купонов указывается количество купонов для скидки. Переход к списку купонов для данной скидки осуществляется с помощью ссылки "Перейти к списку купонов этой скидки".
    Группы пользователей, которые могут воспользоваться скидкой В данном поле выбираются группы пользователей, имеющие право на использование данной скидки. Если ни одна группа не выбрана, то скидка будет доступна для пользователей всех групп.
    Типы цен, к которым применима скидка В данном поле выбираются типы цен товаров, к которым может быть применена данная скидка. Если ни один тип цен не выбран, то скидка может быть применена ко всем типам цен товаров.
    Разделы, к которым применима скидка С помощью данных полей указываются разделы и подразделы сайта, к товарам в которых может быть применена создаваемая скидка.
    • Для добавления нового раздела используйте кнопку "Добавить".
    • Для удаления выбранного раздела воспользуйтесь кнопкой "Удалить".
    Если ни один раздел не выбран, то подразумевается, что скидка может быть применена к товарам всех разделов.
    Товары, на которые дается скидка Данное поле предназначено для выбора товаров, при покупке которых может быть использована данная скидка. Для выбора товара служит кнопка "Добавить". Если ни один из товаров не выбран, то подразумевается, что скидка может быть применена ко всем товарам из раздела, выбранного в предыдущем поле.

    Кнопки управления

    Кнопка Описание
    Сохранить Сохранение внесённых изменений. Переход на страницу со списком скидок.
    Применить Применение внесённых изменений. Продолжение редактирования параметров скидки.
    Отменить Отмена внесённых изменений. Возврат первоначальных значений параметров.
