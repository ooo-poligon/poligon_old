    Форма Настройки торгового каталога предназначена для управления общими настройками модуля. Для удобства работы форма разделена на несколько закладок, на каждой из которых выполняется настройка определённой группы параметров модуля.

    Закладка "Экспорт/Импорт".

    Параметр Описание
    Тип информационного блока для загрузки в формате CommerceML (из 1С) Определяет, в каталог какого типа будут выгружены данные из 1C.
    Автоматически деактивировать товар при загрузке из ComerceML, если нет цены При включенной опции, если для товара, загружаемого из CML-файла, не задана ни одна цена, данный товар будет деактивирован.
    Период экспорта в Яндекс.Товары (часов) Указывается, с какой частотой данные каталогов, отмеченных в поле Экспорт в Яндекс.Товары таблицы Список информационных блоков, экспортируются в Яндекс.Товары. Подробнее об автоматическом экспорте см. Автоматический экспорт каталогов в Яндекс.Товары
    Товары, которых нет в файле данных CommerceML Определяется действие, выполняемое по умолчанию над товарами, которые уже есть в базе, но отсутствуют в загружаемом CML-файле.
    Например, если выбрано действие Удалять, то при обновлении торгового каталога из файла CommerceML, товары, отсутствующие в данном файле, будут удалены из торгового каталога.
    Экспорт/импорт из CSV
    Доступные поля товара Если в вашей базе для каждого товара создано большое количество полей, содержащих значения свойств и параметров товара, выбор конкретных полей при экспорте или импорте может оказаться затруднительным. Это поле определяет, какие поля из множества хранимых в базе будут доступны для выбора при экспорте.
    Доступные поля цены Определяется, какие поля с параметрами цены будут доступны для выбора при экспорте или импорте из CSV файла.
    Количество уровней групп Указывается, сколько уровней каталога будет доступно в файлах импорта / экспорта.
    Доступные поля групп Определяется, какие поля групп из множества хранимых в базе будут доступны для выбора при экспорте или импорте из CSV файла.
    Доступные валюты Валюты, доступные для экспорта и импорта.

    Закладка "Каталоги"

    Архитектура системы позволяет назначить статус торгового каталога любому информационному блоку, что определяет исключительную гибкость системы. В данной секции вы можете настроить любой из имеющихся информационных блоков на использование в качестве торгового каталога, а также выбрать информационные блоки для продажи контента и указать, данные из каких инфоблоков подлежат экспорту в Яндекс.Товары.

    Поле Описание
    Информационный блок Название информационного блока.
    Является торговым каталогом Установка флага в данное поле позволяет настроить информационный блок на использование в режиме торгового каталога.
    Продажа контента Установка флага в данное поле подразумевает возможность использования торгового каталога для продажи контента.
    Экспорт в Яндекс.Товары Если содержимое информационного блока подлежит экспорту в XML-файл для системы Яндекс.Товары, установите флаг в это поле.

    Закладка "Продажа прав"

    В данной форме вы можете указать группы пользователей, права доступа к ресурсам сайта которых могут продаваться. Например, вы можете осуществлять продажу прав на доступ к определённому разделу сайта, скачиванию файлов, управлению рекламой или любым другим ресурсам сайта и операциям над ними. При этом вы можете определить период времени, в течение которого пользователь может пользоваться предоставленными ему правами.

    В общем случае, для того чтобы организовать продажу прав доступа, необходимо выполнить следующую последовательность действий:

    • создать группу пользователей, наделенную необходимыми правами (Настройки -> Пользователи -> Группы пользователей);
    • разрешить продажу прав данной группе пользователей (Настройки -> Настройки продукта -> Настройки модулей -> Торговый каталог);
    • привязать пользователя, купившего право доступа, к соответствующей группе и задать период времени, на который осуществляется привязка (Настройки -> Пользователи -> Список пользователей).
    Поле Описание
    Группа пользователей Название группы пользователей.
    Разрешена продажа Установка флажка в данном поле позволяет осуществлять продажу прав доступа соответствующей группе пользователей.

    Закладка "Доступ"

    Эта форма позволяет установить уровень доступа к модулю по умолчанию для всех категорий пользователей сайта, а также дополнительно определить индивидуальный, отличный от заданного по умолчанию, уровень доступа для каждой группы пользователей в отдельности.

    Поле Описание
    По умолчанию Уровень доступа групп пользователей, для которых установлено право "по умолчанию".
    [Группа пользователей] Изменение настроек доступа для конкретной группы. Возможно назначение следующих прав доступа к модулю Торговый каталог:
    • закрыт- запрет на доступ;
    • просмотр всех данных модуля - разрешается только просмотр файлов;
    • запись- полный доступ к ресурсам модуля.

    Кнопки управления

    Кнопка Описание
    Сохранить Сохранение внесённых изменений.
    Сбросить Отмена внесённых изменений. Возврат первоначальных значений параметров.
    По умолчанию Установка значений свойств модуля равными значениям по умолчанию.
