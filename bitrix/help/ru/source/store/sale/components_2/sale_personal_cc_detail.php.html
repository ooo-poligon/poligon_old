Компоненты 2.0::Редактирование пластиковых карт

Одностраничный компонент.
Позволяет редактировать пластиковые карты пользователя.

Пример вызова

<?
$APPLICATION->IncludeComponent(
		"bitrix:sale.personal.cc.detail",
		"",
		Array(
			"PATH_TO_LIST" => "plastic_cards.php", 
			"SET_TITLE" => "Y" 
		)
);?>

Описание параметров

Дополнительные настройки
PATH_TO_LIST
Страница со списком пластиковых карт. Указывается путь к странице со списком пластиковых карт. Если страница находится в той же директории, что и данная, то достаточно указать название страницы. Такая страница может быть создана с помощью компонента Список пластиковых карт текущего пользователя.
SET_TITLE
[Y|N] В качестве заголовка страницы выводится «Добавление новой карты».
