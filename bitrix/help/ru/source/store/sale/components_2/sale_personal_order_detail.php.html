Компоненты 2.0::Подробная информация о заказе

Одностраничный компонент.
Выводит подробную информацию по заказу.

Пример вызова

<?
$APPLICATION->IncludeComponent(
	"bitrix:sale.personal.order.detail",
	"",
	Array(
		"PATH_TO_LIST" => "order_list.php", 
		"PATH_TO_CANCEL" => "order_cancel.php", 
		"PATH_TO_PAYMENT" => "payment.php", 
		"ID" => $ID, 
		"SET_TITLE" => "Y" 
	)
);?>

Описание параметров

Дополнительные настройки
PATH_TO_LIST
Страница со списком заказов. Такая страница может быть создана с помощью компонента Список заказов.
PATH_TO_CANCEL
Страница отмены заказа. Указывается путь к странице, где можно отменить заказ. Такая страница может быть создана с помощью компонента Отмена заказа. Необходимо передавать ID заказа в качестве параметра.
PATH_TO_PAYMENT
Страница подключения платежной системы. Указывается путь к странице подключения платежной системы. Такую страницу можно создать с помощью компонента Подключение платежной системы.
ID
Указывается параметр, в котором хранится ID заказа.
SET_TITLE
[Y|N] В качестве заголовка будет установлено «Мой заказ № #ID_заказа#».
