Компоненты 2.0::Редактирование профиля

Одностраничный компонент.
Позволяет редактировать профиль пользователя.

Пример вызова

<?
$APPLICATION->IncludeComponent(
		"bitrix:sale.personal.profile.detail",
		"",
		Array(
			"PATH_TO_LIST" => "profile_list.php", 
			"PATH_TO_DETAIL" => "profile_detail.php?ID=#ID#", 
			"ID" => $ID, 
			"SET_TITLE" => "Y" 
		)
);?>

Описание параметров

Дополнительные настройки
PATH_TO_LIST
Страница со списком профилей. Путь к странице со списком профилей. Если страница находится в текущей директории, то достаточно указать ее название. Страница может быть создана с помощью компонента Cписок профилей текущего пользователя.
PATH_TO_DETAIL
Страница редактирования профиля. Путь к странице редактирования профиля. В качестве параметра необходимо указать ID профиля. Если поле пусто, то пользователь после применения изменений останется на той же странице.
ID
Указывается параметр, в котором хранится ID профиля.
SET_TITLE
[Y|N] В качестве заголовка будет установлено «Профиль № #ID#».
