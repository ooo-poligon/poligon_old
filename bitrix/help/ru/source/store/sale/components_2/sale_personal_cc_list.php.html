Компоненты 2.0::Список пластиковых карт текущего пользователя

Одностраничный компонент.
Выводит cписок пластиковых карт текущего пользователя.

Пример вызова

<?
$APPLICATION->IncludeComponent(
		"bitrix:sale.personal.cc.list",
		"",
		Array(
			"PATH_TO_DETAIL" => "card_detail.php", 
			"PER_PAGE" => "20", 
			"SET_TITLE" => "Y" 
		)
);?>

Описание параметров

Дополнительные настройки
PATH_TO_DETAIL
Страница c подробной информацией о пластиковой карте. Указывается путь к странице с подробной информацией о карте. Если страница находится в той же директории, что и данная, то достаточно указать название страницы. Такая страница может быть создана с помощью компонента Редактирование пластиковых карт.
PER_PAGE
Количество карт, выводимых на страницу. Указанное количество определяет число карт на одной странице, остальные будут выведены с помощью постраничной навигации.
SET_TITLE
В качестве заголовка будет установлено «Мои пластиковые карты».
