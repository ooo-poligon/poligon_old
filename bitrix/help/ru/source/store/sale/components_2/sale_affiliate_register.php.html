Компоненты 2.0::Регистрация аффилиата

Одностраничный компонент.
Компонент служит для создания страницы регистрации аффилиата.

Пример вызова

<?
APPLICATION->IncludeComponent(
		"bitrix:sale.affiliate.register",
		"",
	Array(
		"REDIRECT_PAGE" => $_REQUEST["REDIRECT_PAGE"], 
		"AGREEMENT_TEXT_FILE" => "/bitrix/php_interface/agreement-ru.htm", 
		"SET_TITLE" => "Y" 
		)
);
?>

Описание параметров

Основные параметры
AGREEMENT_TEXT_FILE
Путь к файлу с пользовательским соглашением.
Дополнительные настройки
REDIRECT_PAGE
Указывается путь к странице, на которую будет переведен пользователь после регистрации. По умолчанию поле содержит ={$_REQUEST["REDIRECT_PAGE"]}.
SET_TITLE
[Y|N] В качестве заголовка страницы будет установлено "Регистрация аффилиата".
