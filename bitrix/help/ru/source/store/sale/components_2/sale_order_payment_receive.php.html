Компоненты 2.0::Подключение обработчика результата платежной системы

Одностраничный компонент.
Компонент осуществляет подключение скрипта для получения результата от платежной системы.

Пример вызова

<?
$APPLICATION->IncludeComponent(
		"bitrix:sale.order.payment.receive",
		"",
		Array(
			"PAY_SYSTEM_ID" => "4" 
		)
);?>

Описание параметров

Основные параметры
PAY_SYSTEM_ID
Платежная система. Выбирается из выпадающего списка одна из платежных систем, результат от которой должен быть получен. В выпадающем списке подключены все платежные системы, которые содержат в своей структуре файл result_rec.php
