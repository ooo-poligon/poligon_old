Компоненты 2.0::Отчет аффилиата

Одностраничный компонент.
Если на сайте авторизован аффилиат, то ему будет показано движение денег на его счете за установленный период времени.
Если авторизован пользователь, не являющийся аффилиатом, то компонент переведет его на страницу регистрации аффилиата, указанную в настройках компонента в поле Страница регистрации.
Если пользователь не авторизован, то он будет перенаправлен на страницу регистрации (по умолчанию register.php)

Пример вызова

<?
$APPLICATION->IncludeComponent(
		"bitrix:sale.affiliate.account",
		"",
		Array(
			"REGISTER_PAGE" => "register.php", 
			"SET_TITLE" => "Y" 
		)
);
?>

Описание параметров

Дополнительные настройки
REGISTER_PAGE
Путь к странице регистрации аффилиата. Она может быть создана с помощью компонента Регистрация аффилилата.
SET_TITLE
[Y|N] Если "Да", то в качестве заголовка страницы будет установлено «Счет аффилиата».
