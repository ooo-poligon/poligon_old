Компоненты 2.0::Аффилиатские планы

Одностраничный компонент.
Список доступных аффилиатских планов. Компонент выводит список всех планов, определённых в системе. Может использоваться на странице регистрации аффилиата для показа имеющихся планов.

Пример вызова

<?
$APPLICATION->IncludeComponent(
		"bitrix:sale.affiliate.plans",
		"",
		Array(
			"SET_TITLE" => "Y" 
		)
	);
?>

Описание параметров

Дополнительные настройки
SET_TITLE
[Y|N] В качестве заголовка страницы будет установлено «Планы комиссий аффилиатов».
