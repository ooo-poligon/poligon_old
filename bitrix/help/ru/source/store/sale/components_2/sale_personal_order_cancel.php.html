Компоненты 2.0::Отмена заказа

Одностраничный компонент.
Компонент отменяет заказ.

Пример вызова

<?
$APPLICATION->IncludeComponent(
		"bitrix:sale.personal.order.cancel",
		"",
		Array(
			"PATH_TO_LIST"	=> "order_list.php", 
			"PATH_TO_DETAIL" 	=> "order_detail.php", 
			"SET_TITLE" 	=> "Y", 
			"ID" 		=> $ID 
		)
);?>

Описание параметров

Дополнительные настройки
PATH_TO_LIST
Страница со списком заказов. Указывается путь к странице со списком заказов. Такая страница может быть создана с помощью компонента Список заказов.
PATH_TO_DETAIL
Страница c подробной информацией о заказе. Указывается путь к странице с детальной информацией о заказе. Такая страница может быть создана с помощью компонента Подробная информация о заказе. Необходимо передавать ID заказа в качестве параметра.
SET_TITLE
В качестве заголовка будет установлено «Отмена заказа №#ID_заказа».
Дополнительно
ID
Указывается php код, результатом которого является получение ID заказа. По умолчанию ={$ID}.
