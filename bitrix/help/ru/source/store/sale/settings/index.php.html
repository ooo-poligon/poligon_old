Настройки интернет-магазина

Настройка свойств и основных операций интернет-магазина выполняется на странице Настройки интернет-магазина: Магазин > Настройки магазина.

На данной странице расположен список параметров, настройку которых необходимо выполнить для полноценного функционирования Интернет-магазина. Названия параметров представлены в виде ссылок для перехода в соответствующий раздел административной части сайта.

Скидки на заказ
Форма Скидки на заказ служит для настройки скидок, предоставляемых на весь заказ. Величина скидки зависит от общей суммы заказа.

Службы доставки
Службы доставки делятся на Настраиваемые и Автоматизированные. Для добавления настраиваемой системы доставки достаточно заполнить стандартную форму. При добавлении автоматизированной системы необходимо создать специальный обработчик системы доставки, в котором можно организовать программный рассчет стоимости доставки, также необходимо определить список профилей доставки, применимость профилей доставки к заказу и т.д.

Платежные системы
Форма Платежные системы служит для настройки платежных систем, с помощью которых клиенты магазина могут оплачивать свои заказы.

Типы плательщиков
Форма Типы плательщиков служит для настройки списка типов плательщиков, которые могут осуществлять заказы на сайте. Для каждого типа плательщика создаётся собственный набор полей (свойств) заказа, которые плательщик должен заполнить при оформлении заказа, а так же обработчики платежных систем, с помощью которых плательщик может оплатить заказ.

Статусы
Форма Статусы заказа служит для настройки информационных статусов заказов. Статусы заказа служат для оповещения клиентов и сотрудников магазина о текущем состоянии заказа.

Свойства заказа
Форма Свойства заказа служит для настройки списка полей (свойств) заказа, которые клиент магазина должен будет заполнить при оформлении заказа. Набор свойств заказа зависит от типа плательщика, выбранного клиентом при оформлении заказа.

Группы свойств
Форма Группы свойств служит для настройки групп свойств заказов. Группировка свойств применяется для улучшения зрительного восприятия формы ввода свойств заказа. Для каждого типа плательщика создаётся самостоятельный набор групп свойств.

Местоположения
Форма Местоположения служит для настройки списка местоположений, из которого клиент магазина будет выбирать место для доставки заказа. Выбранное местоположение будет затем использовано для вывода списка доступных служб доставки, расчёта стоимости доставки, а также как часть фактического адреса доставки.

Группы местоположений
Форма Группы местоположений служит для объединения местоположений в группы. Это облегчает работу с настройкой служб доставки.

Налоги
Форма Налоги служит для настройки налогов, применяемых для расчёта стоимости заказа.

Ставки налогов
Форма Ставки налогов служит для задания ставок налогов в зависимости от местоположения покупателя и его типа плательщика.

Освобождение от налогов
Форма Освобождение от налогов служит для определения покупателей, освобожденных от уплаты части налогов.

Интеграция с 1С
Форма Интеграция с "1С:Предприятие" служит для настройки параметров импорта каталога и обмена заказами.
