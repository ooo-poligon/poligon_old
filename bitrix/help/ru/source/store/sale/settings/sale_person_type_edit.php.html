Создание и редактирование типа плательщика

В этом разделе

  • Контекстная панель
  • Форма редактирования
  • Кнопки управления

    • Данная форма служит для создания нового или настройки параметров уже существующего типа плательщика.

    Контекстная панель

    КнопкаОписание
    Список типов плательщиковПереход на страницу со списком типов плательщика.
    Создать новый типПереход к форме создания нового типа плательщика. Кнопка отображается при редактировании существующего типа плательщика.
    Удалить тип Удаление типа плательщика. Кнопка отображается при редактировании существующего типа плательщика.

    Форма редактирования

    ПолеОписание
    IDИдентификатор типа плательщика.
    *СайтСайт, на котором используется данный тип плательщика.
    *НазваниеНазвание типа плательщика.
    Индекс сортировкиОтносительный "вес", влияющий на положение типа плательщика в общем списке.

    * - Поля, обязательные для заполнения.

    Кнопки управления

    КнопкаОписание
    СохранитьСохранение параметров типа плательщика. Переход к списку типов плательщика.
    ПрименитьСохранение внесённых изменений. Продолжение редактирования параметров типа плательщика.
    ОтменитьОтмена выполненных изменений. Возврат первоначальных значений параметров.
