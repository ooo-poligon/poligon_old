Платежные системы

В этом разделе

  • Форма поиска
  • Контекстная панель
  • Список платежных систем

    • Форма Платежные системы служит для настройки платежных систем, с помощью которых клиенты магазина могут оплачивать свои заказы. Данная форма позволяет:

    • просматривать список существующих платежных систем;
    • осуществлять переход к созданию новой или редактированию существующей платежной системы;
    • удалять платежные системы.

    Форма поиска

    С помощью данной формы вы можете выполнить поиск платежных систем в соответствии с указанными условиями.

    ПолеОписание
    СайтОпределяет платежные системы каких сайтов будут участвовать в поиске.
    ВалютаОпределяет в какой валюте платежные системы будут участвовать в поиске.
    АктивностьОпределяет какие платежные системы будут участвовать в поиске: активные, неактивные или все (активные и неактивные).

    Чтобы отобрать платежные системы по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех служб нажмите кнопку Отменить.

    Контекстная панель

    КнопкаОписание
    Добавить платежную системуПереход к форме создания новой платежной системы. По нажатии на кнопку в выпадающем списке необходимо выбрать сайт, к которому будет относиться платежная система.
    НастроитьПозволяет перейти к диалогу настройки внешнего вида отчетной формы.
    ExcelЭкспортирует данные из отображаемой таблицы в формат MS Excel.

    Список платежных систем

    ПолеОписание
    Колонка флажковПоле предназначено для выбора платежных систем, к которым предполагается применить какое-либо правило или действие.
    ДействияДействия над платежной системой:
    • Изменить - изменение параметров платежной системы;
    • Удалить - удаление платежной системы.
    IDИдентификатор платежной системы.
    НазваниеНазвание платежной системы.
    СайтСайт, на котором действует данная платежная система. В зависимости от сайта клиентам магазина предоставляется на выбор разный набор платежных систем.
    ВалютаВалюта, с которой работает данная платежная система.
    Акт.Признак активности платежной системы.
    Сорт.Относительный "вес" платежной системы. Влияет на положение данной системы в общем списке платежных систем.
