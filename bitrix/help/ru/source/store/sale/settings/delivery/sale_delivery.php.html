Настраиваемые

В этом разделе

  • Форма поиска
  • Контекстная панель
  • Список служб доставки

    • В продукте имеется два типа служб доставки: настраиваемые и автоматизированные. Форма Настраиваемые службы доставки служит для настройки служб доставки товаров и услуг из магазина к покупателям. С помощью этой формы можно:

    • просматривать имеющиеся службы доставки и их основные параметры;
    • переходить к изменению параметров службы доставки;
    • удалять службы доставки.

    Форма поиска

    С помощью данной формы вы можете выполнить поиск служб доставки в соответствии с указанными условиями.

    ПолеОписание
    МестоположениеПоиск служб по возможному месту доставки.
    СайтСайт, на котором действует служба доставки.
    ВесПозволяет производить выборку служб по допустимому весу доставляемого заказа.

    Вес заказа может быть задан как в виде диапазона значений, так и с помощью указания нижней или верхней границы веса.
    Стоимость заказаПозволяет произвести выборку служб по стоимости доставляемого заказа.
    Стоимость заказа может быть задана как в виде диапазона значений, так и с помощью нижней или верхней границы стоимости.
    АктивностьУказывается, какие службы будут участвовать в поиске:
    • да - в поиске будут участвовать только активные службы доставки;
    • нет - в поиске будут участвовать только неактивные службы доставки (неактивные службы не предлагаются для доставки товаров);
    • все - в поиске будут участвовать как активные, так и неактивные службы доставки.

    Чтобы отобрать службы доставки по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех служб нажмите кнопку Отменить.

    Список служб доставки

    ПолеОписание
    Колонка флажковПоле предназначено для выбора служб доставки, к которым предполагается применить какое-либо правило или действие.
    ДействияДействия над службой доставки:
    • Изменить параметры службы доставки - редактирование службы доставки;
    • Удалить службу доставки - удаление службы доставки.
    IDИдентификатор службы доставки.
    НазваниеНазвание службы доставки, как оно будет показано клиентам магазина при оформлении заказа.
    СайтСайт, на котором действует эта служба доставки. В зависимости от сайта клиентам магазина предоставляется на выбор разный набор служб доставки.
    ВесДиапазон общего веса заказа, при котором возможен выбор этой службы доставки. Вес должен быть задан в тех же единицах, что и вес товаров, помещаемых в корзину (по умолчанию - в граммах).
    Стоимость заказаДиапазон стоимости заказа, при котором возможен выбор этой службы доставки. Если валюта диапазона отличается от валюты заказа, то происходит автоматический пересчет в валюту заказа по текущему курсу системы.
    Сорт.Относительный "вес" службы доставки. Влияет на положение данной службы в общем списке служб доставки.
    Акт.Признак активности службы доставки. Неактивные службы не предлагаются для доставки товаров.
    ЦенаСумма, которая добавляется к стоимости товаров в заказе для оплаты услуг службы доставки. Если валюта цены отличается от валюты заказа, то происходит автоматический пересчет в валюту заказа по текущему курсу системы.
