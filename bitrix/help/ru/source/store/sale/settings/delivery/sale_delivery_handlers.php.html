Автоматизированные

В этом разделе

  • Контекстная панель
  • Список служб доставки

    • Автоматизированные системы доставки имеют специальный обработчик, в котором можно организовать программный расчёт стоимости доставки, определить список профилей доставки, применимость профилей доставки к заказу и т.д.

    В форме Автоматизированные службы доставки представлен весь список доставок этого типа. С помощью этой формы можно просматривать имеющиеся службы и их основные параметры, переходить к изменению параметров служб доставки.

    Вверху страницы выведен адрес магазина с индексом (либо адреса, если для каждого сайта системы задан свой адрес). А также ссылка на изменение адреса. Далее выведена подсказка, где хранить пользовательские обработчики доставки. Путь берется из настроек модуля Интернет-магазин.

    Контекстная панель

    КнопкаОписание
    Настраиваемые службыПереход к списку настраиваемых служб доставки.

    ExcelЭкспортирует данные из отображаемой таблицы в формат MS Excel.

    Список автоматизированных служб доставки

    ПолеОписание
    Колонка флажковПоле предназначено для выбора служб доставки, к которым предполагается применить какое-либо правило или действие.
    ДействияДействия над службой доставки:
    • Редактировать - редактирование службы доставки;
    • Очистить настройки - все настройки службы доставки будут удалены, обработчик будет деинсталлирован.
    Акт.Значок активности службы доставки.
    Сорт.Индекс сортировки службы доставки, определяющий ее положение в общем списке.
    SIDСимвольный идентификатор службы доставки.
    НазваниеПолное название службы доставки.
    ОбработчикОтносительный путь к файлу-обработчику системы доставки.
