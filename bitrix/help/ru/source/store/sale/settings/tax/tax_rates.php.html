Ставки налогов

В этом разделе

  • Форма поиска
  • Контекстная панель
  • Список ставок налогов

    • В форме Ставки налогов представлен список всех заданных ставок налогов. С помощью данной формы вы можете:

    • добавлять новые ставки налогов;
    • переходить к форме редактирования ставок;
    • удалять ставки налогов.

    Форма поиска

    С помощью данной формы вы можете выполнить поиск ставок налога в соответствии с указанными условиями.

    Поле Описание
    Налог Название налога, ставки для которого следует отобрать.
    Сайт Сайт, на котором действует ставка.
    Тип плательщика Тип плательщика, ставки для которого следует отобрать.
    Местоположение Местоположение, ставки налогов для которых следует отобрать.

    Для того чтобы отобрать ставки налога по заданным критериям поиска, нажмите кнопку "Найти". Для отображения всех ставок нажмите кнопку "Отменить".

    Контекстная панель

    Кнопка Описание
    Новая ставка налога Переход к форме создания новой ставки налога.
    Excel Экспортирует данные из отображаемой таблицы в формат MS Excel.
    Настроить Позволяет перейти к диалогу настройки внешнего вида отчетной формы.

    Список ставок налогов 

    Поле Описание
    Колонка флажков Поле предназначено для выбора ставок налога, к которым предполагается применить какое-либо правило или действие.
    Действия Действия, которые могут быть применены к ставке:
    • Изменить параметры ставки налога - редактирование ставки налога;
    • Удалить ставку налога - удаление ставки налога.
    ID Код ставки налога.
    Дата изменения Дата и время последнего изменения ставки налога.
    Название Название налога, для которого задана ставка.
    Тип плательщика Тип плательщика, к которому применима данная ставка налога.
    Ставка Величина ставки налога.
    Входит в цену Указывает, включен ли данный налог в цену товара.
    Порядок применения Ставки налога с одинаковым порядком применения складываются, с разными - образуют сложные проценты (проценты на проценты).
