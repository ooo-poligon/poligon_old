Создание и редактирование информационного статуса заказа

В этом разделе

  • Контекстная панель
  • Поля формы
  • Кнопки управления

    • Данная форма служит для создания новых и изменения параметров уже существующих информационных статусов заказа.

    Контекстная панель

    КнопкаОписание
    Список статусовПереход на страницу со списком статусов заказа.
    Добавить новый статусПереход к форме создания нового статуса заказа. Кнопка отображается при редактировании существующего статуса.
    Удалить статусУдаление редактируемого статуса заказа. Кнопка отображается при редактировании существующего статуса.

    Поля формы

    ПолеОписание
    *КодУникальный код статуса. Задается в виде 1 буквы латинского алфавита.
    СортировкаОтносительный "вес" статуса. Используется для определения положения данного статуса в общем списке статусов.
    *НазваниеУкажите название статуса для всех языков, используемых в системе.
    ОписаниеЗадайте описание статуса для всех языков, используемых в системе.
    Права на доступ
    В данной части формы устанавливаются права на доступ к заказам в данном статусе.
    ПолеОписание
    Группа пользователейВыведены те группы пользователей, которые имеют в настройках модуля Интернет-магазин уровень доступа Обработка заказов.
    ПросмотрОпция возможности просмотра заказа, который находится в редактируемом статусе, пользователями данной группы.
    Флаг отменыПри отмеченной опции пользователь данной группы будет иметь возможность изменять флаг отмены в форме детального просмотра заказа, т.е. сможет отменить заказ.
    Флаг доставкиПри отмеченной опции пользователь данной группы будет иметь возможность изменять флаг доставки в форме детального просмотра заказа, т.е. сможет разрешить доставку товара.
    Флаг оплатыПри отмеченной опции пользователь данной группы будет иметь возможность изменять флаг оплаты в форме детального просмотра заказа, т.е. переводить заказ в состояние "Оплачен".
    Перевод в этот статусОпция дает возможность пользователю данной группы переводить заказ в редактируемый статус.
    ИзменениеПри наличии данной опции пользователю данной группы будет доступно действие Изменить параметры заказа в меню действий заказа.
    УдалениеПри наличии данной опции пользователю данной группы будет доступно действие Удалить заказ в меню действий заказа.

    * - Поля являются обязательными для заполнения.

    Кнопки управления

    КнопкаОписание
    СохранитьСохранение внесённых изменений. Переход на страницу со списком статусов заказа.
    ПрименитьПрименение внесённых изменений. Продолжение редактирования параметров статуса заказа.
    ОтменитьОтмена внесённых изменений. Возврат первоначальных значений параметров.
