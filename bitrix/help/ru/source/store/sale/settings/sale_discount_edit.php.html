    • Контекстная панель

    КнопкаОписание
    Список скидокПереход на страницу со списком скидок.
    Создать новую скидкуПереход к форме создания скидки. Кнопка отображается при редактировании существующей скидки.
    Удалить скидкуУдаление редактируемой скидки.

    Поля формы

    ПолеОписание
    IDКод скидки.
    *СайтСайт, на котором действует эта скидка. В зависимости от сайта используются разные скидки.
    Применяется при сумме заказаДиапазон стоимости заказа (без доставки), для которой устанавливается скидка.
    Допустимо также указание только одной границы диапазона (например, если в поле "от" указано 100, а в поле "до" ничего не указано, то скидка применяется при любой стоимости заказа больше либо равной 100).
    Период активностиУказывается период активности скидки.
    *Величина скидкиВеличина скидки. Если в выпадающем списке выбрано значение "%", то скидка задается в процентах от стоимости заказа. В противном случае скидка задается в фиксированном виде в валюте диапазона цены (диапазона стоимости заказа).
    АктивностьПризнак активности скидки. При расчёте стоимости товара могут применяться только активные скидки.
    Индекс сортировкиОтносительный "вес", определяющий положение скидки в списке.

    * - Поля, обязательные для заполнения.

    Кнопки управления

    КнопкаОписание
    СохранитьСохранение изменений, выполненных на данной странице.
    ПрименитьСохранение внесённых изменений. Продолжение редактирования параметров скидки.
    ОтменитьОтмена внесённых изменений. Возврат первоначальных значений параметров.
