    Контекстная панель

    КнопкаОписание
    Список платежных системПереход к странице со списком платежных систем.
    Добавить системуПереход к форме создания новой платежной системы. Кнопка отображается при редактировании существующей платежной системы.
    Удалить системуУдаление редактируемой платежной системы. Кнопка отображается при редактировании существующей платежной системы.

    Форма редактирования

    Закладка "Платежная система"

    Данная форма служит для управления основными параметрами платежной системы.

    ПолеОписание
    IDИдентификатор платежной системы.
    СайтСайт, на котором действует данная платежная система. В зависимости от сайта клиентам магазина предоставляется на выбор разный набор платежных систем.
    *ВалютыВалюта, с которой работает данная платежная система.
    *НазваниеНазвание платежной системы для отображения в административном разделе сайта.
    АктивностьПризнак активности платежной системы.
    СортировкаОтносительный "вес" платежной системы. Влияет на положение данной системы в общем списке платежных систем.
    ОписаниеТекст длиной до 250 символов, информирующий клиента об особых условиях работы платежной системы.

    * - Поля являются обязательными для заполнения.

    Закладка "<Тип плательщика>"

    В форме с названием типа плательщика выполняется настройка параметров платежной системы для данного типа плательщика.

    Применяется для данного типа плательщикаОпределяется, будет ли создаваемая платежная система доступна для данного типа плательщика.
    *НазваниеНазвание платежной системы, отображаемое в публичном разделе сайта для данного типа плательщика.
    *ОбработчикВыбирается файл-обработчик платежей, осуществляемых с помощью данной платежной системы.
    Открывать в новом окнеФлаг в данном поле означает, что если по результатам работы обработчика платежей пользователю должен быть представлен какой-либо результат (например, платежный документ), то он будет выведен в отдельном окне веб-браузера.
    Свойства обработчикаЕсли выбранный вами обработчик платежной системы требует задания значений дополнительных свойств, то внизу секции будет выведен список этих свойств.
    • Для того чтобы скрыть список свойств, воспользуйтесь ссылкой Свернуть свойства обработчика.
    • Чтобы раскрыть список свойств, воспользуйтесь ссылкой Развернуть свойства обработчика.

    * - Поля являются обязательными для заполнения.

    Кнопки управления

    КнопкаОписание
    СохранитьСохранение внесённых изменений. Переход на страницу со списком платежных систем.
    ПрименитьПрименение внесённых изменений. Продолжение редактирования параметров платежной системы.
    ОтменитьОтмена внесённых изменений. Возврат первоначальных значений параметров.

    Файлы-обработчики

    Файлы работы с платежной системой обрабатываются следующим образом.

    • Файл-обработчик вызывается непосредственно после оформления заказа, а так же при выборе клиентом функции повторения платежа в персональном разделе. Этот файл может содержать скрипт, отображающий на экране необходимый платежный документ, или форму для отправки данных электронной платежной системе. Подробности смотрите в разделе Обработка платежных систем.
    • Обработчик результатов вызывается сайтом для запроса параметров оплаты заказа у электронной платежной системы. Если платежная система не возвращает параметры заказа (успешна ли оплата, сколько реально оплачено и т.п.), то задавать обработчик результатов не нужно.
