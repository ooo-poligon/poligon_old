Группы свойств

В этом разделе

  • Форма поиска
  • Контекстная панель
  • Список групп свойств

    • Форма Группы свойств служит для настройки групп свойств заказов. Данная группировка применяется при построении формы для ввода значений свойств заказа, что позволяет улучшить ее зрительное восприятие.
    Показ группы свойств в форме заказа зависит от типа плательщика, выбранного покупателем, т.е. для каждого типа плательщика отображается самостоятельный набор групп свойств.

    Форма поиска

    С помощью данной формы вы можете выполнить поиск групп свойств заказа в соответствии с указанными условиями.

    Поле Описание
    Тип плательщика Поиск групп свойств по типу плательщика.

    Для того чтобы отобрать группы свойств по заданным критериям поиска, нажмите кнопку "Найти". Для отображения всех групп свойств нажмите кнопку "Отменить".

    Контекстная панель

    Кнопка Описание
    Новая группа Переход к форме создания новой группы свойств.
    Excel Экспортирует данные из отображаемой таблицы в формат MS Excel.
    Настроить Позволяет перейти к диалогу настройки внешнего вида отчетной формы.

    Список групп свойств

    ПолеОписание
    Колонка флажков Поле предназначено для выбора групп свойств, к которым предполагается применить какое-либо правило или действие.
    Действия Действия над группой свойств заказа:
    • Изменить группу- редактирование параметров группы свойств;
    • Удалить группу- удаление группы свойств.
    ID Идентификатор группы свойств.
    Название Название группы свойств, как оно будет показано клиенту магазина при оформлении заказа.
    Тип плательщика Тип плательщика, к которому привязана данная группа свойств.
    Сорт. Относительный "вес" группы свойств. Определяет порядок, в котором группы свойств будут выводиться при оформлении заказа.
    Свойства Ссылка для перехода к списку свойств, относящихся к данной группе.
