Группы местоположений

В этом разделе

  • Форма поиска
  • Контекстная панель
  • Список групп местоположений

    • Форма Группы местоположений служит для объединения местоположений в группы. Это облегчает работу с настройкой служб доставки. Вы можете создать группу местоположений и привязать к ней все службы доставки, которые осуществляют доставку в местоположения этой группы.

    При появлении нового местоположения или изменения тарифов по какому-либо из существующих местоположений будет достаточно добавить или удалить это местоположение из группы местоположений. Изменять каждую из служб доставки, к которым имеет отношение это местоположение, не нужно.

    Форма поиска

    С помощью данной формы вы можете выполнить поиск групп местоположений в соответствии с указанными условиями.

    Поле Описание
    Местоположение Поиск групп по названию входящего в них местоположения.

    Для того чтобы отобрать группы местоположений по заданным критериям поиска, нажмите кнопку "Найти". Для отображения всех групп местоположений нажмите кнопку "Отменить".

    Контекстная панель

    Кнопка Описание
    Новая группа Переход к форме создания новой группы местоположений.
    Excel Экспортирует данные из отображаемой таблицы в формат MS Excel.
    Настроить Позволяет перейти к диалогу настройки внешнего вида отчетной формы.

    Список групп местоположений

    Поле Описание
    Колонка флажков Поле предназначено для выбора групп местоположений, к которым предполагается применить какое-либо правило или действие.
    Действия Действия над группой местоположений:
    • Изменить параметры группы местоположений - редактирование группы местоположений;
    • Удалить группу местоположений - удаление группы местоположений. Удаление группы не влечет за собой удаление местоположений, входящих в эту группу.
    ID Идентификатор группы в системе.
    Название Название группы местоположений.
    Сортировка Относительный "вес", влияющий на положение группы в общем списке.
