Типы плательщиков

В этом разделе

  • Форма поиска
  • Контекстная панель
  • Список типов плательщика

    • Форма Тип плательщика служит для управления списком типов плательщика, которые могут осуществлять заказы на сайте. Для каждого типа плательщика создаётся собственный список свойств, которые плательщик должен заполнить при оформлении заказа, а также указываются обработчики платежных систем, с помощью которых плательщик может оплатить заказ.

    Форма поиска

    С помощью данной формы вы можете выполнить поиск типов плательщика в соответствии с указанными условиями.

    ПолеОписание
    СайтПозволяет отобрать типы плательщика, используемые на указанном сайте.

    Чтобы отобрать типы плательщика по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех типов плательщика нажмите кнопку Отменить.

    Контекстная панель

    КнопкаОписание
    Добавить типПереход к форме создания нового типа плательщика.
    НастроитьПозволяет перейти к диалогу настройки внешнего вида отчетной формы.
    ExcelЭкспортирует данные из отображаемой таблицы в формат MS Excel.

    Список типов плательщика

    ПолеОписание
    Колонка флажковПоле предназначено для выбора типов плательщиков, к которым предполагается применить какое-либо правило или действие.
    ДействияДействия над типом плательщика:
    • Изменить - редактирование выбранного типа плательщика;
    • Удалить - удаление выбранного типа плательщика.
    IDИдентификатор типа плательщика.
    НазваниеНазвание типа плательщика, как оно будет показано клиентам магазина при оформлении заказа.
    СайтСайт, на котором действует этот тип плательщика. В зависимости от сайта клиентам магазина предоставляется на выбор разный набор типов плательщиков.
    Сорт.Относительный "вес" данного типа плательщика. Используется для определения порядка показа типов плательщика в списке.
    СвойстваСсылка для перехода к просмотру свойств заказа, заданных для данного типа плательщика.
