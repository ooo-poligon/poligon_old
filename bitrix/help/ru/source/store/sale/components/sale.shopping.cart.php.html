Компоненты 1.0::Корзина

$APPLICATION->IncludeFile("sale/sale_basket/basket.php",
    Array(
    		"ORDER_PAGE" => "/ru/catalog_wizard/order.php",   
				// Путь к странице процедуры заказа
		    "HIDE_COUPON"	=>	"Y",	
				// Спрятать поле ввода купона
		 	"COLUMNS_LIST"	=>	Array("NAME","PRICE","TYPE","QUANTITY","DELETE","DELAY","WEIGHT"),	
				// Выводимые колонки
    		)
);

Компонент выводит полнофункциональную корзину.

Параметры

ORDER_PAGE
Путь к странице процедуры заказа.
HIDE_COUPON
[Y|N] Спрятать поле ввода купона.
COLUMNS_LIST
Выбираются выводимые характеристики товаров в таблице корзины магазина.
