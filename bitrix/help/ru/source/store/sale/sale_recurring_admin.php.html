Продление подписки

В этом разделе

  • Форма поиска

  • Список продлений подписки

    • Форма Продление подписки (Магазин > Продление) содержит список записей о продлении прав пользователей на пользование платным контентом сайта.

    Форма поиска

    Поиск записей о продлении в соответствии с указанными условиями. По умолчанию выведеоно только поле Пользователь. С помощью кнопки Дополнительно можно подключить поиск по другим полям.

    ПолеОписание
    ПользовательПоиск записи по имени, отчеству, фамилии и/или e-mail адресу пользователя, для которого было оформлено продление подписки. Возможно использование условий сложного поиска.
    Код пользователяПоиск записи по коду пользователя, для которого было оформлено продление подписки. Данный код автоматически присваивается пользователю при регистрации на сайте.
    Логин пользователяПоиск записи по логину (имени входа) пользователя, для которого было оформлено продление подписки.
    ОтмененоПоиск записи по состоянию подписки:
    • да - в поиске участвуют отменённые записи о продлении подписки;
    • нет - в поиске участвуют действующие (активные) продления подписки;
    • все - в поиске участвуют как отменённые, так и действующие продления подписки.
    Дата последнего продленияПоиск по дате оформления продления подписки.
    Значение поля может быть задано как количеством дней, прошедших со дня оформления подписки, так и в виде временного диапазона (с помощью on-line календаря).
    Дата очередного продленияПоиск по дате оформления очередного продления подписки.
    Значение поля может быть задано как количеством дней, оставшихся до дня оформления очередной подписки, так и в виде временного диапазона (с помощью on-line календаря).
    Номер базового заказаНомер заказ, запись о продлении которого присутствует в списке.
    Успешное продлениеПоиск записи по статусу продления подписки:
    • да - в поиске участвуют записи только об успешно продленной подписке;
    • нет- в поиске участвуют записи о подписке, продление которой прошло с какой-либо ошибкой;
    • все - в поиске участвуют записи обо всех продлениях подписки (успешных и неуспешных).

    Чтобы отобрать записи о продлении по заданным критериям поиска, нажмите кнопку Найти. Для отображения всех записей нажмите кнопку Отменить.

    Контекстная панель

    КнопкаОписание
    Добавить записьПереход к форме создания нового продления.
    ExcelЭкспорт данных из отображаемой таблицы в формат MS Excel.
    НастроитьПереход к диалогу настройки внешнего вида отчетной формы.

    Список продлений подписки

    ПолеОписание
    Колонка флажковПоле предназначено для выбора записей, к которым предполагается применить какое-либо правило или действие.
    ДействияДействия, доступные для выполнения над записью о продлении подписки:
    • Изменить параметры продления - редактирование продления подписки;
    • Удалить продление- удаление записи о продлении подписки.
    IDИдентификатор записи о продлении подписки.
    ПользовательРегистрационный код, имя и e-mail адрес пользователя, для которого было оформлено продление подписки.
    Отм.Признак действительности продления подписки.
    Если значение поля "Да", то продление подписки считается недействительным (т.е. продление отменяется).
    Предыдущее продлениеДата и время последнего продления подписки.
    Очередное продлениеДата и время предстоящего продления подписки.
    Успешный платежПризнак успешности оплаты продления подписки.
