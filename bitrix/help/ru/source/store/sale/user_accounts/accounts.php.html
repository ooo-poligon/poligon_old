Внутренние счета пользователей

В этом разделе

  • Форма поиска
  • Контекстная панель
  • Список счетов

    • Форма Внутренние счета пользователей (Магазин -> Счета покупателей -> Счета) содержит список внутренних счетов клиентов и отражает остаток средств на данных счетах.

    Форма поиска

    С помощью данной формы вы можете выполнить поиск внутренних счетов пользователей в соответствии с указанными условиями.

    Поле Описание
    Пользователь Поиск счета по имени, отчеству, фамилии и/или e-mail адресу пользователя-владельца счета. Возможно использование условий сложного поиска.
    ID пользователя Идентификационный номер пользователя-владельца счета.
    Логин пользователя Поиск счета по логину (имени входа) пользователя-владельца счета.
    Валюта Валюта счета.
    Заблокирован Поиск счета по признаку активности:
    • да - в поиске будут участвовать только заблокированные (неактивные) счета;
    • нет - в поиске будут участвовать только активные счета;
    • все- в поиске будут участвовать все счета (активные и заблокированные).

    Для того чтобы отобрать внутренние счета пользователей по заданным критериям поиска, нажмите кнопку "Найти". Для отображения всех счетов нажмите кнопку "Отменить".

    Контекстная панель

    Кнопка Описание
    Добавить счет Переход к форме создания нового внутреннего счета.
    Excel Экспортирует данные из отображаемой таблицы в формат MS Excel.
    Настроить Позволяет перейти к диалогу настройки внешнего вида отчетной формы.

    Список счетов

    Поле Описание
    Колонка флажков Поле предназначено для выбора счетов пользователей, к которым предполагается применить какое-либо правило или действие.
    Действия Действия, доступные для выполнения над внутренним счетом пользователя:
    • Изменить параметры счета - редактирование внутреннего счета пользователя;
    • Удалить счет- удаление счета.
    ID Идентификатор счета в системе.
    Пользователь Идентификатор, имя, логин и e-mail адрес пользователя-владельца счета.
    Сумма на счете Размер денежной суммы на счете (в валюте счета).
    Заблокирован Признак активности счета (т.е. доступности выполнения транзакций по счету). Для того чтобы разблокировать счет, отметьте данный счет с помощью флага в крайнем левом поле, затем выберите операцию Разблокировать на панели управления внизу страницы и нажмите кнопку "Применить".
    Транзакции Количество выполненных транзакций по данному счету. При нажатии на ссылку с количеством транзакций будет открыта страница со список транзакций по данному счету.
    Выбрано Количество элементов в списке.
    Отмечено Количество элементов, отмеченных в поле [1].
    Итого Сумма средств на счетах отдельно по каждому виду валюты.
