Пластиковые карты

В этом разделе

  • Форма поиска
  • Контекстная панель
  • Список пластиковых карт

    • Форма Пластиковые карты (Магазин -> Счета покупателей -> Пластиковые карты) содержит список всех пластиковых карт, с помощью которых клиенты магазина осуществляют оплату за товары.

    Важно! Для обеспечения безопасности работы с пластиковыми картами на странице настроек модуля Интернет-магазин необходимо задать пароль для шифрования номеров карт. Если пароль не задан, то соответствующее сообщение будет выведено вверху страницы.

    Форма поиска

    С помощью данной формы вы можете выполнить поиск пластиковых карт в соответствии с указанными условиями.

    Поле Описание
    Пользователь Поиск карты по имени, отчеству, фамилии и/или e-mail адресу пользователя, осуществляющего платежи. Возможно использование условий сложного поиска.
    Код пользователя Поиск карты по коду пользователя, осуществляющего платежи. Данный код автоматически присваивается пользователю при регистрации на сайте.
    Логин пользователя Поиск карты по логину (имени входа) пользователя, осуществляющего платежи.
    Активность Поиск пластиковой карты с использованием признака активности карты:
    • да - в поиске участвуют только активные (доступные для использования при оплате) карты;
    • нет- в поиске участвуют только неактивные карты;
    • все - в поиске участвуют все пластиковые карты (активные и неактивные).

    Для того чтобы отобрать пластиковые карты по заданным критериям поиска, нажмите кнопку "Найти". Для отображения всех пластиковых карт, нажмите кнопку "Отменить".

    Контекстная панель

    Кнопка Описание
    Добавить карту Переход к форме создания новой пластиковой карты.
    Excel Экспортирует данные из отображаемой таблицы в формат MS Excel.
    Настроить Позволяет перейти к диалогу настройки внешнего вида отчетной формы.

    Список пластиковых карт

    Поле Описание
    Колонка флажков Поле предназначено для выбора пластиковых карт, к которым предполагается применить какое-либо правило или действие.
    Действия Действия, доступные для выполнения над пластиковой картой:
    • Изменить параметры пластиковой карты - редактирование пластиковой карты;
    • Удалить пластиковую карту - удаление записи с параметрами пластиковой карты.
    ID Идентификатор пластиковой карты.
    Пользователь Идентификатор, имя и e-mail адрес пользователя, осуществляющего платежи с помощью данной пластиковой карты.
    Акт. Признак активности пластиковой карты.
    Неактивная карта не может использоваться для оплаты товаров.
    Сорт. Относительный "вес" пластиковой карты. Используется для определения положения пластиковой карты в общем списке.
    Валюта Наименование валюты, в которой осуществляются платежи с помощью данной пластиковой карты.
    Тип Наименование типа пластиковой карты.
