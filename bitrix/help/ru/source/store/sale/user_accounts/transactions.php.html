Транзакции

В этом разделе

  • Форма поиска
  • Контекстная панель
  • Список транзакций

    • Форма Транзакции (Магазин -> Счета покупателей -> Транзакции) содержит список всех транзакций, выполняемых по счету клиента интернет-магазина (например, зачисление или списание денежных средств).

    Форма поиска

    С помощью данной формы вы можете выполнить поиск транзакций в соответствии с указанными условиями.

    Поле Описание
    Пользователь Поиск транзакции по имени, отчеству, фамилии и/или e-mail адресу пользователя, по счету которого была выполнена операция. Возможно использование условий сложного поиска.
    ID пользователя Идентификационный номер пользователя, по счету которого была выполнена операция.
    Логин пользователя Поиск транзакции по логину (имени входа) пользователя, по счету которого была выполнена операция.
    Валюта Поиск транзакции по валюте операции.
    Дата транзакции Поиск операции по дате проведения.
    Возможно указание как количества дней, прошедших со дня выполнения операции, так и периода времени, в течение которого могла быть выполнена транзакция.
    Номер заказа Поиск транзакции по номеру заказа, в рамках которого она была проведена.

    Для того чтобы отобрать транзакции по заданным критериям поиска, нажмите кнопку "Найти". Для отображения всех транзакций нажмите кнопку "Отменить".

    Контекстная панель

    Кнопка Описание
    Добавить транзакцию Переход к форме создания новой транзакции.
    Excel Экспортирует данные из отображаемой таблицы в формат MS Excel.
    Настроить Позволяет перейти к диалогу настройки внешнего вида отчетной формы.

    Список транзакций

    Поле Описание
    ID Уникальный идентификатор транзакции в системе.
    Дата транзакции Дата проведения операции по счету.
    Пользователь Регистрационный код, имя и e-mail адрес пользователя, по счету которого осуществляется данная операция.
    Сумма Сумма операции.
    Заказ Номер заказа, в рамках которого была проведена данная операция.
    Номер заказа представлен в виде ссылки, позволяющей перейти к форме с подробной информацией о заказе.
    Тип Тип операции по счету.
    Описание Регистрационный код, имя и логин пользователя, выполнившего транзакцию.
