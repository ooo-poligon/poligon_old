Выполнение новой транзакции

В этом разделе

  • Контекстная панель
  • Поля формы
  • Кнопки управления

    • Данная форма служит для проведения новой операции (транзакции) по счету клиента.

    Контекстная панель

    Пункт Описание
    Список транзакций Переход на страницу со списком транзакций.

    Поля формы

    Поле Описание
    *Пользователь Идентификатор пользователя, по счету которого выполняется операция. Идентификатор пользователя выбирается в списке, открываемом с помощью кнопки [...].
    *Дата транзакции Дата и время проведения операции.
    Если значение поля указано для будущего периода, то операция будет выполнена при наступлении указанной даты и времени.
    *Сумма Сумма операции.
    Валюта Валюта операции.
    Тип записи Тип выполняемой операции (транзакции).
    Номер заказа Если транзакция выполняется в рамках какого-либо заказа, то в данном поле указывается номер этого заказа.
    Заметки Произвольный комментарий к выполняемой операции.

    * - Поля, обязательные для заполнения.

    Кнопки управления

    Кнопка Описание
    Сохранить Сохранение транзакции. Переход на страницу со списком транзакций.
    Отменить Отмена транзакции.
