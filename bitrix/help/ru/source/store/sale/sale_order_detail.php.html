    Форма просмотра заказа

    В форме детального просмотра заказа представлена полная информация о выбранном заказе.

    Закладка "Заказ"

    Основные параметры заказа.

    ПолеОписание
    Заказ № <...> от <...>
    Дата последнего измененияДата последнего изменения заказа.
    Сайт заказаСайт, на котором был оформлен заказ.
    Текущий статус заказаСтатус заказа. Например, возможны следующие значения:
    • Принят
    • В обработке
    • Отгружен
    • Доставлен
    Для изменения текущего статуса заказа нажмите ссылку Сменить статус заказа.
    Сумма заказаОбщая сумма заказа к оплате (включает налоги, доставку и т.п.)
    АффилиатВыводится код и имя аффилиата, если товар приобретен через аффилиата.
    ОтменёнСтатус отмены заказа:
    • да- заказ отменён;
    • нет - заказ принят.
    Для изменения текущего статуса отмены заказа нажмите ссылку Сменить флаг отмены заказа.
    Бюджет покупателя
    ПокупательКод покупателя, присвоенный ему при регистрации в системе.
    Имя входа покупателя (логин)Регистрационное имя покупателя (login).
    E-Mail покупателяАдрес электронной почты покупателя.
    Параметры заказа
    Тип плательщикаТип плательщика, выбранный покупателем при оформлении заказа (физическое лицо, юридическое лицо и т.п.).
    Комплектация
    КомплектацияКомплектация, выбранная покупателем при заказе товара. 
    Адрес доставки
    E-MailАдрес электронной почты получателя товара.
    МестоположениеНазвание местоположения, выбранного для доставки товара.
    ГородНазвание города получателя заказа.
    ИндексПочтовый индекс для адреса доставки товара.
    Адрес (без города)Адрес получателя товара без указания названия города.
    Контактное лицоПолучатель товара.
    Оплата
    Платежная системаПлатежная система, выбранная для оплаты товара.
    ОплаченСтатус оплаты заказа:
    • да- заказ оплачен;
    • нет - заказ не оплачен.
    Для изменения текущего статуса оплаты заказа нажмите ссылку Сменить флаг оплаты.
    Служба доставки
    Служба доставкиНазвание выбранной службы доставки заказа.
    Доставка разрешенаСтатус разрешения доставки. Для изменения статуса воспользуйтесь ссылкой Сменить флаг.
    Комментарий
    Произвольный комментарий персонала интернет-магазина по заказу (не виден покупателю).

    Закладка "Состав заказа"

    Информация о составе заказа.

    ПолеОписание
    НазваниеНазвание товара.
    Внешний кодВнешний код товара.
    СвойстваСвойства товара, передаваемые в заказ (например, ID, цвет, размер и т.д.).
    Скидка на товарСуммарная скидка на товар.
    Тип ценыТип цены, по которой покупатель приобретает товар.
    КоличествоКоличество товара, приобретаемое покупателем.
    ЦенаОбщая стоимость данной позиции заказа.
    СкидкаОбщая сумма применяемых скидок
    НалогиСумма налогов, применяемых при расчёте стоимости заказа.
    ДоставкаСтоимость доставки по данному заказу.
    ИтогоОбщая сумма заказа.

    Закладка "Транзакции по заказу"

    Список записей о движении денежных средств на счете клиента в рамках данного заказа.

    ПолеОписание
    ДатаДата и время совершения операции по счету.
    ПользовательКод, имя и фамилия пользователя, выполнившего операцию.
    СуммаСумма операции.
    • Отрицательная сумма (со знаком "-") означает операцию списания денежных средств со счета покупателя.
    • Положительная сумма (со знаком "+") означает операцию занесения денежных средств на счет клиента.
    ОписаниеОписание содержания операции.
    КомментарииПроизвольный комментарий.
