    • Форма создания/редактирования уровня доступа предназначена для настройки параметров существующего или вновь создаваемого уровня доступа. Также данная форма используется для просмотра параметров системных уровней доступа.

    Контекстная панель

    Кнопка Описание
    Список уровней доступа Переход на страницу со списком уровней доступа.
    Добавить Переход на страницу с формой создания нового уровня доступа. Кнопка отображается при редактировании существующего уровня.
    Копировать Копирование редактируемого уровня.
    Удалить Удаление редактируемого уровня доступа.

    Закладка "Параметры"

    На данной закладке выполняется настройка основных параметров уровня доступа.

    Поле Описание
    *Название Осмысленное название уровня доступа.
    *Модуль Указывается модуль, к которому относится уровень доступа.
    Системный Отметка о том, что уровень является системным или нет.
    Привязка Выберите привязку уровня к файлам и папкам или же ко всему модулю.
    Буква Буква уровня доступа.
    Описание Произвольное описание уровня.

    * - Поля, обязательные для заполнения.

    Закладка "Включаемые операции"

    Поля формы служат для настройки операций, которые возможно проводить, имея данный уровень доступа.

    Поле Описание
    [Название операции] Для возможности выполнения данной операции при этом уровне доступа необходимо, чтобы она была отмечена флагом.

    Кнопки управления

    Кнопка Описание
    Сохранить Сохранение параметров уровня доступа. Переход на страницу со списком уровней.
    Применить Сохранение параметров уровня доступа. Продолжение редактирования.
    Отменить Отмена внесённых изменений. Возврат первоначальных значений параметров.
