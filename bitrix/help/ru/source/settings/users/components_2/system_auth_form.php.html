Компоненты 2.0::Форма авторизации

Данный компонент служит для вывода формы авторизации. Используется обычно в шаблоне дизайна сайта. Компонент является стандартным и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:system.auth.form","",Array(
     "REGISTER_URL" => "register.php",
     "PROFILE_URL" => "profile.php"
     )
);?>

Описание параметров

Дополнительно
REGISTER_URL
Путь к странице регистрации пользователя.


PROFILE_URL
Путь к странице с параметрами пользователя.
