Список пользователей

Форма Список пользователей находится в административной части сайта: Настройки > Пользователи > Список пользователей. Данная форма содержит полный список пользователей сайта и позволяет:

  • добавлять новых пользователей сайта;
  • переходить к управлению параметрами и правами доступа пользователей;
  • удалять пользователей сайта.

Фильтр

Форма фильтра используется для выбора из списка пользователей в соответствии с указанными условиями. Нижеследующая таблица описывает параметры, по которым может выполняться поиск.

ПараметрОписание
НайтиПозволяет найти пользователей по их основным параметрам: логину, e-mail или имени, фамилии, отчеству. Это поле присутствует, даже если фильтр свернут.
IDИдентификатор пользователя. Соответствует полю ID в списке пользователей. Допустимо использование сложного поиска.
Дата измененияПозволяет отобрать пользователей, параметры которых были изменены в течение интересующего вас периода.
Последняя авторизацияПозволяет осуществить поиск пользователей по дате последней авторизации.
АктивенОпределяется, какие пользователи будут участвовать в поиске:
  • активные;
  • неактивные;
  • все (активные и неактивные).
ЛогинПоле позволяет осуществить отбор пользователей по логину. Допустимо использование сложного поиска.
E-MailСлужит для отбора пользователей по адресу электронной почты. Допустимо использование сложного поиска.
Фамилия, имя, отчествоСлужит для отбора пользователей по имени. Допустимо использование сложного поиска.
Поля профайлаПозволяет осуществлять поиск пользователей по другим полям профиля: телефон, номер ICQ и пр. Допустимо использование сложного поиска.
ГруппыСписок групп, пользователей из которых следует отобрать. Для выбора нескольких групп удерживайте клавишу Ctrl.

Для того чтобы отобрать пользователей по заданным критериям поиска, нажмите кнопку "Найти". Для отображения всех групп нажмите кнопку "Отменить".
