Создание и редактирование избранного

В этом разделе

  • Контекстная панель
  • Форма редактирования
  • Кнопки управления

    • Форма создания/редактирования избранного предназначена для настройки параметров существующей или вновь создаваемой ссылки.

    Для того чтобы пользователь имел право добавлять страницы административного раздела в избранное, он должен:

    • Иметь доступ к административному разделу сайта;
    • Иметь уровень доступа Изменение своего профайла в настройках главного модуля.

    Контекстная панель

    Кнопка Описание
    Список избранного Переход на страницу со списком избранного.
    Добавить Переход на страницу с формой создания новой ссылки. Кнопка отображается при редактировании существующей ссылки.
    Удалить Удаление редактируемой ссылки.

    Форма редактирования

    Поле Описание
    Изменено Дата и время последнего изменения ссылки. А также ID, логин и имя пользователя, который внес изменения. ID представлено в виде ссылки на страницу с формой редактирования параметров пользователя. Отображается в режиме редактирования ссылки.
    Создано Дата и время создания ссылки. А также ID, логин и имя пользователя, который создал ссылку. ID представлено в виде ссылки на страницу с формой редактирования параметров пользователя. Отображается в режиме редактирования ссылки.
    *Наименование Наименование ссылки, которое будет отображаться в списке избранного.
    *Ссылка Путь к избранной странице.
    Порядок сортировки Относительный "вес" ссылки. Определяет положение ссылки в общем списке.
    Доступна на языке интерфейса Устанавливается, на каком языке интерфейса будет доступна создаваемая ссылка.
    Комментарий Произвольный комментарий к создаваемой ссылке.
    Настройки администратора
    Эта часть формы отображается при редактировании избранной ссылки администратором сайта.
    Ссылка общая для всех пользователей Опция определяет, отображать ли создаваемую ссылку для всех пользователей, имеющих доступ к административном разделу или только владельцу ссылки. Если данная опция отмечена, то доступно поле Доступна при наличии прав на модуль, а поле Принадлежит пользователю закрыто.
    Принадлежит пользователю Доступно для ввода при снятой опции Ссылка общая для всех пользователей. С помощью кнопки ... можно выбрать одного из пользователей системы. По умолчанию в данное поле вписывается создатель ссылки.
    Доступна при наличии прав на модуль Если опция Ссылка общая для всех пользователей отмечена, то в данном поле можно установить один из модулей системы, и только при наличии прав доступа к этому модулю ссылка будет доступна. Можно выбрать пункт (не проверять).

    * - Поля, обязательные для заполнения.

    Кнопки управления

    Кнопка Описание
    Сохранить Сохранение параметров ссылки. Переход на страницу со списком избранного.
    Применить Сохранение параметров ссылки. Продолжение редактирования.
    Отменить Отмена внесённых изменений. Возврат первоначальных значений параметров.
