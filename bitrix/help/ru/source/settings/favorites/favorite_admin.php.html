    • Управление избранными страницами осуществляется на странице Список избранного в административном разделе сайта: Настройки > Избранное.

    Также на страницу Список избранного можно перейти по ссылке административной панели Избранное > Организовать избранное...

    Форма Список избранного предназначена для:

    • просмотра списка избранного;
    • редактирования избранных ссылок;
    • создания и удаления избранных ссылок.

    Фильтр

    Форма фильтра используется для выбора из списка избранного тех ссылок, которые соответствуют указанным условиям. Нижеследующая таблица описывает параметры, по которым может выполняться поиск.

    ПараметрОписание
    Заголовок ссылкиПоиск ссылок по заголовку. Допустимо использование сложного поиска.
    URL ссылкиПоиск избранного по URL ссылок. Допустимо использование сложного поиска.
    IDИдентификатор ссылки избранного. Соответствует полю ID списка избранного. Допустимо использование сложного поиска.
    Язык интерфейсаОпределяется, какому языку будут соответствовать избранные ссылки в результатах поиска:
    • все - будут отображаться избранные ссылки, соответствующие всем языкам системы.
    • далее в выпадающем списке перечислены все языки, установленные в системе. По каждому из них можно осуществлять поиск.
    Дата измененияПозволяет отобрать избранные ссылки, параметры которых были изменены в течение интересующего вас периода.
    Кем измененоПозволяет отобрать избранные ссылки по пользователю, который изменил параметры избранного. Допустимо использование сложного поиска.
    Кем созданоПозволяет отобрать избранные ссылки по пользователю, который их создал. Допустимо использование сложного поиска.
    ОписаниеДопустимо использование сложного поиска.
    Общая ссылкаОпределяется, какие ссылки будут отображаться в результатах поиска:
    • да - общие для всех пользователей;
    • нет - ссылки, не являющиеся общими для всех;
    • все - все ссылки (и общие, и необщие).
    Принадлежит пользователюПозволяет отобрать избранные ссылки по пользователю, который является их владельцем. Допустимо использование сложного поиска.
    Требуются права на модульПозволяет установить один из модулей системы, права на доступ к которому требуются для перехода по избранным ссылкам.

    Для того чтобы отобрать избранные ссылки по заданным критериям поиска, нажмите кнопку "Найти". Для отображения всех ссылок нажмите кнопку "Отменить".

    Контекстная панель

    ПунктОписание
    ДобавитьПереход к форме создания новой избранной ссылки.
    ExcelЭкспортирует данные из отображаемой таблицы в формат MS Excel.
    НастроитьПозволяет перейти к диалогу настройки внешнего вида отчетной формы.

    Список групп пользователей

    ПолеОписание
    Колонка флажковПоле предназначено для выбора избранных ссылок, к которым предполагается применить какое-либо правило или действие.
    ДействиеДействия над группой пользователей:
    • Изменить - переход к редактированию ссылки;
    • Удалить - удаление избранной ссылки.
    ЗаголовокЗаголовок избранной ссылки. Представлен в виде ссылки на избранную страницу.
    СсылкаСсылка на избранную страницу. По данной ссылке осуществляется переход к редактированию параметров избранной ссылки.
    Сорт.Индекс сортировки. Определяет положение ссылки в общем списке.
    ЯзыкЯзык интерфейса, на котором будет доступна избранная ссылка.
    Общая"Да", если ссылка является общей для всех пользователей. "Нет", если не является.
    ВладелецID, логин и имя пользователя, который является владельцем ссылки. ID представлено в виде ссылки на форму редактирования параметров пользователя.
    МодульПрава на доступ к какому модулю необходимы, чтобы данная ссылка была доступна.
    ИзмененоДата и время изменения записи о ссылке.
    КемID, логин и имя пользователя, изменившего ссылку.
    IDИдентификатор избранной ссылки.
