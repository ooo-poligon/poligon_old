Страница "Контроль файлов" используется для осуществления контроля за изменением файлов в системе. Вы можете ввести известное только вам слово или строку и произвести расчёт контрольных сумм всех файлов, расположенных на вашем сайте. Введенное слово нигде не сохраняется. Полученный журнал (файл с верификационными данными) рекомендуется сохранять на локальном компьютере, чтобы исключить его потерю. В дальнейшем если при выполнении контроля файлов будет определено, что текущее состояние системы не соответствует исходному (сохраненному в файле с верификационными данными), это может означать, что проверяемые файлы могли быть подвергнуты нежелательному изменению.

Перейти на страницу контроля файлов можно из административного меню: Настройки > Инструменты > Контроль файлов. Для удобства работы страница разделена на закладки, на каждой из которых выполняется определённый набор действий.

Закладка "Проверка файлов"

Форма "Проверка файлов" служит для сбора информации о файлах сайта и последующей проверки неизменности этих файлов со времени последнего запуска скрипта контроля целостности файлов.

ПолеОписание
*ПарольПароль, используемый при сборе информации о текущем состоянии системы. В дальнейшем указанный пароль будет использоваться для выполнения контроля файлов.
Пароль должен состоять только из символов латинского алфавита и цифр. Рекомендуемая длина пароля не менее 10 символов.
*Пароль ещё разПодтверждение пароля.
*Обработать папкиВ этом поле выбираются директории, участвующие в контроле.
  • Ядро продукта (папка /bitrix/modules/);
  • Служебная область (папка /bitrix/, за исключением /bitrix/modules/ и /bitrix/cache/);
  • Публичная часть (все папки сайта, за исключением /bitrix/ и /upload/).

Закладка "Загрузка файла"

Форма "Загрузка файла" служит для загрузки и выгрузки файла с верификационными данными, т.е. с данными о состоянии файлов системы.

ПолеОписание
Поле с информацией о созданном файле с верификационными данными и ссылкой для выгрузки файла на локальный компьютер. Данное поле отображается после выполнения сбора информации о файлах системы.

Примечание: Для исключения возможности нежелательного изменения файл с верификационными данными рекомендуется хранить вне системы, например, на локальном компьютере.
Файл с верификационными даннымиДля выполнения контроля предварительно нужно загрузить данные об исходном состоянии файлов системы. С помощью кнопки "Обзор" в данном поле указывается имя и путь к загружаемому файлу.
Распаковать из архива (tar.gz)Если на сайте доступна библиотека Zlib, то установка флага в данное поле позволяет:
  • выполнять автоматическую архивацию выгружаемого на локальный компьютер файла с верификационными данными;
  • автоматически распаковывать загружаемый в систему файл с верификационными данными.

Закладка "Целостность скрипта"

Форма "Целостность скрипта контроля" служит для проверки целостности самого скрипта контроля целостности файлов.

При первом запуске скрипта введите в эту форму произвольный пароль, а так же произвольное кодовое (ключевое) слово, и нажмите на кнопку "Установить новый ключ". При повторных запусках скрипта введите в этой форме заданный при установке ключа пароль.

ПолеОписание
*ПарольПароль для проверки целостности скрипта. Состоит из латинских букв и цифр. Рекомендуемая длина пароля не менее 10 символов.
*Пароль ещё разПодтверждение пароля.
*Новый ключПроизвольное кодовое слово (отличное от пароля), для идентификации возможных изменений скрипта.

Кнопки управления

КнопкаОписание
Установить новый ключЗадание нового кодового слова для проверки целостности скрипта.
Проверить целостность скриптаВыполнение проверки целостности скрипта.

В сообщении о результатах проверки скрипт должен указать кодовое слово, которое вы ввели при установке ключа. Если вы не увидели своего кодового слова в сообщении о результатах проверки, то скрипт контроля целостности файлов скомпрометирован (т.е. он был изменен и его результатам доверять нельзя).

Важно! Если вы установили обновления через Систему обновлений и скрипт контроля целостности файлов при этом обновился, то необходимо заново установить ключ.
