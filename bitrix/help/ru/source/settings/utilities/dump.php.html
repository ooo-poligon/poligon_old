Резервное копирование

В продукте «1С-Битрикс: Управление сайтом» для решения задачи переноса сайта на удаленный хостинг имеется специальный механизм резервного копирования, предоставляющий следующие возможности:

  • создание архивной версии файлов сайта (формат tar.gz);
  • исключение из архива папки, содержащей ядро продукта;
  • исключение из архива файлов, размер которых превышает некоторое заданное значение;
  • создание дампа базы данных (формат tar.gz);
  • исключение из дампа таблиц статистики и поискового индекса.

Выполнить перечисленные действия можно на странице Резервное копирование ( Настройки > Инструменты > Резервное копирование).

Форма копирования

Данная форма служит для задания параметров резервного копирования.

ПолеОписание
Архивировать по шагамДанная опция позволяет архивировать данные по шагам. Если она отмечена, то необходимо указать значение следующего поля.
ШагКоличество секунд выполнения одного шага резервного копирования.
Файлы
Архивировать публичную частьОтмеченная опция позволяет архивировать публичную часть сайта.
Архивировать ядроЕсли вам необходима резервная копия ядра, то отметьте данную опцию.
Исключить из архива файлы размером болееФайлы, размер которых превышает указанное число (в килобайтах), не будут включены в архив.
База Данных
Архивировать базу данныхПри отмеченной опции будет архивирована база данных.
Исключить из архиваДанное поле позволяет исключить из архива таблиц статистики и поискового индекса.

Для начала процесса резервного копирования служит кнопка "Архивировать". Если вам необходимо будет прервать процесс копирования, то для этого воспользуйтесь кнопкой "Остановить".

Список резервных копий

ПолеОписание
Колонка флажковПоле предназначено для выбора файлов архива, к которым предполагается применить какое-либо правило или действие.
ДействиеДействия над файлами архива:
  • Скачать - скачивание файла на локальный компьютер;
  • Распаковать - распаковка файла архива;
  • Удалить - удаление файла.
ИмяИмя файла архива.
Размер файла, МБ Размер файла, указанный в мегабайтах. 
ИзмененДата и время изменения файла.

С помощью ссылки Скачать, расположенной на данной странице после таблицы со списком резервных копий, вы можете скачать скрипт восстановления restore.php.
