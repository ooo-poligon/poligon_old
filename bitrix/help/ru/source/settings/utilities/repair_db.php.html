Проверка БД

Форма Проверка/восстановление таблиц служит для проверки и восстановления таблиц базы данных. Восстановление таблиц работает только для базы данных MySQL и таблиц типа MyISAM.

Внимание! Не открывайте эту страницу одновременно в нескольких копиях - это может повредить вашу базу данных. Операция может занять длительное время, если размер базы данных велик!

Кнопка Проверить/восстановить таблицы запускает процесс проверки таблиц базы данных. В результате проверки выводится таблица, в которую входят следующие поля:

Поле Описание
Название таблицы Выведено название проверенной/восстановленной таблицы.
Количество записей Количество записей в таблице.
Размер таблицы (байт) Выводится размер таблицы в байтах.
Результат проверки Выводится статус проверки. Сообщение зеленого цвета (OK). Сообщения красного цвета выводятся при неполадках с таблицей.
Результат восстановления Сообщение выводится в случае, если таблицу пришлось восстанавливать и проверять. Также выводится сообщение об успешности этого действия.
