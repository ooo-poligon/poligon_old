Создание и редактирование агента

В этом разделе

  • Контекстная панель
  • Форма редактирования
  • Кнопки управления

    • На данной странице вы можете создать нового или изменить параметры существующего агента.

    Контекстная панель

    КнопкаОписание
    Список агентов Переход на страницу со списком агентов.
    ДобавитьПереход на страницу с формой создания нового агента. Кнопка отображается при редактировании существующего агента.
    УдалитьУдаление редактируемого агента.

    Форма редактирования

    ПолеОписание
    Дата последнего запуска Дата и время последнего запуска агента. Данное поле отображается при редактировании агента.
    *Дата и время следующего запуска Указывается дата и время следующего запуска агента.
    АктивенПризнак активности агента. Если агент неактивен, то он не может быть запущен.
    МодульНазвание модуля, к которому относится агент.
    *Функция агента Полное название функции агента.
    ID пользователя Задается идентификатор пользователя. Если пользователь не выбран, то владельцем агента является система.
    СортировкаОтносительный "вес" агента. Чем меньше указанное число, тем выше в списке агентов будет показан данный агент.
    ПериодическийПри отмеченной опции агент будет периодическим и вызываться через интервал, заданный в следующем параметре.
    ИнтервалИнтервал вызова агента, задаваемый в секундах.

    * - Поля, обязательные для заполнения.

    Кнопки управления

    КнопкаОписание
    СохранитьСохранение параметров агента. Переход на страницу со списком всех агентов.
    ПрименитьСохранение параметров агента. Продолжение редактирования.
    ОтменитьОтмена внесённых изменений. Возврат первоначальных значений параметров.
