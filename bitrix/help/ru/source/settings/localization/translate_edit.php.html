Перевод сообщений

В этом разделе

  • Контекстная панель
  • Поля формы
  • Кнопки управления

    • На странице показаны языковые сообщения для языков, загруженных в систему. Вы можете редактировать языковые сообщения для показанных языков, а также удалять сообщения, установив флаг Удалить.

    Контекстная панель

    КнопкаОписание
    СписокПереход к списку файлов языковых сообщений.
    Показать все сообщения/ Показать только непереведенныеПереключение между режимами показа сообщений: отображать либо все сообщения, либо только непереведенные. Режим отображения по умолчанию устанавливается в настройках модуля.

    Поля формы "Перевод сообщений"

    ПолеОписание
    Имя файлаИмя файла языковых сообщений.
    Всего сообщенийКоличество языковых сообщений в файле.
    Не переведеноКоличество непереведенных сообщений для каждого языка.
    Список сообщений
    IDИдентификатор языкового сообщения.
    УдалитьЧтобы удалить языковое сообщение, установите флаг в это поле и нажмите кнопку Применить.
    [Язык]Текст языкового сообщения для каждого языка.

    В исходных строках сообщений возможно наличие шаблонов вида "#ID#", "#COUNT#". Если в оригинале подобные шаблоны присутствуют, в переводе их также необходимо сохранить. В момент использования данного сообщения шаблон будет заменен на конкретное значение.

    Важно! Использование каких-либо PHP команд в языковых сообщениях запрещено. Однако, допустимо использование всех HTML-тегов.
    Удалить всеЧтобы удалить все сообщения, установите флаг в это поле и нажмите кнопку Применить.

    Кнопки управления

    КнопкаОписание
    СохранитьСохранение выполненных изменений. Переход к списку файлов с языковыми сообщениями.
    ПрименитьСохранение выполненных изменений. Продолжение редактирования языковых сообщений.
    ОтменитьОтмена внесённых изменений. Возврат первоначальных значений.
