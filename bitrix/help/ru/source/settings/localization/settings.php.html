Настройки

Данная форма (Настройки > Настройки продукта > Настройки модулей > Перевод) предназначена для задания общесистемных параметров модуля перевода.

Закладка "Настройки"

ПараметрОписание
Начальный каталогУкажите каталог, который должен открываться при переходе на страницу Перевод языковых файлов.
Автоматически считать разницу в языковых файлахЕсли флаг установлен, то при открытии раздела будет показано общее количество языковых файлов, количество переведенных и непереведенных фраз в нем и его подразделах (кнопка Показать различия отображаться не будет).

Если флаг снят, то при открытии языковых файлов какого-либо раздела будет отображен только список его подразделов, а также кнопка Показать различия (Не показывать различия) на контекстной панели.
При редактировании показать по умолчанию только ошибкиЕсли флаг установлен, при открытии файла на редактирование показываются только непереведенные фразы (ошибки).

Если флаг сброшен, при открытии файла показываются все фразы.

Сменить режим отображения можно нажатием соответствующей кнопки на форме редактирования фраз.

Закладка "Доступ"

Эта форма позволяет настроить права доступа для всех имеющихся в системе типов пользователей, кроме администратора, который всегда имеет полный доступ. Кроме того, настроив уровень доступа "по умолчанию", можно легко изменить политику доступа сразу для всех пользователей, которым присвоен уровень доступа "по умолчанию".

ПолеОписание
По умолчаниюПозволяет изменять настройки "по умолчанию", влияющие на все группы пользователей, имеющие настройку доступа "по умолчанию".
[Группа пользователей]Изменение настроек доступа для конкретной группы. В качестве уровней доступа возможен выбор между:
  • закрыт - доступ невозможен;
  • просмотр всех данных модуля - доступ открыт только на просмотр файлов, но не на их изменение;
  • запись - доступ открыт и на чтение, и на изменение файлов.
