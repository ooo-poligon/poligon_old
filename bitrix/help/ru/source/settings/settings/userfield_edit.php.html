    Форма редактирования

    ПолеОписание
    IDУникальный идентификатор поля. Присваивается после сохранения.
    *Тип данныхВыбирается один из возможных типов данных:
    • Целое число
    • Дата/Время
    • Файл
    • Строка
    • Список
    • Число
    • Да/Нет

    Обратите внимание: параметр можно задать только для нового поля, после сохранения изменить его нельзя.

    *ОбъектКод объекта, к которому относится поле. Например, для пользователя объектом является USER, для раздела инфоблока объект строится так: IBLOCK_#SECTION_ID#_SECTION.
    *Код поляКод самого поля. Можно задать только для нового поля. Начинается с UF_.
    XML_IDУказывается идентификатор для работы с внешними источниками.
    СортировкаИндекс сортировки, определяющий положение поля в общем списке.
    МножественноеЯвляется ли значение поля множественным. Указывается только для нового поля.
    ОбязательноеПризнак обязательности поля для заполнения.
    Показывать в фильтре спискаВыбирается одна из возможностей:
    • не показывать - поле не будет показано в фильтре;
    • точное совпадение - поле будет показано в фильтре. Для нахождения элемента требуется точное совпадение условию поиска;
    • поиск по маске - поле будет показано в фильтре. Для поиска элемента в списке по данному полю можно использовать маску;
    • поиск по подстроке - поле будет показано в фильтре. Для нахождения элемента выполнятся поиск по подстроке.
    Не показывать в спискеПри отмеченной опции поле не будет отображаться в списке элементов в месте применения.
    Не разрешать редактирование пользователемПри отмеченной опции значение поля нельзя редактировать, только вычислять и устанавливать через API.
    Значения поля участвуют в поискеПри отмеченной опции значения поля будут участвовать в поиске в публичной части после переиндексации сайта.
    Дополнительные настройки поля (зависят от типа)
    Целое число
    Значение по умолчаниюУкажите, если требуется, значение поля по умолчанию.
    Размер поля ввода для отображенияУкажите размер поля, каким он будет отображен в форме, где используется данное поле. По умолчанию введено 20.
    Минимальное значение (0 - не проверять)Укажите, если требуется, минимальное значение поля. При вводе значения поля это будет проверяться. Введите 0, чтобы не проверялось минимальное значение.
    Максимальное значение (0 - не проверять)Укажите, если требуется, максимальное значение поля. При вводе значения поля будет это проверяться. Введите 0, чтобы не проверялось максимальное значение.
    Дата/Время
    Значение по умолчаниюВыбирается один из возможных вариантов:
    • нет - значения по умолчанию не будет;
    • текущая дата - значением по умолчанию будет текущая дата на момент заполнения;
    • также значение по умолчанию может быть установлено вручную с использованием календаря.
    Файл
    Размер поля ввода для отображенияУкажите размер поля, каким он будет отображен в форме, где используется данное поле. По умолчанию введено 20.
    Максимальные ширина и высота для отображения в спискеПоле используется для изображений. Если размер изображения больше заданного, то изображение будет уменьшено (<img src="..." width="<w>" height="<h>">).
    Максимально допустимый размер для показа в списке (0 - не ограничивать)Ограничение указывается в байтах. Если больше указанного, то в списке будет выведена просто ссылка, а не показываться картинка.
    Максимально допустимый размер файла для загрузки (0 - не проверять)Поле служит для ограничения размера файла. Задайте максимальный размер файла в байтах. 0 - неограниченный размер.
    РасширенияВведите допустимые расширения файлов через запятую.
    Строка
    Значение по умолчаниюМожно определить значение по умолчанию.
    Размер поля ввода для отображенияУкажите размер поля, каким он будет отображен в форме, где используется данное поле. По умолчанию введено 20.
    Количество строчек поля вводаУкажите количество строк поля ввода. По умолчанию равно 1.
    Минимальная длина строки (0 - не проверять)Есть возможность ограничить размер строки по минимальному значению. Если 0, то не ограничивать.
    Максимальная длина строки (0 - не проверять)Есть возможность ограничить размер строки по максимальному значению. Если 0, то не ограничивать.
    Регулярное выражение для проверкиМожно ввести регулярное выражение для ограничения вводимых значений поля. Регулярное выражение необходимо ограничить любыми удобными символами, например, /^T.+N$/ либо #^T.+N$#.
    Список
    Внешний видУказывается тип внешнего вида: Список или Флажки
    Высота спискаУказывается высота списка
    Пункты списка задаются после сохранения изменений (по кнопке Применить) на закладке Список.
    Число
    Точность (количество знаков после запятой)Определяется точность числа - количество знаков после запятой.
    Значение по умолчаниюМожно определить значение по умолчанию.
    Размер поля ввода для отображенияУкажите размер поля, каким он будет отображен в форме, где используется данное поле. По умолчанию введено 20.
    Минимальное значение (0 - не проверять)Можно установить минимальное допустимое значение. Если установлено 0, то не проверять.
    Максимальное значение (0 - не проверять)Можно установить максимальное допустимое значение. Если установлено 0, то не проверять.
    Да/Нет
    Значение по умолчаниюУстанавливается, что будет значением по умолчанию, да или нет.
    Внешний видВыбирается один из возможных внешних видов поля:
    • Флажок;
    • Радио кнопки;
    • Выпадающий список.
    Языковые настройки
    ЯзыкПодпись в форме редактированияЗаголовок в спискеПодпись фильтра в спискеСообщение об ошибке (не обязательное)Помощь
    [1][2][3][4][5][6]
    Язык [1]Перечислены все языки, установленные в системе.
    Подпись в форме редактирования [2]Можно определить для каждого из установленных языков подпись поля в форме редактирования. Если поле пусто в форме редактирования будет выведен код поля.
    Заголовок в списке [3]Можно определить для каждого из установленных языков подпись поля в списке, т.е. в отчетной форме. Если поле пусто в списке поле будет представлено кодом (в случае, если настроено, что поле должно отображаться в списке).
    Подпись фильтра в списке [4]Можно определить для каждого из установленных языков подпись поля в фильтре, т.е. как поле будет отображено в форме поиска. Если поле пусто в фильтре поле будет представлено кодом (в случае, если настроено, что поле должно отображаться в фильтре).
    Сообщение об ошибке (не обязательное) [5]Сообщение об ошибке, которое будет выводиться в форме, где применяется поле. Действует для типа данных Строка, например, при несоответствии введенного значения регулярному выражению, определённому для поля.
    Помощь [6] Поле для ввода сообщений помощи. Поле ведется внутри системы, но пока нигде не выводится. Вы можете вывести его самостоятельно там, где необходимо.
