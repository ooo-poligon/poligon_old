На данной странице вы можете создать новое или изменить параметры существующего правила обработки адресов.

Контекстная панель

КнопкаОписание
Список записейПереход на страницу списком правил обработки адресов.
ДобавитьПереход на страницу с формой создания нового правила обработки адресов. Кнопка отображается при редактировании существующего правила.
УдалитьУдаление редактируемого правила.

Форма редактирования

ПолеОписание
*УсловиеЗадайте условие, по которому будет работать правило обработки адресов.
КомпонентУказывается полное имя компонента (пространство_имен:имя_компонента), для которого будет работать правило.
ФайлАдрес страницы, которая должна быть показана с учетом правила.
ПравилоПравило вызова страницы, путь к которой указан в поле Файл.

* - Поля, обязательные для заполнения.

Кнопки управления

КнопкаОписание
СохранитьСохранение параметров правила обработки адресов. Переход на страницу со списком правил.
ПрименитьСохранение параметров правила. Продолжение редактирования.
ОтменитьОтмена внесённых изменений. Возврат первоначальных значений параметров.
