Компоненты 2.0::Элемент управления "Календарь"

Компонент используется для ввода даты/времени.

Пример вызова

<?$APPLICATION->IncludeComponent("bitrix:main.calendar","",Array(    
     "SHOW_INPUT" => "Y",
     "INPUT_NAME" => "date_fld",
     "INPUT_NAME_FINISH" => "date_fld_finish",
     "INPUT_VALUE" => "",
     "INPUT_VALUE_FINISH" => "", 
     "SHOW_TIME" => "N" 
     )
);?>
Параметры компонента
SHOW_INPUT
[Y|N] Параметр служит для выбора способа отображения календаря:
  • Y- показ иконки календаря с полем ввода;
  • N - показ только иконки календаря.

INPUT_NAME
Название переменной первого поля временного интервала.

INPUT_NAME_FINISH
Название переменной второго поля временного интервала.

INPUT_VALUE
Значение по умолчанию первого поля временного интервала.

INPUT_VALUE_FINISH
Значение по умолчанию второго поля временного интервала.

SHOW_TIME
[Y|N] При выборе значения "Y" пользователь имеет возможность указать не только дату, но и время.
