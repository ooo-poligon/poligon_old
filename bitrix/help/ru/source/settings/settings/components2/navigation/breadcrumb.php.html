Навигационная цепочка

Компонент позволяет выводить навигационную цепочку в шаблоне. Настройки этого компонента позволяют регулировать уровень вложенности разделов (корень сайта - ноль), начиная с которого начнет выводиться навигационная цепочка. Также возможно переопределить путь, для которого цепочка будет построена и т.д.

Пример вызова

<?$APPLICATION->IncludeComponent(
"bitrix:breadcrumb",
".default",
Array(
	"START_FROM" => "0", 
	"PATH" => "", 
	"SITE_ID" => "" 
)
);?>

Описание параметров

Дополнительные настройки
START_FROM
Номер пункта, начиная с которого будет построена навигационная цепочка. В поле может быть задано числовое значение, определяющее уровень вложенности раздела на сайте, начиная с которого начнется построение навигационной цепочки. Например, 0 (ноль) в данном поле означает, что построение навигационной цепочки начнется от корня сайта. В случае если заполнено поле Путь, для которого будет построена навигационная цепочка, то номер пункта считается в данном пути.

PATH
Путь, для которого будет построена навигационная цепочка (по умолчанию, текущий путь). В данном поле можно переопределить путь, для которого будет построена навигационная цепочка. Если поле не заполнено, то навигационная цепочка будет строиться для текущего пути, в противном случае для пути, определенном в данном поле.

SITE_ID
Идентификатор сайта. Параметр устанавливается в случае многосайтовой версии, когда DOCUMENT_ROOT у сайтов разный.
