Меню

Данный компонент позволяет располагать меню в шаблоне сайта, устанавливать для него один из заданных шаблонов, указывать уровень вложенности и т.д.

Пример вызова

<?$APPLICATION->IncludeComponent(
"bitrix:menu",
"horizontal_multilevel",
Array(
	"ROOT_MENU_TYPE" => "top", 
	"MAX_LEVEL" => "3", 
	"CHILD_MENU_TYPE" => "left", 
	"USE_EXT" => "Y" 
	)
);?>

Описание параметров

Основные параметры
ROOT_MENU_TYPE
Указывается один из заданных в системе (в настройках модуля Управление структурой) типов меню первого уровня.

Дополнительные настройки
MAX_LEVEL
Уровень вложенности меню (возможные значения от 1 до 4).
Примечание: уровень вложенности, отличный от 1, можно задать только для многоуровневых шаблонов меню. 

CHILD_MENU_TYPE
Тип меню для остальных уровней. Аналогично с типом меню для первого уровня, определяется тип меню для остальных уровней. В выпадающем списке выбирается один из заданных в системе (в настройках модуля Управление структурой) типов меню. Также можно указать другой тип меню латинскими буквами.

USE_EXT
[Y|N] Подключать файлы с именами вида .тип_меню.menu_ext.php. Если опция отмечена, то при вызове меню из шаблона в каждом разделе будет происходить проверка на наличие в данном разделе файла с именем .тип_меню.menu_ext.php. В таком файле определяется массив параметров, который может быть выведен в качестве пунктов меню указанного типа.
Например, компонент Пункты меню позволяет определить массив, состоящий из разделов определенного инфоблока.
