Карта сайта

С помощью данного компонента осуществляется показ карты сайта. Компонент является стандартным и входит в дистрибутив модуля.

Пример вызова

<?$APPLICATION->IncludeComponent("bitrix:main.map","",Array(
     "LEVEL" => "3",
     "COL_NUM" => "1",
     "SHOW_DESCRIPTION" => "Y",
     "SET_TITLE" => "Y",
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "3600"
     )
);?>

Описание параметров

Настройки кеширования
CACHE_TYPE
Тип кеширования:
  • A - Авто: действует при включенном кешировании в течение заданного времени;
  • Y - Кешировать: для кеширования необходимо определить только время кеширования;
  • N - Не кешировать: кеширования нет в любом случае.

CACHE_TIME
Время кеширования, указанное в секундах.

Дополнительные настройки
SET_TITLE
[Y|N] При выборе значения "Y" будет установлен заголовок страницы Карта сайта.

Настройки компонента
LEVEL
Максимальный уровень вложенности разделов при показе в карте сайта.

COL_NUM
Количество колонок карты сайта.

SHOW_DESCRIPTION
[Y|N] При выборе значения "Y" будут выведены описания разделов, если они заданы.
