Интерфейс

Управление оформлением административного раздела осуществляется на странице Настройка интерфейса пользователя (Настройки > Настройки продукта > Интерфейс).

Закладка "Настройки"

ПолеОписание
Персональные настройки
*Визуальная тема интерфейсаВ данном поле отображается список доступных тем: указывается название темы и имя папки, в которой хранятся файлы темы.
В поставку продукта входит визуальная тема По умолчанию (.default).
Разрешить контекстное меню в спискахДля того чтобы отображалось контекстное меню в списках (таблицах отчета), необходимо отметить данную опцию.
Показывать контекстное меню по нажатиюЕсли разрешено показывать контекстное меню в списках, то необходимо выбрать способ его вызова: по клику правой кнопки мыши или по клику правой кнопки мыши+Ctrl.
Меню "Пуск"
Количество часто используемых ссылокКоличество ссылок на часто используемые страницы, список которых отображается в третьей секции меню "Пуск".
Общие настройки
Установить данные настройки по умолчанию для всех пользователейПри отмеченной опции выбранная тема будет использоваться по умолчанию для всех зарегистрированных пользователей.

* - Поле, обязательное для заполнения.

Закладка "Удаление"

ПолеОписание
Очистить все собственные настройки интерфейсаЕсли вам необходимо очистить все собственные настройки интерфейса, то отметьте данную опцию.
Очистить все настройки по умолчаниюПри отмеченной опции будут очищены все настройки по умолчанию.
Очистить персональные настройки всех пользователейДанная опция позволяет очистить персональные настройки всех пользователей.

Для сохранения настроек нажмите "Применить".
