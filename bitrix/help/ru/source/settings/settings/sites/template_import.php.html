Импорт шаблона

В этом разделе

  • Поля формы
  • Кнопки управления

    • Форма Импорт шаблона предназначена для загрузки готовых шаблонов сайта в систему. Форма доступна по кнопке Загрузить шаблон со страницы Шаблоны сайта.

    Поля формы

    Поле Описание
    *Файл с шаблоном для загрузки (*.gz): С помощью кнопки "Обзор" на локальном компьютере выбирается файл с шаблоном сайта.
    Код шаблона При загрузке по умолчанию шаблон будет распакован и помещен в папку с именем, соответствующим имени загружаемого файла. Например, если имя загружаемого файла template1.tar.gz, то шаблон будет автоматически помещен в папку /template1/, а самому шаблону будет присвоен идентификатор (ID) template1.

    Для того чтобы шаблону был присвоен другой идентификатор, а сам шаблон был размещён в папке с соответствующим именем, в данном поле указывается нужный код шаблона. Например, demo.
    Привязать как шаблон по умолчанию к сайту В данном поле можно выбрать сайт, для которого загружаемый шаблон будет использоваться как шаблон по умолчанию.
    Перейти после сохранения на страницу редактирования шаблона Для того чтобы после завершения загрузки шаблона перейти к его редактированию, установите флаг в этом поле.
