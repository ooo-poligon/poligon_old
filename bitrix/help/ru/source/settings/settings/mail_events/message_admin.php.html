Почтовые шаблоны

    • Форма Почтовые шаблоны служит для управления шаблонами почтовых сообщений. При возникновении на сайте события на отправку какого-либо письма, система ищет подходящий шаблон для такого письма. Если шаблон найден, то письмо отправляется в соответствии с этим шаблоном.

    Перейти к форме Почтовые шаблоны можно из административного меню: Настройки > Настройки продукта > Почтовые события > Почтовые шаблоны.

    Шаблоны почтовых сообщений различаются по типам сообщений и сайтам, для которых шаблоны предназначены.

    Каждый шаблон почтового сообщения может содержать два типа данных:

    • Неизменяемые данные (константы) не изменяются от письма к письму по этому шаблону. Другими словами, все получатели письма получат эти данные в таком виде, в котором они занесены в шаблон письма. Примером неизменяемых данных является подпись внизу письма "С уважением, служба поддержки".
    • Изменяемые данные (параметры) меняются от письма к письму в зависимости от разных условий. Другими словами, каждый получатель письма получит те данные, которые предназначены именно ему. Примером изменяемых данных является e-mail адрес получателя данного письма. Доступные для данного типа сообщения изменяемые данные перечислены в форме редактирования шаблона почтового сообщения в секции Доступные поля.

    ПолеОписание
    НайтиПоле позволяет найти требуемые почтовые шаблоны по наиболее типичным критериям, выбираемым в списке: тема, отправитель, получатель или содержимое шаблона.
    IDПоиск по идентификатору шаблона.
    Тип почтового событияПоиск шаблона по типу, к которому он относится.
    Дата изменения:(DD.MM.YYYY)Интервал даты последнего изменения почтового шаблона.
    СайтСайт(ы), на котором используется шаблон.
    АктивностьПоиск только активных, только неактивных или всех шаблонов.
    От когоАдрес отправителя почтового сообщения.
    КомуАдрес получателя почтового сообщения.
    Скрытая копия (BCC)Адрес получателя скрытой копии сообщения.
    ТемаПоиск шаблона по теме почтового сообщения.
    Тип тела сообщенияПоиск по типу тела сообщения: текст, html или по обоим типам.
    Тело сообщенияПоиск шаблона по содержимому тела сообщения.

    Список шаблонов

    ПолеОписание
    Колонка флажковПоле предназначено для выбора шаблонов, к которым предполагается применить какое-либо правило или действие.
    ДействияДоступные операции над шаблоном:
    • Изменить - изменение почтового шаблона;
    • Добавить копию - копирование почтового шаблона;
    • Удалить - удаление почтового шаблона.
    IDИдентификатор почтового шаблона.
    Дата изм.Дата и время последнего изменения шаблона.
    Акт.Признак активности. Если шаблон не активен, то данный шаблон не будет использоваться для генерации почтовых сообщений.
    СайтИдентификатор сайта, для отправки почтовых сообщений с которого используется данный шаблон.
    Тип почтового событияНазвание типа почтового события, при возникновении которого будет сгенерировано почтовое сообщение на основе данного шаблона.
    Тип почтового событияКод и название типа почтового события, при возникновении которого будет сгенерировано почтовое сообщение на основе данного шаблона.
    ТемаТема почтового сообщения. При генерации сообщения подставляется в поле Тема.
    ОтправительАдрес отправителя почтового сообщения.
    ПолучательАдрес получателя почтового сообщения.
    Скрытая копия (BCC)Адрес получателя скрытой копии сообщения.
    Тип тела сообщенияУказывается формат тела сообщения: текст или html.
