    • Данная форма предназначена для создания нового или изменения уже существующего типа почтового события.

    Контекстная панель

    Кнопка Описание
    Список типов Переход на страницу со списком типов почтовых событий.
    Добавить тип Переход к форме создания нового типа почтового события.
    Удалить тип Удаление редактируемого типа.
    Список шаблонов Переход к списку почтовых шаблонов, привязанных к редактируемому типу. Отображается при наличии привязанных шаблонов к типу.
    Добавить шаблон Переход к форме добавления почтового шаблона.

    Поля формы

    Тип события
    Поле Описание
    *Тип почтового события Кодовое название типа почтового события (латинскими буквами без пробелов).
    Для каждого из установленных в системе языков:
    ID Идентификатор типа события для каждого из языков.
    Сортировка Индекс сортировки, определяющий положение типа события в общем списке.
    Название Осмысленное название типа события.
    Описание Определяется набор специальных полей (placeholder'ов) которые могут быть использованы в почтовом шаблоне, например,
    #ID# - ID
    #IBLOCK_ID# - ID информационного блока

    * Поля обязательные для заполнения.

    Шаблоны

    На данной закладке выводятся все почтовые шаблоны, привязанные к данному типу. Выводится ID шаблона, представленный в виде ссылки на форму редактирования данного шаблона, название шаблона и сайт (или сайты), для отправки почтовых сообщений с которого используется данный шаблон.

    Кнопки управления

    Кнопка Описание
    Сохранить Сохранение внесённых изменений. Переход на страницу со списком типов почтовых событий.
    Применить Применение внесённых изменений. Продолжение редактирования почтового шаблона.
    Отменить Отмена внесённых изменений. Возврат первоначальных значений параметров.
