Типы почтовых событий

В этом разделе

  • Форма поиска
  • Контекстная панель
  • Список шаблонов

    • С помощью формы Типы почтовых событий можно добавлять новые типы почтовых событий, просматривать и изменять существующие, удалять ненужные. Каждый почтовый шаблон привязывается к определённому типу почтового события. Тип определяет набор специальных полей (placeholder'ов), которые могут быть использованы в почтовом шаблоне. В момент создания почтового события (т.е. готового сообщения) эти поля будут инициализированы конкретными значениями.

    Перейти к форме Типы почтовых событий можно из административного меню: Настройки > Настройки продукта > Почтовые события > Типы почтовых событий.

    Форма поиска

    ПолеОписание
    НайтиЭто поле позволяет найти требуемые типы почтовых шаблонов по наиболее типичным критериям, выбираемым в списке: тип шаблона, его название или описание.
    Идентификатор типа почтового событияПоиск типов почтовых событий по ID.
    Идентификатор шаблонаПоиск типов почтовых событий по ID шаблона.

    Для того чтобы отобрать типы почтовых событий по заданным критериям поиска, нажмите кнопку "Найти". Для отображения всех типов нажмите кнопку "Отменить".

    Контекстная панель

    КнопкаОписание
    Добавить тип Переход к форме создания нового типа.
    ExcelЭкспортирует данные из отображаемой таблицы в формат MS Excel.
    НастроитьПозволяет перейти к диалогу настройки внешнего вида отчетной формы.

    Список шаблонов

    ПолеОписание
    Колонка флажковПоле предназначено для выбора типов, к которым предполагается применить какое-либо правило или действие.
    ДействияДоступные операции над шаблоном:
    • Изменить - изменение типа почтового события;
    • Удалить - удаление почтового шаблона.
    IDИдентификаторы типа почтового события для каждого из языков, по которым тип определён.
    ЯзыкВыведены идентификаторы языков, для которых тип определён.
    ТипКодовое название типа почтового события (латинскими буквами без пробелов).
    НазваниеНазвание типа почтового события на текущем языке системы.
    ОписаниеОписание типа почтового события на текущем языке системы.
    ШаблоныПеречислены ID шаблонов, относящихся к данному типу.
