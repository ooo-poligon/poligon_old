Компоненты 2.0::Таблица курсов валют

Данный компонент служит для отображения таблицы курсов валют в соответствии с их форматом. Компонент является стандартным и входит в дистрибутив модуля.

Пример вызова


<?$APPLICATION->IncludeComponent("bitrix:currency.rates","",Array(
     "arrCURRENCY_FROM" => Array("EUR","USD"),
     "CURRENCY_BASE" => "RUB",
     "RATE_DAY" => "",
     "SHOW_CB" => "Y",
     "CACHE_TYPE" => "A",
     "CACHE_TIME" => "86400"
     )
);?>

Описание параметров

Настройки кеширования
CACHE_TYPE
Тип кеширования.
  • A - Авто
    Действует при включенном Автокешировании на странице Настройки кеширования и при установленном времени кеширования в поле CACHE_TIME.
  • Y - Кешировать
    При данном типе кеширования время кеширования (CACHE_TIME) должно быть устанолвено. Настройки кеширования не влияют на данный тип кеширования.
  • N - Не кешировать
    При выборе этого типа кеширование не происходит независимо от настроек кеширования.


CACHE_TIME
Время кеширования, указанное в секундах.
Дополнительно
arrCURRENCY_FROM
Перечислите валюты, курсы которых должны быть показаны.

CURRENCY_BASE
Валюта, по отношению к которой считаются курсы остальных валют (базовая валюта).

RATE_DAY
День, на который показывать курсы валют (в формате "Y-m-d"). Если не установлен - текущий курс.

SHOW_CB
[Y|N] При установленной опции отдельной табличкой будет выведен курс с ЦБ РФ.
