Компоненты 1.0::Таблица курсов валют

Компонент показывает таблицу курсов валют.

Пример вызова

$APPLICATION->IncludeFile("currency/show_rates.php",
    Array(
        "arrCURRENCY_FROM" => Array("USD"),
        // Массив 'исходных' валют для перевода
        "CURRENCY_BASE" => "RUB",
        // Валюта, к которой приводятся 'исходные' валюты
        "RATE_DAY" => "",
        // День, на который показывать
        // курсы валют (в формате 'Y-m-d')
	"SHOW_CB"	=> "Y",	// Показывать курс валют с ЦБ РФ
        "CACHE_TIME" => "0",
        // Время кэширования (сек.)
    )
);

Параметры

arrCURRENCY_FROM
Массив валют, курсы которых отображаются.

CURRENCY_BASE
Базовая валюта, т.е. валюта, курсы по отношению к которой считаются.

RATE_DAY
День, на который показывать курсы валют (в формате 'Y-m-d'). Если не установлен - текущий курс.
SHOW_CB
[Y|N] Показывать курс валют с ЦБ РФ. Если "Да", то отдельной табличкой будет выведен курс ЦБ РФ.

CACHE_TIME
Период (в секундах), на который кешируется результат. Установите это поле в ноль для отключения кеширования.
