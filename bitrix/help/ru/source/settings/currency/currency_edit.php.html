    Контекстная панель

    Кнопка Описание
    Список Переход на страницу со списком валют.
    Создать Переход к форме создания новой валюты. Кнопка отображается при редактировании существующей валюты.
    Удалить Удаление редактируемой валюты.

    Форма редактирования

    Поле Описание
    *Валюта Трехбуквенное обозначение валюты (например, RUB, EUR, USD).
    Номинал Номинал, или количество валюты, для которого устанавливается обменный курс. Обычно равен номиналу, устанавливаемому ЦБ РФ. Для большинства валют номинал будет равен 1 (например, 30.5 RUB за 1 USD). У некоторых валют он будет отличен от 1 (например, 14.25 RUB за 1000 BYR).
    *Курс по умолч. Курс валюты, который будет использоваться при конвертации валюты, если в форме "Курсы валют" для данной валюты не задан другой курс. Для базовой валюты курс должен быть равен 1. Для остальных валют курс устанавливается относительно базовой валюты. Другими словами, при добавлении валюты поставьте напротив этой валюты сумму, которую дают за нее в базовой валюте.

    Например, установите курс для RUB в 1, тогда курс для USD может быть равен 30.5, а для EUR - 34.2. Подробнее о курсах валют смотрите раздел помощи Курсы валют.
    Индекс сортировки Относительный "вес", влияющий на положение валюты в общем списке.
    Настройки валют для языков (English, Russian и т.д.)
    Название Полное название валюты для данного языка.
    Шаблон формата Шаблон, по которому суммы в данной валюте будут выводиться на экран.

    При выборе шаблона вывода в нижеследующие поля будут подставлены значения по умолчанию. При необходимости вы можете изменить значения этих полей.
    *Строка формата для вывода валюты Формат для вывода суммы в данной валюте на экран.

    Все символы строки формата выводятся как есть, за исключением символа "#". Символ "#" заменяется при выводе на реальную сумму (например, строка формата # руб за 1 шт. может быть выведена как 18 586,95 руб за 1 шт.). Формат вывода суммы зависит от описанных ниже полей.
    Десятичная точка при выводе Символ-разделитель целой и дробной части числа.
    Разделитель тысяч при выводе Символ-разделитель тысяч целой части числа. 

    Для разработчиков. Обратите внимание, что для обозначения символа-разделителя можно использовать шестнадцатеричное значение, например, xA0.

    Количество десятичных знаков Количество знаков в дробной части числа (например, количество копеек для RUB).

    * - Поля, обязательные для заполнения.

    Кнопки управления

    Кнопка Описание

    Применить Сохранение внесённых изменений. Продолжение редактирования параметров валюты.
    Отменить Отмена внесённых изменений. Возврат первоначальных значений параметров.
