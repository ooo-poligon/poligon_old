    • Эта форма предназначена для создания нового или изменения существующего правила сортировки результатов поиска.

    Примечание. Применение правила предполагает изучение системой поиска строки запроса; нахождение документов, соответствующих запросу; поиск правил, соответствующих найденным документам; в случае нахождения подходящих правил - ранжирование результатов поиска в соответствии с весами сортировки, указанных в правилах.

    Например, пользователь желает найти документы, соответствующие запросу "телефоны nokia", и существует правило:

    "модуль Информационные блоки :: тип блока Каталог :: блок Телефоны Nokia :: сорт. 1"

    После поиска система предпримет попытку найти подходящее запросу правило и обнаружит, что некоторые результаты поиска включают в себя элементы информационного блока Телефоны Nokia. Таким образом, к результатам поиска будет применено вышеуказанное правило, что повлечет за собой назначение веса сортировки 1 результатам из элементов информационного блока Телефоны Nokia. В большинстве случаев, назначение веса сортировки 1 автоматически поднимает результаты поиска наверх списка.

    Свойства правила - вкладка "Правило"

    ПолеОписание
    СайтВыберите сайт, в пределах которого должно действовать данное правило. Правило не будет применяться к результатам поиска на других сайтах.
    МодульМодуль, к элементам которого применимо данное правило. Система позволяет определять правила, которые будут применяться к:
    • статическим файлам сайта. Область действия правила ограничивается обязательно указываемым в параметре 3 полным путём к требуемому файлу;
    • информационным блокам. Область действия правила может быть ограничена: определённым типом блоков, определённым информационным блоком, а также элементом блока. Можно создать правило без ограничений по типам или элементам информационных блоков. В таком случае правило сортировки будет распространятся на все типы и элементы всех информационных блоков.
    • форумам. Область действия правила может быть ограничена: форумом, определённой темой форума, а также сообщением. Правило сортировки будет распространятся на все сообщения всех форумов, если не указаны ни один из трех параметров правила (форум, тема, сообщение) .
    Параметр 1Первый параметр правила. В зависимости от выбранного модуля, параметр может определять:
    • для статических файлов - не используется;
    • для модуля Информационных блоков - тип информационного блока, к которому применимо правило;
    • для модуля Форумов - форум, к которому применимо правило.
    Параметр 2Второй параметр правила. Может определять:
    • для статических файлов - не используется;
    • для модуля Информационных блоков - информационный блок, к которому применимо правило;
    • для модуля Форумов - тему форума, к которому применимо правило.
    Параметр 3Третий параметр правила. Может определять:
    • для статических файлов - имя файла;
    • для модуля Информационных блоков - элемент информационного блока;
    • для модуля Форумов - текст в сообщении темы форума.
    ВесВес (индекс сортировки) правила. Документы, соответствующие критериям правила, будут расположены в списке результатов поиска с соответствии с индексом сортировки. Чем меньше значение веса, тем выше в списке будет отображен документ.
