Компоненты 2.0::Форма поиска

Компонент служит для вывода формы поиска. В поставляемом продукте форма поиска подключается в шаблоне дизайна и отображается на всех страницах сайта. В строке ввода формы Поиск по сайту вводится поисковая фраза или слово и после нажатия кнопки Поиск пользователь перенаправляется на страницу Поиска (созданную с помощью компонента Стандартная страница поиска bitrix:search.page).

Пример вызова

<?$APPLICATION->IncludeComponent(
"bitrix:search.form","",
Array(
	"PAGE" => "#SITE_DIR#search/index.php" 
)
);?>

Описание параметров

Шаблоны ссылок
PAGE
В этом параметре необходимо указать путь к странице поиска (доступен макрос #SITE_DIR#).
