    Форма поиска

    Форма поиска (фильтр) используется для выбора серверов из списка в соответствии с указанными условиями. Нижеследующая таблица описывает параметры, по которым может выполняться поиск.

    ПолеОписание
    НазваниеПоиск сервера по его названию.
    IDИдентификатор сервера.
    АктивныйПоиск сервера по признаку активности.

    Для того чтобы отобрать серверы по заданным критериям поиска, нажмите кнопку "Найти". Для отображения всех серверов нажмите кнопку "Отменить".

    Контекстная панель

    КнопкаОписание
    ДобавитьПереход к форме создания записи о новом сервере.
    НастроитьПозволяет перейти к диалогу настройки внешнего вида отчетной формы.
    ExcelЭкспортирует данные из отображаемой таблицы в формат MS Excel.

    Список серверов

    ПолеОписание
    Колонка флажковПоле предназначено для выбора серверов, к которым предполагается применить какое-либо правило или действие.
    ДействияДоступные действия над сервером:
    • Изменить параметры сервера - редактирование параметров сервера;
    • Удалить - удаление записи сервера.
    IDИдентификатор сервера.
    Дата изм.Дата и время последней модификации записи сервера.
    НазваниеНазвание сервера.
    Акт.Признак активности сервера.
    UTF-8Если указано "Да", то названия конвертируются из UTF-8. В противном случае конвертации не происходит.
    Симв. кодМнемонический код (префикс при авторизации).
    СерверАдрес корпоративного сервера.
